Bookmarks

Outro método para navegar no workspace é usar os Bookmarks. Você pode adicionar até 9 "bookmarks" para IDE janelas ou seções de código/GML Visual em scripts ou object eventos, e então você pode pular rapidamente entre eles usando a combinação de teclas correta.

Para adicionar um bookmark, basta selecionar a janela ou linha de código que você deseja marcar e depois + + NumberKey[1 - 9] no WIndows ou + + NumberKey[1 - 9] no Mac e isto irá definir o bookmark, marcando o item no IDE da seguinte forma:

Na imagem acima atribuímos marcadores numerados 1, 2 e 4 a diferentes partes do IDE, e agora podemos navegar para longe destas janelas ou mesmo fechá-las completamente, pois o marcador de página nos permitirá voltar diretamente a ele a qualquer momento posterior. Isto é feito pressionando / + NumberKey[1 - 9], e se o bookmark foi fechado será aberto, caso contrário, o workspace irá panar para o bookmark selecionado.

Os BookmarksIMPORTANTES não são salvos com o projeto ou quando o IDE é fechado. Eles só são válidos para a sessão em que foram criados.

Se você quiser remover um marcador de página, basta selecionar a janela ou linha de código marcado e então repetir o + / + NumberKey[1 - 9] que você usou para criá-lo.
