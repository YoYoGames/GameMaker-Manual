Referência doCódigo GML

Esta seção do manual é um guia de referência para a GameMaker Idioma (GML). Você pode encontrar todas as funções disponíveis documentadas aqui junto com os argumentos e exemplos de código necessários para mostrar como elas podem ser usadas. Para informações sobre como usar GML, consulte o Visão geral da GML seção do manual.

NOTA Em qualquer lugar onde o manual tenha "N/A" como valor de retorno para uma função, significa que a função não tem a intenção de retornar nenhum valor como parte de sua operação. Se você tentar verificar um valor de retorno destas funções, então você receberá o valor undefined.
