gif_save_buffer

Com esta função, você pode salvar uma animação GIF. Você fornece o índice GIF (como retornado pela função gif_open()) e a função retornará um crescimento de 1 byte-alinhado buffer com os dados do GIF. Observe que os dados finais do GIF terão sido paletizados usando a Paleta Universal 884 (veja aqui para mais informações).

Sintaxe:

gif_save_buffer(gif_index);

ArgumentoTipo Descrição
gif_index The ID of gif to save

Devoluções:

Buffer ID

Exemplo:

if save_gif == true
{
    if count == 0
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if count < 30
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        global.capture_buff = gif_save_buffer(gif_image);
        count = 0;
        save_gif = false;
    }
    count++;
}

O código acima criará um arquivo de imagem GIF com 30 quadros tirados da superfície da aplicação e depois o salvará em um arquivo buffer.
