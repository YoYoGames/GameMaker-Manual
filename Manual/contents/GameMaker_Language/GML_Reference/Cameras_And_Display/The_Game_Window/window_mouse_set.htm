window_mouse_set

With this function you can change or set the position of the mouse within the game window. The function will only work while the game is in focus and using alt + tab will unlock the mouse.

 This function can only be used on the desktop platforms (Windows, Ubuntu and macOS).

 For regular mouse functions see the section on Mouse Input.

  You can lock the mouse within the window by calling window_mouse_set_locked which removes the need to set the mouse position manually.

 

Syntax:

window_mouse_set(x, y);

Argument Type Description
x Real The x coordinate in the window.
y Real The y coordinate in the window.

 

Returns:

N/A

 

Example:

window_mouse_set(window_get_width() / 2, window_get_height() / 2);

The above code centers the mouse in the game window.
