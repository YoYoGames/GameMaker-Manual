window_set_position

Com esta função você pode configurar a janela do jogo para uma posição específica dentro da tela (em macOS, Linux(Ubuntu) e Windows) ou dentro do navegador (HTML5).

NOTA: Se seu jogo HTML5 usa um index.html personalizado e que define a tela para uma posição fixa, então esta função não terá efeito sobre a posição da janela.

Sintaxe:

window_set_position(x, y);

ArgumentoTipo Descrição
x The x coordinate of where to position the window.
y The y coordinate of where to position the window.

Devoluções:

Exemplo:

window_set_position(0, 0);

O código acima posicionará a janela do jogo no canto superior esquerdo do navegador ou da tela (dependendo do módulo alvo que estiver sendo usado).
