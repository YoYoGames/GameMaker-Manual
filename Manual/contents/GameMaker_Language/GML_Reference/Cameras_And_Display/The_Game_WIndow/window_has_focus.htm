window_has_focus

Com esta função você pode pesquisar o estado da janela (ou aba) e se ela perder o foco a função retornará false caso contrário retornará true. Na maioria dos casos, você pode simplesmente usar a função os_is_paused() para testar isso, mas em alguns casos muito específicos (por exemplo, jogos em aplicativos cromados) essa função não será acionada, caso em que você deve usar essa função em seu lugar.

NOTA: Esta função só é válida para o HTML5, Windows, e MacOS.

Sintaxe:

window_has_focus();

Devoluções:

Exemplo:

if !window_has_focus()
{
    PauseGame();
}

O código acima verificará se a janela do jogo está em foco ou não, e se a função retorna false uma função será chamada.
