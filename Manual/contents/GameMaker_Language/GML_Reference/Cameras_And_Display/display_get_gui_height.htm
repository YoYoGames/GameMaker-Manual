altura_do_agulhas_de_visualização

Com esta função você pode obter a altura (em pixels) da GUI como utilizada no Evento Draw GUI.

Sintaxe:

display_get_gui_height();

Devoluções:

Real

Exemplo:

ads_move(0, display_get_gui_height() - ads_get_display_height(0), 0);

O código acima irá definir um anúncio a ser exibido no canto inferior esquerdo do visor.
