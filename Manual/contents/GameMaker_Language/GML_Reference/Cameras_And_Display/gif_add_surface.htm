gif_add_surface

Com esta função você pode salvar um quadro em um arquivo GIF a partir de uma superfície. Você fornece o ID do arquivo GIF (como retornado pela função gif_open()) e uma identificação de superfície (conforme retornado pela função surface_create()), bem como um tempo de atraso entre os quadros. O tempo de atraso é calculado em 1/100 de segundo, mas note que uma vez que o GIF tenha sido criado, se você o utilizar para fins promocionais, por exemplo, navegadores diferentes podem interpretar o atraso do frame de forma ligeiramente diferente - veja aqui para mais detalhes. Você pode (opcionalmente) fornecer valores de desvio X e Y para a superfície a ser usada para adicionar ao GIF, neste caso a peça desenhada para o GIF começará a partir do desvio e não da posição padrão (0,0), permitindo que você selecione apenas uma seção da superfície a ser adicionada. Você também pode fornecer um valor de quantização, que é um inteiro entre 0 e 3, onde 0 é quantização total e 3 é quantização zero (valores mais baixos terão uma degradação subsequente na qualidade, mas criarão um GIF menor). Note que não há muita diferença entre a qualidade padrão de 2, e a mais alta de 3, e usar 3 irá retardar significativamente a criação do GIF, portanto, deve-se ter cuidado ao usar o valor máximo.

Observe que não há limites embutidos no número de quadros ou no tamanho desses quadros em seu GIF, mas se você usar muita memória, a função falhará e retornará -1, caso contrário, retornará 0.

Sintaxe:

gif_add_surface(gif_index, surface, delay_time, [xoffset], [yoffset]);

Argumento Tipo Descrição
gif_index GIF ID The ID of gif to add the surface data to
surface Identificação da superfície The ID of the surface to use as the added frame
delay_time Real The delay time for the frame (in 100ths of a second)
[xoffset] Real OPTIONAL O deslocamento ao longo do eixo X da superfície a adicionar de
[yoffset] Real OPTIONAL O deslocamento ao longo do eixo Y da superfície para adicionar a partir
[quantization] Real OPTIONAL A quantidade de quantização de 0 a 3 (0 qualidade mais baixa, 3 qualidade mais alta, padrão é 2)

Devoluções:

Real

Exemplo:

if save_gif == true
{
    if count == 0
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if count < 30
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

O código acima criará um arquivo de imagem GIF com 30 quadros tirados da superfície da aplicação e depois o salvará.
