camera_get_proj_mat

Esta função pode ser usada para recuperar a matriz de projeção. A função retorna a matriz que pode ser usada em outras funções da matriz ou para definir a matriz de projeção de outra câmera (usando camera_set_proj_mat()).

Sintaxe:

camera_get_proj_mat(camera_id)

ArgumentoTipo Descrição
camera_id The unique camera ID value returned when you created the camera

Devoluções:

Matrix Array

Exemplo:

mat = camera_get_proj_mat(camera_view[0]);

O código acima obtém a matriz de projeção para a câmera designada para visualizar a porta[0].
