camera_get_proj_mat

This function returns the projection matrix of the given camera. The function returns the matrix which can then be used in other Matrix Functions or to set the projection matrix of another camera (using camera_set_proj_mat).

 

Syntax:

camera_get_proj_mat(camera_id)

Argument Type Description
camera_id Camera ID The unique camera ID value returned when you created the camera

 

Returns:

Matrix

 

Example:

mat = camera_get_proj_mat(camera_view[0]);

The above code gets the projection matrix for the camera assigned to viewport[0].
