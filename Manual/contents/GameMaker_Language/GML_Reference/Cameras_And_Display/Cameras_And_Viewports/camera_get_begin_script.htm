camera_get_begin_script

This function can be used to retrieve the ID of the script function assigned as the begin script for the given camera. If no script function is assigned then the function will return an invalid handle (-1).

 

Syntax:

camera_get_begin_script(camera_id)

Argument Type Description
camera_id Camera ID The unique camera ID value returned when you created the camera

 

Returns:

 

Example:

var _scr = camera_get_begin_script(view_camera[0]);
if (_scr != cutscene)
{
    camera_set_begin_script(view_camera[0], cutscene);
}

The above code checks the script function assigned as the begin script for the camera assigned to viewport[0] and if it is not cutscene it is set to that function.
