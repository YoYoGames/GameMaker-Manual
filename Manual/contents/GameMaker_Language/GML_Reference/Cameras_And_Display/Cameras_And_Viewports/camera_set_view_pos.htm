camera_set_view_pos

Você pode usar esta função para atualizar a posição da visão da câmera dentro do room. Você dá o valor único de identificação da câmera (como retornado pelas diferentes funções camera_create() ) e então dá as posições x e y para ajustar a câmera.

Sintaxe:

camera_set_view_pos(camera_id, x, y)

ArgumentoTipo Descrição
camera_id The unique camera ID value returned when you created the camera.
x The x position to place the view at (in the room)
y The y position to place the view at (in the room)

Devoluções:

Exemplo:

camera_set_view_pos(view_camera[0], x - (view_wport[0] / 2), y - (view_hport[0] / 2));

O código acima definirá a posição de visão da câmera atribuída à porta de visão[0].
