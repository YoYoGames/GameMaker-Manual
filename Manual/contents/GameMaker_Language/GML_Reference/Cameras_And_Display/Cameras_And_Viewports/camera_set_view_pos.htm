camera_set_view_pos

You can use this function to update the position of the camera view within the room. You give the unique camera ID value (as returned by the different camera_create functions) and then give the x and y positions to set the camera to.

 

Syntax:

camera_set_view_pos(camera_id, x, y)

Argument Type Description
camera_id Camera ID The unique camera ID value returned when you created the camera
x Real The x position to place the view at (in the room)
y Real The y position to place the view at (in the room)

 

Returns:

N/A

 

Example:

camera_set_view_pos(view_camera[0], x - (view_wport[0] / 2), y - (view_hport[0] / 2));

The above code will set the view camera position for the camera assigned to viewport[0].
