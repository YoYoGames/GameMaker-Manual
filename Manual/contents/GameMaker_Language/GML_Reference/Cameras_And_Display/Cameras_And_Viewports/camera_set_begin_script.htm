camera_set_begin_script

Esta função pode ser usada para definir uma função de script que será chamada no início de cada quadro de jogo que a câmera for atribuída a uma porta de visualização visível e ativa. Você dá o valor único de identificação da câmera (como retornado pelas diferentes funções camera_create() ) e a identificação da função a ser chamada. A ordem em que scripts é anexada às câmeras e a renderização real da visão da câmera é a seguinte:

  • As câmeras para todas as portas de visualização visíveis e ativas têm sua função de atualização chamada
  • Depois, para cada vista:
    • A função de início da câmera para essa visão é chamada
    • Os eventos do sorteio são executados para essa visão
    • A função final para a câmera é chamada
  • Passar para a próxima visão visível ativa e repetir...

Sintaxe:

camera_set_begin_script(camera_id, script)

Argumento Tipo Descrição
camera_id Identificação da câmera The unique camera ID value returned when you created the camera.
script Função do Script The script function to run each game frame

Devoluções:

N/A

Exemplo:

camera_set_begin_script(view_camera[0], beginCamera);

O código acima define a função begin script para a câmera designada para visualizar a porta [0].
