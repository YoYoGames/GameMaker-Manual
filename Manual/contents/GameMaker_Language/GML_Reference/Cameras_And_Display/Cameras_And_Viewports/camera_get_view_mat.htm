camera_get_view_mat

Esta função pode ser usada para recuperar a matriz de visualização. A função retorna a matriz que pode então ser usada em outras funções de matriz ou para definir a matriz de visão de outra câmera (usando camera_set_view_mat()). Note que esta função é geralmente para uso com câmeras criadas usando camera_create()mas também pode ser usado naqueles criados no editor room (ou com camera_create_view()) - neste caso, a matriz devolvida só será válida após a utilização da câmera dada em uma porta de visualização pela primeira vez.

Sintaxe:

camera_get_view_mat(camera_id)

ArgumentoTipo Descrição
camera_id The unique camera ID value returned when you created the camera

Devoluções:

Matrix Array

Exemplo:

mat = camera_get_view_mat(camera_view[0]);

O código acima obtém a matriz de visualização para a câmera designada para visualizar a porta[0].
