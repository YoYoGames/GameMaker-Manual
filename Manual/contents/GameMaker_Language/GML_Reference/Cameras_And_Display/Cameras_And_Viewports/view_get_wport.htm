view_get_wport

This function returns the width of the given viewport.

 

Syntax:

view_get_wport(viewport)

Argument Type Description
viewport Real The viewport to target (0 - 7)

 

Returns:

Real

 

Example:

if (view_get_wport(0) != (display_get_width () / 2))
{
    view_set_wport(0, display_get_width() / 2);
}

The above code will check the width of the viewport[0] and if it is not half of the display width it is set to that value.
