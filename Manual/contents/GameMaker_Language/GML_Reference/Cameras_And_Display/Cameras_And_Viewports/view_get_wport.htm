view_get_wport

Esta função pode ser usada para recuperar a largura da porta de visualização dada.

Sintaxe:

view_get_wport(view_port)

ArgumentoTipo Descrição
view_port The view port to target (0 - 7)

Devoluções:

Real

Exemplo:

if view_get_wport(0) != (display_get_width () / 2)
{
    view_set_wport(0, display_get_width() / 2);
}

O código acima verificará a largura da porta de visualização[0] e, se não for metade da largura do visor, será ajustado para esse valor.
