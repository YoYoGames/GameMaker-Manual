camera_get_view_border_x

Esta função pode ser usada para recuperar o valor da borda do objeto/stância que segue a câmera dada ao longo do eixo x (borda horizontal). O valor de retorno será em pixels.

Sintaxe:

camera_get_view_border_x(camera_id)

ArgumentoTipo Descrição
camera_id The unique camera ID value returned when you created the camera

Devoluções:

Exemplo:

var xb = camera_get_view_border_x(view_camera[0]);
var yb = camera_get_view_border_y(view_camera[0]);
if xb != 200 || yb != 200
{
    camera_set_view_border(view_camera[0], 200, 200);
}

O código acima recupera os valores xborder e yborder da câmera designada para visualizar a porta[0] e então verifica se ela corresponde ao valor dado. Caso não corresponda, a câmera de visualização é ajustada para o valor dado.
