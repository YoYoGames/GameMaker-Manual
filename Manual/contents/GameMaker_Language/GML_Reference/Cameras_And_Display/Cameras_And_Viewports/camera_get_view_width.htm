camera_get_view_width

Esta função pode ser usada para recuperar a largura (em pixels) da câmera em questão. Note que esta função só é válida para câmeras criadas usando camera_create_view() ou para aqueles adicionados na sala editor.

Sintaxe:

camera_get_view_width(camera_id)

ArgumentoTipo Descrição
camera_id The unique camera ID value returned when you created the camera

Devoluções:

Exemplo:

var vw = camera_get_view_width(view_camera[0]) / 2;
var vh = camera_get_view_height(view_camera[0]) / 2;
camera_set_view_pos(view_camera[0], x - vw, y - vh);

O código acima recupera a largura e a altura da câmera designada para visualizar a porta[0] e então define sua posição em relação ao centro.
