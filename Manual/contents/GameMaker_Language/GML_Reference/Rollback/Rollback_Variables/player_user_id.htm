This variable stores the GX.games ID of the current player instance, for use with direct API calls to the server.

This variable only exists in player instances created by rollback_define_player(), and is different for each player.

Guest players will have an empty string assigned to this variable.

 

Syntax:

player_user_id;

 

Returns:

String
