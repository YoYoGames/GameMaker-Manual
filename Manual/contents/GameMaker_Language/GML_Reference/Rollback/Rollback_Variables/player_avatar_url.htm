This variable stores the URL to the player's avatar, which can be loaded using sprite_add().

You can use player_avatar_sprite which already has the sprite loaded.

This variable only exists in player instances created by rollback_define_player(), and is different for each player.

Guest players will display a placeholder sprite.

 

Syntax:

player_avatar_url;

 

Returns:

String
