This is a global variable that stores the number of frames that have passed since the multiplayer game was started.

This should be used instead of current_time and other similar variables, as explained in Time Variables.

 

Syntax:

rollback_current_frame

 

Returns:

Real

 

Example:

image_angle = dsin(rollback_current_frame) * 90;

The code above uses rollback_current_frame with dsin() to calculate an oscillating value between 1 and -1, which is multiplied by 90 and applied to the instance's angle.
