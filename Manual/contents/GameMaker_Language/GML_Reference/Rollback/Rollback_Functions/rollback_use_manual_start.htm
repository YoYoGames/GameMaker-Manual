rollback_use_manual_start

Por padrão, um jogo multiplayer começa automaticamente quando todos os jogadores se juntam.

Chamar essa função desativa esse comportamento de partida automática, então o jogo espera que você o inicie manualmente.

Se utilizada, esta função deve ser executada antes da função de junção/criação.

Sintaxe:

rollback_use_manual_start();

Devoluções:

N/A

Exemplo:

rollback_use_manual_start();

if (!rollback_join_game())
{
    rollback_create_game(2);
}

O código acima permite o início manual, e depois se junta ou cria um novo jogo multiplayer.
