title

By default, when you run the game, you see debug messages in the top-left corner of your game, displaying multiplayer events as they happen. You can disable those messages by calling this function with false as its argument, and enable them again by specifying true.

 

Syntax:

rollback_display_events(enabled);

Argument Type Description
enabled Boolean Specifies whether event display should be enabled (true) or disabled (false). By default, it's enabled.

 

Returns:

N/A

 

Example:

rollback_display_events(false);

The code above disabled the displaying of events.
