Rollback Functions

The following functions are used to start, join or leave a multiplayer game:

  • rollback_join_game
  • rollback_create_game
  • rollback_start_game
  • rollback_leave_game

 

The following functions are used to define and get information for your game:

  • Set
    • rollback_define_player
    • rollback_define_input
    • rollback_define_mock_input
    • rollback_define_input_frame_delay
    • rollback_use_manual_start
    • rollback_use_player_prefs
    • rollback_set_player_prefs
  • Get
    • rollback_get_input
    • rollback_get_info
    • rollback_get_player_prefs

 

The following functions are provided for use during an active game:

  • rollback_chat
  • rollback_sync_on_frame

 

The following functions are used for testing, or to enable/disable features related to testing:

  • rollback_use_random_input
  • rollback_display_events
