Funções de Rollback

As seguintes funções são usadas para iniciar, entrar ou sair de um jogo multiplayer:

  • rollback_join_game
  • rollback_create_game
  • rollback_start_game
  • rollback_leave_game

As seguintes funções são usadas para definir e obter informações para seu jogo:

  • Conjunto
    • rollback_define_player
    • rollback_define_input
    • rollback_define_mock_input
    • rollback_define_input_frame_delay
    • rollback_use_manual_start
    • rollback_use_player_prefs
    • rollback_set_player_prefs
  • Obtenha
    • rollback_get_input
    • rollback_get_info
    • rollback_geter_player_prefs

As seguintes funções são fornecidas para uso durante um jogo ativo:

  • rollback_chat
  • rollback_sync_on_frame

As seguintes funções são usadas para testes, ou para habilitar/desabilitar recursos relacionados a testes:

  • rollback_use_random_input
  • rollback_display_events
