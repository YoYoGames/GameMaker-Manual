flexpanel_node_remove_all_children

This function removes all direct children from the given node (but doesn't delete them).

Read flexpanel_node_remove_child for information on how it affects each child node.

 

Syntax:

flexpanel_node_remove_all_children(node);

Argument Type Description
node Flex Panel Node The node from which all children will be removed

 

Returns:

N/A

 

Example:

flexpanel_node_remove_all_children(n_root);

This removes all children from the node n_root.
