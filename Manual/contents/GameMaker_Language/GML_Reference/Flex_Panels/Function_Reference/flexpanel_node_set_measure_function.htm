flexpanel_node_set_measure_function

This function sets the Measure Function of the given Flex Panel Node. The Measure Function for a node will run when its layout is calculated and a measurement for the node is required, which may not always be the case.

The provided function will receive four arguments: (width, width_type, height, height_type). The type arguments can be one of the following values:

  • 0: Undefined
  • 1: Exactly
  • 2: At Most

The Measure Function must return a struct with width and height variables (or at least one of them). These will be the dimensions applied to the node.

For information on Measure Functions, see Integrating with external layout systems.

 

Syntax:

flexpanel_node_set_measure_function(node, function);

Argument Type Description
node Flex Panel Node The node to modify
function Function The Measure Function to use

 

Returns:

N/A

 

Example:

var _measure_func = flexpanel_node_get_measure_function(_node);

if (_measure_func == undefined)
{
    flexpanel_node_set_measure_function(_node, text_measure_function);
}

This gets the measure function of the node _node and stores it in a local variable. If it's undefined, it applies an existing Method to the node as the measure function.
