flexpanel_node_get_parent

This function returns the parent node of the given node, or undefined if it has no parent.

 

Syntax:

flexpanel_node_get_parent(node);

Argument Type Description
node Flex Panel Node The node to get the parent of

 

Returns:

Flex Panel Node or undefined

 

Example:

var _parent = flexpanel_node_get_parent(_node);

This gets the parent of the node _node and stores it in a local variable.
