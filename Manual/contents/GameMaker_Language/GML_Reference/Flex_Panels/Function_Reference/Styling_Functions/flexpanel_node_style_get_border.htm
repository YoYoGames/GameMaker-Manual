flexpanel_node_style_get_border

This function returns the border for the given edge(s) of a Flex Panel Node.

See: Border / flexpanel_node_style_set_border

 

Syntax:

flexpanel_node_style_get_border(node, edge);

Argument Type Description
node Flex Panel Node The node to read.
edge Flex Panel Edge Constant The edge to get the border for.

 

Returns:

Flex Panel Unit-Value Struct

 

Example:

var _border = flexpanel_node_style_get_border(_node, flexpanel_edge.all_edges);

This gets the general border of a node and stores it in a local variable.
