flexpanel_node_style_set_flex_grow

This function sets the flexGrow property of the given Flex Panel Node.

See: Flex Grow

 

Syntax:

flexpanel_node_style_set_flex_grow(node, grow);

Argument Type Description
node Flex Panel Node The node to modify
grow Real The grow value

 

Returns:

N/A

 

Example:

flexpanel_node_style_set_flex_grow(_node, 1);

This sets the flex grow value of a node to 1.
