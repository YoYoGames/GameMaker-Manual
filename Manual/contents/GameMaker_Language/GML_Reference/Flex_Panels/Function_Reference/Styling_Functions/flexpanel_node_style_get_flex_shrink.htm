flexpanel_node_style_get_flex_shrink

This function returns the flexShrink property of the given Flex Panel Node.

See: Flex Shrink / flexpanel_node_style_set_flex_shrink

 

Syntax:

flexpanel_node_style_get_flex_shrink(node);

Argument Type Description
node Flex Panel Node The node to read.

 

Returns:

Real

 

Example:

var _flex_shrink = flexpanel_node_style_get_flex_shrink(_node);

This gets the flex shrink value of a node and stores it in a local variable.
