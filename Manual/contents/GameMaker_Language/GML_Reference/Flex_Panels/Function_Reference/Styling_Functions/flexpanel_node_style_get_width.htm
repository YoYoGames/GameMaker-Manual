flexpanel_node_style_get_width

This function returns the width property of the given Flex Panel Node.

See: Width and Height / flexpanel_node_style_set_width

 

Syntax:

flexpanel_node_style_get_width(node);

Argument Type Description
node Flex Panel Node The node to read.

 

Returns:

Flex Panel Unit-Value Struct

 

Example:

var _width = flexpanel_node_style_get_width(_node);

This gets the width of a node and stores it in a local variable.
