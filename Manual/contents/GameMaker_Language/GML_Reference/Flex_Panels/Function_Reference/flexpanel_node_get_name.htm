flexpanel_node_get_name

This function returns the name set for the given node. Will return undefined if no name was set.

This is originally set either through the name property in the initial struct or through flexpanel_node_set_name.

 

Syntax:

flexpanel_node_get_name(node);

Argument Type Description
node Flex Panel Node The node to get the name of

 

Returns:

String or undefined

 

Example:

var _name = flexpanel_node_get_name(_node);

This gets the name of the node _node and stores it in a local variable.
