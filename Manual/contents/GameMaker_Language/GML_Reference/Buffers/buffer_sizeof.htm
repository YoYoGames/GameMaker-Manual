tamanho_do_tampão

Esta função retornará o tamanho (em bytes) do tipo de dados tampão constante (listado aqui).

Sintaxe:

buffer_sizeof(type);

Argumento Tipo Descrição
type Constante do tipo de dados tampão The type of data that is to be checked (see the list of constants here).

Devoluções:

Real

Exemplo:

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

O código acima calcula primeiro o tamanho do buffer para criar multiplicando o tipo de dado não assinado de 8 bits por 12 (já que usaremos o buffer para guardar 12 pedaços de dados), e depois usa este valor para definir um buffer fixo para o tamanho correto.
