buffer_decompress

Com esta função, você pode descomprimir uma compressão anterior buffer usando a compressão zlib. Você fornece o ID buffer para descomprimir, e a função retornará um novo ID buffer que contém os dados não comprimidos. Se a descompressão falhou (por exemplo, você está fornecendo um buffer que não foi comprimido), então a função retornará um valor menor que 0.

Sintaxe:

buffer_decompress(buffer);

ArgumentoTipo Descrição
buffer The index of the buffer to decompress.

Devoluções:

Buffer ID

Exemplo:

var cmpBuff = buffer_load("Player_Save.sav");
var srcBuff = buffer_decompress(cmpBuff);
global.DataString = buffer_read(srcBuff, buffer_string);

O código acima carregará primeiro um salvo buffer, depois o descomprimirá e finalmente lerá os dados string do buffer descomprimido em uma variável global.
