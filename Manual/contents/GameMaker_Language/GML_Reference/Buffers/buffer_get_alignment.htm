buffer_get_alignment

Com esta função você pode obter o alinhamento de bytes para o ID dado buffer.

Sintaxe:

buffer_get_alignment(buffer);

ArgumentoTipo Descrição
buffer The index of the buffer to check.

Devoluções:

Exemplo:

alignment = buffer_get_alignment(buff);

O código acima obterá o alinhamento do buffer a partir do valor indexado na variável "buff" e o armazenará em uma variável.
