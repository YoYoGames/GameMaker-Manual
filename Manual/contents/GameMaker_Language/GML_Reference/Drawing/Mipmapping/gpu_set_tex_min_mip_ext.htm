gpu_set_tex_min_mip_ext

Com esta função você pode definir o nível mínimo de mipmap que é usado atualmente para um determinado amostrador shader. Você fornece o valor do índice para o amostrador shader (como retornado pela função shader_get_sampler_index()e depois dar um valor, onde 0 é para resolução total, 1 é para o primeiro mipmap, 2 para o segundo etc...

Sintaxe:

gpu_set_tex_min_mip_ext(sampler_index, minmip);

ArgumentoTipo Descrição
sampler_index The index of the shader sampler to get
minmip The minimum mipmap level to use

Devoluções:

Exemplo:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_min_mip_ext(_sampleIndex) != 0
{
    gpu_set_tex_min_mip_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

O código acima estabelece o nível mínimo do mipmap para o sampler shader texture dado, se ainda não tiver sido definido como 0.
