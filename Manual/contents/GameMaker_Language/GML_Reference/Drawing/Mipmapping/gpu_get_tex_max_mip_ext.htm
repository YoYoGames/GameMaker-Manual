gpu_get_tex_max_mip_ext

Com esta função você pode obter o nível máximo de mipmap atualmente definido, que será usado para um dado amostrador shader. Você fornece o valor do índice para o amostrador shader (como retornado pela função shader_get_sampler_index()e a função retornará o nível máximo atual do mipmap, onde um valor de 0 é a maior resolução, 1 é usar o primeiro mipmap, 2 é o segundo etc... Note que isto pode ser bastante útil para evitar artefatos de sangramento ao renderizar textures, por exemplo, ajustando a borda da página texture para 8px e depois ajustando o nível máximo do mipmap para 3 garantirá que você não terá nenhum problema de sangramento a distâncias maiores de renderização.

Sintaxe:

gpu_get_tex_max_mip_ext(sampler_index);

ArgumentoTipo Descrição
sampler_index The index of the shader sampler to get

Devoluções:

(default: 16)

Exemplo:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_max_mip_ext(_sampleIndex) != 4
{
    gpu_set_tex_max_mip_ext(_sampleIndex, 4);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

O código acima estabelece o nível máximo de mipmap para o sampler shader texture dado, caso ainda não tenha sido definido para 4.
