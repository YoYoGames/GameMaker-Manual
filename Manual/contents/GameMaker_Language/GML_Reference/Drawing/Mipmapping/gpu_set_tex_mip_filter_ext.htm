gpu_set_tex_mip_filter_ext

Com esta função, você pode configurar o modo de filtro mip para um determinado amostrador shader. Você fornece o valor do índice para o amostrador shader (como retornado pela função shader_get_sampler_index()e depois uma das constantes de valor de modo listadas abaixo.

Constante Descrição
tf_point Isto significa que a mistura entre os níveis do mipmap está desativada, o que pode causar transições visíveis texture, mas dá o melhor desempenho.
tf_linear Isto significa que a mistura entre níveis de mipmap está habilitada (também conhecida como filtragem trilinear), o que suaviza as transições texture, mas dará um pequeno golpe no desempenho.
tf_anisotropic Isto significa que a filtragem anisotrópica está habilitada, o que melhora muito a qualidade de transição texture e pode reduzir o embaçamento visível com outros modos de filtragem, mas tem o maior impacto no desempenho.

Sintaxe:

gpu_set_tex_mip_filter_ext(sampler_index, filter);

Argumento Tipo Descrição
sampler_index Amostrador de amostras de Shader Handle The index of the shader sampler to get.
filter Filtro de Mipmapping Constante The mip filter mode to use (a constant, default: tf_point).

Devoluções:

N/A

Exemplo:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_mip_filter_ext(_sampleIndex) != tf_point
{
    gpu_set_tex_mip_filter_ext(_sampleIndex, tf_point);
}
texture_set_stage(_sampleIndex, _spriteTex);
shader_reset();

O código acima define o modo de filtro mip para tf_point (desabilitando o mipmapping) para o sampler shader texture dado, caso ainda não tenha sido definido.
