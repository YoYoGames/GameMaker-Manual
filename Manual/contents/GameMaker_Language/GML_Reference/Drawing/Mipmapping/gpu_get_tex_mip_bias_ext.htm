gpu_get_tex_mip_bias_ext

Com esta função você pode recuperar o valor do viés do mipmap para um determinado amostrador shader. Você fornece o valor do índice para o amostrador shader (como retornado pela função shader_get_sampler_index()e a função retornará um valor de 0,0 para nenhum viés, ou um valor maior onde 1 é igual ao primeiro mipmap, 2 é igual ao segundo mipmap etc... Isto controla a taxa na qual o mapa mip é trocado e geralmente fará com que o shader textures fique mais borrado quanto maior o valor e maior a "distância" sendo visualizada. Note que isto também pode retornar valores negativos, caso em que shader textures será mais nítido sobre uma distância maior quanto menor o valor.

Sintaxe:

gpu_get_tex_mip_bias_ext(sampler_index);

ArgumentoTipo Descrição
sampler_index The index of the shader sampler to get

Devoluções:

(default: 0)

Exemplo:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_mip_bias_ext(_sampleIndex) != 0
{
    gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

O código acima define o viés do filtro mip como 0 para o sampler shader texture dado, caso ainda não tenha sido definido como 0.
