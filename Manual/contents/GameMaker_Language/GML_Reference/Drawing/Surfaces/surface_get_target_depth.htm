surface_get_target_depth

This function returns the surface whose depth buffer is currently used for drawing. A surface different than the current draw target can be set for depth when calling surface_set_target.

 

Syntax:

surface_get_target_depth()

 

Returns:

Surface or -1 if no depth buffer is used

 

Example:

var _surf_depth = surface_get_target_depth();

The code above gets the surface whose depth buffer is currently used for drawing and stores the result in a temporary variable _surf_depth.
