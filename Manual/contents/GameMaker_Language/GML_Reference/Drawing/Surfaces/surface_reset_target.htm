surface_reset_target

Com esta função, você repõe todos os desenhos adicionais da superfície do alvo de volta para a tela. Observe que para começar a desenhar para uma superfície você deve ter primeiro chamado a função surface_set_target() e então este depois de terminado, para cada alvo de superfície que você tiver definido ou então nada será desenhado na tela, pois todos os outros desenhos (mesmo em outros casos) serão feitos na superfície. Você também deve perceber que nada será visto se a própria superfície não for desenhada na tela no evento de desenho de uma instância.

NOTA: Se você não definiu previamente uma meta de renderização com a função surface_set_target()A utilização desta função terminará silenciosamente (ou seja, sem qualquer mensagem de erro) toda a execução de códigos adicionais para o evento.

Sintaxe:

surface_reset_target();

Devoluções:

Exemplo:

if view_current = 0
{
    surface_set_target(surf);
    with (obj_Effect)
    {
        draw_self();
    }
    surface_reset_target();
}
else
{
    draw_surface(surf, 0, 0);
}

O código acima verificará para ver qual vista está sendo desenhada atualmente, e se é vista[0] define o alvo de sorteio para uma superfície e desenha todas as instâncias do "obj_Effect" object antes de redefinir o alvo de sorteio novamente. Se a view não for view[0], a superfície é desenhada para a tela.
