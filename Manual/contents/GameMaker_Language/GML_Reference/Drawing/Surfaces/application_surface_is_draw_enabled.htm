application_surface_is_draw_enabled

This function returns whether automatic drawing of the application surface is enabled.

Automatic drawing of the application surface can be enabled or disabled using application_surface_draw_enable.

 

Syntax:

application_surface_is_draw_enabled()

 

Returns:

Boolean

 

Example:

var _enabled = application_surface_is_draw_enabled();
show_debug_message("Automatic drawing of application surface enabled: {0}", _enabled);

The code above retrieves whether automatic drawing of the application surface is enabled, stores the result in a temporary variable _enabled and outputs it in a debug message.
