application_surface_is_draw_enabled

This function returns true if default drawing of the application surface is enabled, otherwise it returns false.

 

Syntax:

application_surface_is_draw_enabled();

 

Returns:

Boolean

 

Example:

if (keyboard_check_pressed(vk_space))
{
    if (application_surface_is_draw_enabled())
    {
        application_surface_draw_enable(false);
    }
    else
    {
        application_surface_draw_enable(true);
    }
}

The above code checks for a key press and the toggles the application surface's automatic drawing on or off depending on its state (like in an options menu).
