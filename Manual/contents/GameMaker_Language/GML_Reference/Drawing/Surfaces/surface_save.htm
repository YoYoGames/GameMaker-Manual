superfície_save

Esta função salvará uma superfície para o disco usando o nome de arquivo dado. A superfície deve ser salva como um arquivo no formato *.png.

Sintaxe:

surface_save(surface_id, fname);

ArgumentoTipo Descrição
surface_id The ID of the surface to set as the drawing target.
fname The name of the saved image file.

Devoluções:

Exemplo:

if keyboard_check_pressed(ord("S")
{
    surface_save(surf, "test.png");
}

O código acima verificará se o usuário pressiona a tecla "S" no teclado e se o faz salvará a superfície indexada na variável "surf" ao disco.
