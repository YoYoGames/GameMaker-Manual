draw_surface_ext

Esta função irá desenhar a superfície dada como na função draw_surface() mas com opções adicionais para alterar a escala, mistura, rotação e alfa da superfície a ser desenhada. A mudança destes valores não modifica o recurso de forma alguma (apenas como ele é desenhado).

Sintaxe:

draw_surface_ext(id, x, y, xscale, yscale, rot, col, alpha);

Argumento Tipo Descrição
id Identificação da superfície The unique ID value of the surface to draw.
x Real The x position of where to draw the surface.
y Real The y position of where to draw the surface.
xscale Real The horizontal scale.
yscale Real The vertical scale.
rot Real The rotation or angle to draw the surface.
col Cor The colour with which to blend the surface.
alpha Real The alpha transparency for drawing the surface.

Devoluções:

N/A

Exemplo:

draw_surface_ext(surf, 0, 0, 2, 2, 0, c_red, 0.5);

O código acima desenhará uma superfície indexada na variável "surf" na posição (0,0) no room, com o dobro da escala original, misturada em vermelho e semi transparente.
