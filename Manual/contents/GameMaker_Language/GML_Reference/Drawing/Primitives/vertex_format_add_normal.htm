vértice_formato_add_normal

Diga ao GameMaker para aceitar dados normais de superfície (nx, ny e nz) como parte do novo formato vertex que está sendo criado.

Sintaxe:

vertex_format_add_normal();

Devoluções:

Exemplo:

vertex_format_begin();
vertex_format_add_textcoord();
vertex_format_add_normal();
my_format = vertex_format_end();

O código acima criará um novo formato vertex com apenas texture e os valores normais de superfície e então armazenará o id de formato na variável "my_format".
