vértice_posição_3d

Esta função adicionará dados de posição 3D ao vertex que está sendo definido atualmente para os primitivos personalizados. Você fornece o buffer para escrever os dados, assim como as posições x, y e z para o desenho.

Sintaxe:

vertex_position_3d(buffer, x, y, z);

Argumento Tipo Descrição
buffer Vertex Buffer ID The buffer to write the information to.
x Real The x position.
y Real The y position.
z Real The z position.

Devoluções:

N/A

Exemplo:

vertex_position_3d(buff, x - 100, y - 125, 0);

O código acima definirá a posição do vertex que está sendo definido.
