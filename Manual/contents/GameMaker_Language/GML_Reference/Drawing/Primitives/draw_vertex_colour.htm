draw_vertex_colour

Esta função define a posição de um vertex para um primitivo, com sua própria cor e ajuste alfa. O aspecto final do primitivo dependerá do tipo primitivo escolhido para desenhar e da ordem com que você adicionar os vértices a ele (veja draw_primitive_begin() para maiores informações) e os vértices com cores e alfas diferentes se misturarão suavemente de um para o outro. Para terminar e desenhar o primitivo você deve chamar draw_primitive_end().

Sintaxe:

draw_vertex_colour(x, y, col, alpha)

ArgumentoTipo Descrição
x The x coordinate of the vertex.
y The y coordinate of the vertex.
col The colour to draw this vertex with.
alpha The alpha to draw this vertex with (0-1).

Devoluções:

Exemplo:

draw_primitive_begin(pr_trianglelist);
draw_vertex_colour(100, 100, c_blue, 0.1);
draw_vertex_colour(100, 200, c_red, 0.1);
draw_vertex_colour(150, 150, c_green, 1);
draw_primitive_end();

O código acima desenhará um triângulo semitransparente com cada vertex colorido de uma cor diferente.
