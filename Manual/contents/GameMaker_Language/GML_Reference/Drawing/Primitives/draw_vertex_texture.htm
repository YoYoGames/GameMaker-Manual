draw_vertex_texture

Esta função define a posição de um texturizado vertex para um primitivo. O aspecto final do primitivo dependerá do tipo primitivo escolhido para desenhar (Veja draw_primitive_begin() para mais informações), a ordem com que você adiciona os vértices a ele e a posição do ponto de partida que você definiu para o texture. Para terminar e desenhar o primitivo você deve chamar draw_primitive_end().

Você também pode escolher desenhar apenas parte do texture sobre a superfície do primitivo, definindo a posição xtex/ytex apropriada para a amostragem (estas coordenadas também são conhecidas como pontos UV ). Este ponto é definido por um valor entre 0 e 1, sendo o canto superior esquerdo da imagem a utilizar o ponto (0,0) e o canto inferior direito o ponto (1,1). Desta forma, você pode texture um primitivo com apenas uma parte da imagem de origem asset.

Ao desenhar um primitivo texturizado, o texture será misturado com a cor definida por draw_set_colour. Isto significa que para desenhar o texture sem mistura, este deve ser ajustado para c_white, como mostrado no exemplo abaixo.

NOTA Para que um texture repita deve ser um poder de dois em tamanho, ou seja: 32x32, 128x128, etc...

Sintaxe:

draw_vertex_texture(x, y, xtex, ytex)

Argumento Tipo Descrição
x Real The x coordinate of the vertex.
y Real The y coordinate of the vertex.
xtex Real The width of the texture sample to use.
ytex Real The height of the texture sample to use.

Devoluções:

N/A

Exemplo:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

O código acima desenhará uma faixa triangular 4 vertex (fazendo um retângulo) texturizada com o texture mantido na variável "tex", e o conjunto texture será usado para cobrir o primitivo completo.
