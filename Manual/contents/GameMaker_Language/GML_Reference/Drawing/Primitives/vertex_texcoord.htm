vértex_texcoord

Esta função definirá as coordenadas texture a serem utilizadas para o vertex que está sendo definido atualmente para o primitivo personalizado. Você fornece o buffer para escrever os dados, bem como a posição UV dentro do texture a ser usado.

Sintaxe:

vertex_texcoord(buffer, u, v);

Argumento Tipo Descrição
buffer Vertex Buffer ID The buffer to write the information to.
u Real The first texture coordinate to use (0 - 1).
v Real The second texture coordinate to use (0 - 1).

Devoluções:

N/A

Exemplo:

vertex_texcoord(buff, 0, 0);

O código acima definirá os valores UV do vertex que está sendo definido.
