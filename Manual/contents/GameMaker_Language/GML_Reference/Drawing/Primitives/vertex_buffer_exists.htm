vertex_buffer_exists

This function returns whether the given vertex buffer exists in memory or not.

 

Syntax:

vertex_buffer_exists(buffer)

Argument Type Description
buffer Vertex Buffer The value to check

 

Returns:

Boolean

 

Example:

var _exists = vertex_buffer_exists(vbuff);

The code above calls vertex_buffer_exists to check if a variable holds a vertex buffer and assigns the result to a local variable.
