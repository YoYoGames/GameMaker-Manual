draw_vertex_texture_colour

Esta função define a posição de um texturizado vertex para um primitivo. O aspecto final do primitivo dependerá do tipo primitivo escolhido para desenhar (Veja draw_primitive_begin() para mais informações), a ordem com que você adiciona os vértices, a posição dos pontos de início e fim que você dá para a amostra texture e os valores de cor e alfa que você definiu. Para manter a aparência texture enquanto altera apenas o alfa, um valor de -1 (ou c_white) pode ser usado para o argumento da cor. Para terminar e desenhar o primitivo você deve chamar draw_primitive_end().

NOTA Para que um texture repita deve ser um poder de dois em tamanho, ou seja: 32x32, 128x128, etc...

Sintaxe:

draw_vertex_texture_colour(x, y, xtex, ytex, col, alpha)

ArgumentoTipo Descrição
x The x coordinate of the vertex.
y The y coordinate of the vertex.
xtex The x coordinate within the texture.
ytex The y coordinate within the texture.
col The colour to blend with the texture at this vertex (-1 or c_white para não misturar).
alpha The alpha to draw this vertex with (0-1).

Devoluções:

Exemplo:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture_colour(0, 0, 0, 0, c_fuchsia, 1);
draw_vertex_texture_colour(640, 0, 1, 0, c_yellow, 1);
draw_vertex_texture_colour(0, 480, 0, 1, c_aqua, 1);
draw_vertex_texture_colour(640, 480, 1, 1, c_lime, 1);
draw_primitive_end();

O código acima desenhará uma faixa triangular 4 vertex (fazendo um retângulo) texturizada com o texture mantido na variável "tex", e o conjunto texture será usado para cobrir o primitivo completo, e será misturado com quatro cores diferentes.
