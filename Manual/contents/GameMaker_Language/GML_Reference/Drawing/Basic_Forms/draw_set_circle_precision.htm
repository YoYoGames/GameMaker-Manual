draw_set_circle_precision

Para otimizar o desempenho ao desenhar círculos, o GameMaker basicamente desenha uma forma poligonal com lados suficientes para fazê-lo parecer circular. Entretanto, dependendo de quão grandes ou pequenos você precisa que seus círculos sejam, você pode descobrir que mudando este valor pode ajudar a aumentar o desempenho de seu jogo, ou fazer os círculos parecerem melhores. O valor de precisão que você pode inserir nesta função é o número de lados que o polígono do círculo tem, e este número pode ser qualquer coisa entre 4 e 64, mas deve ser um número divisível por 4, com um valor padrão de 24.

NOTA: Se você estiver querendo desenhar uma forma usando um shader, você deve estar ciente de que a maioria shaders espera as seguintes entradas: vertex, texture, Cor. Entretanto, ao usar esta função, apenas vertex e dados de cor estão sendo passados, e assim o shader não pode desenhar nada (ou desenhar algo, mas não corretamente). Se você precisar desenhar formas desta forma, então o shader deve ser personalizado com isto em mente.

Sintaxe:

draw_set_circle_precision(precision);

ArgumentoTipo Descrição
precision The precision of all circles drawn after this function is called.

Devoluções:

Exemplo:

draw_set_circle_precision(12);

O código acima desenhará todos os círculos usando um polígono de 12 lados.
