draw_point

Com esta função você pode desenhar um único pixel em qualquer lugar da tela, usando a cor e o alfa atualmente definidos.

Sintaxe:

draw_point(x, y);

ArgumentoTipo Descrição
x The x coordinate of the point to be drawn.
y The y coordinate of the point to be drawn.

Devoluções:

Exemplo:

draw_set_colour(c_yellow);
draw_point(100,100);

Isto desenhará um pixel amarelo na posição (100.100).
