With this function you can draw a single pixel anywhere on the screen with a colour that you define. The colour settings will override the base colour set with the function draw_set_colour().

 

Syntax:

draw_point_colour(x, y, col1);

Argument Type Description
x Real The x coordinate of the point.
y Real The y coordinate of the point.
col1 Colour The colour of the point.

 

Returns:

N/A

 

Example:

draw_point_colour(50, 50, c_red);

This would draw a red pixel at (50,50).
