draw_healthbar

Com esta função você pode desenhar uma barra colorida para mostrar um valor constante. Embora a função utilize a palavra "barra de saúde", você pode usá-la para exibir o que desejar, desde que a quantidade a ser exibida seja um valor percentual entre 0 e 100 (uma porcentagem de qualquer valor pode ser calculada com a fórmula (CURRENT_Value / MAXIMUM_value) * 100), assim, por exemplo, você pode usá-la para exibir potência, saúde, mana, tempo ou qualquer outra coisa que se beneficiaria de uma exibição de barra.

Você pode definir várias coisas diferentes que mudam o aspecto visual da barra e estas estão ilustradas na imagem abaixo (note que as cores usadas aqui são c_green para o valor máximo e c_red para o valor mínimo, mas você pode usar as cores que desejar):

NOTA: Se você estiver querendo desenhar uma forma usando um shader, você deve estar ciente de que a maioria shaders espera as seguintes entradas: vertex, texture, Cor. Entretanto, ao usar esta função, apenas vertex e dados de cor estão sendo passados, e assim o shader não pode desenhar nada (ou desenhar algo, mas não corretamente). Se você precisar desenhar formas desta forma, então o shader deve ser personalizado com isto em mente.

Sintaxe:

draw_healthbar(x1, y1, x2, y2, amount, backcol, mincol, maxcol, direction, showback, showborder);

Argumento Tipo Descrição
x1 Real The x coordinate of the left of the healthbar
y1 Real The y coordinate of the top of the healthbar
x2 Real The x coordinate of the right of the healthbar
y2 Real The y coordinate of the bottom of the healthbar
amount Real The variable which defines total health (between 0 and 100)
backcol Cor The colour of the background for the bar
mincol Cor The colour of the bar when at no health (0)
maxcol Cor The colour of the bar when at full health (100)
direction Real Where the bar is "anchored" (0 = left, 1 = right, 2 = top, 3 = bottom)
showback Booleano Whether the bar's background is drawn (true) ou não (false). Se falso, o argumento do backcol é desconsiderado.
showborder Booleano Whether the bar's elements have a 1px wide black border (true) ou não (false).

Devoluções:

N/A

Exemplo:

var pc;
pc = (hp / max_hp) * 100;
draw_healthbar(100, 100, 500, 200, pc, c_black, c_red, c_lime, 0, true, true)

O código acima usa o valor percentual que você obtém das variáveis "hp" e "hp_max" para desenhar uma barra de saúde padrão vermelha/verde.
