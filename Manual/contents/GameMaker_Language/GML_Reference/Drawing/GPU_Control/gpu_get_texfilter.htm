gpu_get_texfilter

Com esta função você pode verificar se a filtragem texture (interpolação linear) está habilitada (retorna true) ou não (retorna false). Para mais informações sobre a filtragem texture, veja a função gpu_set_texfilter().

NOTA: No alvo HTML5, esta função só funcionará com WebGL habilitado.

Sintaxe:

gpu_get_texfilter();

Devoluções:

Boolean

Exemplo:

if gpu_get_texfilter()
{
    gpu_set_texfilter(false);
}
else
{
    gpu_set_texfilter(true);
}

O código acima verifica se a filtragem texture está ligada ou desligada e a liga de acordo.
