gpu_get_stencil_enable

This function returns whether the stencil test is currently enabled.

 

Syntax:

gpu_get_stencil_enable()

 

Returns:

Boolean

 

Example:

var _enabled = gpu_get_stencil_enable();

The code above retrieves whether the stencil test is currently enabled with a call to gpu_get_stencil_enable and stores the result in a temporary variable _enabled.
