gpu_set_set_state

Esta função definirá o estado atual da GPU usando o Mapa DS aprovado. O mapa fornecido pode ser criado usando a função gpu_get_state().

Sintaxe:

gpu_set_state(ds_map);

ArgumentoTipo Descrição
ds_map The GPU state to set as a DS Map.

Devoluções:

Exemplo:

gpu_set_state(gpu_map);

O código acima define o estado da GPU usando o mapa fornecido na variável "gpu_map".
