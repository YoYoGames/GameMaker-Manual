gpu_get_stencil_func

This function gets the comparison function currently used for comparisons in the stencil buffer.

 

Syntax:

gpu_get_stencil_func()

 

Returns:

Comparison Function Constant

 

Example:

var _func = gpu_get_stencil_func();

The above code calls gpu_get_stencil_func to get the stencil function that's currently used and stores the result in a temporary variable _func.
