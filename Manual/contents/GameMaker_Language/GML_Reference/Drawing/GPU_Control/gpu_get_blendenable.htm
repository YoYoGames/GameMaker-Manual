gpu_get_blendenable

Esta função pode ser usada para recuperar o estado de mistura alfa. Se ela retornar true, a mistura alfa está habilitada e se retornar false, está desabilitada. Por padrão, esta função está ativada e, portanto, a função retornará true

Sintaxe:

gpu_get_blendenable();

Devoluções:

Exemplo:

if gpu_get_blendenable() == false
{
    gpu_set_blendenable(true);
}

O código acima verifica o estado da mistura alfa e, se estiver desativado, é ativado novamente.
