gpu_get_blendmode_ext

Esta função pode ser usada para recuperar o atual modo de mistura estendida que está sendo usado para desenho. A função retorna uma matriz de 2 elementos 1D com os seguintes elementos:

  • [0] = Fonte Fator do modo de mistura constante (o padrão é bm_src_alpha)
  • [1] = Destino Fator do modo de mistura constante (o padrão é bm_inv_src_alpha)

Os valores da matriz serão uma das seguintes constantes ("s" denota um valor retirado da fonte enquanto um "d" denota um valor do destino):

Fator do modo de mistura constante
Constante Fator de mistura (Vermelho, Verde, Azul, Alfa)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As)
bm_src_alpha (As, As, As, As, As)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (Anúncio, Anúncio, Anúncio, Anúncio)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) onde f = min(As, 1-Ad)

Note que você pode mudar esses valores e passar o conjunto completo para o gpu_set_blendmode_ext() funcionam como um único argumento.

Sintaxe:

gpu_get_blendmode_ext();

Devoluções:

Array (2 elements only; see above for constants)

Exemplo:

var bm = gpu_get_blendmode_ext();
bm[0] = bm_src_alpha;
gpu_set_blendmode_ext(bm);

O código acima obtém o modo de mistura estendida atual, modifica a fonte e, em seguida, define novamente o modo de mistura estendida.
