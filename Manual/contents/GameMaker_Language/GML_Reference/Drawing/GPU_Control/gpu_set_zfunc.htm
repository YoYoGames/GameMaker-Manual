gpu_set_zfunc

Esta função pode ser usada para definir o modo de comparação de teste z-buffer (ver gpu_set_ztestenable() para mais informações). Os valores disponíveis para uso são qualquer uma das seguintes constantes (o valor padrão é cmpfunc_lessequal):

Constante Descrição
cmpfunc_never Nunca
cmpfunc_less Menos
cmpfunc_equal Igual
cmpfunc_lessequal Menos ou Igual
cmpfunc_greater Maior
cmpfunc_notequal Não Igual
cmpfunc_greaterequal Maior ou Igual
cmpfunc_always Sempre

Sintaxe:

gpu_set_zfunc(cmp_func);

Argumento Tipo Descrição
cmp_func Função de comparação Constante The comparison mode to use (see list above)

Devoluções:

N/A

Exemplo:

gpu_set_ztestenable(true);
gpu_set_zfunc(cmpfunc_always);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

O código acima liga o teste de z-buffer e estabelece seu modo de comparação antes de desenhar um fundo sprite e depois desliga-o novamente para continuar o desenho.
