gpu_set_ztestenable

Esta função pode ser usada para ativar ou desativar o teste de z-buffer, afetando a forma como as coisas serão desenhadas (em geral, só é útil quando se trabalha com projetos 3D). Essencialmente, por padrão, quando o teste z está desligado e você tem dois objects desenhando para o mesmo espaço, ambos objects serão prestados independentemente do fato de um desenhar em demasia o outro, resultando em chamadas desnecessárias de saque. Se você switch isto em seguida, o z-buffer é testado para ver se um object será "visível" e não desenhado se for não. Observe que este é o comportamento padrão, mas você pode mudar isso mudando o tipo de comparação usada para testes com z-buffer (veja a função gpu_set_zfunc().

Por padrão, o teste z-buffer está desligado (false).

Sintaxe:

gpu_set_ztestenable(enable);

ArgumentoTipo Descrição
enable Enable or disable z-buffer testing (true ou false).

Devoluções:

Exemplo:

gpu_set_ztestenable(true);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

O código acima liga o teste de z-buffer para desenhar um fundo sprite e depois o desliga novamente para continuar o desenho.
