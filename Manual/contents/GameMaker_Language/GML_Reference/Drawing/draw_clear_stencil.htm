draw_clear_stencil

This function clears the stencil buffer of the current render target to the given stencil value, an integer value from 0 to 255.

 

Syntax:

draw_clear_stencil(stencil);

Argument Type Description
stencil Real The value to clear the stencil buffer to. An integer value in the range [0, 255].

 

Returns:

N/A

 

Example:

draw_clear_stencil(1);

The above code clears the stencil buffer to the value 1.
