video_get_status

Esta função retorna o status do vídeo atualmente carregado. O status pode ser qualquer uma das seguintes constantes:

Status do vídeo Constante
Constante Descrição
video_status_closed Nenhum vídeo está atualmente carregado, ou o vídeo foi fechado com video_close()
video_status_preparing O vídeo está atualmente em preparação e ainda não começou a ser reproduzido.
video_status_playing O vídeo está sendo reproduzido atualmente
video_status_paused O vídeo é pausado (ver video_pause())

Sintaxe:

video_get_status();

Devoluções:

Video Status Constant

Exemplo:

var _status = video_get_status();

if (keyboard_check_pressed(vk_space))
{
    if (_status == video_status_playing)
    {
        video_pause();
    }
    else if (status == video_status_paused)
    {
        video_resume();
    }
}

O código acima obtém o status do vídeo e então verifica se o player pressionou Espaço. Nesse caso, ele pausa o vídeo se estiver tocando, e o retoma se estiver pausado.
