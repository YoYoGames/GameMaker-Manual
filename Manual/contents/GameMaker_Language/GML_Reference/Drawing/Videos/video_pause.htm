video_pause

Esta função faz uma pausa no arquivo de vídeo que está atualmente carregado. Você pode retomá-lo ligando para video_resume() a qualquer momento após esta função.

Sintaxe:

video_pause();

Devoluções:

N/A

Exemplo:

var _status = video_get_status();

if (keyboard_check_pressed(vk_space))
{
    if (_status == video_status_playing)
    {
        video_pause();
    }
    else if (status == video_status_paused)
    {
        video_resume();
    }
}

O código acima obtém o status do vídeo e então verifica se o player pressionou Espaço. Nesse caso, ele pausa o vídeo se estiver tocando, e o retoma se estiver pausado.
