draw_set_colour

Com esta função, você pode definir a cor do sorteio base para o jogo.

Isto afetará o desenho de fonts, formas, primitivas e 3D, porém não afetará sprites (desenhado manualmente ou por uma instância). Se qualquer gráfico afetado for desenhado com seus próprios valores de cor, este valor será ignorado.

Sintaxe:

draw_set_colour(col);

Argumento Tipo Descrição
col Cor The colour to set for drawing.

Devoluções:

N/A

Exemplo:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

O código acima desenhará algum texto na posição especificada com um efeito de sombra criado pelo desenho alfa e cor modificados.
