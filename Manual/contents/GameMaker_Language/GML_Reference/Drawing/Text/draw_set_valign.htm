draw_set_valign

Esta função é utilizada para alinhar o texto ao longo do eixo vertical e mudar o alinhamento vertical mudará a posição e a direção na qual todo o texto adicional é desenhado, com o valor padrão sendo fa_top. As seguintes constantes são aceitas:

Constante Alinhamento
fa_top
fa_middle
fa_bottom

Sintaxe:

draw_set_valign(valign);

Argumento Tipo Descrição
valign Constante de Alinhamento Vertical Vertical alignment constant (from the table above).

Devoluções:

N/A

Exemplo:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text(100, 32, "Score: " + string(score));

O código acima desenhará a pontuação centrada em torno do centro do texto.
