draw_get_font

Esta função receberá o texto font atualmente designado para o desenho. A função retornará -1 se nenhum font estiver definido, ou o valor de identificação (um inteiro positivo) para o font asset atribuído.

Sintaxe:

draw_get_font();

Devoluções:

Font Asset or -1

Exemplo:

var _cur_font = draw_get_font();
var _y_offset = 0;

switch (_cur_font)
{
    case ft_small:
        _y_offset = 10;
    break;

    case ft_medium:
        _y_offset = 22;
    break;

    case ft_big:
        _y_offset = 34;
    break;

    default:
        _y_offset = 8;
}

draw_text(room_width / 2, 200 + _y_offset, "MENU");

O código acima recebe o código atualmente aplicado font e roda uma declaração switch sobre ele, aplicando um valor de offset em Y diferente, dependendo do font. Em seguida, ele usa esse valor de offset enquanto desenha algum texto.
