draw_get_halign

Esta função é usada para obter o ajuste do alinhamento do texto ao longo do eixo horizontal, e retornará uma das constantes listadas abaixo.

Sintaxe:

draw_get_halign();

Devoluções:

Constant

Constante de Alinhamento Horizontal
Constante Alinhamento
fa_left
fa_center
fa_right

Exemplo:

var _cur_halign = draw_get_halign();
var _cur_valign = draw_get_valign();

draw_set_halign(fa_right);
draw_set_valign(fa_bottom);

draw_text(100, 32, "Score: " + string(score));

draw_set_halign(_cur_halign);
draw_set_valign(_cur_valign);

O código acima salva os valores "halign" e "valign" aplicados atualmente às variáveis locais, e depois muda os alinhamentos para desenhar algum texto. Após desenhá-lo, ele redefine os alinhamentos de volta aos valores armazenados nas variáveis locais.
