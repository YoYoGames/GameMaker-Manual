texturegroup_delete

This function deletes a Texture Group that was previously created using texturegroup_add. The name to be specified in this function is the same name that was passed into texturegroup_add for creating the Texture Group.

The function returns true if the Texture Group was successfully deleted, otherwise it returns false.

 This function is not supported on HTML5.

 

Syntax:

texturegroup_delete(groupname);

Argument Type Description
groupname String The name of the texture group to delete, must have been created with texturegroup_add

 

Returns:

Boolean

 

Example:

/// Create event
texturegroup_add("MyTextureGroup", "image.png", _sprite_data);

/// Destroy event
texturegroup_delete("MyTextureGroup");

After a Texture Group is created in the Create event (see the full example code on the texturegroup_add page), this function is used to delete the same Texture Group in the Destroy event. The name that was specified when creating the Texture Group is used to delete it.
