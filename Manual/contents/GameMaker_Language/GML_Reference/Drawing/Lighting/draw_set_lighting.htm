draw_set_lighting

Esta função é utilizada para permitir todos os efeitos de iluminação. O padrão está desativado (false).

Sintaxe:

draw_set_lighting(enable);

Argumento Tipo Descrição
enable Booleano Enable or disable all lighting (true ou falso)

Devoluções:

N/A

Exemplo:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

O código acima permitirá a iluminação de toda a cena, então definirá uma luz direcional branca no espaço room, e então finalmente acenderá essa luz.
