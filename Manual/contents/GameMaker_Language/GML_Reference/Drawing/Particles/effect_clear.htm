effect_clear

This function clears all particles that are currently visible in the room which have been made using the effect_create_* functions.

 

Syntax:

effect_clear();

 

Returns:

N/A

 

Example:

effect_clear();

Calling this will clear the room of any particles made with the effect_create_* functions.
