part_type_destroy

Com esta função você pode remover o tipo de partícula especificado do jogo. Quando você usar esta função, todas as partículas do tipo especificado desaparecerão do room e a própria partícula será removida da memória, portanto esta função deve ser usada somente quando você não precisar mais da partícula.

Sintaxe:

part_type_destroy(ind);

Argumento Tipo Descrição
ind Identificação do tipo de partícula The index of the particle type to destroy.

Devoluções:

N/A

Exemplo:

if part_particles_count(smoke_sys) = 0
{
    part_type_destroy(smoke_part);
    part_system_destroy(smoke_sys);
    instance_destroy();
}
else alarm[0] = 1;

O código acima verifica um sistema de partículas para ver se alguma partícula está visível no momento no room e se não estiver, as partículas, o sistema e a instância são destruídos.
