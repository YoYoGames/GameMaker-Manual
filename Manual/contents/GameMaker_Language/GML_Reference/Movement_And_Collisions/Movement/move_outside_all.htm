move_outide_all

Com esta função você pode dizer a uma instância para sair de uma colisão em qualquer direção e qualquer número de pixels a cada passo, com um valor de -1 ou 0 para o maxdist sendo um 1000px padrão, ou seja: GameMaker movimentará a instância continuamente até 1000 pixels até que ela esteja fora de colisão.

Sintaxe:

move_outside_all(dir, maxdist);

ArgumentoTipo Descrição
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Devoluções:

Exemplo:

if place_meeting(x, y, all)
{
    move_outside_all(90, 1);
}

O código acima moverá a instância para cima de 1 pixel a cada passo que ela estiver em colisão com qualquer outra instância.
