position_empty

Esta função verificará se uma determinada posição entra em colisão com qualquer instância com uma máscara de colisão válida na posição em questão.

Sintaxe:

position_empty(x, y);

ArgumentoTipo Descrição
x The x position to check.
y The y position to check.

Devoluções:

Exemplo:

var xx, yy;
xx = random(room_width);
yy = random(room_height);
if position_empty(xx, yy)
{
    instance_create_layer(xx, yy, "Bullets", obj_Bomb);
}

Isto verificará uma posição aleatória em algum lugar no room para uma colisão e se não houver nenhuma, criará uma instância de obj_Bomba.
