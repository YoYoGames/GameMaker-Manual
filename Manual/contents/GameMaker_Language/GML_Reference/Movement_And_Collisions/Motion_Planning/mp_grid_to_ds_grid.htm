mp_grid_to_ds_grid

Com esta função você pode copiar a grade MP dada em uma grade DS. A grade DS deve ter o mesmo tamanho que a grade MP, embora não tenha que ser (os dados serão perdidos se forem menores, e se forem maiores, todas as células extras da grade serão 0). Uma célula na grade DS conterá o valor -1 se ela foi marcada como ocupada na grade MP, ou 0 se não foi.

Sintaxe:

mp_grid_to_ds_grid(source, destination);

Argumento Tipo Descrição
source ID da grade MP Index of the mp_grid that is to be used
destination DS Grid ID Index of the ds_grid that is to be used to copy the grid data to.

Devoluções:

Boolean

Exemplo:

motion_grid = ds_grid_create(room_width / 32, room_height / 32);
mp_grid_to_ds_grid(mp_grid, motion_grid);

O código acima criará uma nova grade DS e então copiará os dados da grade MP contidos na variável "mp_grid" para a nova grade DS.
