mp_linear_path

Esta função calcula uma linha reta path da posição atual da instância até o ponto especificado pelos valores "xgoal" e "ygoal". Ela utiliza o tamanho do passo indicado, o mesmo que na função mp_linear_step(). O path indicado já deve existir e será substituído pelo novo path e a função retornará se um path completo for encontrado (verdadeiro) ou não (falso). Se falso for retornado, então um path ainda será gerado, mas só chegará até a posição onde o path foi bloqueado.

NOTA: Esta função não move a instância. Ela define apenas um path, e você deve usar as funções de Caminho para se movimentar.

Sintaxe:

mp_linear_path(path, xgoal, ygoal, stepsize, checkall);

ArgumentoTipo Descrição
path The index of the path to be used.
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check for collisions with all instances (true) or just solid ones (false).

Devoluções:

Exemplo:

if mp_linear_path(path, obj_Player.x, obj_Player.y, 2, 0)
{
    path_start(path, 2, 0, 0);
}

Isto faz com que o object verifique e veja se há um linear path de sua posição atual para a posição de "obj_Player". Se houver, então inicia o path.
