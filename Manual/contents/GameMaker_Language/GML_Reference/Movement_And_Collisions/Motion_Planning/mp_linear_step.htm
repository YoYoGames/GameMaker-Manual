mp_linear_step

Com esta função, você diz a uma instância para dar um "passo" em direção a um ponto específico, especificado pelos valores xgoal e ygoal. O tamanho da etapa (que é quantos pixels a instância deve mover cada etapa) é indicado pelo tamanho da etapa, e se a instância já estiver na posição ela parará e não se moverá mais, ao contrário de outras funções mais simples, como move_towards_point(). O tamanho dos degraus é também a distância "à frente" que o object verificará cada degrau para uma colisão, e você pode escolher se a instância pára em uma colisão com qualquer instância ou apenas aquelas que estão marcadas como sólidas. A função retornará se atingiu a posição de objetivo (verdadeiro) ou se falhou (falso).

NOTA Esta função não tenta fazer desvios se encontrar um obstáculo, ela simplesmente falha e pára de se mover.

Sintaxe:

mp_linear_step(xgoal, ygoal, stepsize, checkall);

ArgumentoTipo Descrição
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check all instances (true) or just solid ones (false).

Devoluções:

Exemplo:

if mp_linear_step(mouse_x, mouse_y, 5, 0)
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

O código acima fará com que o object se mova em direção ao mouse a uma velocidade de 5 pixels por passo. Uma vez alcançada a posição do mouse, ele criará uma "obj_Explosão" object e se destruirá.
