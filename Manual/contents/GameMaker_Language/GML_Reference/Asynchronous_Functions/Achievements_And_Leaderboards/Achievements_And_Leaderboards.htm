Conquistas e Quadros de Líderes

Os jogos que são feitos para direcionar dispositivos que rodam iOS ou Android também podem se conectar a uma das duas API's de liderança e de realização. Para iOS você tem o iTunes Connect, e para Android você pode usar o Google Play, com ambas as API's de utilizando as funções encontradas nesta página. Estas funções permitem que seu jogo envie informações sobre pontuações e realizações para os quadros de líderes que você criou na Developers Account para o jogo que você está fazendo e é uma excelente maneira de envolva os jogadores no lado social do jogo, pois eles poderão então ver o que outras pessoas que jogaram seu jogo conseguiram e tentar melhorá-los.

Ao utilizar estas funções, você deve observar que elas não acompanham as realizações e pontuações automaticamente para você. Elas apenas mostram os dados que você lhes envia de seu jogo, portanto você deve codificar suas próprias realizações e sistema de hi-score para acompanhar o status dos indivíduos no jogo, então use estas funções para adicionar os dados à rede escolhida (entretanto você pode solicitar informações de progresso para realizações usando o achievement_load_progress() função).

Algumas dessas funções desencadearão um Evento Assíncrono Social de retorno, que conterá um Mapa DS de dados relacionados para a função que a desencadeou. As funções que acionam o evento também detalham os pares chave/valor do mapa que são devolvidos pela callback, entretanto, um Evento Social de callback também pode ser acionado por seu jogo recebendo notificações externas sobre desafios, etc...

NOTA: Para que estas funções funcionem com iOS você deve ter uma conta Game Center válida e um aplicativo iOS id para seu jogo.

NOTA: Para que estas funções funcionem com Android você deve submeter seu jogo ao Google Play, e ter configurado as seções necessárias, bem como ter a API ativada nas Opções de Jogo do Android para seu jogo.

As funções listadas abaixo são as básicas utilizadas para integrar seu jogo com a plataforma de jogos sociais alvo escolhida:

  • achievement_login_status
  • achievement_login
  • achievement_logout
  • achievement_available
  • achievement_post
  • achievement_post_score
  • achievement_increment
  • achievement_show_achievements
  • achievement_show_leaderboards
  • achievement_show_challenge_notifications
  • achievement_get_challenges
  • achievement_get_pic
  • achievement_load_friends
  • achievement_load_leaderboard
  • achievement_load_progress
  • achievement_send_challenge
  • achievement_reset
