http_get_request_crossorigin

Esta função pode ser usada para obter o conjunto do tipo de origem cruzada para jogos HTML5 e retornará um string (em todas as outras plataformas será retornado um string "" vazio).

Sintaxe:

http_get_request_crossorigin();

Devoluções:

Exemplo:

if string_lower(http_get_request_crossorigin()) != "use-credentials"
{
    http_set_request_crossorigin("use-credentials");
}
sprite_add("sprites/player_outfit_1.png", 0, false, false, 0, 0);

O código acima verificará primeiro o tipo de origem cruzada definido atualmente e se não for "credenciais de uso", então é definido para "credenciais de uso" e então um sprite é adicionado a partir de um arquivo.
