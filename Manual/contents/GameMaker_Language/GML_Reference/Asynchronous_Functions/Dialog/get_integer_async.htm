get_integer_async

Esta função abre uma janela e exibe uma mensagem assim como um espaço para o usuário inserir um valor (que conterá o valor padrão fornecido para começar). Esta é uma função assíncrona, e como tal o GameMaker não bloqueia o dispositivo em que está sendo executado enquanto espera por uma resposta, mas sim continua a executar eventos como de costume. Uma vez que o usuário tenha digitado seu string e pressionado o botão "Ok", uma função assíncrona Diálogo evento é acionado que, somente durante a duração desse evento, terá um mapa DS armazenado na variável async_load.

Este mapa conterá as três teclas, "id", "status", e "value". "id" é o valor que foi retornado pela função quando chamada, o "status" será ou true para o botão "Ok" sendo pressionado, ou false se a mensagem foi cancelada (onde aplicável, pois nem todas as plataformas alvo permitem que as mensagens sejam canceladas). Finalmente "value" retornará o inteiro que o usuário digitar como string (um string vazio "" será retornado se nenhuma entrada tiver sido dada).

NOTA Esta função é apenas para depuração e testee não deve ser usada em jogos lançados. Para isso, você deve criar uma interface de usuário personalizada para receber informações dos jogadores, para que você tenha controle total sobre a aparência e o comportamento dos diálogos.

Sintaxe:

get_integer_async(string, default);

Argumento Tipo Descrição
string Cordão The message to show in the dialog.
default Real The default value.

Devoluções:

Async Request ID

Exemplo ampliado:

O evento de pressão do mouse esquerdo (por exemplo) do object que está mostrando a mensagem teria o seguinte código:

msg = get_integer_async("How old are you?", 0);

O acima mostrará uma mensagem solicitando que o usuário digite um string e pressione "Ok". A identificação da função é armazenada na variável "msg" e será usada no evento de Diálogos Assíncronos, como mostrado abaixo:

var i_d = ds_map_find_value(async_load, "id");
if i_d == msg
{
    if ds_map_find_value(async_load, "status")
    {
        global.Age = ds_map_find_value(async_load, "value");
    }
}

O código acima verifica a chave "id" do retorno DS Map em relação ao valor armazenado na variável "msg". Se forem os mesmos, então verifica se "Ok" foi pressionado (ao invés da janela ser fechada/cancelada) e se retorna true, então define uma variável global a partir do inteiro retornado.
