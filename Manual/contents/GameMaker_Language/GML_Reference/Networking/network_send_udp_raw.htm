network_send_udp_raw

Com esta função você pode enviar dados através da rede usando o UDP para um server. A função leva a ID do soquete para conectar, a URL para conectar e a porta para usar. Você deve então fornecer o id do buffer que contém os dados a serem enviados (para mais informações em buffers veja Referência - Buffers) e finalmente o tamanho (em bytes) dos dados. O UDP é "sem conexão" na medida em que você não faz realmente uma conexão, você apenas envia um pacote diretamente para um IP, e o server recebe os dados recebidos de um endereço IP e tem que lidar com ele "como está". Os dados enviados não são formatados pelo GameMaker de forma alguma e os dispositivos receptores receberão os dados como um fluxo, o que significa que você mesmo terá que lidar com eles. A função retornará o número de bytes de dados enviados, ou um número inferior a 0 se o envio falhar.

NOTA Esta função não funcionará quando utilizada em um projeto em execução no alvo HTML5, e os projetos HTML5 também não poderão receber UDP.

Sintaxe:

network_send_udp_raw(socket, url, port, bufferid, size);

ArgumentoTipo Descrição
socket The id of the socket to use.
url The url or IP to connect to (a string).
port The port to connect to.
bufferid The id of the buffer to get the data from.
size The size (in bytes) of the data.

Devoluções:

Real

Exemplo:

network_send_udp_raw(sock, "www.macsweeneygames.com", 6510, buff, buffer_tell(buff));

O código acima enviará um pacote UDP cru para o endereço server definido pela URL na porta 6510. Os dados são retirados do buffer indexado na variável "buff".
