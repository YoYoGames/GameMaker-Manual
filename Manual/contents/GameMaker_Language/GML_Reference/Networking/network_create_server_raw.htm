rede_criar_servidor_servidor_desenhar

Esta função é usada para criar uma nova rede server para seu jogo, usando um dos protocolos de conexão permitidos (veja as constantes listadas abaixo). Você fornece o tipo server, depois lhe dá uma porta para usar, e finalmente o número máximo de conexões que devem ser permitidas a qualquer momento para o server (note que este número depende de você, mas muitos clientes conectados saturarão a rede ou a CPU do dispositivo não será capaz de lidar com o processamento deste número de jogadores, então use com cuidado). A função retorna uma identificação única que deve ser usada armazenada em uma variável e usada para identificar o server em todas as outras funções da rede, ou um valor inferior a 0 se a conexão falhar.

Como isso cria um "cru" server, ele não aceitará nem utilizará nenhum dos cabeçalhos de dados construídos em GameMaker para comunicação, e por isso você deve estar usando as funções network_send_raw() e network_send_udp_raw() para enviar dados não-formatados para o server criado.

Sintaxe:

network_create_server_raw(type, port, max_client);

ArgumentoTipo Descrição
type The type of server to create (see the constants listed below).
port The port that the server will use.
max_client The maximum number of clients that can connect at once.

Devoluções:

Network Socket ID

Exemplo:

var port = 6510;
server = network_create_server_raw(network_socket_tcp, port, 32);
while (server < 0 && port < 65535)
{
    port++
    server = network_create_server(network_socket_tcp, port, 32);
}

O código acima tentará criar um server usando TCP através da porta 6510. Se essa porta não estiver disponível, ela então loop através das portas para encontrar uma que esteja.
