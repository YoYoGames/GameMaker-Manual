network_create_socket

Esta função é utilizada para criar um novo soquete de cliente para que seu jogo se comunique através da rede. Você deve definir o tipo de socket (veja a lista de constantes abaixo) e a função retornará uma identificação única para aquele socket, que deve ser usada em todas as chamadas adicionais de função para aquele socket, ou um valor inferior a 0 se a conexão falhar.

DICA Você pode usar o IP "127.0.0.1" para conectar de volta ao mesmo dispositivo que está rodando o jogo.

Sintaxe:

network_create_socket(type);

Argumento Tipo Descrição
type Tipo de soquete Constante The type of socket connection to create (see the constants listed above).

Devoluções:

Network Socket ID

Exemplo:

if os_browser == browser_not_a_browser
{
    client = network_create_socket(network_socket_tcp);
    network_connect( client, "192.134.0.1", 6510 );
}
else
{
    client = network_create_socket(network_socket_ws);
    network_connect_raw_async( client, "192.134.0.1", 6520 );
}

O código acima verificará se o jogo está rodando em um navegador ou não e criará um novo TCP ou socket Web antes de tentar se conectar através dele ao endereço IP dado na porta dada.
