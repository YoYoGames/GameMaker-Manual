network_create_socket_ext

This function creates a new client socket for your game to communicate over the network.

You must define the socket type (see the list of constants below) and give a port to use, and the function will return a unique id which should be used in all further function calls for that socket, or a value of less than 0 if the connection fails. When you no longer need the socket, remove it from memory with network_destroy.

 

Syntax:

network_create_socket_ext(protocol, port);

Argument Type Description
protocol Socket Type Constant The network protocol to use
port Real The port to use

 

Returns:

Network Socket ID

 

Example:

client = network_create_socket_ext(network_socket_udp, 6510);

The above code will create a new UDP socket on port 6510.
