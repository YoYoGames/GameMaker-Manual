dbg_set_view

This function sets the current view to a view that was previously created using dbg_view. Any new debug controls will be created in the view set using this function.

 

Syntax:

dbg_set_view(view);

Argument Type Description
view Debug View Pointer A pointer to a debug view

 

Returns:

N/A

 

Example:

var _view1 = dbg_view("View 1", true);
var _view2 = dbg_view("View 2", true);
dbg_slider(ref_create(global, "value2"), 0, 20, "Value2");
dbg_set_view(_view1);
dbg_slider(ref_create(global, "value"), 0, 20, "Value1");

This creates two views and adds a slider into View 2. Then it uses dbg_set_view() to set the current view to View 1 and adds a slider into it.
