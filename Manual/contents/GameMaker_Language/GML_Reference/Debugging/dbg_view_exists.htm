dbg_view_exists

This function returns whether the given custom debug view, created with dbg_view, still exists.

 

Syntax:

dbg_view_exists(view)

Argument Type Description
view Debug View Pointer A pointer to a debug view returned by a call to dbg_view.

 

Returns:

Boolean

 

Example:

view = dbg_view("View 1", true);
show_debug_message(dbg_view_exists(view));
dbg_view_delete(view);
show_debug_message(dbg_view_exists(view));

The above code first creates a new debug view using dbg_view and stores the pointer to it in a variable view. dbg_view_exists is called a first time to check if the debug view stored in view exists and the result is output using show_debug_message. The view is then deleted with a call to dbg_view_delete. At this point, the variable view still holds a pointer to a debug view, but the debug view doesn't exist anymore and the pointer has become invalid. dbg_view_exists is called a second time and its return value output in a debug message.
