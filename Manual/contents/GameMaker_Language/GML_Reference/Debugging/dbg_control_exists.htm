dbg_control_exists

This function returns whether the given debug control still exists.

 

Syntax:

dbg_control_exists(control)

Argument Type Description
control Debug Control A debug control value returned by any debug control function, e.g. dbg_button, dbg_slider, dbg_text_separator, etc.

 

Returns:

Boolean

 

Example:

var _view1 = dbg_view("View", true);
var _sec1 = dbg_section("Section 1");
var _slider = dbg_slider(ref_create(global, "value"), 0, 20, "Value1");
show_debug_message($"Slider exists? { dbg_control_exists(_slider) }");
dbg_control_delete(_slider);
show_debug_message($"Slider exists? { dbg_control_exists(_slider) }");

This code creates a debug view, section and slider. It prints a message to the Output Log to check if the slider exists, then deletes the slider and prints the same message again.

This code outputs the following:

Slider exists? 1
Slider exists? 0
