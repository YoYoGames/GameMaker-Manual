dbg_section_exists

This function returns whether the given debug control still exists.

 

Syntax:

dbg_section_exists(section)

Argument Type Description
section Debug Control A pointer to a debug section returned by a call to dbg_section.

 

Returns:

Boolean

 

Example:

section = dbg_section("Section 1", true);
show_debug_message(dbg_section_exists(section));
dbg_section_delete(section);
show_debug_message(dbg_section_exists(section));

The above code first creates a new debug section using dbg_section and stores the pointer to it in a variable section. dbg_section_exists is called a first time to check if the debug section stored in section exists and the result is output using show_debug_message. The section is then deleted with a call to dbg_section_delete. At this point, the variable section still holds a pointer to a debug section, but the debug section doesn't exist anymore and the pointer has become invalid. dbg_section_exists is called a second time and its return value output in a debug message.
