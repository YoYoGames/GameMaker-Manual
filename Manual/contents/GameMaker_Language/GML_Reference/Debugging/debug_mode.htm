debug_mode

This read-only variable returns true when the game is being run in debug mode and false when being played as normal.

 

Syntax:

debug_mode

 

Returns:

Boolean

 

Example:

if (debug_mode)
{
    ini_open("Cheats.ini");
}
else
{
    ini_open("Game.ini");
}

The above code opens a different INI file depending on the value of the read-only variable debug_mode.
