ds_priority_destroy

Esta função removerá da memória a estrutura de dados queue dada prioridade, liberando os recursos que estava utilizando e removendo todos os valores que continha. Esta função deve ser sempre utilizada quando você terminar de usar a prioridade DS queue para evitar vazamentos de memória que podem retardar e travar seu jogo.

IMPORTANTE! Quando você cria uma estrutura de dados, o valor do índice para identificá-lo é um valor inteiro começando em 0. Isso significa que estruturas de dados de diferentes tipos podem ter o mesmo valor de índice, portanto, em caso de dúvida, você deve estar usando o ds_exists() antes de acessá-los. Observe também que os índices são reutilizados, portanto, um valor de índice de estrutura de dados destruído pode ser usado por um índice recém-criado posteriormente, por isso recomendamos sempre definir a variável que manteve o índice DS em -1 após a destruição.

Sintaxe:

ds_priority_destroy(id);

ArgumentoTipo Descrição
id The id of the data structure to remove.

Devoluções:

Exemplo:

if (lives == 0)
{
    ds_priority_destroy(AI_queue);
    AI_queue = -1;
    room_goto(rm_Menu);
}

O código acima verificará o valor da variável global "vidas" construída e, se for 0, destruirá a prioridade DS queue indexada na variável "AI_queue" e, em seguida, muda rooms.
