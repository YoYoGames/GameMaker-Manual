ds_priority_delete_delete_value

Esta função irá simplesmente apagar o valor dado, juntamente com sua prioridade, da prioridade indexada queue.

Sintaxe:

ds_priority_delete_value(id,val);

ArgumentoTipo Descrição
id The id of the priority queue to use.
val The value to delete from the priority queue.

Devoluções:

Exemplo:

if (ai_move == false)
{
    ds_priority_delete_value(ai_priority, AI_Move);
}

O código acima verifica uma variável de instância e se ela retornar false, removerá a função indexada script da prioridade queue.
