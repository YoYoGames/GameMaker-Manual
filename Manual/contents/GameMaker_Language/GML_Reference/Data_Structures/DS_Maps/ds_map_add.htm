ds_map_add

Esta função deve ser usada para adicionar conjuntos de pares chave/valor no mapa DS especificado. Você pode verificar esta função para ver se foi bem sucedida ou não (ela retornará true no caso de sucesso ou false caso contrário), pois pode falhar se já existir a mesma chave no mapa DS ou se você especificar um mapa DS inexistente como o ID do mapa a ser adicionado. As chaves e os valores que você fornece podem ser compostos de qualquer combinação de tipos de dados, portanto, todos os seguintes - e mais - são aceitáveis (embora, na prática, você usaria mais comumente um string para a chave):

ds_map_add(map, 5, undefined);
ds_map_add(map, "level", 1);
ds_map_add(map, 89.6, "hello world");
ds_map_add(map, 5, buffer_get_address(buff));

Você também pode adicionar a um mapa usando o acessor "?", como mostrado abaixo:

map[? 5] = undefined;
map[? "level"] = 1;
map[? 89.6] = "hello world";
map[? 5] = buffer_get_address(buff);

NOTA: Ao contrário de outras estruturas de dados no GameMaker, esta chave não irá para o início (nem para o fim) do mapa DS, mas irá apenas para o mapa DS em algum lugar.

Sintaxe:

ds_map_add(id, key, val);

ArgumentoTipo Descrição
id The id of the map to add to.
key The key of the value to add.
val The value to add to the map.

Devoluções:

Exemplo:

inventory = ds_map_create();
ds_map_add(inventory, "hp potion", 1);
ds_map_add(inventory, "gold", 100);

Isto criará um novo mapa, atribuindo sua identificação à variável "inventário". Em seguida, acrescenta duas novas chaves ao mapa, "poção hp" e "ouro", e define seus valores iniciais como 1 e 100.
