ds_map_is_list

Com esta função, você pode verificar se uma lista DS está armazenada na chave do mapa dado. Se a chave fornecida contém um DS list ID, então a função retornará true caso contrário, retornará false.

Note que isto só detectará listas que foram adicionadas usando a função ds_map_add_list() .

Sintaxe:

ds_map_is_list(id, key)

ArgumentoTipo Descrição
id The id of the map to use.
key The key to check.

Devoluções:

Exemplo:

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++)
{
    if ds_map_is_list(inventory, key)
    {
        ds_list_destroy(inventory[? key]);
    }
    key = ds_map_find_next(inventory);
}
ds_map_destroy(inventory);

O código acima loops através de um mapa DS e verifica se alguma das chaves dentro dele é para uma lista DS. Se forem, então a lista DS é destruída, e no final do loop o mapa DS é destruído também.
