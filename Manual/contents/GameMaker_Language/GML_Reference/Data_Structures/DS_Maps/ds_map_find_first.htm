ds_map_find_first

Esta função retorna a primeira chave armazenada no mapa DS. Esta não é a primeira chave na ordem em que você as adicionou! Os mapas DS não são armazenados de forma linear, para isso utilize a lista DS, então tudo o que isso faz é encontrar a primeira chave como armazenada pelo computador. Isto pode ser útil se você tiver que iterar através do mapa DS procurando por algo, mas deve ser evitado se possível, pois pode ser lento.

Note que esta função retornará undefined se o mapa DS dado estiver vazio.

Sintaxe:

ds_map_find_first(id);

ArgumentoTipo Descrição
id The id of the map to use.

Devoluções:

Variable or undefined

Exemplo:

var size = ds_map_size(inventory) ;
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
    if key != "gold"
    {
        key = ds_map_find_next(inventory, key);
    }
    else break;
}

O código acima cria algumas variáveis temporárias e depois obtém o tamanho do mapa DS e encontra a primeira chave como armazenada pelo computador no mapa. Em seguida, ele usa um for loop para iterar através do mapa DS procurando o valor da chave "ouro". Se ele o encontrar, ele rompe o loop.
