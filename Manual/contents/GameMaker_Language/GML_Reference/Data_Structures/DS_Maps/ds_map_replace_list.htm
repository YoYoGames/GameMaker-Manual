ds_map_replace_list

Com esta função você pode substituir uma lista DS que tenha sido armazenada na "chave" dada por outra lista que tenha sido criada anteriormente. Esta função foi projetada para criar mapas compatíveis com JSON que você então codificaria usando json_encode() e só deve ser usado em conjunto com essa funcionalidade.

Sintaxe:

ds_map_replace_list(id, key, value)

Argumento Tipo Descrição
id DS Map ID The id of the map to use.
key Cordão The key to replace.
value DS List ID The id of the ds_list to use to replace the one previously stored in the given key.

Devoluções:

N/A

Exemplo:

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
ds_map_replace_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_list_destroy(j_list);

O código acima criará um DS List e o preencherá com os valores de várias variáveis globais antes de substituir uma lista previamente armazenada no DS Map "j_map".
