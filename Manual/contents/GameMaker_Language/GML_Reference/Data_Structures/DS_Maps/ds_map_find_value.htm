ds_map_find_value

Com esta função você pode obter o valor a partir de uma chave especificada. Os valores de entrada da função são o mapa DS (previamente criado) a ser usado e a chave a ser verificada.

NOTA: Se essa chave não existir, a função retornará undefined. Você deve sempre verificar isto usando a função is_undefined() função

Sintaxe:

ds_map_find_value(id, key)

ArgumentoTipo Descrição
id The id of the map to use.
key The key to find.

Devoluções:

Variable or undefined

Exemplo:

amount = ds_map_find_value(inventory, "food");

Ou, usando o acessor de mapa "?":

amount = inventory[? "food"];

O código acima obterá o valor da chave "alimento" e o armazenará na variável "quantidade".
