ds_list_find_value

Com esta função, você pode verificar a posição dada na lista e o valor mantido dentro da lista para aquela posição será devolvido. Observe que se você der uma posição fora do tamanho da lista (ou seja: posição 11 em uma lista de 10 valores), então a função pode retornar undefined ou 0. Isto porque quando você cria a lista, internamente as primeiras entradas na lista são definidas como 0 para minimizar os problemas de desempenho ao adicionar itens à lista inicialmente (embora o ds_list_size() ainda retornará 0 em uma lista recém-criada). Se você deseja garantir que a lista esteja "verdadeiramente" vazia ao criar, então você deve chamar ds_list_clear() depois de criar a lista, o que significa que quaisquer valores retornados para os slots da lista não-populosa serão undefined, que você pode então verificar consistentemente usando o is_undefined() função.

Sintaxe:

ds_list_find_value(id, pos);

Argumento Tipo Descrição
id DS List ID The id of the list to use.
pos Real The position to look at, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.

Devoluções:

Variable or undefined

Exemplo:

val = ds_list_find_value(list, ds_list_size(list) - 1);

O código acima verifica a lista indexada na variável "list" na última posição da lista e armazena o valor retornado na variável "val".
