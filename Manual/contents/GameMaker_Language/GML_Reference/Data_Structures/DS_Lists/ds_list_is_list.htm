ds_list_is_list

Com esta função, você pode verificar se outra lista DS está armazenada na posição determinada dentro de uma lista DS. Se a posição dada contém um DS list ID, então a função retornará true caso contrário, retornará false.

Note que isto só detectará listas que foram marcadas manualmente usando a função ds_list_mark_as_list() .

Sintaxe:

ds_list_is_list(id, pos);

ArgumentoTipo Descrição
id The id of the list to check.
pos The position within the list to check.

Devoluções:

Exemplo:

var size = ds_list_size(ships);
for (var i = 0; i < size; i++)
{
    if ds_list_is_list(ships, i)
    {
        ds_list_destroy(ships[| i]);
    }
}
ds_list_destroy(ships);

O código acima loops através de uma lista DS e verifica se alguma das entradas contém outras identificações da lista. Se eles contêm, estas listas são destruídas, e então a lista principal é destruída depois que o loop é terminado.
