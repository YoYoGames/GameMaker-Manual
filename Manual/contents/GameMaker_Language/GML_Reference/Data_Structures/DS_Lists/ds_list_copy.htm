ds_list_copy

Com esta função você pode copiar o conteúdo de uma lista para outra. Ambas as listas devem ter sido criadas anteriormente e se a lista que está sendo copiada já tiver informações dentro dela, esta lista será limpa primeiro. O resultado final são duas listas independentes que contêm as mesmas informações.

Sintaxe:

ds_list_copy( id, source );

ArgumentoTipo Descrição
id The id of the list being copied to.
source The id of the list to be copied from.

Devoluções:

Exemplo:

if !ds_list_empty(main_list)
{
    old_list = ds_list_create();
    ds_list_copy(old_list, main_list);
    ds_list_clear(main_list);
}

O código acima verificará uma lista DS para ver se ela está vazia. Se não estiver vazia, será copiada para outra lista DS (que foi criada anteriormente) e então a lista original será limpa.
