ds_list_add

Esta função pode ser usada para adicionar um novo valor (real ou string) à lista, que será adicionado no final. A função pode levar mais argumentos opcionais (quantos forem necessários), permitindo adicionar vários valores consecutivamente à lista em uma única chamada.

Sintaxe:

ds_list_add(id, val1 [, val2, ... max_val]);

Argumento Tipo Descrição
id DS List ID The id of the list to add to.
val Variável The value to add to the list.
[val2, ... max_val] Variável Optional values to be added to the list.

Devoluções:

N/A

Exemplo:

ds_list_add(sc_list, score);

O código acima adicionará o valor armazenado na variável "score" à lista indexada na variável "sc_list".
