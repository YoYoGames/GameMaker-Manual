ds_grid_get_max

Esta função pode ser usada para encontrar o valor máximo para todas as células encontradas dentro da região definida de uma grade, como mostrado na imagem abaixo:

Sintaxe:

ds_grid_get_max(index, x1, y1, x2, y2);

ArgumentoTipo Descrição
index The index of the grid.
x1 The left cell column of the region.
y1 The top cell row of the region.
x2 The right cell column of the region.
y2 The bottom cell row of the region.

Devoluções:

Real or String

Exemplo:

val = ds_grid_get_max(grid, 0, 0, 5, 5)

O código acima definirá a variável "val" para o valor máximo contido dentro da região determinada da grade DS indexada na variável "grid".
