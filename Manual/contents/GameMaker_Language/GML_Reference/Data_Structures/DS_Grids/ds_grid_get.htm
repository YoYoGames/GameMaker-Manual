ds_grid_get

Esta função pode ser usada para obter o valor de qualquer célula dentro de uma determinada grade DS. Se você passar coordenadas de grade inválidas para a função, então o valor retornado será undefined e um erro será mostrado na janela de saída.

Sintaxe:

ds_grid_get(index, x, y);

ArgumentoTipo Descrição
index The index of the grid.
x The x position of the cell you want to find the value of.
y The y position of the cell you want to find the value of.

Devoluções:

Variable

Exemplo:

var xx = irandom(ds_grid_width(grid) - 1);
var yy = irandom(ds_grid_height(grid) - 1);
val = ds_grid_get(grid, xx, yy)

O código acima seleciona uma célula aleatória da grade DS indexada na variável "grid" e armazena seu valor na variável "val".
