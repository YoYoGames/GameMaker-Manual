ds_grid_shuffle

Esta função pode ser usada para aleatorizar as posições de todos os valores em todas as células dentro de uma grade.

NOTA Esta função embaralhará as células da grade para as mesmas posições toda vez que o jogo for executado novamente devido ao fato de que o GameMaker gera a mesma semente inicial aleatória toda vez para tornar o código debugging uma tarefa muito mais fácil. Para evitar que este comportamento usa randomise() no início de seu jogo. Isto só é verdade ao testar e debugging o jogo, como o executável final. não mostrará este comportamento e será aleatório a cada jogada.

Sintaxe:

ds_grid_shuffle(index);

ArgumentoTipo Descrição
index The index of the grid to shuffle.

Devoluções:

Exemplo:

ds_grid_shuffle(grid)

Isto pegaria todos os valores da grade DS indexados na variável "grid" e os embaralharia para obter uma ordem diferente.
