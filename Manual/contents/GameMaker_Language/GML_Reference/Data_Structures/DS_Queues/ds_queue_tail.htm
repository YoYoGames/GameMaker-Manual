ds_queue_tail

Esta função só lerá o último valor do queue (aquele que está "na cauda"). Ela não irá decifrar o valor, o que significa que ainda poderá ser lido no futuro por esta função ou pelo ds_queue_dequeue().

Sintaxe:

ds_queue_tail(id);

ArgumentoTipo Descrição
id The id of the data structure to read from.

Devoluções:

(Data type value stored in the queue)

Exemplo:

num = ds_queue_tail(control_queue);

O código acima lerá o valor do queue indexado na variável "control_queue" e armazenará o valor de retorno na variável "num".
