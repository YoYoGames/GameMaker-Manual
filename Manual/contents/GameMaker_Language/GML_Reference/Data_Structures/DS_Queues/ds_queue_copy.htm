ds_queue_copy

Esta função pode ser usada para copiar o conteúdo de um queue em outro. Note que isto NÃO remove o conteúdo do original queue, nem destrói o original queue. Ao utilizar esta função, o queue copiado deve ter sido previamente criado e se continha algum item antes da cópia, estes serão limpos primeiro (significando que esta informação será perdida).

Sintaxe:

ds_queue_copy(id, source);

Argumento Tipo Descrição
id DS Queue ID The id of the new queue.
source DS Queue ID The original queue to copy from.

Devoluções:

N/A

Exemplo:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    queue = ds_queue_create();
    ds_queue_copy(queue, other.queue);
}

A função acima cria uma nova instância e, nessa instância, cria uma nova fila DS e copia o conteúdo da queue na instância que executa o bloco de código, para a instância recém-criada queue.
