ds_stack_write

Esta função retorna um string que pode então ser armazenado ou transferido para outra estrutura de dados usando o ds_stack_read() função.

OBSERVAÇÃO: O retorno string não é uma leitura humana string, mas sim um despejo do conteúdo da estrutura de dados

Sintaxe:

ds_stack_write(id);

ArgumentoTipo Descrição
id The id of the data structure to write.

Devoluções:

Exemplo:

ini_open("save.ini");
var str = ds_stack_write(stack);
ini_write_string("Stacks", "0", str);
ds_stack_clear(stack);
ini_close();

O código acima abre um arquivo ini e então escreve um string contendo as informações armazenadas na pilha DS indexada na variável "stack". A pilha é então limpa e o arquivo ini é fechado.
