ds_stack_empty

Com esta função você pode verificar a pilha DS dada para ver se ela está vazia (retorna true) ou não (retorna false).

Sintaxe:

ds_stack_empty(id);

ArgumentoTipo Descrição
id The id of the data structure to check.

Devoluções:

Exemplo:

if (ai_count == 15 && !ds_stack_empty(AI_stack))
{
    ds_stack_clear(AI_stack);
    alarm[0] = room_speed;
    ai_count = 0;
}

O código acima verifica uma variável para ver se ela atingiu um valor específico e se ela tem a pilha DS indexada na variável "AI_stack", define um alarme e redefine a variável para 0.
