ds_stack_clear

Com esta função, você pode limpar todos os dados da estrutura de dados da pilha. Isto NÃO destrói a estrutura de dados (para isso você deve usar ds_stack_destroy()) ele apenas limpa todos os dados e devolve uma pilha vazia.

Sintaxe:

ds_stack_clear(id);

ArgumentoTipo Descrição
id The id of the data structure to clear.

Devoluções:

Exemplo:

if (ai_count = 15 && !ds_stack_empty(AI_stack))
{
    ds_stack_clear(AI_stack);
    alarm[0] = room_speed;
    ai_count = 0;
}

O código acima verifica uma variável para ver se ela atingiu um valor específico e se ela tem a pilha DS indexada na variável "AI_stack", define um alarme e redefine a variável para 0.
