data_é_data_data

Esta função retornará true se o valor de data/hora determinado for o dia em que está sendo verificado (ou seja: hoje), ou false caso contrário. Esta pode ser uma função útil para coisas como Ovos de Páscoa em seus jogos, ou para desbloquear conteúdo sazonal. Note que esta função será afetada pelo fuso horário definido (o padrão é a hora local) que você pode alterar usando o date_set_timezone() função.

Sintaxe:

date_is_today(date);

ArgumentoTipo Descrição
date The datetime to use.

Devoluções:

Exemplo:

if date_is_today(global.Halloween)
{
    global.Max_Levels = 200;
}

O código acima verificará a data/hora armazenada na variável global "Halloween" para ver se ela é hoje, e se ela é outra variável global para um novo valor.
