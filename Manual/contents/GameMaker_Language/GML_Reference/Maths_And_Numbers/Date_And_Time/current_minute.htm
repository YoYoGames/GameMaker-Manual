minuto_actual

Esta variável lida somente retornará as atas que correspondem ao momento atual.

Sintaxe:

current_minute;

Devoluções:

Exemplo:

draw_text(32, 32, "The time is " + string(current_hour) + ":" + string(current_minute) + "." + string(current_second));

O código acima desenharia a hora internacional atual na tela.
