current_year

This read-only variable will return the current year.

 

Syntax:

current_year

 

Returns:

Real

 

Example:

draw_text((32, 32, "Today is " + string(current_day) + "/" + string (current_month) + "/" + string(current_year) +".");

The above code will draw the day, month and year.
