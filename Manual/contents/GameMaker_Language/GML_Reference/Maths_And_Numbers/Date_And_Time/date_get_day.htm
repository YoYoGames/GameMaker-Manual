This function returns the day (from 1 to 31) of the given datetime.

 

Syntax:

date_get_day(date);

ArgumentType Description
date The date to check.

 

Returns:

 

Example:

myday = date_get_day( date_current_datetime() );

This would set "myday" to the current day.
