darctan

This function returns the inverse tangent (in degrees) of x.

This will accept any number as, unlike dtan, darctan asymptotes are on the y axis so it just means you'll never get returned a number greater than 90 or less than -90.

 This function returns a value in degrees, not radians.

 

Syntax:

darctan(x)

Argument Type Description
val Real The value to return the inverse tangent of

 

Returns:

Real

 

Example:

val = darctan(1);

This will set val to -45.
