tan

This function returns the tangent of an angle in radians. In a right-angled triangle the tangent is defined as tan(val) = opposite / adjacent, where val is one of the three angles.

 pi/2, (3*pi/2), -pi/2 and a vast number of other values will error with this function due to their returning infinity, a graph representation of this would produce asymptotes at these values.

 

Syntax:

tan(val)

Argument Type Description
val Real The angle (in radians) to return the tangent of

 

Returns:

Real

 

Example:

val = tan(pi);

This will set val to 0.
