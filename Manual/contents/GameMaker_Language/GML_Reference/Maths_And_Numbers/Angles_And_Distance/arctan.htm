arctan

This function returns the inverse tangent (in radians) of x.

This will accept any number as, unlike tan, arctan asymptotes are on the y axis which means you'll never get returned a number greater than pi/2 or less than -pi/2.

 This function returns a value in radians, not degrees.

 

Syntax:

arctan(x)

Argument Type Description
x Real The value to return the inverse tangent of

 

Returns:

Real

 

Example:

val = arctan(0);

This will set val to 0.
