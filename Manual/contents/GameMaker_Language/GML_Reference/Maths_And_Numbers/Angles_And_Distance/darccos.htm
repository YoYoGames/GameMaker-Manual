darccos

This function returns the inverse cosine (in degrees) of x, in that if dcos(val)=n, darccos(n)=val, and the resulting number will be between 0 and 180.

 This will only accept a number between -1 and 1 (anything else will throw an error).

 This function returns a value in degrees, not radians.

 

Syntax:

darccos(val)

Argument Type Description
x Real The value to return the inverse cosine of

 

Returns:

Real

 

Example:

val = arccos(-1);

This will set val to 180.
