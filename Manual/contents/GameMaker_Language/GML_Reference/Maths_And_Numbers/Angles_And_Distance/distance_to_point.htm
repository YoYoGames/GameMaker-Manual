distância_a_ponto

Esta função calcula a distância da borda da caixa delimitadora da instância chamadora até a posição x/y especificada no room, com o valor de retorno em pixels. Note que se a chamada object não tiver sprite ou nenhuma máscara definida, os resultados serão incorretos.

Sintaxe:

distance_to_point(x, y);

ArgumentoTipo Descrição
x The x position to check.
y The y position to check.

Devoluções:

Exemplo:

if (distance_to_point(obj_Player.x, obj_Player.y) < range)
{
    canshoot = true;
}

O código acima verificará a distância até a posição do jogador object x/y e se for menor que o valor armazenado na variável "faixa", a variável "canhoot" é definida como verdadeira.
