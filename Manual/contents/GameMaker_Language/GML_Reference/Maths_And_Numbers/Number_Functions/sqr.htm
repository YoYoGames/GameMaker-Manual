sqr

Multiplica um número por si só e assim retorna o quadrado desse número. EG: sqr(5) retorna 25 desde 5*5=25.

Sintaxe:

sqr(val);

ArgumentoTipo Descrição
val The number to square.

Devoluções:

Exemplo:

score += sqr(dmg);

O código acima irá adicionar no quadrado da variável "dmg" à pontuação.
