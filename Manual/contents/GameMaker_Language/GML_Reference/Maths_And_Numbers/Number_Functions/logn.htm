logn

This function returns the logarithm base n of the given number, which is the number of times that you need to multiply n by itself to get the value.

This function is similar to the log2 and log10 functions, only you stipulate the log base value. For example, logn(5,25) will return how many 5's we need to multiply to get 25 (which is 2).

 

Syntax:

logn(n, val)

Argument Type Description
n Real The log base
val Real The input value

 

Returns:

Real

 

Example:

logval = logn(5, num);

The above code gets the log5 of the value stored in num.
