logn

Esta função é similar à função de log2(n) e log10(n), somente você estipula o valor base de registro. Por exemplo, logn(5,25) devolverá quantos 5's precisamos para multiplicar para conseguir 25 (que é 2).

Sintaxe:

logn(n, val);

ArgumentoTipo Descrição
n The log base.
val The input value.

Devoluções:

Exemplo:

logval = logn(5, num);

O código acima recebe o log5 do valor armazenado em "num".
