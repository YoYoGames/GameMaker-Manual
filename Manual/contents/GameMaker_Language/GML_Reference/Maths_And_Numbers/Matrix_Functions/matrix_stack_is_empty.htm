matrix_stack_is_empty

Esta função pode ser usada para verificar se a pilha matriz está vazia (retorna true) ou não (retorna false).

Sintaxe:

matrix_stack_is_empty();

Devoluções:

Exemplo:

if !matrix_stack_is_empty()
{
    matrix_stack_clear();
}

O código acima verifica a pilha matricial para ver se está vazia e se não está limpa.
