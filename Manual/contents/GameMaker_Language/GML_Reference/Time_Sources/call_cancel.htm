call_cancel

Esta função cancela uma função de chamada de retorno que foi iniciada anteriormente por call_later.

Sintaxe:

call_cancel(handle);

Argumento Tipo Descrição
handle Identificação da fonte de tempo The handle to a Time Source retornado por call_later.

Devoluções:

N/A

Exemplo:

call_cancel(handle);

Voltar: call_later
