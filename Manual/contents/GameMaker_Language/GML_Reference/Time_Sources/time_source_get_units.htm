unidades_de_recurso_de_tempo

Esta função retorna as unidades definidas para a Fonte de Tempo em questão. O valor do período para a Fonte de Tempo é expresso nestas unidades.

Sintaxe:

time_source_get_units(id);

Argumento Tipo Descrição
id Identificação da fonte de tempo The Time Source para obter as unidades para

Devoluções:

Time Source Unit Constant

Voltar: Fontes de tempo
