This function returns the period value set for the given Time Source.

This period value will be expressed in the units set for the Time Source (seconds or frames).

 

Syntax:

time_source_get_period(id);

Argument Type Description
id Time Source ID The Time Source to get the period value for

 

Returns:

Real
