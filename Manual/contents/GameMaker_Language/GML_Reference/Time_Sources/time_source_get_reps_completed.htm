time_source_get_reps_completed

Esta função retorna o número de repetições que a Fonte de Tempo determinada completou desde seu último reset. Esse seria o número de vezes que ela expirou e executou seu método de retorno.

Sintaxe:

time_source_get_reps_completed(id);

Argumento Tipo Descrição
id Identificação da fonte de tempo The Time Source para obter informações para

Devoluções:

Real

Voltar: Fontes de tempo
