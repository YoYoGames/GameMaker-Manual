call_later

Esta função aciona uma função de retorno após o tempo especificado ou após um certo intervalo. Ela faz isto criando uma Fonte de Tempo (ver Fontes de Tempo).

Para cancelar a função de retorno de chamada, veja call_cancel.

IMPORTANTE Enquanto esta função retorna um ID de fonte de tempo real, este ID não pode ser usado com as funções de fonte de tempo, pois as fontes de tempo usadas por call_later e call_cancel são "invisíveis" - elas não aparecerão como filhos de nenhuma fonte de tempo embutida e não podem ser usadas com outras funções de fonte de tempo. Elas estão, entretanto, incluídas na contagem da fonte de tempo retornada pelo debug_event("ResourceCounts").

Argumentos

A seguir, uma explicação dos argumentos desta função:

Período

Este é um período de tempo que pode ser expresso em segundos ou quadros, dependendo da unidade especificada no próximo argumento.

Ao utilizar armações como unidade, o período deve ser um número inteiro. Os valores não inteiros são arredondados para baixo (piso), com exceção dos valores inferiores a 1, que são arredondados para cima para 1.

Unidade

Esta é a unidade em que o período é expresso, e pode ser tempo_unidades_de_fonte_segundos ou tempo_unidades_de_fonte_quadro.

Veja: Unidades de fonte de tempo

Você pode usar um valor de batidas por minuto usando time_bpm_to_seconds().

Callback

Você deve especificar uma "ligação de retorno". method que é chamado quando a Fonte de Tempo expira.

Loop

Se a fonte de tempo deve loop.

Sintaxe:

call_later(period, unit, callback, [loop=false]);

Argumento Tipo Descrição
period Real The period that the Time Source funciona para, nas unidades dadas
units Unidade Fonte de Tempo Constante The units that the given period is in
callback Método ou Função Script The method or script function to call after the specified time
loop Booleano OPTIONAL Se a fonte de tempo deve ser laçada (o padrão é falso)

Devoluções:

Time Source ID

Exemplo 1:

var _callback = function()
{
    show_debug_message("Five seconds have passed!");
}
var _handle = call_later(5, time_source_units_seconds, _callback);

Neste exemplo, criamos uma função de chamada de retorno que mostra uma mensagem de depuração após cinco segundos passados. Depois disso, a fonte de tempo subjacente é destruída.

Exemplo 2:

var _create_enemy = function()
{
    var _x = random(room_width);
    var _y = random(room_height);
    instance_create_depth(_x, _y, depth, obj_enemy);
}

var _handle = call_later(10, time_source_units_seconds, _create_enemy, true);

Neste exemplo, criamos uma função de callback que cria um novo inimigo em uma posição aleatória no room. Esta função é então executada a cada 10 segundos, passando-a para call_later. O último argumento é definido para verdadeiro Assim, a função continua sendo chamada a cada 10 segundos.

Use o call_cancel com a alça devolvida pelo call_later para parar a fonte de tempo e as chamadas para o retorno da chamada.
