GM_runtime_version

Esta constante contém o número da versão runtime, conforme definido nas Preferências, como o runtime que está sendo usado para construir o projeto. O valor é armazenado como um string.

Sintaxe:

GM_runtime_version;

Devoluções:

String

Exemplo:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);
draw_text(32, 96, "Runtime " + GM_runtime_version);

O código acima pega a data GM, a versão e as constantes runtime e as desenha para a tela.
