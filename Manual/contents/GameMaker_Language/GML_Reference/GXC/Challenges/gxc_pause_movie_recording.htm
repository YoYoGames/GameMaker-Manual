gxc_pause_movie_recording

Esta função é usada para pausar uma gravação GXC que foi previamente iniciada com gxc_start_movie_recording().

Sintaxe:

gxc_pause_movie_recording();

Devoluções:

Exemplo:

if (keyboard_check_pressed(vk_escape))
{
    gxc_pause_movie_recording();
}

Isto fará uma pausa na gravação do filme do GXC se a tecla escape for pressionada.
