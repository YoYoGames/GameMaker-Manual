gxc_resume_movie_recording

Esta função é usada para retomar uma gravação GXC que foi previamente pausada com gxc_pause_movie_recording().

Sintaxe:

gxc_resume_movie_recording();

Devoluções:

Exemplo:

if (keyboard_check_pressed(vk_enter))
{
    gxc_resume_movie_recording();
}

Isto retomará a gravação do filme GXC atualmente em pausa, se a tecla enter for pressionada.
