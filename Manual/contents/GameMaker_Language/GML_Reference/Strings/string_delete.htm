string_delete

Esta função é utilizada para remover uma parte do dado string. Você fornece a entrada string, a posição inicial dentro desse string para remover caracteres de (o índice começa em 1) e o número de caracteres a serem removidos. A função retornará um novo string sem essa parte nele.

Sintaxe:

string_delete(str, index, count);

ArgumentoTipo Descrição
str The string to copy and delete from.
index The position of the first character to remove (from 1).
count The number of characters to remove.

Devoluções:

String

Exemplo:

str1 = "Helloo World";
str2 = string_delete(str1, 5, 1);

Isto colocará str2 em "Hello World", pois remove o "o" extra de "Hello" (que está presente na posição 5).
