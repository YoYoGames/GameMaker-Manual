ord

Esta função recebe uma entrada de caractere único string e retorna o valor Unicode (UTF8) para esse caractere. Note que quando usado com a função keyboard_check* a entrada string pode ter apenas um caractere e só pode ser um número de 0 a 9 ou um caractere romano capitalizado de A a Z.

Sintaxe:

ord(string);

Argumento Tipo Descrição
string Cordão The string with which to find the Unicode value.

Devoluções:

Real

Exemplo:

if keyboard_check(ord("W"))
{
    y -= 4;
}

Isto moverá a instância de chamada quatro pixels para cima se a chave W for mantida para baixo.
