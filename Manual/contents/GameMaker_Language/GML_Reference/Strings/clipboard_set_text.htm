clipboard_set_text

Esta função irá definir a prancheta para segurar o definido string. Você pode defini-la como vazia string "" para limpar efetivamente a área de transferência de texto.

Sintaxe:

clipboard_set_text(string);

Argumento Tipo Descrição
string Cordão The text (as a string) to set the clipboard to hold.

Devoluções:

N/A

Exemplo:

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

O código acima verifica o texto na prancheta e se ela contém algum, ele é lido como um string na variável "str". Finalmente, a área de transferência é limpa, definindo-a como um string vazio.
