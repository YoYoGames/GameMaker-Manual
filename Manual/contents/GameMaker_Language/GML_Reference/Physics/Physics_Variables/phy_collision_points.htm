phy_collision_points

Esta variável somente leitura retorna o número de pontos de colisão detectados entre os dois objects na colisão

NOTA: Esta variável só está disponível no evento de colisão de uma instância física habilitada.

Sintaxe:

phy_collision_points;

Devoluções:

(integer value, or undefined if the instance is not physics enabled)

Exemplo:

for(var i = 0; i < phy_collision_points; i += 1;)
{
    part_particles_create(global.Sname, phy_collision_x[i], phy_collision_x[1], global.Spark, 5);
}

O código acima cria partículas em todos os pontos definidos de uma colisão entre duas instâncias físicas habilitadas.
