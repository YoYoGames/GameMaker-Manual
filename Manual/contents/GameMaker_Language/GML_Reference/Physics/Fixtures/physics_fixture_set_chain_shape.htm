forma_da_cade_física

Esta função define uma forma de "corrente" de fixação. Uma fixação em cadeia é composta de vários pontos, que são então conectados entre si usando formas de bordas para uni-los. A própria função leva o índice (ID) da fixação e pode dizer a fixação para loop ou não. Ajustando isto para true conectará o último ponto ao primeiro ponto com uma borda, enquanto que o ajuste para false não o fará. Note que esta função prepara a fixação para aceitar os pontos necessários para fazer a cadeia, e estes devem ser adicionados após chamar esta função usando physics_fixture_add_point()como você faria na construção de um polígono.

Ao criar um acessório de corrente, você deve dar-lhe pelo menos dois pontos, mas não está limitado no número de pontos subseqüentes que você pode adicionar para compor o acessório acabado.

Sintaxe:

physics_fixture_set_chain_shape(fixture, loop)

ArgumentoTipo Descrição
fixture The index of the fixture
loop Whether to close the chain (true) ou não (false).

Devoluções:

Exemplo:

var xx = -100;
var yy = room_height / 2;
var fix = physics_fixture_create(); physics_fixture_set_chain_shape(fix, false);
for (var i = 0; i < 10; i++;)
{
    physics_fixture_add_point(fix, xx, yy);
    xx += 50 + random(150);
    yy = (room_height / 2) - 250 + irandom(500);
}
physics_fixture_bind(fix, id);

O código acima criará uma linha de "bordas" com uma variedade de alturas ao longo do comprimento do room.
