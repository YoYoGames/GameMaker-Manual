physics_get_friction

This function gets the friction value of the given bound fixture (not the "base" fixture).

 

Syntax:

physics_get_friction(fixture)

Argument Type Description
fixture Physics Fixture ID the ID of the bound fixture

 

Returns:

Real

 

Example:

var _fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, _fric + 0.1);

The code above gets the current friction value for the bound physics properties of the instance and then sets them to a different value.
