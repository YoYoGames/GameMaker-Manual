valor_junto_físico

Certas propriedades da junta podem ser alteradas e definidas mesmo após a criação da junta. Há uma série de constantes que podem ser usadas nesta função e que podem ser encontradas aqui: Constantes de Articulação Física.

Sintaxe:

physics_joint_set_value(joint, field, value)

Argumento Tipo Descrição
joint Identificação Conjunta Física The index of the joint that you wish to change
field Constante da Junta Física The constant for the joint property that you wish to change
value Real The new value for the joint property

Devoluções:

N/A

Exemplo:

if physics_joint_get_value(revJoint, phy_joint_max_motor_torque) < 2
{
    physics_joint_set_value(revJoint, phy_joint_max_motor_torque, 2);
}

O código acima verifica se o torque máximo do motor das juntas está ajustado para menos de 2 e se é 2.
