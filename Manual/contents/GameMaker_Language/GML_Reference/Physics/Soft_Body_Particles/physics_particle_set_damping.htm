fisica_particle_set_damping

Com esta função você pode definir o amortecimento linear de partículas na simulação. O amortecimento é usado para reduzir a velocidade de simulação física de instâncias ao longo do tempo, muito parecido com a resistência do ar no mundo real. Esta função é global no escopo, na medida em que mudará o amortecimento não apenas para novas partículas criadas após a mudança, mas também para aquelas já presentes na simulação.

Sintaxe:

physics_particle_set_damping(damping)

ArgumentoTipo Descrição
damping The linear damping to be applied to the particle fixture.

Devoluções:

Exemplo:

physics_particle_set_radius(15);
physics_particle_set_density(0.5);
physics_particle_set_damping(1);
physics_particle_set_gravity_scale(1);

O código acima definirá as propriedades básicas para todas as partículas na simulação.
