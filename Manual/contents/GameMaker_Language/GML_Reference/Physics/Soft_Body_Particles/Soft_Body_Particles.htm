Partículas macias do corpo

GameMaker utiliza a biblioteca de física corporal rígida Box2D, e como tal não permite a física corporal suave. Entretanto, o Diversão líquida extension adiciona partículas ao sistema físico, o que permite a criação de corpos moles simulados usando a física rígida do corpo. Basicamente, pode-se criar uma forma de partículas que mantém a coesão e assim simular um corpo completo, o que, devido à interação das partículas, dá a impressão de um corpo macio. Você também pode usar as partículas físicas para simular líquidos e outros sistemas físicos baseados em partículas.

Antes de começar a criar suas partículas físicas, você precisará definir as propriedades físicas básicas para elas, como se fosse uma fixação na física regular. Note que a definição destas propriedades afetará todas as partículas criadas anteriormente, bem como quaisquer novas partículas que você criar, de modo que você normalmente só usaria estas funções no início do jogo ou nível. Entretanto, há funções específicas disponíveis para definir as bandeiras de partículas para partículas individuais (e grupos de partículas) que mudarão suas propriedades como "ressalto" ou "viscosidade", permitindo que você crie diferentes tipos de partículas dentro das faixas destas configurações globais.

As seguintes funções estão disponíveis para a configuração e obtenção de dados do sistema global de partículas:

  • physics_particle_count
  • physics_particle_group_count
  • physics_particle_set_max_count
  • physics_particle_set_radius
  • physics_particle_set_density
  • physics_particle_set_damping
  • physics_particle_set_gravity_scale
  • physics_particle_get_max_count
  • physics_particle_get_radius
  • physics_particle_get_density
  • physics_particle_get_damping
  • physics_particle_get_gravity_scale

Você também tem as seguintes funções disponíveis para colocar bandeiras em partículas individuais (ou grupos de partículas), bem como para obter informações, como posição ou cor, de uma única partícula ou de um único grupo:

  • physics_particle_set_flags
  • physics_particle_set_category_flags
  • physics_particle_set_group_flags
  • physics_particle_get_group_flags
  • physics_particle_group_get_inertia
  • física_grupo_de_partículas_get_mass
  • physics_particle_group_get_centre_x
  • physics_particle_group_get_centre_y
  • physics_particle_group_get_vel_x
  • physics_particle_group_get_vel_y
  • physics_particle_group_get_ang_vel
  • physics_particle_group_get_x
  • physics_particle_group_get_y
  • physics_particle_group_get_angle
  • physics_particle_get_data
  • physics_particle_get_data_particle
  • physics_particle_group_get_data

Você pode criar partículas de duas maneiras - individualmente ou em grupos. As partículas individuais se comportarão exatamente como definido pelas configurações globais e pelo flags que foram definidas quando foram criadas, mas grupos de partículas podem exibir comportamentos diferentes dependendo de suas bandeiras de grupo. Estas bandeiras de grupo podem ser definidas em valores diferentes para dar ao grupo propriedades diferentes, como penetração e rigidez.

As seguintes funções estão disponíveis para a criação de partículas:

  • physics_particle_create
  • physics_particle_group_begin
  • physics_particle_group_circle
  • physics_particle_group_box
  • physics_particle_group_polygon
  • physics_particle_group_add_point
  • physics_particle_group_end
  • physics_particle_group_join
  • physics_particle_group_delete

Você não só pode criar partículas, mas também eliminá-las, seja individualmente ou em uma região. As seguintes funções estão disponíveis para apagar as partículas do sistema:

  • physics_particle_delete
  • physics_particle_delete_region_circle
  • physics_particle_delete_region_box
  • physics_particle_delete_region_poly

A menos que você tenha ativado o desenho debug usando physics_draw_debug()Não se vêem partículas na tela ao criá-las. Portanto, para ver realmente as partículas, você precisa desenhá-las para a tela no Evento de Desenho de uma instância usando as seguintes funções (note que você também pode desenhá-las usando as funções regulares draw_sprite se você obtiver os dados para cada partícula usando a função physics_particle_get_data()):

  • physics_particle_draw
  • physics_particle_draw_ext
