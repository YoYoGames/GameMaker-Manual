physics_debug

This function sets whether to output physics errors to The Output Window.

By default, physics errors are not output.

 

Syntax:

physics_debug(enable);

Argument Type Description
enable Boolean Whether to output physics errors or not

 

Returns:

N/A

 

Example:

physics_debug(true);

The code above enables the output of physics errors.
