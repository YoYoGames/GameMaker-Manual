uwp_suspend

Esta função indica ao sistema que o aplicativo terminou de salvar dados após a detecção de um pedido de suspensão. A chamada desta função é um requisito para a apresentação do Xbox One, e deve ser sempre usada em conjunto com a função uwp_is_suspending()como mostrado no exemplo abaixo.

Sintaxe:

uwp_suspend();

Devoluções:

Exemplo:

if uwp_is_suspending()
{
    scr_Save_Game_Data();
    uwp_suspend();
}

O código acima verifica se o aplicativo está entrando em suspensão e se está chamando um script para salvar os dados do jogo antes de suspender o jogo.
