xboxlive_matchmaking_start

Esta função deve ser chamada antes que você possa iniciar qualquer sessão multi-jogador para o usuário. Ela pega o ponteiro User ID, que você pode recuperar para um determinado bloco de jogos chamando o xboxlive_user_for_pad() e inicializará a API multiplayer.

Sintaxe:

xboxlive_matchmaking_start(user_id);

ArgumentoTipo Descrição
user_id The user ID pointer to use

Devoluções:

Exemplo:

global.UserID = xboxlive_user_for_pad(global.PadIndex);
xboxlive_matchmaking_start(global.UserID);

O código acima obtém a identificação do usuário para o índice de teclado dado e, em seguida, inicia as capacidades multiplayer.
