xboxlive_id_de_utilizador_para_utilizador

Esta função converte o ponteiro de identificação do usuário em um string.

Sintaxe:

xboxlive_user_id_for_user(user_id);

ArgumentoTipo Descrição
user_id The ID pointer of the user to check.

Devoluções:

Exemplo:

var _uid = xboxlive_user_for_pad(0);
uid_string = xboxlive_user_id_for_user(_uid);

O código acima obtém o ponteiro de identificação do usuário para o usuário atribuído ao slot do gamepad [0], e então salva essa identificação do usuário como uma variável de instância string.
