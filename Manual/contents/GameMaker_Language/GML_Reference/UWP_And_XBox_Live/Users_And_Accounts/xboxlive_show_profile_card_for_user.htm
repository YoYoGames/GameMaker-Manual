xboxlive_show_profile_card_for_user

Com esta função, você pode solicitar que a XBox mostre o cartão de perfil para o ponteiro de identificação do usuário alvo. A função requer tanto o ponteiro de identificação do usuário para o usuário que está solicitando as informações quanto o ponteiro de identificação do usuário para o usuário alvo e obter o cartão de perfil do usuário.

Sintaxe:

xboxlive_show_profile_card_for_user(requesting_user_id, target_user_id);

ArgumentoTipo Descrição
requesting_user_id The user ID (a pointer) of the requesting user
target_user_id The user ID (a pointer) of the user to get the profile card of

Devoluções:

Exemplo:

if (gamepad(0, gp_start))
{
    xboxlive_show_profile_card_for_user(user[0], user[1]);
}

O código acima verifica se há um botão de controle de jogo e, se for detectado, mostra o cartão de perfil para o usuário em questão.
