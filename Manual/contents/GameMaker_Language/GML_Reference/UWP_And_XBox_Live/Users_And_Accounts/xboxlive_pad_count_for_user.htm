xboxlive_pad_count_for_user

Com esta função você pode descobrir quantos "slots" de bloco estão ativos para o usuário atual (veja xboxlive_pad_for_user() para mais detalhes).

Sintaxe:

xboxlive_pad_count_for_user(user_id, slot);

ArgumentoTipo Descrição
user_id The user ID (a pointer) to check

Devoluções:

Exemplo:

global.slots[0] = xboxlive_pad_count_for_user(user_id[0])

O código acima armazena o número de blocos associados com o ponteiro de identificação do usuário dado.
