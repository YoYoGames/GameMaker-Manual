shader_set_uniform_matrix_array

Com esta função você pode definir uma constante shader para manter uma matriz de valores. Você deve ter obtido previamente a "manipulação" da constante usando a função shader_get_uniform()e você terá que ter previamente rubricado a matriz como uma matriz de valores de ponto flutuante, onde cada grupo sucessivo de 16 flutuadores é uma matriz 4x4.

NOTA: Todos os uniformes devem ser ajustados após a chamada da função shader_set()e antes de chamar shader_reset().

Sintaxe:

shader_set_uniform_matrix_array(handle, array);

Argumento Tipo Descrição
handle Cabo Uniforme Shader The handle of the shader constant to set.
array Array A previously initialised array of floating point values.

Devoluções:

N/A

Exemplo:

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_tint, "cMatrixArray");
shader_set_uniform_matrix_array(shader_params, matrix_array);
draw_self();
shader_reset();

O código acima obterá o comando da constante "cMatrixArray" shader e então definirá essa constante para a matriz dada.
