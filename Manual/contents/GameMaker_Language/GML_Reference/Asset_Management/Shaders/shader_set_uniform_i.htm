shader_set_uniform_i

Com esta função você pode definir o valor (ou valores) de uma constante shader. Você deve ter obtido previamente a "manipulação" da constante usando a função shader_get_uniform()e você terá que saber que tipo de constante é para passar o número correto de valores inteiros, ou seja: se você tem um vec2 você terá que passar dois valores para a função.

NOTA: Todos os uniformes devem ser ajustados após a chamada da função shader_set()e antes de chamar shader_reset().

Sintaxe:

shader_set_uniform_i(handle, value1 [, value2, value3, value4]);

Argumento Tipo Descrição
handle Cabo Uniforme Shader The handle of the shader constant to set.
value1 ... value4 Real The integer value (or values) to set the shader constant to.

Devoluções:

N/A

Exemplo:

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_i(shader_params, 0, 65, 255);
draw_self();
shader_reset();

O código acima receberá o comando da constante shader "u_vParams" ( a vec3), e então definirá essa constante para os valores inteiros dados.
