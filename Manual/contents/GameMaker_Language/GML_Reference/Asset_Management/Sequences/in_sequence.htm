in_sequence

Esta é uma variável incorporada que faz parte das variáveis de instância criadas para cada object instância em seu jogo. Se a instância estiver sendo controlada por um sequence, esta variável retornará true, caso contrário retornará false. Esta é uma variável somente de leitura e não pode ser alterada.

Note que esta variável se tornará false após o fim do controle Sequence, mesmo que seu elemento Sequence ainda exista, e se o elemento Sequence for reproduzido novamente, se tornará true.

Você pode usar esta variável em seu player (ou CPU) controlado objects para ter certeza de que eles só serão movidos por código quando não estiverem em um Sequence. Isto pode ser útil para cutscenes, quando você precisar usar as mesmas instâncias em um Sequence (usando a função sequence_instance_override_object() ) e você precisa ter certeza de que eles não podem ser movidos por seu código habitual enquanto aquela cutscene estiver ativa.

Sintaxe:

in_sequence

Devoluções:

Exemplo:

if (!in_sequence)
{
    x += move_x;
    y += move_y;
}

O código acima verifica a variável in_sequence, e se for false (o que significa que a instância não está sendo controlada por um sequence), então acrescenta move_x e move_y à variável posição, certificando-se de que ele só se move quando não está em uma Sequence.
