extensão_existe

Esta função informa se existe ou não um extension com o nome dado(verdadeiro)(falso).

Sintaxe:

extension_exists(extension_name);

Argumento Tipo Descrição
extension_name Cordão The name of your extension asset as a string

Devoluções:

Boolean

Exemplo:

if (extension_exists("MyExtension"))
{
    var _options = extension_get_options("MyExtension");
    var _enabled = _options.enabled;
}

Este código verifica se o MyExtension extension existe, e se existe, recebe suas opções como struct, e lê um valor a partir dele.
