animcurve_get_channel_index

Esta função retornará o valor do índice para qualquer canal animation curve. Você fornece o ID animation curve ou struct, onde o ID da curva é o nome do animation curve como foi definido no Asset Browser, ou o ponteiro struct como retornado pela função animcurve_create(). Você então fornece o nome do canal, como um string, e a função retornará o valor do índice associado a esse canal. Note que se a curva ou canal não existir, então você receberá um erro.

Sintaxe:

animcurve_get_channel_index(curve_struct_or_id, channel_name);

Argumento Tipo Descrição
curve_struct_or_id Estruturas Curvas de Animação The ID or struct pointer of the animation curve to target
channel_name Cordão The channel name (a string).

Devoluções:

Real

Exemplo:

var _channelindex = animcurve_get_channel_index(ac_ButtonTween, "x_pos")
var _channeldata = animcurve_get_channel(ac_ButtonTween, _channelindex);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
    _points[i].value += 1;
}

O código acima recupera o canal struct para o canal chamado "x_pos" na curva asset "ac_ButtonTween", depois loops através dos pontos na curva do canal e adiciona um ao seu valor.
