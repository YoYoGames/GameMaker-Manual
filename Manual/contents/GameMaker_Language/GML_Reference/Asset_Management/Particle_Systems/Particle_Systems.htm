Particle System Assets

The following functions are available when working with particle system assets, which you either create using The Particle System Editor or at runtime using code. These functions are only for dealing with an asset itself, and not for working with particle system instances at runtime in your game (for that see the appropriate functions on the section Particles).

The functions available for working with a Particle System Asset are:

  • particle_exists
  • particle_get_info
  • particle_add
  • particle_delete
