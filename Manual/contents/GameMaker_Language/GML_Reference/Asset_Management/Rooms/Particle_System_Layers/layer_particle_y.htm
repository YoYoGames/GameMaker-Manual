layer_particle_y

This function sets the position on the y axis of the given particle system element.

 This value is the y offset of the element, which is different from the value you set using part_system_position.

 

Syntax:

layer_particle_y(particle_element_id, y);

Argument Type Description
particle_element_id Particle System Element ID The unique ID value of the particle system element
y Real The position on the y axis of the particle system element

 

Returns:

N/A
