layer_particle_get_angle

This function gets the angle of the given particle system element.

 This function gets the angle of the element, which is different from the value set using part_system_angle.

 

Syntax:

layer_particle_get_angle(particle_element_id)

Argument Type Description
particle_element_id Particle System Element ID The unique ID value of the particle system element

 

Returns:

Real

 

Example:

var _element_id = layer_particle_get_id("Particles", "particle_smoke");
var _angle = layer_particle_get_angle(_element_id);

The code above gets the element ID of a particle system element named "particle_smoke"  on a layer "Particles". It then retrieves the angle of the element and stores it in a variable for further use.
