layer_particle_yscale

This function sets the scale on the y axis of the given particle system element.

 

Syntax:

layer_particle_yscale(particle_element_id, yscale);

Argument Type Description
particle_element_id Particle System Element ID The unique ID value of the particle system element
yscale Real The scale on the y axis of the particle system element

 

Returns:

N/A
