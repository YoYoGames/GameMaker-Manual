tile_set_index

Esta função pode ser usada para definir o índice tile (a posição do tile dentro da imagem tile set ) para um conjunto de dados de azulejos. Você dá os dados do azulejo, que podem ser recuperados usando a função tilemap_get()e depois o valor inteiro que representa a posição do tile (seu índice) dentro do tile set. A função retornará um conjunto modificado de dados de azulejos que pode então ser aplicado usando o tilemap_set() função.

Sintaxe:

tile_set_index(tiledata, index)

Argumento Tipo Descrição
tiledata Dados sobre os azulejos the tile-data to set
index Real The integer index value of the tile to use

Devoluções:

Tile Data

Exemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var ind = tile_get_index(data);
data = tile_set_index(data, irandom(23));
tilemap_set(map_id, data, mx, my);

O código acima obtém o ID do mapa tile da camada dada e depois usa-o para obter os dados do azulejo para a célula sob a posição do mouse. Estes dados são então usados para definir o índice tile para um número aleatório.
