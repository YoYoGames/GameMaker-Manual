tile_set_empty

Esta função pode ser usada para definir um determinado conjunto de dados de azulejos para ser um conjunto vazio tile. Você dá os dados de azulejos, que podem ser recuperados usando a função tilemap_get(), e a função retornará true se o tile estiver vazio, ou false se houver um índice tile. A função retornará um conjunto modificado de dados de azulejos que poderá então ser aplicado usando a função tilemap_set().

Sintaxe:

tile_set_empty(tiledata)

Argumento Tipo Descrição
tiledata Dados sobre os azulejos the tile-data to set

Devoluções:

Tile Data

Exemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
for (var i = 0; i < tilemap_get_width(map_id); i++;)
{
    for (var j = 0; j < tilemap_get_height(map_id); j++;)
    {
        var data = tilemap_get(map_id, i, j);
        if !tile_get_empty(data)
        {
            data = tile_set_empty(data)
            tilemap_set(map_id, data, i, j);
        }
    }
}

O código acima obtém o ID do mapa tile da camada dada e então procede à verificação de cada célula tile no mapa para ver se ela tem ou não dados. Se tiver, o tile está configurado para esvaziar.
