tile_get_index

Esta função pode ser usada para obter o índice tile (a posição do tile dentro da imagem tile set ) a partir de um conjunto de dados de azulejos. O usuário especifica os dados do azulejo, que podem ser recuperados usando a função tilemap_get()e a função retornará um valor inteiro para o índice.

Ao utilizar esta função, certifique-se de que os dados do azulejo dado sejam válidos, caso contrário o valor do índice retornado também não será válido, o que pode resultar em comportamento inesperado.

Sintaxe:

tile_get_index(tiledata)

Argumento Tipo Descrição
tiledata Dados sobre os azulejos the tile-data to check

Devoluções:

Real

Exemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var ind = tile_get_index(data);
data = tile_set_index(data, irandom(23));
tilemap_set(map_id, data, mx, my);

O código acima obtém o ID do mapa tile da camada dada e depois usa-o para obter os dados do azulejo para a célula sob a posição do mouse. Estes dados são então usados para definir o índice tile para um número aleatório.
