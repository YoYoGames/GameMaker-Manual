tilemap_x

Esta função controla a posição ao longo do eixo x do room do elemento do mapa asset tile na camada. Você dá o ID do elemento de mapa tile (que você obtém quando cria um elemento de mapa tile usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), e então definir o valor x a ser usado (com base nas coordenadas room ).

Sintaxe:

tilemap_x(tilemap_element_id, x);

Argumento Tipo Descrição
tilemap_element_id Identificação do Elemento do Mapa de Telha The unique ID value of the tile map element to change
x Real The x position for the tile map

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_x(map_id, irandom(room_width));

tile O código acima obtém o valor de ID do mapa asset atribuído à camada "Asset_sky" e então define sua posição x para um valor aleatório entre 0 e a largura do room.
