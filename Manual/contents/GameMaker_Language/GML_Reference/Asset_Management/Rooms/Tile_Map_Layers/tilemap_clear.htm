tilemap_clear

Usando esta função, você pode limpar/ajustar todo o tiles em um determinado mapa de azulejos. Você dá o ID do elemento do mapa tile (que você obtém ao criar um elemento do mapa tile usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), e depois fornecer os dados tile com os quais você deseja limpar a camada. Um valor padrão de 0 limpará todos os tiles da camada (essencialmente tornando todos tiles "vazios"), enquanto você pode usar as funções dedicadas tile_* para criar seus próprios dados tile para limpar o mapa tile.

Sintaxe:

tilemap_clear(tilemap_element_id, tiledata)

Argumento Tipo Descrição
tilemap_element_id Identificação do Elemento do Mapa de Telha The unique ID value of the tile map element to change
tiledata Dados do Ladrilho (Real) The tile data to use to clear the layer

Devoluções:

N/A

Exemplo:

var layer_id = layer_get_id("Forest");
var tile_id = layer_tilemap_get_id(layer_id);
tilemap_clear(tile_id, 0);

O código acima obtém o valor de identificação de um mapa tile criado no editor room, e depois o limpa usando "vazio" tiles.
