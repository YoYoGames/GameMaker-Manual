tilemap_set

Esta função pode ser usada para definir qualquer célula (quadrado de grade) dentro do elemento do mapa tile na camada para um novo tile. Você dá o ID do elemento do mapa tile (que você obtém ao criar um elemento do mapa tile usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), e depois os dados a definir em tile, bem como a posição dentro do mapa tile. Você pode recuperar os dados tile usando a função tilemap_get() e depois usar as funções tile_ para mudá-lo antes de configurar a célula usando esta função. A função retornará true se o tile foi configurado com sucesso e false se houve um problema e não foi configurado.

Sintaxe:

tilemap_set(tilemap_element_id, tiledata, xcell, ycell)

Argumento Tipo Descrição
tilemap_element_id Identificação do Elemento do Mapa de Telha The unique ID value of the tile map element to change
tiledata Dados do Ladrilho (Real) The tile data to set
xcell Real The cell (grid) position to set along the x-axis
ycell Real The cell (grid) position to set along the y-axis

Devoluções:

Boolean

Exemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

O código acima obtém a identificação para o mapa tile na camada "Tiles_Sky" e depois a usa para obter os dados do tile na célula (0, 0). Estes dados do tile são então virados antes de serem usados para definir novamente o tile no mapa tile.
