tilemap_get_cell_x_at_pixel

Usando esta função você pode recuperar a posição do eixo x de uma célula do mapa individual tile dando a posição relativa do eixo x dentro do room. Você dá o ID do elemento do mapa tile (que você obtém ao criar um elemento do mapa tile usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), assim como a posição x e y dentro da sala e a função devolverá a posição x da célula dentro do mapa tile para esse ponto. Observe que se o valor estiver fora da área do mapa tile, e não houver célula disponível, ele retornará -1.

Sintaxe:

tilemap_get_cell_x_at_pixel(tilemap_element_id, x, y);

ArgumentoTipo Descrição
tilemap_element_id The unique ID value of the tile map element to get the cell x position of
x The x position within the room to use for getting the cell
y The y position within the room to use for getting the cell

Devoluções:

(x-axis cell position or -1 if there is an error)

Exemplo:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var _y = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
tiledata = tilemap_get(map_id, _x, _y);

O código acima usa o ID do mapa tile recuperado para obter a posição da célula a partir de um conjunto de coordenadas room e então armazenar os dados para qualquer tile encontrado ali em uma variável de instância.
