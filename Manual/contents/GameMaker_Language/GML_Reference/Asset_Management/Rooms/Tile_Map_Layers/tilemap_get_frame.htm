tilemap_get_frame

Como tiles pode ser animado, às vezes pode ser útil saber qual quadro está sendo desenhado atualmente e reagir de acordo, portanto, com esta função você pode recuperar o índice atual do quadro para um dado mapa tile. Você dá o ID do elemento do mapa tile (que você obtém quando cria um elemento do mapa tile usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), e a função retornará o índice do quadro.

Sintaxe:

tilemap_get_frame(tilemap_element_id)

Argumento Tipo Descrição
tilemap_element_id Identificação do Elemento do Mapa de Telha The unique ID value of the tile map element to get the frame index of

Devoluções:

Real (between 0 (inclusive) and the maximum number of frames of animation (exclusive))

Exemplo:

var lay_id = layer_get_id("Tiles_Traps");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_frame(map_id) >= 2 && tilemap_get_frame(map_id) < 4
{
    global.activate = true;
}
else
{
    global.activate = false;
}

O código acima verifica o quadro de animação atual para o mapa tile dado na camada dada, e define uma variável global com base no valor de retorno.
