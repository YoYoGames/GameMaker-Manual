tilemap_get_at_pixel

Usando esta função, você pode recuperar os dados tile de uma posição (dentro do room) do elemento do mapa tile. Você dá o ID do elemento do mapa tile (que você obtém quando cria um elemento do mapa tile usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()) assim como a posição x e y na sala para obter os dados tile e a função devolverá o "blob" de dados tile. Estes dados são essencialmente uma máscara de bit que contém o índice tile, as booleanas flip/rotate/mirror e quaisquer dados de máscara que tenham sido aplicados (veja tilemap_set_mask() para detalhes), e o valor dos dados resultantes pode então ser usado nas funções tile_* para alterar uma propriedade tiles. Se você precisar obter os dados tile de uma célula específica do tile, você deve estar usando a função tilemap_get() ao invés disso.

IMPORTANTE! Se o tiles no mapa tile não tiver sido alterado (ou seja: eles não estão girados ou invertidos etc...), então o valor de retorno dos dados do tile set será exatamente igual ao índice do tile no tile set. Assim você pode criar "mapas de colisão" de tiles usando um tile no índice 1 no tile set - por exemplo - então use esta função para verificar 1 ou 0 (um tile vazio) para calcular as colisões.

Sintaxe:

tilemap_get_at_pixel(tilemap_element_id, x, y);

Argumento Tipo Descrição
tilemap_element_id Identificação do Elemento do Mapa de Telha The unique ID value of the tile map element to get the tile-data of
x Real The position along the x-axis to get the tile data from (in room pixels)
y Real The position along the y-axis to get the tile data from (in room pixels)

Devoluções:

Tile Data (-1 if there is an error)

Exemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 64, 128);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 64, 128);

O código acima obtém a identificação para o mapa tile "Clouds" na camada "Tiles_Sky" e depois usa isso para obter os dados do tile na posição (64, 128). Estes dados do tile são então virados antes de serem usados para definir novamente o tile no mapa tile.
