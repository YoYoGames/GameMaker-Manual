room_goto_next

Com esta função você pode fazer seu jogo ir para o próximo, conforme listado no Gerente de Sala no momento em que o jogo foi compilado. Se este room não existir, um erro será lançado e o jogo será forçado a fechar. Note que o room não mudará até o final do evento onde a função foi chamada, portanto, qualquer código após este ter sido chamado ainda será executado se no mesmo evento.

Sintaxe:

room_goto_next();

Devoluções:

Exemplo:

if room_exists(room_next(room))
{
    room_goto_next();
}

O código acima irá verificar se há outro room após o atual e, se houver, irá para room.
