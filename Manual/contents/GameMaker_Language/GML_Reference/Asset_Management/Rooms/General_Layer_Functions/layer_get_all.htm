layer_get_all

This function will return an array populated with the handles of each layer in the current room. This will also include UI Layers as well as any layers created at runtime using layer_create in the current room, unless a target room has been set, then it will only return the room layers created in the IDE for that particular room (even if it is set to the current room).

 

Syntax:

layer_get_all()

 

Returns:

Array (populated with Layer IDs)

 

Example:

var a = layer_get_all();
for (var i = 0; i < array_length(a); i++;)
{
    layer_destroy(a[i]);
}

The above code retrieves all the layers in the current room and adds their handles to an array. This array is then parsed to destroy the room layers.
