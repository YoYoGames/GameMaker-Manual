layer_get_all

Esta função irá retornar uma matriz preenchida com os valores de identificação exclusivos de cada camada no room.

Sintaxe:

layer_get_all()

Devoluções:

Array (populated with Layer IDs)

Exemplo:

var a = layer_get_all();
for (var i = 0; i < array_length(a); i++;)
{
    layer_destroy(a[i]);
}

O código acima recupera todas as camadas em um room e adiciona seus valores de identificação a uma matriz. Este array é então analisado para destruir ou as camadas room.
