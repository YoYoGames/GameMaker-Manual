layer_get_type

This function returns the type of the given layer. You supply either the name of the layer as a string or the Layer handle, and the function returns one of the following constants:

Layer Type Constant
Constant Description
layer_type_unknown An unknown layer.
layer_type_room A room layer.
layer_type_ui_viewports A UI layer with "Game View" set to "Viewports".
layer_type_ui_display A UI layer with "Game View" set to "Display".

 

Syntax:

layer_get_type(layer_id)

Argument Type Description
layer_id String or Layer The handle of the layer to get the type of (or the layer name as a string)

 

Returns:

Layer Type Constant

 

Example:

if (layer_get_type(layer) == layer_type_ui_display)
{
    layer_set_visible(layer, false);
}

This checks if the layer that the instance is on is a Display UI layer. In that case, it makes the layer invisible.
