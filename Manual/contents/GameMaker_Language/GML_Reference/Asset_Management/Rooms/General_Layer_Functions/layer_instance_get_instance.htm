layer_instance_get_instance

Esta função pode ser usada para obter a identificação única da instância do elemento da instância em questão. Você dá o ID do elemento de instância (veja o exemplo de código abaixo para obter isto), e a função retornará um valor real que representa o ID da instância única para o elemento. Se o elemento não for uma instância, a função retornará -1.

Sintaxe:

layer_instance_get_instance(element_id)

Argumento Tipo Descrição
element_id Identificação do elemento de instância The unique ID value of the instance element to get the ID of

Devoluções:

Instance ID

Exemplo:

elements = layer_get_all_elements("Instances");
for (var i = 0; i < array_length(elements); i++)
{
     if (layer_get_element_type(elements[i]) == layerelementtype_instance)
     {
         var layerelement = elements[i];
         var inst = layer_instance_get_instance(layerelement);
         inst.x = inst.x + 10;
     }
}

O código acima verificará todos os elementos de instância em uma camada, depois obterá seu valor de identificação único e o utilizará para movê-los 10px para a direita.
