layer_set_visible

Com esta função, você pode alternar a visibilidade de uma camada. Você fornece a identificação da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho), bem como o valor de comutação da camada onde visível é true e invisível é false.

Quando uma camada é invisível, nada que é atribuído à camada será sorteado, e se alguma instância for atribuída à camada, ela nem mesmo executará seu evento de sorteio (muito parecido com se você definir a variável de instância visible para false). Note que quaisquer instâncias já marcadas como invisíveis não serão marcadas como visíveis se a camada em que estão marcadas for definida como visível, pois estas são duas configurações independentes, embora seu comportamento seja o mesmo.

Sintaxe:

layer_set_visible(layer_id, visible)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer to target (or the layer name as a string)
visible Booleano Whether the layer should be visible (true) ou não (false)

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("Instances");
if layer_get_visible(lay_id)
{
    layer_set_visible(lay_id, false);
}
else
{
    layer_set_visible(lay_id, true);
}

O código acima obtém o valor de ID para a camada chamada "Instances" no editor room, então usa o ID para verificar se a camada está visível ou não, alternando a visibilidade da camada dependendo do valor retornado.
