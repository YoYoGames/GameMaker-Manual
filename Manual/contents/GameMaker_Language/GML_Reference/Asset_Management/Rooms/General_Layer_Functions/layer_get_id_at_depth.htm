layer_get_id_id_at_depth

Você pode usar esta função para obter as identificações de todas as camadas com uma profundidade específica. Você dá a profundidade a ser verificada e a função retornará uma matriz com 1 ou mais entradas, dependendo se existem ou não camadas na profundidade determinada. Se não houver camadas na profundidade determinada, então o array terá uma única entrada na posição [0] com um valor -1, mas, se houver camadas na profundidade, então será feita uma entrada no array para cada camada encontrada - o valor de entrada será o valor de identificação único para uma camada.

Sintaxe:

layer_get_id_at_depth(depth)

Argumento Tipo Descrição
depth Real The depth to check and retrieve the layer IDs from.

Devoluções:

Array

Exemplo:

var a = layer_get_id_at_depth(0);
if a[0] != -1
{
    for (var i = 0; i < array_length(a); i++;)
    {
        layer_destroy(a[i]);
    }
}

O código acima recupera dados sobre as camadas com uma profundidade de 0. É feita uma verificação para ver se existe alguma camada a essa profundidade, e se há então a matriz retornada é analisada e cada uma das camadas encontradas é destruída.
