layer_set_fx

Esta função é utilizada para atribuir um FX struct a um Room Layer. Você especifica o ID ou o nome da camada que deseja modificar, e então um FX struct (como retornado de fx_create() ou layer_get_fx()) que será aplicado à camada especificada, tornando o efeito visível no room.

Se você especificar uma camada que não seja uma camada de filtro/efeito, mas que seja de outro tipo (como uma camada de instância, Asset Layer, Tile Layer, etc.), então o efeito será aplicado ao conteúdo dessa camada, bem como a quaisquer camadas abaixo dela.

Se a camada especificada já tiver um FX struct atribuído a ele, ele será substituído pelo novo.

OBSERVAÇÃO Chamando esta função ao nível da raiz de um script não fará nada, pois scripts são executados antes das primeiras cargas de room. Você pode definir primeiro uma sala de destino e depois chamar esta função.

Sintaxe:

layer_set_fx(layer_name_or_id, filter_or_effect);

Argumento Tipo Descrição
layer_name_or_id Cordão ou Identificação da camada The name or ID of the layer to modify
filter_or_effect Estruturas FX The FX struct to apply to the layer

Devoluções:

N/A

Exemplo:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

O código acima cria um novo FX struct a partir do efeito "_filter_tintfilter ", que é o filtro "Colour Tint" encontrado no editor Room. Ele atribui um valor a seu parâmetro "g_TintCol " que é a cor da tonalidade, e como é um vec4 internamente, ele toma uma matriz contendo quatro valores (correspondentes a seus valores vermelho, verde, azul e alfa). O FX struct para a tonalidade é então aplicado a uma camada FX.
