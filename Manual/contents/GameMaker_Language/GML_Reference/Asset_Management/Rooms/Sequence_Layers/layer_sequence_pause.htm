layer_sequence_pause

Com esta função você pode pausar a reprodução do dado sequence. Você fornece o ID do elemento sequence conforme retornado por layer_sequence_create() ou por uma das funções de elementos de camada e a função fará uma pausa no sequence até que você comece a reprodução novamente usando a função layer_sequence_play().

Sintaxe:

layer_sequence_pause(sequence_element_id)

Argumento Tipo Descrição
sequence_element_id Identificação do Elemento de Sequência The unique ID value of the sequence element to target

Devoluções:

N/A

Exemplo:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
if global.Pause
{
    layer_sequence_pause(_seq);
}

O código acima cria um novo sequence e armazena sua identificação em uma variável local. Em seguida, ele verifica se o jogo está em pausa e se é uma pausa na reprodução do sequence.
