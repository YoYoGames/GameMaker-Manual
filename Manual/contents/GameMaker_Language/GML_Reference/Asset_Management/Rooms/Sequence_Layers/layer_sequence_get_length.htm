layer_sequence_get_length

This function returns the length of the sequence referenced by the given sequence element.

You supply the sequence element ID - as returned by layer_sequence_create or by one of the layer element functions - and the function will return the length of its sequence. This is the number of frames that the sequence will run for.

 

Syntax:

layer_sequence_get_length(sequence_element_id)

Argument Type Description
sequence_element_id Sequence Element ID The unique ID value of the sequence element to target

 

Returns:

Real

 

Example:

var _frames = layer_sequence_get_length(my_seq);
alarm[0] = _frames;

The above code retrieves the number of frames that a sequence will run, then uses this value to set an alarm.
