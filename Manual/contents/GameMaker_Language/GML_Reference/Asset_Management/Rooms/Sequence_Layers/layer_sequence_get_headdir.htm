layer_sequence_get_headdir

Com esta função você fornece o ID do elemento sequence - como retornado por layer_sequence_create() ou por uma das funções dos elementos da camada - e retornará a direção atual do playhead, que será uma das constantes listadas abaixo.

Sintaxe:

layer_sequence_get_headdir(sequence_element_id)

ArgumentoTipo Descrição
sequence_element_id The unique ID value of the sequence element to target

Devoluções:

Constante Descrição Valor
seq_dir_right O sequence está jogando quadros em uma ordem incremental da esquerda para a direita 1
seq_dir_left O sequence está jogando quadros em uma ordem decrescente da direita para a esquerda -1

Exemplo:

if layer_sequence_get_headdir(title_sequence) != seq_dir_left
{
    layer_sequence_headdir(title_sequence, seq_dir_left);
}

O código acima verifica a direção atual do playhead do elemento ID sequence armazenado na variável "title_sequence", e se não estiver configurado para seq_dir_left, está configurado para este valor.
