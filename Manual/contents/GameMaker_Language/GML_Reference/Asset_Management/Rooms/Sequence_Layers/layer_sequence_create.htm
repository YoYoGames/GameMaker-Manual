layer_sequence_create

Com esta função, você pode criar uma instância de um sequence asset na camada dada. Você fornece o ID da camada que pode ser um string do nome da camada - como definido no editor room - ou o ID exclusivo da camada - como retornado pela função layer_get_id()bem como a posição X e Y no room para criar o sequence em, e finalmente o ID do sequence para criar. O ID sequence é o nome constante que você definiu no navegador Asset para o sequence. A função retornará o ID único do elemento sequence, que pode então ser usado em todas as outras funções de camada para sequences, ou pode ser usado para recuperar a instância sequence struct usando a função layer_sequence_get_instance().

Vale notar que se o sequence contém alguma faixa object, suas instâncias serão criadas assim que o próprio elemento sequence for criado, independentemente de onde suas chaves asset estiverem posicionadas na Folha de Drogas. O controlador Sequence simplesmente alterna a visibilidade da instância para escondê-la e mostrá-la dependendo da posição e duração da chave asset da pista e não a cria e destrói repetidamente. Como resultado, as instâncias executarão seus eventos Create quando o elemento sequence for criado e não quando suas chaves de ativos começarem.

Sintaxe:

layer_sequence_create(layer_id, x, y, sequence_id)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the instance layer to target
x Real The x position in the room to create the sequence at
y Real The y position in the room to create the sequence at
sequence_id Sequência Ativo The sequence asset to use, as defined in the Asset Browser

Devoluções:

Sequence Element ID

Exemplo:

var _s = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(_s);

O código acima cria um novo sequence na camada "Background" e depois o faz uma pausa.
