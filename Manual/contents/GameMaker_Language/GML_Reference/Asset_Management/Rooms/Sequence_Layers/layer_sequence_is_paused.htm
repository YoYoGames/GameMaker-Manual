layer_sequence_is_paused

This function returns whether the given sequence is currently paused.

You supply the sequence element ID - as returned by layer_sequence_create or by one of the layer element functions - and it will check if the sequence is currently paused or not, returning true if it is paused, false if it isn't.

 

Syntax:

layer_sequence_is_paused(sequence_element_id)

Argument Type Description
sequence_element_id Sequence Element ID The unique ID value of the sequence element to target

 

Returns:

Boolean

 

Example:

if (layer_sequence_is_paused(title_sequence) != 0)
{
    layer_sequence_play(title_sequence);
}

The above code checks if the sequence element ID stored in the variable title_sequence is paused, and if it is it, resumes it.
