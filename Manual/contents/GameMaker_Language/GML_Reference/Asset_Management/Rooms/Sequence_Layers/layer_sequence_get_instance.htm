layer_sequence_get_instance

Com esta função você fornece o ID do elemento sequence - como retornado por layer_sequence_create() ou por uma das funções dos elementos da camada - e devolverá a estrutura sequence da instância. Você pode saber mais sobre o formato da estrutura de instância sequence nesta página.

Sintaxe:

layer_sequence_get_instance(sequence_element_id)

Argumento Tipo Descrição
sequence_element_id Identificação do Elemento de Sequência The unique ID value of the sequence element to target

Devoluções:

Sequence Instance Struct

Exemplo:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_instance(_seq);
_struct.speedScale = 0.5;

O código acima cria um novo elemento sequence e depois recupera sua identificação de instância sequence. Isto é então usado para alterar a propriedade da escala de velocidade de reprodução da instância sequence.
