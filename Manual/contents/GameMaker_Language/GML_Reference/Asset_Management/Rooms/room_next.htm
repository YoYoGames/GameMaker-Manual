room_next

Com esta função, você pode recuperar o índice do room após a entrada do room na função. Por exemplo, você pode usar a variável room para obter o índice do atual room e depois usar esta função para encontrar o room que o segue, conforme listado no Room Manager. Se não houver room após a entrada, então -1 é devolvido. Note que esta função não reconhecerá ou levará em consideração rooms que tenham sido adicionados dinamicamente usando room_add() ou room_duplicate().

Sintaxe:

room_next(numb);

ArgumentoTipo Descrição
numb The index of the room to get the next one after.

Devoluções:

Room Asset

Exemplo:

if room_next(room) != -1
{
    room_goto_next();
}

O código acima verificará se o próximo room existe e, em caso afirmativo, ele irá até ele.
