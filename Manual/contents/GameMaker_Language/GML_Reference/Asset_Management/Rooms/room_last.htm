room_last

This read-only variable holds the handle for the very last room in the game (this is defined by the order in which the rooms appear in The Room Manager and not by the order in which they were created).

 This variable will not recognise or take into consideration rooms that have been added dynamically using room_add or room_duplicate.

 

Syntax:

room_last

 

Returns:

Room Asset

 

Example:

if (keyboard_check_pressed(ord("Q")))
{
    room_goto(room_last);
}

The above code checks to see if a key has been pressed and if so it goes to the last room in the game.
