layer_sprite_destroy

Esta função destruirá o dado elemento sprite. Você fornece o ID sprite (que você obtém ao criar o sprite usando layer_sprite_create() ou quando você usa a identificação da camada junto com layer_get_sprite_id()) e isto irá removê-lo. Note que isto não remove a camada, apenas o sprite dela, e se o sprite for um que foi adicionado no editor room, então na próxima vez que você deixar o room e depois retornar, o sprite será recriado novamente. Entretanto, se o room for persistente, o sprite será removido, a menos que a persistência do room seja desligada novamente.

Sintaxe:

layer_sprite_destroy(sprite_element_id)

ArgumentoTipo Descrição
sprite_element_id The unique ID value of the sprite to be destroyed

Devoluções:

Exemplo:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.Asset_spr_trees)
{
    layer_sprite_destroy(lay_id);
}

O código acima verifica a camada "Asset_Trees" para ver se o elemento dado sprite existe e se existe, então ele é destruído (mas não a camada).
