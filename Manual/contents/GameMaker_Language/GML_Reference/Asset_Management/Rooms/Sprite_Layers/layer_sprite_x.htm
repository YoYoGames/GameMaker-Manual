layer_sprite_x

Esta função controla a posição ao longo do eixo x do elemento room do elemento asset sprite na camada. Você dá o ID do elemento sprite (que você obtém quando cria um elemento sprite usando layer_sprite_create() ou quando você usa a função layer_sprite_get_id()), e então definir o valor x a ser usado (com base nas coordenadas room ).

Sintaxe:

layer_sprite_x(sprite_element_id, x);

ArgumentoTipo Descrição
sprite_element_id The unique ID value of the sprite element to change
x The x position for the sprite

Devoluções:

Exemplo:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_x(spr_id, irandom(room_width));

O código acima obtém o valor de identificação do sprite asset "Clouds" atribuído à camada "Asset_sky" e então define sua posição x para um valor aleatório entre 0 e a largura do room.
