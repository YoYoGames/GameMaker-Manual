layer_sprite_change

Usando esta função, você pode alterar o recurso sprite atribuído a um determinado elemento sprite em uma camada. Você dá o ID do elemento sprite (que você obtém quando cria um elemento sprite usando layer_sprite_create() ou quando você usa a função layer_sprite_get_id()), e depois fornecer um elemento de índice sprite asset será alterado. Note que se o sprite tem sub-imagens, então ele também animará (isto pode ser controlado usando o layer_sprite_index() e layer_sprite_speed() funções). Note que você pode atribuir um valor de -1 como o novo índice sprite e não sprite será mostrado, embora o elemento ainda exista e ainda possa ser alterado mais tarde.

Sintaxe:

layer_sprite_change(sprite_element_id, sprite_index)

Argumento Tipo Descrição
sprite_element_id Identificação do Elemento Sprite The unique ID value of the sprite element to change
sprite_index Ativo Sprite The new sprite index to use

Devoluções:

N/A

Exemplo:

var s = layer_sprite_get_id(layer, global.Asset_sprite);
if layer_sprite_get_sprite(s) != spr_Nighttime
{
    layer_sprite_change(s, spr_nighttime);
}

O código acima verifica o índice sprite do elemento com o ID armazenado na variável global global.Asset_sprite e se não for "spr_Nighttime" então que sprite é atribuído ao elemento.
