layer_sprite_get_alpha

Esta função pode ser usada para obter o valor alfa do elemento sprite. Você dá o ID do elemento sprite (que você obtém ao criar um elemento sprite usando layer_sprite_create() ou quando você usa a função layer_sprite_get_id()), e a função retornará um valor entre 0 (totalmente transparente) e 1 (totalmente opaco).

Sintaxe:

layer_sprite_get_alpha(sprite_element_id);

ArgumentoTipo Descrição
sprite_element_id The unique ID value of the sprite element to get the information from

Devoluções:

(from 0 to 1)

Exemplo:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_alpha(spr_id) < 0.1
{
    layer_sprite_destroy(spr_id);
}

O código acima obterá o ID da camada chamada "sprite_sky" e depois o usará para obter o ID do elemento sprite nessa camada. Este ID é então usado para verificar o elemento alfa e se for inferior a 0,1, então o elemento da camada é destruído.
