layer_sprite_exists

Você pode usar esta função para verificar e ver se existe um elemento sprite em qualquer camada. Você fornece a identificação da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho) e o ID do elemento sprite (que você obtém quando cria um elemento sprite usando layer_sprite_create() ou quando você usa a função layer_sprite_get_id()) e a função retornará ou true se o elemento existir, ou false se não existir.

NOTA: Esta função funciona dentro do escopo do alvo atual room - por padrão o room no qual a função é chamada - que pode ser definido usando a função layer_set_target_room().

Sintaxe:

layer_sprite_exists(layer_id, sprite_element_id)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer to target (or the layer name as a string)
sprite_element_id Identificação do Elemento Sprite The unique ID value of the sprite element to check

Devoluções:

Boolean

Exemplo:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.TreesSprites)
{
    layer_sprite_destroy(lay_id, global.TreesSprites);
}

O código acima verifica a camada "Asset_trees" para ver se o elemento dado sprite existe e se existe, então ele é destruído (mas não a camada).
