layer_sprite_index

Esta função pode ser usada para definir o índice de imagem de um sprite asset que tem múltiplas subimagens em uma camada. Você dá o ID do elemento sprite (que você obtém ao criar um elemento sprite usando layer_sprite_create() ou quando você usa a função layer_sprite_get_id()), e então definir o índice de imagem a ser utilizado. Se você definir um valor fora do intervalo de sub-imagens, então o índice de imagem será loop em torno.

Sintaxe:

layer_sprite_index(sprite_element_id, image_index);

Argumento Tipo Descrição
sprite_element_id Ativo Sprite The unique ID value of the sprite element to set
index Real The image index to use for the sprite

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("sprite_trees");
var spr_id = layer_sprite_get_id(lay_id, "gfc_trees");
layer_sprite_index(spr_id, 1);

O código acima obterá a identificação da camada chamada "sprite_trees" e depois a utilizará para obter a identificação do elemento sprite dado nessa camada. Este ID é então usado para mudar o índice de imagem do elemento.
