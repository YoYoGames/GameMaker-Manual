Sprites

Os Sprites são geralmente as representações visuais de objects dentro do jogo que você está criando, embora possam ser usados para muitos outros fins. Um sprite é ou uma imagem única, ou um conjunto de imagens que, quando jogado um após o outro, parece uma animação em loop. Sprites pode ser criado em muitos formatos, sendo o "tradicional" sprite composto de quadros de imagens pixel, importados para o GameMaker como arquivos *.png. Entretanto, você também pode importar vetores sprites que foram salvos no formato *.swf, ou animações esqueléticas sprites que foram salvas como arquivos *.json com arquivos acompanhando *.atlas e *.png (para mais informações sobre estes tipos de arquivos de imagem e como importá-los para o GameMaker, consulte a seção Importando Sprites Não-Bitmap).

Como sprites é parte integrante de qualquer jogo, o GameMaker tem um grande número de funções e variáveis que podem ser usadas para obter informações sobre sprites específicas, ou para alterar as que estão em seu jogo, ou até mesmo para carregar o sprites personalizado em runtime a partir de um arquivo de imagem de origem.
