draw_enable_enable_skeleton_blendmodes

Esta função é utilizada para ativar(verdadeiro) ou desativar(falso) os modos de mistura por ranhura para o esqueleto sprites.

Sintaxe:

draw_enable_skeleton_blendmodes(enable)

Argumento Tipo Descrição
enable Booleano Whether to enable or disable blend modes

Devoluções:

N/A

Exemplo:

draw_enable_skeleton_blendmodes(true);

O código acima permite modos de mistura por ranhura para o esqueleto sprites.
