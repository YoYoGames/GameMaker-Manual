Com esta função você pode obter uma cor de ranhura de fixação. Você fornece o nome da ranhura (a string) e a função retornará um valor inteiro para a cor.

Sintaxe:

skeleton_slot_colour_get(slot);

Argumento Tipo Descrição
slot Cordão The slot name to check, a string

Devoluções:

Real

Exemplo:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if !ds_list_empty(slot_list)
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_colour_get(slot_list[| i]) != c_white
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

O código acima cria uma lista DS e depois a preenche com os nomes dos slots encontrados na posição do mouse no room. Em seguida, loops através da lista de slots e repõe a cor dos slots encontrados para branco com um alfa de 1 se a cor ainda não for branca.
