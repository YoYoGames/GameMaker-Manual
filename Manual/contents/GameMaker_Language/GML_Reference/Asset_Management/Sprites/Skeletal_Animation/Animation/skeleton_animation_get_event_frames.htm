esqueleto_animação_get_event_frames

Esta função pode ser usada para recuperar todos os quadros para o evento dado, na animação dada. Você fornece o nome do esqueleto da animação (como um string, como definido no programa usado para fazer a animação, ou como retornado usando a função skeleton_animation_get()) e um nome do evento. A função retorna um array com os frames para aquele evento.

Se o nome do evento especificado não existir, a função retornará uma matriz com -1 como seu primeiro (e único) elemento.

Sintaxe:

skeleton_animation_get_event_frames(anim_name, event_name);

Argumento Tipo Descrição
anim_name Cordão The animation name to get the frames of.
event_name Cordão The event name to get the frames for.

Devoluções:

Array of Reals

Exemplo:

var _frames = skeleton_animation_get_event_frames(skeleton_animation_get(), "Footstep");

if (_frames[0] != -1)
{
    var _count = array_length(_frames);
    var _current_frame = skeleton_animation_get_frame(0);

    for (var i = 0; i < _count; i ++)
    {
        if (_frames[i] == _current_frame)
        {
            audio_play_sound(snd_footstep, 1, false);
            break;
        }
    }
}

O código acima recebe os quadros para o evento "Footstep". Se ele devolveu um array com os frames válidos, ele loops através dele, e verifica se o frame atual é igual a qualquer um dos frames do array. Nesse caso, ele toca o efeito sonoro do "footstep" e quebra (pára) o loop.
