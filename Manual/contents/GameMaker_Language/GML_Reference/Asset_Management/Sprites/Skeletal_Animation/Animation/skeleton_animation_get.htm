esqueleto_animação_get

Com esta função você pode obter o conjunto de animação atual sendo usado por sua animação esquelética sprite. O valor de retorno é um string, que será o nome do conjunto como você o definiu em seu programa de animação esquelética.

Sintaxe:

skeleton_animation_get();

Devoluções:

Exemplo:

if keyboard_check_pressed(vk_space)
{
    if skeleton_animation_get() != "jump"
    {
        skeleton_animation_set("jump");
    }
}

O código acima mudará o conjunto de animação sendo usado para o conjunto de "salto" quando a tecla de espaço for pressionada, mas somente se o conjunto atual sendo usado ainda não estiver "pulando".
