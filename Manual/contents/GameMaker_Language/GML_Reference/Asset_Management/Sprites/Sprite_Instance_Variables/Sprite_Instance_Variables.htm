Variáveis de Sprite Instance

Estas variáveis estão incluídas por padrão no GameMaker objects e estão todas relacionadas ao sprite atribuído à instância no room. Estas variáveis podem ser acessadas a qualquer momento dentro do código ou ações de uma instância e usadas para alterar o que é desenhado e como.

  • sprite_index
  • sprite_width
  • sprite_height
  • sprite_xoffset
  • sprite_yoffset
  • image_alpha
  • image_angle
  • image_blend
  • image_index
  • image_number
  • image_speed
  • image_xscale
  • image_yscale
  • mask_index
  • bbox_bottom
  • bbox_left
  • bbox_right
  • bbox_top
