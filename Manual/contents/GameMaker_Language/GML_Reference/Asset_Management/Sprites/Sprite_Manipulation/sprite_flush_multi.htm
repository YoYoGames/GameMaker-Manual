sprite_flush_multi

Com esta função, você pode remover qualquer número de páginas texture para o dado sprites da memória texture. Você fornece os índices sprite como uma matriz e as páginas texture para as quais eles são designados serão removidas da memória texture. Observe que se uma das as páginas texture são usadas em outro lugar no room (por outra instância sprite ou um fundo, etc...) você pode ter um pequeno acerto de desempenho, já que a página é imediatamente recarregada de novo na memória, portanto, deve-se ter cuidado ao usar esta função. Note que a função retornará -1 se a descarga não for suportada pelos recursos escolhidos, ou retornará 0 se todos funcionarem corretamente.

Sintaxe:

sprite_flush_multi(array);

ArgumentoTipo Descrição
array Array with the sprite indices to flush

Devoluções:

(-1 or 0)

Exemplo:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_flush_multi(spr_a);

O código acima cria uma matriz onde cada elemento contém um índice sprite. Este array é então usado para limpar aqueles sprite textures da memória.
