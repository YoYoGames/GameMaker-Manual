sprite_delete

Esta função excluirá um sprite do jogo, liberando qualquer memória que estava reservada para ele. Esta é uma função muito importante para aqueles momentos em que você precisa criar e mudar sprites de fontes externas (como carregar um sprite de um arquivo com sprite_add() ou duplicar um sprite usando sprite_duplicate()) e deve ser sempre usado para remover aqueles assets que não são mais necessário para um jogo, ou para limpar uma variável indexada asset de uma variável antes de reatribuir outra asset a essa variável. Isto não pode ser usado para excluir sprites que estão incluídas no jogo como parte do assets no navegador Asset .

Sintaxe:

sprite_delete(index);

ArgumentoTipo Descrição
index The index of the sprite to be deleted.

Devoluções

Exemplo:

var spr = sprite_create_from_surface(application_surface, 0, 0, 32, 32, false, false, 16, 16);
sprite_merge(spr_Player, spr);
sprite_delete(spr);

O código acima cria uma variável local e depois armazena o índice do sprite criado a partir da superfície da aplicação. Este sprite é então fundido com o asset indexado na variável "spr_Player" antes de ser removido da memória novamente.
