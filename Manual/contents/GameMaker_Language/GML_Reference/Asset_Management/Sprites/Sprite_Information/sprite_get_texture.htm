sprite_get_texture

Esta função retorna um ponteiro especial para a página sprite texture. Este valor pode então ser usado em outras funções de desenho, particularmente nas funções 2D primitivas, bem como nas funções Shader. Você pode obter mais informações sobre a página texture retornada usando as diferentes funções texture_ encontradas aqui.

Sintaxe:

sprite_get_texture(spr, subimg);

Argumento Tipo Descrição
spr Ativo Sprite The index of the sprite to use.
subimg Real The sub-image of the sprite to use.

Devoluções

Texture

Exemplo:

var tex;
tex = sprite_get_texture(spr_Wall, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(480, 0, 1, 0);
draw_vertex_texture(480, 640, 1, 1);
draw_vertex_texture(0, 640, 0, 1);
draw_primitive_end();

O código acima desenhará uma faixa triangular 4 vertex texturizada com o texture mantido na variável "tex".
