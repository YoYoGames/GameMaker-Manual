caminho_mudança_ponto

Com esta função você pode mudar a posição e/ou o fator de velocidade de qualquer ponto previamente definido para um path (o path pode ter sido criado no editor path ou através de código usando path_add()). Se usado em um path do navegador Asset, note que a função mudará o ativo real, e assim afetará permanentemente como o path é usado por todas as instâncias no jogo desde o momento em que a função é usada até o final do jogo. Se isto não for o que você precisa, então você deve usar uma função como path_duplicate() para criar uma cópia do path primeiro, depois chamar esta função no asset duplicado (não se esqueça de chamar path_delete() no asset quando não for mais necessário).

Sintaxe:

path_change_point(index, n, x, y, speed);

Argumento Tipo Descrição
index Ativo do caminho The index of the path to change a point of.
n Real The defining point to change the attributes of.
x Real The new x coordinate (relative to the path) of the point.
y Real The new y coordinate (relative to the path) of the point.
speed Real The new speed factor of the point.

Devoluções:

N/A

Exemplo:

for (var i = 0; i < path_get_number(path0); ++i;)
{
    var px = path_get_point_x(pth_Patrol, i) + 64 - random(128);
    var py = path_get_point_y(pth_Patrol, i) + 64 - random(128);
    path_change_point(pth_Patrol, i, px, py, 100);
}

O código acima loops através de todos os pontos do path indexados como "path0" e reposiciona todos os pontos para uma posição aleatória dentro de uma área de 128x128 pixels.
