Manipulação do caminho

No GameMaker você não está limitado apenas ao paths que você faz usando o Path Editor. Há uma série de funções que podem ser usadas para manipular e mudar esses paths em tempo de execução, ou mesmo criar novos e depois mudá-los e manipulá-los.

As seguintes funções são usadas para mudar os caminhos:

  • path_exists
  • path_add
  • path_add_point
  • path_change_point
  • path_insert_point
  • path_delete_point
  • path_clear_points
  • path_append
  • path_assign
  • path_delete
  • path_duplicate
  • path_flip
  • path_mirror
  • path_reverse
  • path_rotate
  • path_rescale
  • path_set_closed
  • path_set_kind
  • path_set_precision
  • path_shift
