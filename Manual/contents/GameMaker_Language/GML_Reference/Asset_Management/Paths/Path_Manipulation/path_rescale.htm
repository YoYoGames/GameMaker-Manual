caminho_escala

Esta função pode ser usada para redimensionar o dado path ao longo de ambos (ou qualquer um dos dois) os eixos vertical e horizontal, movendo basicamente cada um dos pontos path para uma nova posição correspondente a esta escala em torno do centro do path. Esta função muda o ativo path real, e assim afetará permanentemente como o path é usado por todas as instâncias do jogo desde o momento em que a função é usada até o final do jogo. Se isto não for o que você precisa, então você deve usar uma função como path_duplicate() para criar uma cópia do path primeiro, então ligue para esta função no asset duplicado (não esqueça de ligar para path_delete() no asset quando não for mais necessário).

Sintaxe:

path_rescale(index, xscale, yscale);

ArgumentoTipo Descrição
index The index of the path to scale.
xscale What to multiply the current horizontal scale by. Default scale is 1.
yscale What to multiply the current vertical scale by. Default scale is 1.

Devoluções:

Exemplo:

path_rescale(pth_AI, 3, 3);

Isto redimensionaria o dado path para três vezes seu tamanho original.
