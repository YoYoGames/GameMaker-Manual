pontos_de_caminho

Com esta função você pode remover todas as definições de pontos de um path, efetivamente fazendo um "vazio" path. Isto não elimina o path, para que você possa usar path_delete()No entanto, deve-se notar que esta função altera o ativo path real, e assim afetará permanentemente como o path é usado por todas as instâncias no jogo desde o momento em que a função é usada até o final do jogo. Se isto não for o que você precisa, então você deve usar uma função como path_duplicate() para criar uma cópia do path primeiro, então ligue para esta função na duplicata asset (não esqueça de ligar para path_delete() na asset quando não for mais necessária).

Sintaxe:

path_clear_points(index);

ArgumentoTipo Descrição
index The index of the path to clear.

Devoluções:

Exemplo:

if path_get_number(mypath) > 0
{
    path_clear_points(mypath);
}

O código acima verifica se há algum ponto no path indexado na variável "mypath" e, se houver, limpa o path.
