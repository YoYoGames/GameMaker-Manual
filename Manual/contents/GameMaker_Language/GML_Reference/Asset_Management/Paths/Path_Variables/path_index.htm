path_index

Todos os recursos no GameMaker têm um número de identificação único, e o path_index é uma variável somente leitura que contém o valor do índice para um determinado path asset que foi atribuído a uma instância usando a função path_start(). Se a instância não tiver um path atribuído, a variável será definida como -1.

Sintaxe:

path_index;

Devoluções:

Path Asset

Exemplo:

if (path_index == -1)
{
    path_start(pth_enemy3, 4, path_action_reverse, 0);
}

O código acima verifica se um path foi atribuído à instância, e se não, inicia um novo path (atribuindo-o ao path_index ao mesmo tempo).
