path_get_number

Esta função pode ser usada para retornar o número de pontos em um path.

Sintaxe:

path_get_number(index);

ArgumentoTipo Descrição
index The index of the path to check.

Devoluções:

Exemplo:

if (path_get_number(pth_AI) > 1)
{
    path_start( pth_AI, 4, 3, 0);
}

O código acima verifica se um path tem mais de um ponto e, em caso afirmativo, começa a instância a mover-se ao longo desse path.
