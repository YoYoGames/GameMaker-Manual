audio_emitter_get_vy

Esta função retorna a velocidade da corrente ao longo do eixo y para o emissor de áudio dado.

Sintaxe:

audio_emitter_get_vy(emitter);

Argumento Tipo Descrição
emitter ID do emissor de áudio The index of the emitter to use.

Devoluções:

Real

Exemplo:

if audio_emitter_get_vy(emitter_player) != 0
{
    audio_emitter_velocity(emitter_player, 0, 0, 0);
}

O código acima verifica a velocidade y atual de um determinado emissor e, se não for igual a 0, é ajustado para 0.
