audio_get_name

This function will return the name of a given audio asset as a string.

The "index" value can be that of the asset itself (as seen in The Asset Browser) or the Sound Instance ID that is given when you play the sound using, for example, audio_play_sound.

To get the actual asset reference originally used to play a sound instance, use audio_sound_get_asset.

 The string returned is not the same as the resource ID and cannot be used to access the resource itself, so should only be used for displaying or error checking.

 

Syntax:

audio_get_name(index);

Argument Type Description
index Sound Asset The index of the sound to check.

 

Returns:

String

 

Example:

var snd = audio_play_sound(choose(snd_One, snd_Two, snd_Three), 0, false);
var name = audio_get_name(snd);
show_debug_message("Sound = " + name);

The above code plays a random sound chosen from three different sound resources then shows a debug message with its name.
