audio_play_in_sync_group

Com esta função você pode atribuir um som a um grupo de sincronização previamente criado. O índice do grupo é o valor retornado quando você criou o grupo usando a função audio_create_sync_group()e o índice de som é o índice de qualquer som comprimido (*.ogg ou *.mp3) do Asset Browser. Esta função apenas adiciona o som ao grupo, mas não toca o som. O valor de retorno será >= 0 se tiver sucesso e -1 se falhar, onde um valor bem sucedido indica a ordem do som no grupo de sincronização (assim 0 seria o primeiro som adicionado, 1 seria o segundo, etc...).

NOTA: Esta funcionalidade não está disponível para a plataforma de destino HTML5.

Sintaxe:

audio_play_in_sync_group(group_index, sound_index);

Argumento Tipo Descrição
group_index ID do Grupo de Sincronização de Áudio The group index to assign the sound to.
sound_index Bens sólidos The sound index to be assigned to the group.

Devoluções:

Real

Exemplo:

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_play_in_sync_group(sg, sound3);
audio_play_in_sync_group(sg, sound4);
audio_start_sync_group(sg);

O acima exposto cria um novo grupo de sincronização e atribui o índice do grupo à variável "sg". Quatro sons são então adicionados ao grupo, com o ganho para três deles sendo ajustado para 0. Finalmente, o grupo de sincronização é tocado.
