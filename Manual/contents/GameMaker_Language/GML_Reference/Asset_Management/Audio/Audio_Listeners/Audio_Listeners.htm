Ouvintes de áudio

Ao lidar com situações complexas de áudio, pode ser útil definir onde no ambiente de áudio o ouvinte deve estar situado. Ao trabalhar com as funções de áudio simples, normalmente não é necessário que o ouvinte seja trocado de qualquer forma, entretanto, ao utilizar emissores para gerar áudio realista dentro do espaço de áudio 3D, é importante que o ouvinte seja trocado e movido corretamente, caso contrário, o som estará na posição errada para a pessoa que estiver jogando seu jogo.

Por padrão, há um ouvinte criado automaticamente no jogo, que usa o ID 0 (para referência nas funções de ouvinte).

O GameMaker oferece as seguintes funções de ouvinte:

  • audio_listener_position
  • audio_listener_velocity
  • audio_listener_orientation
  • audio_listener_get_data
  • audio_listener_set_position
  • audio_listener_set_velocity
  • audio_listener_set_orientation

Certas plataformas alvo podem permitir mais de um ouvinte, com cada uma correspondendo a um dispositivo de saída diferente, ou seja: sua plataforma alvo poderia permitir dois ouvintes, um para os fones de ouvido e outro para o monitor. Portanto, o GameMaker oferece uma série de funções extras com as quais você pode encontrar os ouvintes disponíveis, bem como configurá-los para os sons que estão sendo reproduzidos:

  • audio_get_listener_count
  • audio_get_listener_info
  • audio_get_listener_mask
  • audio_set_listener_mask

Há também funções para obter e definir dados de máscara sobre sons individuais:

  • audio_sound_get_listener_mask
  • audio_sound_set_listener_mask

Assim como para obter e definir dados de máscara sobre emissores:

  • audio_emitter_get_listener_mask
  • audio_emitter_set_listener_mask
