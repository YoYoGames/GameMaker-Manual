asset_get_index

Você pode usar esta função para obter o índice de identificação único para um jogo asset a partir de seu nome. Se o asset não for encontrado, a função retornará um valor de -1, caso contrário, retornará o id de índice único para o asset que está sendo verificado. Este id pode então ser usado em outras funções como qualquer outro valor de índice, como o sprite_index ou o path_indexpor exemplo. Observe que embora esta função possa ser usada para fazer referência a assets a partir de strings (ver exemplo abaixo) você deve sempre certificar-se de que o asset existe antes de utilizá-lo, caso contrário você poderá receber erros que irão travar seu jogo.

Sintaxe:

asset_get_index(name);

ArgumentoTipo Descrição
name The name of the game asset to get the index of (a string).

Devoluções:

Asset (any asset type)

Exemplo:

var obj = asset_get_index("obj_Enemy_" + string(global.Level));
if obj > -1
{
    instance_create_layer(random(room_width), random(room_height), obj, "Enemy_Layer");
}

O código acima receberá um índice object de um string, e se esse índice existir, crie uma instância do object no jogo.
