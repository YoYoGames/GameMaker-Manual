tag_get_asset_ids

Com esta função você pode obter todo o assets de um determinado tipo que tem as tags atribuídas a eles. Você fornece uma única tag (como um string) ou uma matriz, onde cada item da matriz é uma tag (como um string), assim como o tipo de asset para verificar. o tipo de asset deve ser uma das constantes a seguir:

Constante Descrição
asset_object O nome dado refere-se a um objeto.
asset_sprite O nome dado refere-se a um duende.
asset_sound O nome dado refere-se a um som.
asset_room O nome dado refere-se a um room.
asset_tiles O nome dado refere-se a um conjunto de azulejos.
asset_path O nome dado refere-se a um caminho.
asset_script O nome dado refere-se a um script.
asset_font O nome dado refere-se a uma fonte.
asset_timeline O nome dado refere-se a uma linha de tempo.
asset_shader O nome dado refere-se a um sombreador.
asset_animationcurve O nome dado refere-se a uma Curva de Animação.
asset_sequence O nome dado refere-se a uma sequência.

A função retornará uma matriz, onde cada item da matriz será um único valor de índice asset. Se não houver assets do tipo que tem a(s) tag(s) dada(s), um array vazio será retornado.

Sintaxe:

tag_get_asset_ids(tags, asset_type);

Argumento Tipo Descrição
tags Cordão ou matriz de cordas A single asset tag string or an array with various asset tags.
asset_type Tipo de ativo Constante An asset type constant (listed above)

Devoluções:

Array

Exemplo:

var _paths = tag_get_asset_ids("enemy", asset_path);
var _num = irandom(array_length(_paths) - 1);
path_start(_paths[_num], 1, path_action_reverse, false);

O código acima usa a tag "enemy" para encontrar todo o path assets com essa tag, antes de escolher uma ao acaso e atribuí-la à instância que executa o código.
