font_get_texture

Esta função retorna um ponteiro especial para a página font texture. Este valor pode então ser usado em outras funções de desenho, particularmente no desenho em geral, quando se usam tanto as funções primitivas quanto as funções Shader. Você pode obter mais informações sobre a página texture retornada usando as diferentes funções texture_ encontradas aqui.

Sintaxe:

font_get_texture(font);

Argumento Tipo Descrição
font Fonte Ativo The index of the font to use.

Devoluções:

Texture

Exemplo:

tex = font_get_texture(fnt_Main);

O código acima receberá o ponteiro texture para o font indexado como "fnt_Main".
