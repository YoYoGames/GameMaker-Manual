Fontes

O GameMaker tem uma série de funções que podem ser usadas para obter informações sobre o font assets incluído, bem como um conjunto adicional de funções que podem ser usadas para alterar, adicionar ou remover fonts de seu jogo em tempo de execução. Em particular, a função font_add() pode ser usada para adicionar fonts não-padrão, como o glifo asiático fonts, a partir de um arquivo *.ttf ou *.otf.

Vale notar que se você adicionar um font ao seu jogo que não tenha os glifos exigidos pelo texto que você deseja escrever, então o caractere Unicode 9647 (▯) é usado para substituir os glifos que faltam no evento do sorteio. Assim, se seu font não tiver, por exemplo, o símbolo º, então escrever 90º produzirá de fato 90▯.

As seguintes funções podem ser usadas para obter informações sobre os recursos font:

  • font_get_name
  • font_get_fontname
  • font_get_first
  • font_get_last
  • font_get_italic
  • font_get_bold
  • font_get_size
  • font_get_texture
  • font_get_uvs
  • font_get_info

As seguintes funções podem ser usadas para mudar e manipular fonts durante um jogo:

  • font_exists
  • font_set_cache_size
  • font_add_enable_aa
  • font_add_get_enable_aa
  • font_add
  • font_add_sprite
  • font_add_sprite_ext
  • font_replace
  • font_replace_sprite
  • font_replace_sprite_ext
  • font_delete
  • font_texture_page_size
