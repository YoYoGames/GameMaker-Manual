timeline_clear

Com esta função você pode limpar uma linha de tempo específica de "momentos", removendo todos os códigos e ações para aquela linha de tempo e deixando-a vazia.

Sintaxe:

timeline_clear(ind);

Argumento Tipo Descrição
ind Linha do tempo Ativo The index of the timeline to clear.

Devoluções:

N/A

Exemplo:

if timeline_position > 200
{
    timeline_clear(global.tl);
    timeline_index = -1;
}

O código acima limpará a linha de tempo especificada indexada pela variável "global.tl" de todos os momentos em que um momento específico tenha sido passado.
