posição_da_classe

Com esta função você pode verificar uma posição para uma colisão com outra instância ou todas as instâncias de uma object. Quando você usa esta função você está verificando um único ponto no room para uma instância ou um object. Esta verificação será feita contra a caixa de delimitação da instância ou contra a máscara da instância se essa instância tiver colisões precisas verificadas e retornará a instância única id. Se você não precisar da identificação da instância de colisão, você deve considerar o uso de position_meeting() ao invés disso. Esta função também aceita as palavras-chave especiais all e other e devolverá a palavra-chave noone se não ocorrer nenhuma colisão ou o valor de identificação único da instância encontrada se ocorrer uma colisão.

Sintaxe:

instance_position( x, y, obj );

Argumento Tipo Descrição
x Real The x position to check for instances.
y Real The y position to check for instances.
obj Objeto Ativo ou ID da instância The object to check for instances of.

Devoluções:

Instance ID or noone

Exemplo:

var inst;
inst = instance_position(mouse_x, mouse_y, obj_Pause_Button);
if (inst != noone)
{
    with (inst) image_index=1;
    instance_create_layer(room_width / 2, 0, "Controllers", obj_Menu);
}

O código acima verificará uma colisão com uma instância de "obj_Pause_Button" na posição do mouse, e se houver uma, então usará o id retornado para definir seu índice de imagem para um novo valor antes de criar uma nova instância do "obj_Menu" object.
