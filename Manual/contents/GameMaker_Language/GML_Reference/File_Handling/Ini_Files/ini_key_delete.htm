ini_key_delete

Com esta função você pode remover a chave selecionada (e seu valor correspondente) de um arquivo ini.

Sintaxe:

ini_key_delete(section, key);

ArgumentoTipo Descrição
section The section to delete a key from.
key The key to delete.

Devoluções:

Exemplo:

ini_open("savedata.ini");
ini_write_real("save1","Score",score);
ini_key_delete("save1","Score");
ini_close();

Este exemplo abrirá "savedata.ini" e escreverá um valor para "save1" > "Score". Em seguida, apagará a tecla "Score" e fechará o arquivo .ini.
