File Directories

When working with files, there will be moments when you need to reference the directory in which the files are stored. GameMaker has some special functions and variables relating to the different possible directories that you may need.

 You cannot create or destroy directories on the HTML5 and GX.games targets.

Function Reference

General

  • directory_exists
  • directory_create
  • directory_destroy

Variable Reference

Paths

  • temp_directory
  • working_directory
  • program_directory
  • cache_directory

Other

  • game_save_id (from General Game Control)
