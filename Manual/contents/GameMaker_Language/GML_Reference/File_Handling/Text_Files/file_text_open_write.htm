file_text_open_write

Esta função abre o arquivo de texto com o nome de arquivo indicado apenas para escrita (se o arquivo não existir, ele é criado), devolvendo a identificação única do arquivo que deve ser armazenado em uma variável, pois ele será usado para todas as outras ações relacionadas a esse arquivo.

Observe que se o arquivo não puder ser criado (por causa de um nome de arquivo ilegal, por exemplo), a função retornará -1.

Se o arquivo já existir, o uso desta função fará com que ele seja sobregravado quando os dados forem salvos. Se você quiser adicionar dados a um arquivo mantendo intacto seu conteúdo existente, use file_text_open_append().

OBSERVAÇÃO: Você só pode ter um máximo de 32 arquivos abertos a qualquer momento. Você também deve sempre fechar os arquivos quando terminar, pois isto escreve as informações e libera a memória associada ao arquivo.

ADVERTÊNCIA! Esta função pode não funcionar como você espera devido ao GameMaker ser lixado! Consulte a seção sobre o Sistema de Arquivos para maiores informações.

Sintaxe:

file_text_open_write(fname);

Argumento Tipo Descrição
fname Cordão The name of the file to write to.

Devoluções:

Text File ID or -1

Exemplo:

var file;
file = file_text_open_write(working_directory + "level.txt");
file_text_write_string(file, level_data);
file_text_close(file);

O código acima abrirá o arquivo "level.txt" para escrever e então escreverá o string armazenado na variável "level_data" antes de finalmente fechar o arquivo novamente.
