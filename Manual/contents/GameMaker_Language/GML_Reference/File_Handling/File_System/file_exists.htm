file_exists

This function will return true if the specified file exists and false if it does not. Note that the function can only be used to check local files, but not any files stored on a remote server.

 On HTML5, some web hosts may have security measures in place that make this function unreliable (such as it returning true when the file does not exist), so this will have to be accounted for and tested before final use.

 

Syntax:

file_exists(fname)

Argument Type Description
fname String The name of the file to check for

 

Returns:

Boolean

 

Example:

if (file_exists("level.txt"))
{
    file = file_text_open_read("level.txt");
}

This would check for a file and if it exists it is opened for reading.
