filename_drive

Esta função retorna as informações de acionamento do nome do arquivo.

Sintaxe:

filename_drive(fname);

ArgumentoTipo Descrição
fname The file to use.

Devoluções:

Exemplo:

drive = filename_drive(file_find_first(working_directory + "*.doc", 0));

O código acima obtém as informações da unidade (como um string) do arquivo especificado.
