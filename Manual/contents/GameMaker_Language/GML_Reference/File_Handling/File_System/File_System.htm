File System

GameMaker has a number of functions that enable you to manipulate files of all types in different ways. This page lists the various functions available for you to use.

Function Reference

General

  • file_exists
  • file_attributes

File Manipulation

  • file_delete
  • file_rename
  • file_copy

File Find

  • file_find_first
  • file_find_next
  • file_find_close

Pathname Manipulation

 These functions do not work on the actual files! They only deal with the strings related to the different path elements and should be used in conjunction with the other file functions.

  • filename_name
  • filename_path
  • filename_dir
  • filename_drive
  • filename_ext
  • filename_change_ext

File Dialogs

These are functions for getting filenames that can be used for loading and saving files with file-type filters. They will open an explorer window, and therefore cause the game to lose focus. This means that any input the game is depending on may not be correctly registered, and as such we recommend that (if using these functions in a keyboard event) you use the key up event and not the pressed or down events.

 These functions are Windows and macOS only.

 These functions may not work as you expect due to GameMaker being sandboxed! Please see the section on The File System for more information.

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
