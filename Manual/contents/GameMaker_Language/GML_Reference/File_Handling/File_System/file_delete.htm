file_delete

Esta função excluirá o arquivo especificado do sistema. Deve-se observar que esta função somente excluirá aqueles arquivos que o GameMaker é capaz de criar e analisar: arquivos ini, arquivos de texto e arquivos binários, ou aqueles arquivos feitos para armazenar recursos criados pelo jogo, como sprites ou superfícies. No entanto, ela não excluirá nenhum outro arquivo. A função também retornará true se o arquivo tiver sido removido com sucesso, ou false em qualquer outra circunstância.

ADVERTÊNCIA! Esta função pode não funcionar como você espera devido ao GameMaker ser lixado! Consulte a seção sobre Limites do Sistema de Arquivos para maiores informações.

Sintaxe:

file_delete(fname);

ArgumentoTipo Descrição
fname The name of the file to delete.

Devoluções:

Exemplo:

if file_exists("level.txt")
{
    file_delete("level.txt");
}

Isto verificaria a existência de um arquivo e, se ele existir, seria eliminado.
