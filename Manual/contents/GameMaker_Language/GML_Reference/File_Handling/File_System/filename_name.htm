nome_nome_do_arquivo

Usando esta função, retorna a parte do nome do arquivo indicado, com o extension mas sem o path.

Sintaxe:

filename_name(fname);

ArgumentoTipo Descrição
fname The file to use.

Devoluções:

String

Exemplo:

name = filename_name(file_find_first("C:/Games/*.doc", 0));

O código acima recebe o nome (como um string) do primeiro arquivo do tipo "doc" encontrado no diretório especificado.
