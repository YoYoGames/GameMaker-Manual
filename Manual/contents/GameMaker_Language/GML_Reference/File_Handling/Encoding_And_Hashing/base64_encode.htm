base64_encode

This function will convert a string into a base64 format encoded string.

Base64 is a commonly used encoding scheme that is often used for any media that needs to be stored or transferred over the internet as text, and renders the output unreadable to the human eye.

 

Syntax:

base64_encode(string)

Argument Type Description
string String The string to encode

 

Returns:

String

 

Example:

var _str, _file;
_str = base64_encode(game_data);
_file = file_text_open_write("save.txt");
file_text_write_string(_file, _str);
file_text_close(_file);

The above code will convert the string stored in game_data into a base64 encoded string which is then stored in an external text file.
