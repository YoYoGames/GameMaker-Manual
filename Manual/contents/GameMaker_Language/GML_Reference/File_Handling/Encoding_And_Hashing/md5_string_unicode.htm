md5_string_unicode

Na criptografia, MD5 (Message-Digest algoritmo 5) é uma função de hash criptográfico amplamente utilizada com um valor de hash de 128 bits e tem sido empregada em uma grande variedade de aplicações de segurança. Também é comumente usado para verificar a integridade de arquivos e strings. Esta função irá pegar um unicode de entrada string (que é 16bits para cada char) e retornar o hash MD5 hexadecimal de 32 caracteres que é único para aquele string. Desta forma, você pode gerar uma chave segura que pode ser armazenada e usada para verificar a integridade das informações enviadas para (ou recebidas de) um server externo (por exemplo).

NOTA Existem dois formatos para a codificação MD5, UTF-8 e unicode. Ambos são fornecidos para facilitar a comunicação com diferentes configurações server, mas o mais comum a ser usado é unicode.

NOTA: O MD5 não é completamente seguro e pode ser quebrado. Veja esta página para mais informações.

Sintaxe:

md5_string_unicode(string)

ArgumentoTipo Descrição
string The string to hash.

Devoluções:

Exemplo:

var hash, str;
str = base64_encode(game_data);
hash = md5_string_unicode(str);
http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?hash=" + hash); http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?data=" + str);

O código acima base64 codificará um string e então gerará um hash MD5. Finalmente, tanto o hash quanto o codificado string são enviados para um server.
