file_bin_position

Esta função retornará a posição atual em bytes, onde 0 é a primeira posição, do arquivo com a identificação do arquivo dado. Você fornece o valor do ID do arquivo, conforme retornado pela função file_bin_open().

NOTA: Estas funções não funcionam quando o módulo alvo é HTML5.

Sintaxe:

file_bin_position(binfile);

ArgumentoTipo Descrição
binfile The ID of the file to get the position in.

Devoluções:

Exemplo:

pos = file_bin_position(file);

Isto armazenaria a posição atual na variável "pos".
