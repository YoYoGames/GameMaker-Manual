variável_set_struct_set

Com esta função você pode definir o valor de uma determinada variável em um struct. Você fornece a referência struct assim como o nome da variável para definir o valor de uma string (veja o código de exemplo abaixo), e então finalmente o valor a ser definido (pode ser qualquer tipo de dado válido). Se a variável não existir já no struct, ela será criada e então será atribuído o valor.

Sintaxe:

variable_struct_set(struct, name, val);

Argumento Tipo Descrição
struct Estrutura The struct reference to set
name Cordão The name of the variable to set (as a string)
val Variável The value to set the variable to

Devoluções:

N/A

Exemplo:

if !variable_struct_exists(mystruct, "shields")
{
    variable_struct_set(mystruct, "shields", 0);
}

O código acima verificará se a variável dada existe no dado struct e se não existe, então ela é criada e definida como 0.
