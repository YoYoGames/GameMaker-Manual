is_array

Esta função pode ser usada para verificar e ver se uma variável contém uma matriz (retornará true) ou não (nesse caso retornará false).

Sintaxe:

is_array(n);

Argumento Tipo Descrição
n Variável The variable to check.

Devoluções:

Boolean

Exemplo:

if is_array(a)
{
    a = -1;
}

O código acima verifica uma variável para ver se ela é um array, e se a função retorna true, o array é apagado definindo a variável como -1.
