array_copy

Com esta função você pode copiar toda ou parte de um array para outro array em qualquer posição. Você precisa fornecer tanto a matriz de origem quanto a de destino (ambas precisam ter sido criadas anteriormente), bem como uma posição dentro da matriz de origem para copiar e uma posição dentro da matriz de destino para copiar. Finalmente, você precisa especificar o comprimento da matriz (ou o comprimento da peça que você deseja) para copiar. Se os dados sendo copiados excederem o comprimento da matriz de destino, a matriz será estendida para aceitar os dados.

Esta função também pode ser usada para matrizes multidimensionais, desde que você especifique qual dimensão deseja copiar ao fornecer o índice da matriz, seguindo este padrão:

// Copy to the first dimension of an array
// from the second dimension of an array
array_copy(item_array, 0, inventory_array[0], 0, len);

// Copy to the third dimension of an array
// from the first dimension of an array
array_copy(item_array[0][0], 0, inventory_array, 0, len);

// etc...

Sintaxe:

array_copy(dest, dest_index, src, src_index, length);

Argumento Tipo Descrição
dest Array The ID of the array to copy to.
dest_index Real The index within the array to copy to.
src Array The ID of the array to copy from.
src_index Real The index with the array to start copying from.
length Real The length (number of array indices) to copy.

Exemplo:

if !array_equals(inventory_array, item_array)
{
    var len = array_length(inventory_array);
    array_copy(item_array, 0, inventory_array, 0, len);
}

O código acima verificará duas arrays para ver se elas possuem valores equivalentes, e se elas não possuem, então o código copiará todo o conteúdo da array "inventory_array" para a array "item_array".
