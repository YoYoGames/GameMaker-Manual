array_set

Com esta função, você pode definir o valor de um índice em uma matriz para um valor. A função requer que você forneça uma variável que contenha o array, bem como o índice a ser definido e o valor a ser definido. Esta função também pode ser usada para matrizes multidimensionais, desde que você especifique qual dimensão deseja definir ao fornecer o índice da matriz, seguindo este padrão:

// 1D array
array_set(my_array, 0, 100);
// 2D array
array_set(my_array[0], 0, 100);
// 3D array
array_set(my_array[0][0], 0, 100);
// etc...

Sintaxe:

array_set(variable, index, value);

Argumento Tipo Descrição
variable Array The variable that holds the array.
index Real The index of the array element to set the value for.
value Variável The value to set.

Devoluções:

N/A

Exemplo:

for (var i = 0; i < 10; ++i;)
{
    array_set(score_array, i, i*100));
}

O código acima definirá os primeiros 10 itens da matriz dada como um valor.
