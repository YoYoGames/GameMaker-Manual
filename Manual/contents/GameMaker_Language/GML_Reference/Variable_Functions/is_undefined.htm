is_undefined

Esta função retorna se uma determinada variável está definida ou não. Em alguns casos, você deseja verificar e ver que tipo de dado uma variável contém no GameMaker e é quando você usaria esta função. Ela retorna true ou false, dependendo se o valor está definido ou não.

NOTA: Esta função não pode ser usada para verificar a existência de uma variável. Use variable_instance_exists() ou variable_global_exists() ao invés disso.

Sintaxe:

is_undefined(n);

Argumento Tipo Descrição
n Variável The argument to check.

Devoluções:

Boolean

Exemplo:

var val = ds_map_find_value(map, 13);
if is_undefined(val)
{
    show_debug_message("Map entry does not exist!");
}

O código acima verifica a variável "val" para ver se ela é undefined ou não e mostra uma mensagem de debug se for.
