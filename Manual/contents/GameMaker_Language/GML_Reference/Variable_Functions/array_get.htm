array_get

Com esta função você pode recuperar o valor de um índice em uma matriz. A função exige que você forneça uma variável que contenha o array e o índice para obter o valor de dentro desse array. Note que se o índice do array dado estiver fora dos limites, então o jogo irá travar com um erro. Esta função também pode ser usada para matrizes multidimensionais, desde que você especifique qual dimensão deseja obter ao fornecer o índice da matriz, seguindo este padrão:

// 1D array
array_get(my_array, 0);
// 2D array
array_get(my_array[0], 0);
// 3D array
array_get(my_array[0][0], 0);
// etc...

Sintaxe:

array_get(variable, index);

Argumento Tipo Descrição
variable Array The variable that holds the array.
index Real The index of the array element to get the value from.

Devoluções:

Variable (Any valid data type that an array can hold)

Exemplo:

for (var i = 0; i < 10; ++i;)
{
    show_debug_message(array_get(my_array, i));
}

O código acima dará os primeiros 10 itens do dado para o console.
