game_set_speed_speed

Esta função pode ser usada para definir a velocidade do jogo. Você pode configurá-la de duas maneiras - como quadros de jogo por segundo (FPS) ou como microssegundos por quadro de jogo (MPF) - usando uma das duas constantes a seguir:

Constante Descrição
gamespeed_fps Define a velocidade do jogo usando quadros por segundo.
gamespeed_microseconds Define a velocidade do jogo usando microssegundos por quadro.

Assim, por exemplo, uma velocidade de jogo de 30 quadros por segundo seria de 33333 microssegundos por quadro de jogo, o que seria então expresso por esta função como um ou outro:

game_set_speed(30, gamespeed_fps);

ou:

game_set_speed(33333, gamespeed_microseconds);

Sintaxe:

game_set_speed(speed, type);

Argumento Tipo Descrição
speed Real The new game speed (as either FPS or MPF).
type Constante de velocidade do jogo The type of method used to set the game speed (see the constants above).

Devoluções:

N/A

Exemplo:

if os_browser == browser_not_a_browser
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}

O código acima verifica se o jogo está rodando em um navegador e define a velocidade do jogo de acordo com um valor FPS.
