cursor_sprite

A definição desta variável instruirá GameMaker para usar o designado sprite como um cursor (basicamente ajustando-o para a posição atual x/y do mouse a cada passo). O valor padrão é -1 que não é sprite para o cursor, mas você pode atribuir qualquer índice sprite do jogo assets ou que tenha sido importado de um recurso externo. Observe que não há como controlar a velocidade da animação ou índice_de_imagem, portanto, se o sprite tiver sub-imagens, estas serão alternadas na mesma velocidade que a velocidade do room. Para remover o cursor sprite, você pode definir esta variável para -1 novamente.

Também vale a pena notar que esta variável não substitui o cursor da janela do jogo, e que ainda será desenhada como normal. Para evitar isto você pode usar a função window_set_cursor() e colocá-lo na constante cr_none, o que tornará o cursor padrão invisível.

Sintaxe:

cursor_sprite;

Devoluções:

Sprite Asset

Exemplo:

cursor_sprite = spr_CustomCursor;

O código acima definirá o sprite indexado na variável "spr_CustomCursor" para ser o cursor sprite para o jogo.
