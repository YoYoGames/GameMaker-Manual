highscore_value

Com esta função você pode recuperar o valor da pontuação que foi armazenado na lista de pontuação alta na posição determinada. Se nenhuma pontuação tiver sido inserida, a função retornará 0.

Sintaxe:

highscore_value(place);

ArgumentoTipo Descrição
place The place on the table (1-10).

Devoluções:

Exemplo:

var i = 9;
repeat(10)
{
    scr[i] = highscore_value(i + 1);
    i -= 1;
}

O código acima irá loop através da lista de pontuação alta e armazenará todas as pontuações em uma matriz.
