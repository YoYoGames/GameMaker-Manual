keyboard_virtual_set_position

When Windows IME is enabled (see keyboard_virtual_show) this function is used to set the position of the IME input window. This position is in the game window space, you can use application_get_position to get the position of the application surface within the window in case the application surface is not at 0, 0.

 

Syntax

keyboard_virtual_set_position(x, y);

Argument Type Description
x Real The x position in the window
y Real The y position in the window

 

Returns:

N/A

 

Example:

var _mx = device_mouse_x_to_gui(0);
var _my = device_mouse_y_to_gui(0);

keyboard_virtual_show(undefined, undefined, undefined, undefined);
keyboard_virtual_set_position(_mx, _my);

This code enables the virtual keyboard on Windows (for IME input) and sets its position to the mouse position on the GUI layer.
