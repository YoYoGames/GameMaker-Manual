gamepad_set_axis_deadzone

Esta função pode ser usada para definir a "zona morta" do eixo do joystick. Você especifica o slot do dispositivo a ser definido, e então define um valor de 0 a 1 e se a quantidade de entrada for inferior ao valor dado, o eixo do joystick é considerado como sendo 0. Observe que esta é uma configuração global que afetará todos os eixos de todos os joysticks conectados ao slot do dispositivo especificado.

O uso desta função não significa que o valor do eixo começará a partir do valor da zona morta, pois a função gamepad_axis_value() sempre retornará um valor normalizado entre -1 Por exemplo, ajustar a zona morta para 0,2 significa que empurrar o bastão para a direita só começará a retornar um valor de 0 - 1 quando o valor do eixo bruto for superior a 0,2 - então quando o valor bruto for 0,2, o valor de retorno será 0, quando o valor bruto estiver em 0,5, o valor de retorno será 0,375, ou se o valor bruto for 0,9, então o valor de retorno será 0,875.

Sintaxe:

gamepad_set_axis_deadzone(device, deadzone);

ArgumentoTipo Descrição
device Which gamepad device "slot" to check.
deadzone The dead zone value from 0 to 1.

Devoluções:

Exemplo:

if gamepad_is_connected(0) gamepad_set_axis_deadzone(0, 0.05);
if gamepad_is_connected(1) gamepad_set_axis_deadzone(1, 0.05);
if gamepad_is_connected(2) gamepad_set_axis_deadzone(2, 0.05);
if gamepad_is_connected(3) gamepad_set_axis_deadzone(3, 0.05);

O código acima ajustará a zona morta do eixo do joystick em qualquer um dos 4 dispositivos conectados a 0,05.
