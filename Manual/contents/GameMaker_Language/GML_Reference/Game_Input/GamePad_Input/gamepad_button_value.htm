gamepad_button_value

Com esta função você pode obter o valor atual de um botão analógico, de 0 a 1, onde 0 é sem pressão e 1 é pressão total. Você fornece o índice de slots para verificar, junto com uma constante de botão (como listado aqui), ou um valor inteiro entre 0 e gamepad_button_count() - 1.

Note que isto retornará o valor bruto para o botão, e não leva em conta a configuração para o limiar (veja aqui para mais informações sobre limiares).

Sintaxe:

gamepad_button_value(device, button);

Argumento Tipo Descrição
device Real Which gamepad device "slot" to check.
button Botão Gamepad Constant Which gamepad button constant to check for.

Devoluções:

Real

Exemplo:

speed = gamepad_button_value(0, gp_shoulderrb) * 4;

O código acima usa o valor do gatilho analógico do controle de jogo conectado ao dispositivo "slot" 0 para definir a velocidade da instância.
