gamepad_set_option

Esta função pode ser usada para definir qualquer uma das opções disponíveis de gamepad. Você fornece o "slot" do controle de jogo para ativar a opção, assim como a opção string para ativar e o valor a ser usado. A opção disponível string dependerá da plataforma na qual o projeto está sendo executado, conforme listado abaixo:

Opção Plataforma Descrição Valor
"allow_rotation" (permitir_rotação) tvOS Definir se o Siri Remote será bloqueado por rotação (ou seja: só pode ser usado verticalmente) ou não. Quando destravado (um valor de true) girando o controle mudará a entrada, de tal forma que uma rotação de 90º para a esquerda fará com que o lado "direito" do touch pad retorne "para cima", etc... de tal forma que toda a entrada será relativa à orientação do dispositivo. Quando travado (um valor de false), a entrada não será alterada em relação à posição do controlador e será sempre tratada como se o controlador estivesse em posição vertical. Booleano
"dpad_absolute" tvOS Definir se o touchpad Siri Remote aceita entrada absoluta, ou entrada relativa. No modo absoluto (um valor de true) o centro do touchpad é considerado a posição (0,0) - ou seja: onde o bastão analógico está em repouso ou onde estaria o centro do dpad - e o movimento em torno desta posição gerará a entrada. Quando ajustado para relativo (um valor de false então a posição (0,0) será considerada como estando onde quer que o toque inicial no pad seja feito e toda a entrada será gerada a partir dessa posição, não o centro do pad. Booleano

Sintaxe:

gamepad_set_option(device, option_name, value);

Argumento Tipo Descrição
device Real Which gamepad device "slot" to set.
option_name Cordão The name of the option to set (a string, see the table above)
value Real, Booleano ou Cordão The value to set the option to (can be boolean, real or string - see the table above)

Devoluções:

N/A

Exemplo:

for(var i = 0; i < 12; ++i;)
{
    if gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote"
    {
        if gamepad_get_option(i, "allow_rotation") == false
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

O código acima loops através de todos os slots e verificações para o "Siri Remote" na plataforma tvOS. Se for detectado, ele então define o controle remoto para permitir a rotação de entrada.
