roda_do_mouse_up

Esta função retorna true se a roda do mouse estiver sendo girada para cima e false caso contrário.

Sintaxe:

mouse_wheel_up();

Devoluções:

Exemplo:

if mouse_wheel_up()
{
    y -= 10;
}

Isto move a instância atual para cima da tela se a roda do mouse for girada para cima.
