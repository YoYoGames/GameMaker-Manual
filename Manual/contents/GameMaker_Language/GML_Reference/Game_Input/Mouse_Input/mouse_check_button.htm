mouse_check_button

Esta função retornará true se o botão do mouse sendo verificado for mantido pressionado ou false se não for. Você fornece o botão do mouse para verificar a partir de uma das seguintes constantes:

Sintaxe:

mouse_check_button(numb);

Argumento Tipo Descrição
numb Botão do Rato Constante Which mouse button constant to check for.

Devoluções:

Boolean

Exemplo:

if mouse_check_button(mb_left)
{
    instance_create_layer(mouse_x, mouse_y, "Effects", obj_Star);
}

O código acima verificará o botão esquerdo do mouse e cada passo que ele for mantido para baixo criará uma instância do object indexado em "obj_Star".
