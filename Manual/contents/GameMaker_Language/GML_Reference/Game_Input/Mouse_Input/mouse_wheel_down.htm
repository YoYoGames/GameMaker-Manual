mouse_wheel_down

Esta função retorna true se a roda do mouse estiver sendo girada para baixo e false caso contrário.

Sintaxe:

mouse_wheel_down();

Devoluções:

Exemplo:

if mouse_wheel_down()
{
    y += 10;
}

Isto move a instância atual para baixo da tela se a roda do mouse for girada para cima.
