gesto_flick_flick_speed

Esta função é utilizada para definir a velocidade necessária para que um evento Flick Gesture seja acionado quando um toque ou clique é liberado. A velocidade é medida em polegadas por segundo e tem um valor padrão de 2,0.

Sintaxe:

gesture_flick_speed(speed);

ArgumentoTipo Descrição
speed The speed (in inches per second) to set for flick gesture event detection.

Devoluções:

Exemplo:

if gesture_get_flick_speed() != 2
{
    gesture_flick_speed(2);
}

O código acima verifica se a velocidade dos gestos está ajustada para 2 polegadas por segundo e se não está ajustada a esse valor.
