gesto_douplo_tap_distância

Esta função é usada para definir a distância dentro da qual você tem que tocar/clicar a tela novamente após um único toque, a fim de acionar um gesto de duplo toque. A distância é medida em polegadas e tem um valor padrão de 0,1.

Sintaxe:

gesture_double_tap_distance(distance);

ArgumentoTipo Descrição
distance The distance (in inches) to set for double tap gesture event detection.

Devoluções:

Exemplo:

if gesture_get_double_tap_distance() != 0.1
{
    gesture_get_double_tap_distance(0.1);
}

O código acima verifica se a distância de toque duplo para gestos está ajustada a 0,1 polegadas e se não está ajustada a esse valor.
