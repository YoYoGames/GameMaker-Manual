keyboard_lastchar

This variable stores a string of the last key pressed. This variable is not read-only and you can change it, for example to set it to "" (an empty string) if you handled it already.

 

Syntax:

keyboard_lastchar

 

Returns:

String

 

Example:

if (keyboard_lastkey != vk_nokey)
{
    str += keyboard_lastchar;
    keyboard_lastkey = vk_nokey;
}

The above code checks to see if the keyboard_lastkey variable stores a valid key, and if it is it adds whatever the last key was as a string to the variable str.
