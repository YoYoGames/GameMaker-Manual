keyboard_clear

Com esta função você pode limpar o estado atual do teclado, o que essencialmente significa que se a tecla estiver sendo mantida pressionada, ela não será mais reconhecida até ser liberada novamente (o que também não acionará o evento Keyboard Key Release Release neste ocasião) e pressionado novamente. A função terá um valor de código de chave como retornado pela função ord() (somente letras maiúsculas de A-Z ou números de 0-9), ou qualquer uma das constantes vk_* listadas na página principal de Entrada de Teclado.

Sintaxe:

keyboard_clear(key);

ArgumentoTipo Descrição
key The key to clear.

Devoluções:

Exemplo:

keyboard_clear(vk_space);

O código acima esclarece o estado da barra de espaço.
