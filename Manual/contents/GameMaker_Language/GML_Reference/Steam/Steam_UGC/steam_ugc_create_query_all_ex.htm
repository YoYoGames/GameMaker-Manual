steam_ugc_create_query_all_ex

Esta função pode ser usada para consultar a base de dados da UGC. A função requer o ID do jogo que vai consumir o item e/ou o ID do jogo que criou o item, e você precisa usar as seguintes constantes para o tipo de consulta a ser criada, o tipo de item a consultar e o número de página a consultar. Note que uma consulta retornará um número máximo de 50 itens.

A função retorna um valor único de manuseio de consulta que deve ser armazenado em uma variável para uso nas outras funções de consulta. Note que esta função apenas prepara a consulta, mas não a envia - para isso você deve chamar a função steam_ugc_send_query() - e você pode usar mais funções de steam_ugc_query_*() para refinar o pedido de busca antes que ele seja realmente enviado.

Sintaxe:

steam_ugc_create_query_all_ex(query_type, match_type, page, creator_app_id, consumer_app_id);

Argumento Descrição
query_type O tipo de consulta de dados a criar (veja as constantes listadas abaixo).
match_type O tipo de itens de UGC a consultar (ver constantes listadas abaixo).
page O número da página a consultar.
creator_app_id O ID do aplicativo criador do item.
consumer_app_id O ID do aplicativo do consumidor.

Devoluções:

Query Handle

Exemplo:

query_handle = steam_ugc_create_query_all_ex(ugc_query_RankedByVote, page, global.AccountID, 0, global.GameID);

O código acima cria um pedido de consulta e armazena o seu manuseio em uma variável para uso futuro.
