vapor_set_stat_int

Com esta função você pode definir uma estatística específica para um novo valor inteiro, assinado. A estatística deve ter sido previamente definida na página de contas do painel de controle Steamworks para seu jogo e o string que é passado para a função deve corresponder ao que é usado como o nome da API no painel de controle. Exemplos de quando você poderia usar isto são para acompanhar quantas vezes o jogador morre ou para acompanhar o progresso em direção a uma conquista.

Sintaxe:

steam_set_stat_int(stat_name, value);

Argumento Descrição
stat_name O nome da estatística a ser definida (a string).
value O valor (um inteiro de 32 bits) para definir a estatística.

Devoluções:

N/A

Exemplo:

xp += 100;
steam_set_stat_int("Total_XP", steam_get_stat_int("Total_XP") + 100);
if steam_get_stat_int("Total_XP") > 1000
{
    if !steam_get_achievement("Ach_1000XP") steam_set_achievement("Ach_1000XP");
}

O código acima estabelece uma estatística e depois verifica o valor final para que ele decida se deve ou não conceder uma conquista.
