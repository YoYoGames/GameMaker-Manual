url_open

This will open the specified URL on the default browser of the chosen target device, or, if you are using the HTML5 module, in the currently open browser.

 

Syntax:

url_open(url);

Argument Type Description
url String The URL (website address) to link to.

 

Returns:

N/A

 

Example:

url_open("http://gamemaker.io");

This would open the GameMaker homepage in the current window.
