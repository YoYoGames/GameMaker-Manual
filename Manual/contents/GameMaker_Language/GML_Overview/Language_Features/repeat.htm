repita

O GameMaker Language tem uma série de maneiras que você pode realizar loops, ou seja: ter uma declaração ou declarações iterativas sobre si mesmo um certo número de vezes. A mais simples delas é a declaração repeat, que tem a forma:

repeat (<expression>)
{
    <statement>;
    <statement>;
    ...
}

Com repeat a declaração dada é repetida o número de vezes indicado pelo valor arredondado da expressão. Por exemplo, o seguinte cria cinco bolas em posições aleatórias:

repeat (5)
{
    instance_create_layer(random(400), random(400), "Instances", obj_ball);
}

Isto pode ser muito útil para evitar digitar o mesmo código várias vezes, ou para usar matrizes, ou para contar através de várias operações, etc... Você também não está limitado a usar uma única declaração, e pode repetir múltiplas declarações, encerrando-as dentro de parênteses curvos {}. Por exemplo:

var _x = 32;
repeat (global.p_lives)
{
    draw_sprite(spr_heart, 0, _x, 32);
    _x += sprite_get_width(spr_heart);
}

O exemplo acima repete as afirmações entre colchetes por mais iteraçõesque a variável global "vive" tenha, e cada iteração desenha o coração sprite na posição _x, depois move a posição um pouco com base na largura do coração sprite.

Quando você deve usar um repeat loop? Sempre que você quiser repetir uma ou mais afirmações um número fixo de vezes sem nenhuma necessidade específica de manter uma contagem das iterações.

Vale notar que você pode usar as declarações especiais break e continue dentro de um repita loop também. Usando break sairá imediatamente do loop e passará para qualquer código que esteja no evento ou função após o loop ter terminado, por exemplo:

var i = 0;
var temp = 0;
repeat (10)
{
    temp += array[i];
    if (temp > max_total)
    {
        break;
    }
    else
    {
        i += 1;
    }
}

O código acima loops até 10 valores de matriz e os adiciona a uma variável local. Se o total da variável local for maior do que o valor dado para max_total, então o loop é encerrado usando o break, caso contrário, o loop continuará.

Um exemplo de utilização do continue em um repeat loop seria:

repeat(10)
{   
    var _x = random(room_width);
    var _y = random(room_height);
    if (instance_position(_x, y, obj_Enemy)
    {
        continue;
    }
    instance_create_layer(_x, _y, "Instances", obj_Enemy);
}

Este código irá repetir 10 vezes, gerando uma posição aleatória room e então verificando se existe uma instância do object obj_Enemy nessa posição. Se existir, a atual iteração do loop é terminada usando continue e uma nova iteração é iniciada, e se não existir, então uma instância do object obj_Enemy é criada na posição aleatória.
