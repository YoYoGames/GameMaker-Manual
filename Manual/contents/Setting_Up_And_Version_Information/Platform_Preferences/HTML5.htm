Preferências HTML5

As preferências HTML5 têm as seguintes opções:

  • Tipo de embalagem padrão: Ao construir um pacote executável HTML5 final, você pode escolher entre Pacote como arquivos soltos - que simplesmente gerará todos os arquivos necessários em uma pasta - ou Pacote como Zip - que criará um arquivo comprimido ZIP que terá o jogo e quaisquer arquivos adicionais dentro dele. Por padrão, você terá a escolha de qual destas duas opções deseja escolher na compilação, mas você pode definir esta preferência para usar sempre uma ou outra. Note que quando esta opção for definida para Mostrar Diálogo, na janela de diálogo real que aparece na compilação, escolhendo "Lembrar Opção de Embalagem", esta preferência será definida para a opção que você selecionar.
  • Porta padrão da Web Server : Isto permite mudar a porta padrão para a Web HTML5 server. O valor padrão é 51264.
  • Endereço padrão da Web Server: Aqui você pode alterar o endereço padrão da web server para a web HTML5 server. O valor padrão é 127,0,0,0,1.
  • Web padrão Server IPs permitidos: Aqui você pode adicionar qualquer IP permitido para a web server e eles serão definidos como padrão para todos os dispositivos adicionados através do Gerenciador de destino.

IMPORTANTE Ao utilizar um determinado dispositivo alvo (ou seja, navegador), os valores de IP e porta definidos para esse dispositivo em particular são usados em vez dessas preferências. Verifique o Gerenciador de Dispositivos para ver uma lista de seus dispositivos com seus detalhes.
