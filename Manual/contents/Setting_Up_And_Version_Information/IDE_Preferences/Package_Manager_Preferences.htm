Package Manager Preferences

This contains preferences for the Package Manager:

  • Packages Install Directory: The directory where packages are installed. The GMRT Preferences should be set to look in this directory.
  • Default Source URL: The default package source.
  • NPM Path: Optional, if you want to use your own version of NPM.
  • Node Path: Optional, if you want to use your own version of Node.js.
  • Build Type: The build of the Package Manager to use.
  • Target Triple: The build combination to get packages for, when using GMRT.
  • Platform: The build platform to get packages for, when using GMRT.
