AI Assistant Preferences

This section contains preferences for the AI Assistant:

  • Show the Welcome screen: Show the welcome screen for the AI Assistant the next time a project is loaded.
  • Default window location: The location in the IDE where the AI Assistant appears.

 

The Confirm Dialogs menu shows the following options:

  • Automatic response to delete conversation: This option controls what happens when you choose to delete a conversation. You can choose to get a confirmation prompt or delete without confirmation.
