Licencing & Logging In

You can start and use GameMaker without making an account. You will be asked to create an account when you create an executable.

Use the Version & Account Details menu to sign in before creating executables. Clicking the Sign In/Register button opens your default web browser and takes you to the sign in page where you can choose to use an Opera account to sign in, or use a Legacy YoYo Account.

Licencing Information

For information on the various GameMaker licences and obtaining them, see the links on Licencing & Version Information.

 After purchasing any new licence for GameMaker, use the Update License option in the Version & Account Details menu so your new licence details are loaded.
