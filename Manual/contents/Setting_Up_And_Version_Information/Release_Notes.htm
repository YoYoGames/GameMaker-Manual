Notas de Lançamento

Cada lançamento do GameMaker vem com seu próprio conjunto de notas de lançamento. Estas contêm informações detalhadas sobre as mudanças que foram feitas, bem como uma lista completa de todos os bugs que foram corrigidos. Esta é uma ferramenta inestimável para decidir seja para atualizar sua versão atual, ou quando você encontrar algo que possa ser um bug após um lançamento.

As notas de lançamento são divididas em dois conjuntos diferentes devido ao fato de que tanto o IDE quanto o atual Runtimes podem ser atualizados independentemente. Você pode encontrar as notas de lançamento atuais para o IDE no link abaixo:

  • Notas de Lançamento IDE
