Prefab Library

main image

The Prefab Library is used to import assets from a source project. A source project from which prefabs are imported is known as a Collection.

When you use an asset from the Prefab Library, that asset is not copied into your project, but is loaded from its source project when your game is built. This means that any changes in the original collection will also be reflected in your project.

visualise use of prefabs

This allows you to use the same assets in multiple projects and have them all be updated when the source collection is updated.

You can also choose to duplicate an asset into your project, at which point the asset is unlinked from its collection, meaning that it will no longer receive updates from its collection however it will be fully modifiable within the own project.

Adding Prefab Packages

Currently you can use the official GameMaker Asset Bundles via the Prefab Library.

Open the Prefab Library via the Windows menu in the menu bar. When you open this for the first time, your Prefab Library will be empty.

Click on Package Manager to add prefab collection packages:

adding packages

This opens the Package Manager window with the "Package source" drop-down set to "Prefabs". From here, you can install any of the official GameMaker Asset Bundles.

Once installed, they will show up in your Prefab Library, from where you can navigate to any individual assets within the collections:

prefabs and assets

Using Prefab Assets

Assets from the Prefab Library can be dragged into compatible asset editors. Currently, you can use assets in the following ways:

  • Use Sprite assets in The Object Editor, The Sequence Editor, The Room Editor, The Tile Set Editor, The Particle System Editor
  • Use Sound assets in The Sequence Editor
  • Use Tile Set assets in The Room Editor

Apart from the editors mentioned above, any assets from prefab collections can be referenced in your code and they will be imported into your builds and be usable during runtime.

Prefab Library Features

The Prefab Library window is divided into the following sections:

sections

The left section displays the Folder List that contains a tree view of your prefab folders. The right section displays the contents of the currently open folder (or the prefab collections if no folder is selected). The divider between these two sections can be dragged to resize the sections.

The Folder List section contains the following parts:

  • Search Bar: This lets you search for any assets across all of the prefab collections that you have added from the Package Manager.
  • Sort/Filter: This menu lets you switch between A-Z and Z-A sorting in both views of the Prefab Library. You can also enable Favourites so only assets that you have favourited are visible. Additionally you can filter assets by asset type, so e.g. you only see sprites, sounds, or any type of asset you're looking for.
  • View Menu: This lets you switch between the following views that affect how the Prefab Library is displayed:
    • Horizontal View: This is the default view. It displays the folder list on the left and the content view on the right.
    • Vertical View: This displays the folder list at the top and the content view at the bottom of the Prefab Library.
    • Simple View: This disables the folder list and only displays the content view.
    • Tree View: This disables the content view and only displays the folder list. The folder list will now also show the assets within a folder so they can be dragged for use in other editors.
  • Package Manager: This button opens the Package Manager where you can import any new prefab collections or delete ones that you have added before.

The Content View contains the following parts:

  • Path View: This displays a hierarchy of folders that have opened. You can click on any folder to navigate back to it, or click on < to navigate back to the root.
  • Switch View: This button lets you switch between the Grid and List views.

RMB Menu

Right-clicking on any item in the Prefab Library displays the following options:

  • Make Favourite: Add the selected folder or asset to Favourites. Your Favourites can be viewed from the Sort/Filter menu in the Folder List section.
  • Duplicate into Project: This duplicates the selected asset into your project as a local asset. The asset becomes unlinked from its collection and is fully editable.
  • Expand/Collapse All: This option is available in the Folder List. It expands or collapses all the folders in the tree view.
