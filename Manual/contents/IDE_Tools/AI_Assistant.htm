AI Assistant

The AI Assistant is a chatbot where you can converse with the AI, ask questions and request images.

When you open a project, you will see this window:

Click "Start New Chat" to open the AI Assistant and start a new chat. The "Don't show again" option is enabled by default which makes sure this window doesn't appear again (this modifies a preference which you can later enable to prompt this window to open the next time you open a project).

The AI Assistant can also be opened from The Windows Menu or the  button in the Quick Buttons menu of the IDE. The AI Assistant window can be docked to any part of the IDE or used as a floating window.

Home Screen

The home screen of the AI Assistant window displays quick options to start chats about various GameMaker concepts.

The following buttons are given at the top:

  • Start New Chat : Start a new chat
  • Menu : In the home screen, this contains two options: 
    • Open chat history: This opens the chat history menu described below
    • Start New Chat: Start a new chat

Chat History

This menu displays a list of all the chats that were previously created. You can search through the chat titles and select multiple chats to  delete them.

The chat list can be sorted by the Last Modified column in an ascending or descending order. The  button next to a chat lets you rename or delete the chat. The  button is used to open the chat.

Chat

The chat menu is where you can converse with the AI Assistant.

The  menu at the top contains options to start a new chat, rename or delete the current chat, and open the home or chat history screens.

You can search through the chat. At the bottom of the window you can write a message to send to the AI Assistant.

 When you send a message for the first time, you will be required to log in and the Opera log-in page will automatically open in your default browser.
