Language And Themes

The Code Editor supports different types of text files, however there are major differences to keep in mind, which this page will cover. We will also look at how themes function.

Supported Files

 

Language Support

Any GML Code documents open in the Code Editor will get code completion, error highlighting, tooltips, and more, which are covered under Feather Features.

Any of the other types of files mentioned in the "Supported Files" list above will only get colour coding.

Themes & Scopes

The Code Editor Preferences allow you to create and edit themes used in the editor. See Theme for detailed information on all menus.
