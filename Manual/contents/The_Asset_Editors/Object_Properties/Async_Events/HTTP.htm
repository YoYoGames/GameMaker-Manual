HTTP

 

The Async HTTP Event is triggered by the callback from one of the http_*() functions, like http_post_string. It isn't triggered for every single data packet that is received, but rather updates within the main game loop during the download.

A DS Map that is exclusive to this event is stored in the special variable async_load and can be accessed in this event (please see the individual functions for code examples that explain the use of this event in further detail).
