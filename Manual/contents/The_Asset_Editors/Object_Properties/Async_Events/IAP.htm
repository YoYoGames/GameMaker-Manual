Na compra de aplicativos

Este evento só pode ser acionado pelos vários In App Purchase (IAP) extensions para as diferentes plataformas alvo disponíveis no Marketplace. Este evento sempre gerará um mapa DS exclusivo para este evento e armazenado na variável especial async_load. Este mapa DS terá diferentes pares de chaves/valores dependendo da função extension que o acionou, mas sempre haverá uma chave "id" que poderá então ser usada para identificar o tipo de evento de compra In App que é. Para detalhes exatos sobre os possíveis valores de retorno e funções que geram este evento, consulte os manuais individuais em PDF incluídos com cada um dos extensions.
