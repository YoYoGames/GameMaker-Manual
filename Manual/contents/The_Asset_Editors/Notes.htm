O Editor de Notas

O Editor de Notas abrirá uma janela de código/script onde você poderá guardar notas e trechos de código ou qualquer outra informação que você considere relevante para o projeto. Você pode criar várias notas e nomeá-las com o botão direito do mouse no menu do Navegador de Ativos. As notas também podem ser agrupadas e reordenadas como qualquer outro asset.

Se você importou um projeto criado usando uma versão anterior do GameMaker, então você também encontrará um Relatório de Compatibilidade incluído como uma nota nesta seção.
