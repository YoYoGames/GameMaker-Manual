Tipos e parâmetros de FX

Esta página lista todos os tipos de filtro/efeito fornecidos no GameMaker, juntamente com as seguintes informações sobre cada FX:

  • Nome interno (por exemplo, "_filter_tintfilter") que pode ser passado para fx_create() para criar um FX em tempo de execução
  • Descrição
  • Exemplo de imagem
  • Parâmetros IDE, que podem ser editados através das propriedades da camada
  • Parâmetros de tempo de execução, que podem ser usados para modificar FXs em runtime, usando as funções de Filtro e Efeito
    • Os parâmetros Runtime também mencionam seus tipos, para que você saiba que tipo de dados aplicar a qual parâmetro.
