Idioma GameMaker

Você pode usar GameMaker Language para programar seus jogos de duas maneiras diferentes: colocando blocos para "escrever" código visualmente, ou escrevendo código manualmente.

A primeira opção, GML Visual, é mais adequada para iniciantes e hobbistas que querem começar a fazer seus próprios jogos rapidamente.

A segunda opção, Código GML, permite que você aproveite todo o poder do GameMaker Language escrevendo seu código manualmente.

Embora o GML Visual seja mais fácil de começar, o Código GML ainda é uma linguagem de programação fácil de usar com características poderosas(Funções, Métodos, Estruturas & Construtores, etc.).

Criação de Scripts Visuais/Códigos

Você é solicitado a escolher entre o Código GML e o GML Visual ao criar um novo evento Object, um novo Script asset, ou qualquer outra coisa que resulte na criação de um roteiro:

Além da escolha entre o Código GML e o GML Visual, esta janela contém duas caixas de seleção:

  • Não peça novamente por este projeto: Quando isto é habilitado, ele define a opção de jogo "Linguagem de script padrão" para sua nova seleção, para que você não seja solicitado a escolher novamente para o projeto atual.
  • Não pergunte nunca mais: Quando isto é habilitado, estabelece a preferência "Linguagem de script padrão" em relação à sua nova seleção, para que você nunca mais seja solicitado a escolher, para qualquer projeto.
