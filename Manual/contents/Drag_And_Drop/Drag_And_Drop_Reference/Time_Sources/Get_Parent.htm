This action gets the parent Time Source for the given Time Source.

The returned value may be the ID of a custom Time Source, or one of the Built-In Time Sources, depending on how the given Time Source was created.

 

Action Syntax:

Arguments:

Argument Description
Time Source The Time Source ID to get the parent of
Target The variable to store the parent Time Source
Temp Whether the variable should be created as a temporary variable
