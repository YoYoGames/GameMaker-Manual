This action allows you to control other actions based on the unit type of the given Time Source.

It checks if the Time Source Unit Constant given in "Units" is (or isn't) the unit type of the Time Source given in "Time Source".

 

Action Syntax:

Arguments:

Argument Description
Time Source The Time Source ID to check the state of
Not Enable this to invert the condition
Units The Time Source Unit Constant to check against
