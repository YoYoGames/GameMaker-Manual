Obter Saúde

Você pode usar esta ação para recuperar o valor da variável de instância criada para a "saúde" do object. Você fornece a variável de destino para armazenar o valor "saúde" retornado (que pode ser marcado como uma variável local temporária), mas observe que se a variável não foi definida anteriormente (usando Set Health), então a variável será criada para você e a ação retornará 0 (ao contrário das variáveis de instância regular, que darão um erro se você tentar para acessá-las sem tê-las definido primeiro).

Sintaxe de ação:

Argumentos:

Argumento Descrição
Target A variável alvo para armazenar o valor "saúde" retornado (pode ser sinalizada como uma variável local temporária)

Exemplo:

O código de bloco de ação acima recuperará o valor para "saúde" e o armazenará em um variável local. Esta variável é então utilizada para desenhar uma barra de saúde, por exemplo.
