Pontuação definida

Com esta ação, você pode criar ou definir uma variável de exemplo para a pontuação. Você fornece o valor para definir a variável de pontuação ou para adicionar à variável de pontuação (se você marcar a caixa de seleção relativa, então o valor será adicionado à "pontuação" atual. valor). Observe que ao contrário das variáveis de instância regulares, o nome da variável para "pontuação" é armazenado internamente, portanto, para recuperar o valor de pontuação necessário para usar a ação Obter Pontuação.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Score A quantia para definir a pontuação para (ou para adicionar à pontuação se relativo for verificado)

O código de bloco de ação acima verificará se o room é o primeiro no jogo e se é ele estabelece as variáveis "pontuação", "vidas" e "saúde", por exemplo.
