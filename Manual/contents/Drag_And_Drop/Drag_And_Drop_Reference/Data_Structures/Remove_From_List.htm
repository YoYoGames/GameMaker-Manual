Remover da lista

Esta ação pode ser usada para remover um item de índice da lista fornecida. Você fornece a variável que armazena o índice da lista (conforme retornado pela ação Criar Lista) e a posição do índice dentro da lista a ser removida, onde o A posição do índice é entre 1 e (comprimento da lista -1). Esta ação não retorna nada, portanto, se você precisar do valor na posição do índice você deve usar Get List Item At antes de remover o índice.

Sintaxe de ação:

Argumentos:

Argumento Descrição
List O índice (armazenado em uma variável) da lista para remover de
Index O índice dentro da lista para remover

Exemplo:

O código de bloco de ação acima verifica se há uma colisão na posição da instância e se é encontrado o valor de identificação único para a instância é armazenado em uma variável temporária e depois verificado para ver se ele existe dentro da estrutura de dados da lista. Se ele existe, então o item é removido da lista e a instância é destruída.
