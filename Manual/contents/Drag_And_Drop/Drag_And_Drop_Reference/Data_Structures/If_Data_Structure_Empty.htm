Se a estrutura de dados Esvaziar

Com esta ação você pode verificar se a dada estrutura de dados está vazia ou não. Primeiro você precisa escolher o tipo de estrutura de dados a ser verificada (ou uma Lista, um Mapa ou uma Pilha - sem grades, já que elas não podem sequer estar vazias) e então você fornece o variável que contém o índice da estrutura de dados (como retornado pelas ações Criar Lista, Criar Mapa e Criar Pilha). A ação retornará true se a estrutura de dados está vazia, e false se não estiver. Observe que você pode flag "não" na ação para verificar se uma estrutura de dados contém dados , verificando se ela não está vazia.
