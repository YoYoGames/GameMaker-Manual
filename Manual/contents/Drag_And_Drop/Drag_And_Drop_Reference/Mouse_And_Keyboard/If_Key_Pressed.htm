Se a tecla foi pressionada

Esta ação verifica a impressão inicial em uma tecla de teclado. Ela retornará true na etapa de jogo único (frame) em que a tecla é inicialmente pressionada e false em todos os outros momentos. Se você ativar o modificador not, esta ação será revertida, e verifique se a tecla não foi pressionada, retornando false na etapa do jogo (moldura) que foi pressionada e true em todos os outros momentos. Se você precisar verificar se uma tecla está sendo pressionada e não apenas o evento inicial da imprensa, então use a ação If Key Down.

Note que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo:

Quaisquer ações abandonadas em outro lugar serão executadas após o bloco "se".

Exemplo:

O código de bloco de ação acima pesquisa o estado do teclado a cada passo e se uma tecla é sendo mantido pressionado, ele verifica se a tecla inicial está pressionada. Se a verificação for true na prensa inicial, um som é tocado, então, enquanto a tecla é mantida pressionada, a mistura de sprite é mudada para vermelho. Se a tecla não estiver sendo mantida pressionada, uma verificação é feito na liberação da chave para repor a cor da mistura para branco.
