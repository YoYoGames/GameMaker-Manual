Conjunto Sequence Posição

Com esta ação você pode definir a posição para o dado elemento sequence no room. Você fornece o ID do elemento sequence (como retornado pela ação Criar Sequência), e então a nova posição X e Y. Se você marque a caixa de seleção "relativo", então os valores dados para X e Y serão adicionados à posição atual sequence, caso contrário, eles serão uma posição absoluta.

Argumentos:

Argumento Descrição

X A nova posição x no room para o sequence.
Y A nova posição no site room para o sequence.
