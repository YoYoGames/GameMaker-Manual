Obter Sequence Posição

Com esta ação você pode recuperar a posição atual para o dado elemento sequence no room. Você fornece o ID do elemento sequence (como retornado pela ação Criar Sequência), e esta ação retornará sua posição X e Y em variáveis-alvo. Observe que as variáveis-alvo podem ser marcadas como variáveis temporárias (locais), que serão criadas para a ação e podem ser usadas em todas as ações subseqüentes para o evento em que a ação estiver.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Variable A identificação do elemento sequence para obter a posição de.
Target X A variável alvo para manter a posição x.
Target Y A variável alvo para manter a posição y.
