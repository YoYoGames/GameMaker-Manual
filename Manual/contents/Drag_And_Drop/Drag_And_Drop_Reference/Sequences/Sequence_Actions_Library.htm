Sequences Action Library

The Sequence action library is where you can create and destroy sequences in the room, as well as play/pause them and get/set information about them. For more information on Sequences, please see here.

The available actions are all listed below:

Create Sequence Element
Destroy Sequence Element
Get Sequence Position
Set Sequence Position
Play Sequence
Pause Sequence
If Sequence Exists
Get Sequence Length
Get Sequence Head
Set Sequence Head
