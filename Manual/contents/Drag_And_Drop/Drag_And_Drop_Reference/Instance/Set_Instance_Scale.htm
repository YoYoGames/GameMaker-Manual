Escala de instância fixa

Este bloco de ação estabelece os valores image_xscale e image_yscale para a instância. Os valores de entrada aqui são modificadores que serão aplicados ao sprite designado para a instância, onde uma escala de 1 (o valor padrão) indica que não escala (1:1), valores menores serão escalados para baixo (0,5, por exemplo, metade da largura do sprite), e valores maiores serão escalados para cima. Você pode usar valores negativos para inverter o sprite e escalá-lo, a menos que o valor usado seja exatamente -1 (neste caso, o sprite é apenas invertida ou espelhada sobre sua origem sem escalas). Você pode optar por definir a escala horizontal ou vertical em relação aos valores atuais, caso em que você estará adicionando ou subtraindo o novo valor dos valores da escala atual.

Observe que a alteração destes valores também mudará a forma como a instância detecta colisões, a menos que você tenha fornecido um mask_index (máscara de colisão) separado no Editor de Objetos. Por favor, note que para que as mudanças nesta variável sejam visíveis, a instância não deve ter nenhum evento de sorteio (e, portanto, o GameMaker irá, por padrão, sortear o sprite) ou ser sorteado usando a ação do Draw Self.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Horizontal O fator de escala vertical a ser usado (o padrão é 1).
Vertical O fator de escala horizontal a ser usado (o padrão é 1).

Exemplo:

O código de bloco de ação acima estabelece um novo sprite assim como uma série de outros propriedades de como esse sprite deve ser exibido, incluindo o ajuste para escalar até três quartos de seu tamanho de base ao longo de ambos os eixos.
