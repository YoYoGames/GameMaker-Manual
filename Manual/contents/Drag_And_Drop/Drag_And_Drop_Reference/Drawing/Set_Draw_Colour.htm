Set Draw Colour

Com esta ação você pode definir a cor a ser usada para qualquer forma (retângulo/elipse/linha) ou para qualquer texto que seja desenhado posteriormente. Note que isto não precisa ser chamado antes de cada ação de desenho, e pode ser definido apenas uma vez no início do jogo. Por exemplo, se todo seu texto for amarelo, você pode chamar esta ação no Evento Criar do primeiro room do jogo, e todos os textos (e formas) serão então desenhados em amarelo a partir de então. Entretanto, se você precisar usar várias cores diferentes, então você precisará chamar isto antes de cada texto ou forma que requeira uma cor diferente.

A ação tem uma caixa de seleção na parte inferior para Usar alfa. Isto permite que o componente alfa da cor seja usado para definir o alfa também (muito parecido com o uso do Set Draw Alpha faz. Se você não quiser incluir o componente alfa e apenas ter uma cor sólida, isto pode ser desativado (é habilitado por padrão).
