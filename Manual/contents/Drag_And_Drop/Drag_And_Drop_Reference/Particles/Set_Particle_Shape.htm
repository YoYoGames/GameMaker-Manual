Forma da partícula do conjunto

Esta ação pode ser usada para definir a forma construída em sprite que uma partícula deve usar ao desenhar. Você fornece o valor de identificação único para o tipo para definir a forma de (como retornado quando você criou o tipo com a ação Criar tipo de partícula) e a forma a ser usada. As formas das partículas são simplesmente brancas sprites que foram criadas para que o GameMaker as utilize especificamente como partículas e consistem das seguintes formas

NOTA: Se você usar esta ação, não deve usar a ação Set Particle Sprite, pois você só pode ter um sprite ou uma forma, mas não ambos.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Type O valor de identificação para o tipo para definir a forma de
Shape A forma a ser usada para o tipo de partícula

Exemplo:

O código de bloco de ação acima criará um novo tipo de partícula e atribuirá seu exclusivo valor de identificação para uma variável global. Em seguida, ele passa a definir todas as propriedades para o tipo de partícula.
