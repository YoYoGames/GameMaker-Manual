Sistema de Atualização de Partículas

Esta ação forçará o sistema de partículas em questão a atualizar a posição, cor, alfa, etc... Você fornece o valor de identificação único para que o sistema seja atualizado (como retornado quando você criou o sistema com a ação Criar Sistema de Partículas), e cada vez que você chamar a ação o sistema atualizará um quadro de jogo, o que significa que você pode - por exemplo - chamar esta ação dentro de um Repeat loop e fazer a atualização do sistema de partículas várias vezes em um único quadro de jogo (útil para quando se deseja que um nível tenha qualquer efeito que requeira a presença de partículas logo no início, como neve ou chuva). Você também pode usar esta ação quando o sistema de partículas tiver sido pausado, usando o Sistema de Partículas de Pausa para atualizar manualmente as partículas no sistema.
