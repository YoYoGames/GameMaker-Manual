Set Particle Alpha

Esta ação definirá o alfa para quando você criar a partícula, bem como o alfa durante sua vida útil. Primeiro você especifica o valor de identificação do tipo de partícula (que deve ser o mesmo que o valor retornado quando você criou a partícula usando a ação Criar Tipo de Partícula ) e então você pode definir os valores alfa a serem usados.

Você pode escolher três valores alfa para usar, e a partícula se misturará entre eles ao longo da vida útil da partícula. Por exemplo, se uma partícula tem 0,5, 1 e 0 como valores alfa, e uma vida útil de 60 quadros de jogo, então quando ela é criada (quadro 0) estará a 0,5 alfa, depois se desvanecerá em direção a um alfa completo de 1 (quadro 30), antes de desvanecer para ser invisível com um alfa de 0 (quadro 60).

Sintaxe de ação:

Argumentos:

Argumento Descrição
Type O valor de identificação único para o tipo de partícula para definir a cor de
Start O alfa inicial a utilizar
Middle O entre alfa a usar
End O alfa final a utilizar

Exemplo:

O código de bloco de ação acima criará um novo tipo de partícula e atribuirá seu exclusivo valor de identificação para uma variável global. Em seguida, ele passa a definir todas as propriedades para o tipo de partícula.
