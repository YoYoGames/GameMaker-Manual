Salve Buffer

Com esta ação você pode salvar um arquivo contendo os dados de um buffer. Você fornece o único ID buffer (como retornado quando você criou o buffer com Create Buffer) e dá um nome para o arquivo (como um string, com o extension de sua escolha) e o buffer será escrito neste arquivo pronto para ser carregado novamente usando o Load Buffer.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Buffer O único valor de identificação buffer do buffer para economizar
Filename O nome (como um string) do arquivo para salvar

Exemplo:

O código de bloco de ação acima criará uma variável para conter dados buffer e então verificará se um buffer salvar arquivo já foi feito anteriormente. Se o arquivo existe, então ele é carregado na variável buffer que criamos, mas se não existir, ele é criado, escrito e então salvo pronto para ser carregado na próxima vez.
