Fechar arquivo de iniciativas

Esta ação fechará o arquivo Ini, atualmente aberto. Esta ação deve ser chamada no momento em que você terminar de ler ou escrever para qualquer arquivo Ini aberto (os arquivos Ini são abertos usando a ação Open Ini File). Se você não usar esta ação após ter usado a ação Gravar no arquivo Ini, nada será gravado em disco, pois as informações do arquivo serão mantidas em memória até que esta ação seja chamada, momento em que ela será retirada da memória. e escrito no arquivo em disco. Observe também que se você tentar abrir um arquivo da Ini sem ter previamente fechado outro (ou o mesmo), você receberá um erro.

Exemplo:

O código de bloco de ação acima abrirá um arquivo ini para uso, então obterá o valor associado a a tecla "name" sob o cabeçalho "player". Se o nome retornado corresponder ao valor padrão do nome (ou seja: o arquivo, seção ou chave não existe), então o arquivo tem uma variável global escrita antes de ser fechado (e escrita os novos dados em disco).
