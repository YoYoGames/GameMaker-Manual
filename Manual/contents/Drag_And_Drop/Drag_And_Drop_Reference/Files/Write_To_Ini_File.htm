Escrever para o arquivo da Ini

Com esta ação, você pode escrever um valor numérico ou um string em um arquivo da Ini. Você deve ter previamente aberto o arquivo Ini para leitura/escrita usando a ação Abrir arquivo Ini, e então com esta ação você dá o tipo de dados para escrever (um número real ou um string incluído em ""), assim como o cabeçalho da seção, a chave dentro dessa seção e o valor para a chave. Para ajudar a visualizar a estrutura de um arquivo da Ini, veja esta imagem:

Como você pode ver, os arquivos da Ini são divididos em seções - onde cada seção tem seu próprio nome de cabeçalho - e então cada seção é subseqüentemente dividida em vários pares chave/valor (muito parecido com uma estrutura de dados do mapa). Observe que você mesmo pode criar arquivos Ini usando qualquer software de processamento de texto (como o Bloco de Notas em Windows) e depois incluí-los como um arquivo incluído para ler e escrever em seu jogo.

IMPORTANTE! Tentar ler dados de um arquivo Ini que não tenha sido aberto anteriormente usando a ação Open Ini File causará um erro grave e travará seu jogo.

IMPORTANTE! Isto só grava os dados na memória e para finalmente gravar os dados no arquivo em disco, você deve chamar Close Ini File, caso contrário os dados não serão salvos.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Type O tipo de dados a escrever, seja um string (entre aspas ""), ou um real number
Section A seção do arquivo da Ini para escrever (como um string)
Key A chave dentro da seção para escrever para (como um string)
Value O valor a escrever, seja um string ou um número real baseado no argumento "tipo" (acima)

Exemplo:

O código de bloco de ação acima abrirá um ini e então escreverá os dados em três chaves diferentes sob o mesmo cabeçalho de seção (um string e dois números reais) antes de fechar novamente o arquivo ini (escrevendo assim os dados em disco).
