Se o fim do tampão

Esta ação pode ser usada para verificar se a posição de leitura/gravação chegou ao final do buffer. Você fornece o valor de identificação único buffer como retornado pela ação Criar Tampão e então a ação retornará true se a posição de leitura/gravação chegou ao final de todos os dados disponíveis no buffer, ou false se não chegou. Você pode flag "não" para fazer a verificação da ação se a posição de leitura/escrita não estiver no final do buffer.
