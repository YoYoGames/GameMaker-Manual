Obter Tile Transformação de dados

Com esta ação você pode descobrir se os dados tile de um tile foram transformados de uma das três maneiras: virados, espelhados ou girados. Os dados de tile são simplesmente um valor que reflete o índice do tile junto com as transformações aplicadas, e você normalmente querem obter os dados tile primeiro usando Tile Data In Cell ou Tile Data At Pixel antes de usar esta ação. A ação retornará true se a transformação escolhida foi aplicada aos dados do azulejo, ou false caso contrário, e o valor retornado será então armazenado na variável alvo que pode ter sido criada anteriormente ou pode ser uma nova variável temporária (se você verificar o "Temp" caixa de seleção). Para mais informações sobre os dados tile, consulte a seção GML sobre Funções do Mapa de Ladrilhos.
