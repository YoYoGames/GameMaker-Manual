This action forces the current multiplayer game to start, before all players have joined. You must first create a multiplayer game using Create Rollback Game.

The system normally waits for all players to join before automatically starting the game. This action can be used to start the game early, before all players have joined.

You can call Use Manual Start to make Rollback wait for this call even after all players have joined.

Running this action will trigger the Rollback Start event.
