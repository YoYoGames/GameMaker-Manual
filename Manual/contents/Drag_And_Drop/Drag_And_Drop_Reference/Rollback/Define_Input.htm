This function is used to define custom inputs for all players. It takes an Input/Value pair, where the Input is the name of the input, and the Value is the input constant assigned to it.

To know how to assign custom inputs, see Custom Controls.

If used, this function must run before the join/create action.

 

Action Syntax:

Arguments:

Argument Description
Input The name of the input to define
Value The input constant to assign (see Custom Controls)

 

Example:

The above action defines the inputs fire, left, right, up and down for all players.
