This action can be used during Sync Test to assign temporary "mock" input to the specified player. This overrides the controls defined for the player and any random input from Sync Test.

 

Action Syntax:

Arguments:

Argument Description
Player ID The player number that gets this mock input set
Input The name of the input to define
Value The input constant to assign (see Custom Controls)

 

Example:

The above action defines the mock inputs fire, left, right, up and down for player 1.
