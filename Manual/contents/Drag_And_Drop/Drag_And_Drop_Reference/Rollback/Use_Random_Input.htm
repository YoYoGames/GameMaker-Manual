When you play a game using Sync Test, all "remote" players receive random values for their defined inputs, as a basic way of testing. You can disable or re-enable this behaviour using this action.

 

Action Syntax:

Arguments:

Argument Description
Enabled Enable or disable random input for remote players while testing
