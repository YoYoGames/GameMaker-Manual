This function returns a struct containing information about a player. This should only be used when managing player instances manually. If you're using Define Player, then each player instance will automatically receive this information as variables.

For information on the contents of the returned struct, see: rollback_get_info

 

Action Syntax:

Arguments:

Argument Description
Player ID The ID of the player to get info for
Target The variable where the returned struct should be stored
