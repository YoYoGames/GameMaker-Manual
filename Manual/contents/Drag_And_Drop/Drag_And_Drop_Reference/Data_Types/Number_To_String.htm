Número para String

Esta ação converterá um número real em um string. Se o número não for um valor inteiro e tiver casas decimais, estas serão arredondadas para as duas casas mais próximas, assim, por exemplo, o número 123.45678 retornaria o string "123.46".

Sintaxe de ação:

Argumentos:

Argumento Descrição
Value O número real para converter
Target A variável alvo para manter o valor retornado

Exemplo:

O código de bloco de ação acima toma um valor decimal e o converte em um número inteiro, armazenando o valor retornado em uma variável local temporária. Este valor é então transformado em um string e atraído para a tela.
