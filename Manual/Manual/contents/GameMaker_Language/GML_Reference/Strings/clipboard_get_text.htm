clipboard_get_text

Esta função retornará um string do texto contido na prancheta. se nenhum texto for armazenado, retornará um string "" vazio.

Sintaxe:

clipboard_get_text();

Devoluções:

String

Exemplo:

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

O código acima verifica o texto na prancheta e se ela contém algum, ele é lido como um string na variável "str". Finalmente, a área de transferência é limpa, definindo-a como um string vazio.
