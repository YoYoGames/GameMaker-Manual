string_replace_all

Você pode usar esta função para analisar um string procurando peças específicas, que podem então ser substituídas pelo novo string que você especificou em todos os lugares que elas ocorrem.

Sintaxe:

string_replace_all(str, substr, newstr);

ArgumentoTipo Descrição
str The string to be copied.
substr The substring within the string to be replaced in all instances.
newstr The new substring to replace the previous one(s).

Devoluções:

Exemplo:

str1 = "Hexxo Worxd";
str2 = string_replace_all(str1, "x", "l");

Isto colocará str2 em str1, mas com suas instâncias de "x" substituídas por "l", resultando em str2 como "Hello World".
