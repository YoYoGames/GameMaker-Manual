matrix_build

Esta função pode ser usada para criar sua própria matriz personalizada e retornará uma matriz, que deve ser armazenada em uma variável para referência e uso futuro. Ela aceita tradução tridimensional(x, y, z), rotação e valores de escala, e os utiliza para construir uma matriz matricial.

A matriz contém 16 valores no total, onde os 4 elementos iniciais são linha/coluna 1os próximos 4 elementos são linha/coluna 2 e assim por diante, como parte de uma matriz 4x4. Se a matriz é ordenada por linhas ou colunas depende da plataforma alvo, já que o renderizador gráfico usado por uma plataforma pode usar matrizes de linha ou de coluna.

NOTA Quando você constrói uma nova matriz desta forma, a ordem de operação é YXZ.

Sintaxe:

matrix_build(x, y, z, xrotation, yrotation, zrotation, xscale, yscale, zscale);

ArgumentoTipo Descrição
x The x component of the translation vector.
y The y component of the translation vector.
z The z component of the translation vector.
xrotation The angle to rotate around the x-axis (in degrees °).
yrotation The angle to rotate around the y-axis (in degrees °).
zrotation The angle to rotate around the z-axis (in degrees °).
xscale The x scale amount.
yscale The y scale amount.
zscale The z scale amount.

Devoluções:

Matrix Array

Exemplo:

t_matrix = matrix_build(x, y, 0, 0, 90, 0, 1, 2, 1);

O código acima irá construir uma nova matriz transformará e armazenará o índice de matriz resultante na variável "t_matrix".
