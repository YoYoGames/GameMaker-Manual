matrix_stack_set

Esta função sobrepõe o topo atual da pilha de matrizes com a matriz especificada.

Sintaxe:

matrix_stack_set(matrix);

ArgumentoTipo Descrição
matrix The matrix index to use.

Devoluções:

Exemplo:

var m = matrix_build(x, y, 0, 0, 0, 0, 1, 1, 1);
matrix_stack_set(m);

O código acima construirá uma nova matriz e armazenará o índice de matriz resultante na variável "m" antes de substituir a parte superior da pilha de matriz por ela.
