ano_actual

Esta variável lida somente retornará o ano corrente.

Sintaxe:

current_year

Devoluções:

Exemplo:

draw_text((32, 32, "Today is " + string(current_day) + "/" + string (current_month) + "/" + string(current_year) +".");

O código acima desenhará o dia, mês e ano.
