ângulo_diferença

Esta função retornará a menor diferença entre os dois ângulos especificados, onde a diferença é calculada a partir do ângulo da fonte em direção ao ângulo de destino. O valor retornado será entre -180 e 180 graus (onde um ângulo positivo é anti-horário e um ângulo negativo é horário).

A relação entre os ângulos "fonte" e "destino" é tal que a diferença que você obtém ao chamar ângulo_diferença(dest, src), quando adicionado de volta ao valor src, lhe dá o ângulo de destino (embora os valores numéricos exatos possam ser diferentes). Você pode usar uma técnica semelhante para mover o ângulo de origem em direção ao ângulo de destino gradualmente a cada passo, onde você adiciona, digamos, 10% da diferença retornada de volta ao src para movê-lo um pouco em direção ao destino (isto é demonstrado no exemplo abaixo).

Sintaxe:

angle_difference(dest, src)

ArgumentoTipo Descrição
dest The destination or target angle
src The source angle

Devoluções:

Exemplo:

var _dir = point_direction(x, y, mouse_x, mouse_y);
var _diff = angle_difference(_dir, image_angle);
image_angle += _diff * 0.1;

O código acima irá obter a direção da instância para o cursor do mouse, então obterá a diferença entre esse ângulo e a corrente image_angleusando este valor para girar gradualmente a instância em direção ao mouse.
