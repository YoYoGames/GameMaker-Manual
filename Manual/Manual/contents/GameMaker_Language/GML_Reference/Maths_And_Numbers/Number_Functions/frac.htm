frac

Esta função retorna a parte fracionada de n, ou seja, a parte atrás do ponto decimal. Ela retornará somente os decimais atrás do ponto de um valor, portanto frac(3.125) retornará 0,125, frac(6.921) retornará 0,921, etc...

Sintaxe:

frac(n);

ArgumentoTipo Descrição
n The number to change.

Devoluções:

Exemplo:

val = frac(3.4);

Isto colocará o valor em 0,4.
