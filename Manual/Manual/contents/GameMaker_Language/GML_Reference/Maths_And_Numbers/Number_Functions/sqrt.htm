sqrt

Se você multiplicar um número consigo mesmo, você obtém o quadrado desse número, mas às vezes você quer fazer o inverso e obter a raiz quadrada de um número. Assim, para descobrir qual número foi multiplicado consigo mesmo para obter qualquer dado valor positivo que usamos esta função. Por exemplo: sqrt(9) retornará 3 desde 3*3=9.

Sintaxe:

sqrt(val);

ArgumentoTipo Descrição
val The number to get the square root of.

Devoluções:

Exemplo:

num = sqrt(exp);

O código acima definirá a variável "num" para manter a raiz quadrada do valor contido em "exp".
