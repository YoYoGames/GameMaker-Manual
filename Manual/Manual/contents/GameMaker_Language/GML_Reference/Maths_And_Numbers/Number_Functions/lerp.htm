lerp

Com esta função você pode encontrar o valor que equivale à posição entre dois outros valores para uma determinada porcentagem. Assim, por exemplo, se você o fizer:

lerp(0, 10, 0.5)

você obteria o valor de retorno de 5, que é 50% dos valores de entrada. Você pode extrapolar usando esta função também, fornecendo um valor positivo ou negativo para o valor da interpolação, de modo que fazendo algo parecido:

lerp(0, 10, 2)

devolverá um valor de 20.

Sintaxe:

lerp(a, b, amt)

ArgumentoTipo Descrição
a The first value.
b The second value.
amt The amount to interpolate.

Devoluções:

Exemplo:

xx = lerp(x, x + hspeed, room_speed);
yy = lerp(y, y + vspeed, room_speed);

O código acima usa a função de interpolação linear para prever para onde uma instância teria se movido após um segundo de tempo de jogo.
