rollback_define_input

Esta função é utilizada para definir entradas personalizadas para todos os jogadores. É necessário um struct que deve conter todas as entradas com suas constantes de entrada atribuídas.

Para saber como atribuir entradas personalizadas, consulte Controles personalizados.

Se utilizada, esta função deve ser executada antes da função de junção/criação.

Sintaxe:

rollback_define_input(input_struct);

Argumento Tipo Descrição
input_struct Estrutura A struct containing input names as variables, and their assigned inputs as values.

Devoluções:

N/A

Exemplo:

rollback_define_input({
    fire: mb_left,
    interact: vk_space,
    left: [ord("A"), vk_left],
    right: [ord("D"), vk_right]
});

O código acima define entradas personalizadas para todos os jogadores. As duas últimas entradas utilizam múltiplos controles, listando-os em uma matriz.

Para um exemplo ampliado, consulte Definindo entradas.
