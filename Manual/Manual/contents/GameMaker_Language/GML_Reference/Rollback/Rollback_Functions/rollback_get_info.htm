rollback_get_info

Esta função retorna um struct contendo informações sobre um jogador. Isto só deve ser usado quando se gerencia manualmente as instâncias de um jogador. Se você estiver usando rollback_define_player()Cada instância de jogador receberá automaticamente estas informações como variáveis.

A função toma a identificação do jogador como argumento, e retorna um struct com as seguintes variáveis:

Variável Tipo Descrição
player_avatar_url Cordão URL que aponta para a imagem do avatar do jogador
player_avatar_sprite Ativo Sprite Sprite contendo o avatar do jogador
player_name Cordão O nome do jogador
player_type Cordão Diz se o jogador está inscrito ou não na GX.games

Será "Convidado " para usuários convidados, e "Usuário" para usuários assinantes
player_user_id Cordão A identificação do jogador no GX.games
player_prefs Qualquer As preferências definidas para o jogador

Sintaxe:

rollback_get_info(player_id);

Argumento Tipo Descrição
player_id Real The ID of the player for which information will be returned.
