player_avatar_sprite

Esta variável armazena o sprite da instância atual do jogador.

Esta variável só existe em instâncias de jogadores criados por rollback_define_player()e é diferente para cada jogador.

Os jogadores convidados exibirão um espaço reservado sprite.

Sintaxe:

player_avatar_sprite;

Devoluções:

Sprite Asset

Exemplo:

// Draw event
draw_sprite_stretched(player_avatar_sprite, 0, x, y - 256, 64, 64);

Isto exibe o avatar da instância do jogador executando o código, em uma área 64x64, 256px acima da instância.
