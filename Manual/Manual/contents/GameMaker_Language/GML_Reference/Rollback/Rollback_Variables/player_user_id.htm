player_user_id

Esta variável armazena o ID GX.games do jogador atual, para uso com chamadas API diretas para o servidor.

Esta variável só existe em instâncias de jogadores criados por rollback_define_player()e é diferente para cada jogador.

Os jogadores convidados terão um string vazio atribuído a esta variável.

Sintaxe:

player_user_id;

Devoluções:

String
