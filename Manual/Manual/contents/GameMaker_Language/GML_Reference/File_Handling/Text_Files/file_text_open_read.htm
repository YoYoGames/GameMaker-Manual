file_text_open_read

Esta função abre o arquivo de texto com o nome de arquivo indicado apenas para leitura, retornando a identificação única do arquivo que deve ser armazenado em uma variável, pois ele será usado para todas as outras ações relacionadas a esse arquivo. Se o arquivo não existir, então a função retornará o valor -1.

OBSERVAÇÃO: Você só pode ter um máximo de 32 arquivos abertos a qualquer momento. Você também deve sempre fechar os arquivos quando terminar, pois isto escreve as informações e libera a memória associada ao arquivo.

ADVERTÊNCIA! Esta função pode não funcionar como você espera devido ao GameMaker ser lixado! Consulte a seção sobre o Sistema de Arquivos para maiores informações.

Sintaxe:

file_text_open_read(fname);

ArgumentoTipo Descrição
fname The name of the file to read from.

Devoluções:

Text File ID or -1

Exemplo:

file = file_text_open_read(working_directory + "level.txt");

Isto abriria "level.txt" do mesmo diretório do jogo e armazenaria a identificação do arquivo na variável "file".
