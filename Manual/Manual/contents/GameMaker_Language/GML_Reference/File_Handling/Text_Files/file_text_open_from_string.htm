file_text_open_from_string

Esta função criará um arquivo de texto de um string e o abrirá para leitura, devolvendo o arquivo "handle" que deve ser usado em todas as outras chamadas de função de arquivo para leitura deste arquivo. Note que este arquivo é temporário e somente de leitura, e como tal será removido da memória no momento em que for fechado.

OBSERVAÇÃO: Você só pode ter um máximo de 32 arquivos abertos a qualquer momento. Você também deve sempre fechar os arquivos quando terminar, pois isto escreve as informações e libera a memória associada ao arquivo.

ADVERTÊNCIA! Esta função pode não funcionar como você espera devido ao GameMaker ser lixado! Consulte a seção sobre o Sistema de Arquivos para maiores informações.

Sintaxe:

file_text_open_from_string(string);

ArgumentoTipo Descrição
string The string to create the file from.

Devoluções:

File ID or -1

Exemplo:

file = file_text_open_from_string(reset_str);

O código acima pega o string armazenado na variável "reset_str" e o usa para criar um arquivo de texto somente leitura. O "handle" para este arquivo é então armazenado na variável "file" para todas as outras funções de arquivo a serem usadas.
