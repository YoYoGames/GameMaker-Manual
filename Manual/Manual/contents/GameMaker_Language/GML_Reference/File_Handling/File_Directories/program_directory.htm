program_directory

Isto devolverá o diretório onde o jogo executável é armazenado. Entretanto, isto pode nem sempre ser útil, particularmente porque alguns dispositivos executam o exe a partir de um arquivo *.zip, então isto retornaria o mesmo não importando de onde o jogo esteja realmente rodando.

ADVERTÊNCIA! Esta função pode não funcionar como você espera devido ao GameMaker ser lixado! Consulte a seção sobre o Sistema de Arquivos para maiores informações.

NOTA: Se você desativar a caixa de areia, então você ainda não poderá escrever neste diretório para evitar qualquer possível problema com a eliminação dos arquivos de jogo necessários.

Sintaxe:

program_directory

Devoluções:

Exemplo:

dir = program_directory;

Isto irá armazenar o diretório onde o executável é armazenado em uma variável.
