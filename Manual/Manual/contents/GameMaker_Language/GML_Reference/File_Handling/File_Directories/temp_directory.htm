temp_directório

Isto pode ser usado para devolver o diretório temporário criado para seu jogo cada vez que ele for executado (a raiz não contém o "\" final). Este diretório conterá arquivos e poderá ser acessado enquanto o jogo estiver sendo executado, mas será removido (ao longo do jogo). com todos os arquivos que contém) quando o jogo é encerrado.

ADVERTÊNCIA! Esta função pode não funcionar como você espera devido ao GameMaker ser lixado! Consulte a seção sobre o Sistema de Arquivos para maiores informações.

Sintaxe:

temp_directory

Devoluções:

Exemplo:

ini_open(temp_directory + "\temp_ini.ini");

Isto abrirá um arquivo ini no diretório temporário do jogo (criando-o se ele ainda não existir).
