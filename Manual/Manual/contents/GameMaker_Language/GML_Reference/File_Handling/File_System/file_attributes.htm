file_attributes

Você pode usar esta função para verificar os atributos de um determinado arquivo. Você pode somar as seguintes constantes para ver o tipo de arquivo que deseja:

Constante Descrição
fa_readonly Arquivos somente leitura
fa_hidden Arquivos ocultos
fa_sysfile Arquivos do sistema
fa_volumeid Arquivos volumétricos
fa_directory Diretórios
fa_archive Arquivos arquivados

NOTA: Esta é uma função apenas Windows.

Sintaxe:

file_attributes(fname, attr);

Argumento Tipo Descrição
fname Cordão The name of the file to check.
attr Arquivo Atributo Constante The attributes to check for.

Devoluções:

Boolean

Exemplo:

if !file_attributes(file, fa_hidden)
{
    file_delete(file);
}

Isto verificaria um arquivo para ver se ele está escondido ou não, e se não estiver apagado.
