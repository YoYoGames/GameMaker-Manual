sha1_file

Na criptografia, SHA-1 é uma função de hash criptográfico projetada pela Agência de Segurança Nacional dos Estados Unidos e é empregada em várias aplicações amplamente utilizadas, populares Git, onde é usada para verificar alterações de arquivos, e os protocolos TLS e SSL, PGP, SSH, S/MIME, e IPsec. Esta função pegará um arquivo de entrada e retornará um resumo de 160 bits de mensagem em formato ASCII exclusivo para esse arquivo a ser usado para verificação de integridade em qualquer data posterior.

NOTA: O SHA-1 não é completamente seguro e pode ser quebrado. Veja esta página para mais informações.

Sintaxe:

sha1_file(filename)

Argumento Tipo Descrição
filename Cordão The file to generate the sha1 hash for.

Devoluções:

String

Exemplo:

hash = sha1_file(working_directory + "game_data.ini")

O código acima irá gerar um hash sha1 para o arquivo especificado e armazenar o valor retornado na variável "hash".
