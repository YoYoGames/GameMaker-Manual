base64_decode

Esta função irá converter um string codificado anteriormente usando o formato base64, em texto padrão. Base64 é um esquema de codificação comumente usado para qualquer mídia que precise ser armazenada ou transferida pela Internet como texto, e renderiza a saída não legível ao olho humano.

Sintaxe:

base64_decode(string)

ArgumentoTipo Descrição
string The string to decode.

Devoluções:

Exemplo:

var str, file;
str = base64_encode(game_data);
file = file_text_open_read("save.txt");
str = file_text_read_string(file); level_data = base64_decode(str);
file_text_close(file);

O código acima abrirá um arquivo de texto e lerá um string a partir dele na variável local "str". Este string é então decodificado e o resultado é armazenado na variável de instância "level_data".
