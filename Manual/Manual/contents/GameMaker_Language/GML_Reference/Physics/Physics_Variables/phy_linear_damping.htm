f_linear_damping

Esta variável pode ser usada para definir o amortecimento linear da instância, ou pode ser usada para obter o amortecimento linear atual. O amortecimento é a quantidade de "resistência" ao movimento de avanço que a instância habilitada pela física tem, com um valor mais baixo permitindo que a instância se mova e acelere mais rapidamente e um valor mais alto fazendo com que ela exija um empurrão mais forte

Sintaxe:

phy_linear_damping;

Devoluções:

(single precision floating point value, or undefined if the instance is not physics enabled)

Exemplo:

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
    phy_linear_damping = 10;
}
else
{
    phy_linear_damping = 3;
}

O código acima verificará se há uma colisão entre a instância de chamada e instâncias de "obj_Water" e alterará o amortecimento linear de acordo.
