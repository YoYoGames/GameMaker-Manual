f_angular_damping

Esta variável pode ser usada para definir o amortecimento angular da instância, ou pode ser usada para obter o amortecimento angular atual. O amortecimento é a quantidade de "resistência" à rotação angular que a instância habilitada pela física possui, com um valor mais baixo permitindo que a instância gire mais rápido após uma colisão (por exemplo) e um valor mais alto fazendo com que ela exija um empurrão mais forte e gire mais lentamente.

Sintaxe:

phy_angular_damping;

Devoluções:

(single precision floating point value, or undefined if the instance is not physics enabled)

Exemplo:

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
    phy_angular_damping = 10;
}
else
{
    phy_angular_damping = 3;
}

O código acima verificará se há uma colisão entre a instância de chamada e instâncias de "obj_Water" e alterará o amortecimento angular de acordo.
