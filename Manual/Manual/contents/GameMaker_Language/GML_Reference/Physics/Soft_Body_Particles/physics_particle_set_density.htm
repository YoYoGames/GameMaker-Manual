densidade_de_set_física

Com esta função você pode definir a densidade das partículas em uma simulação física. A definição da densidade da partícula terá um impacto direto em quanto inércia ela tem, bem como em como ela reage a colisões, portanto, se você fizer uma pequena partícula com uma alta densidade terá uma massa muito grande, mas se você definir uma partícula grande com uma baixa densidade, ela terá uma massa muito menor. Esta função é global no escopo, na medida em que mudará a densidade não apenas para novas partículas criadas após a mudança, mas também para aqueles já presentes na simulação.

Sintaxe:

physics_particle_set_density(density)

ArgumentoTipo Descrição
density The density of the particle fixture.

Devoluções:

Exemplo:

physics_particle_set_radius(15);
physics_particle_set_density(0.5);
physics_particle_set_damping(1);
physics_particle_set_gravity_scale(1);

O código acima definirá as propriedades básicas para todas as partículas na simulação.
