fisic_fixture_set_restitution

Em física, a restituição é definida como "o retorno de um object ou sistema ao seu estado original após deformação elástica", mas como os dispositivos no GameMaker são realmente corpos rígidos e não podem ser deformados, a restituição é realmente uma forma de dizer o quão "saltitante" é o acessório. Este cenário afetará o quanto um object "salta" quando colide com outro objects e é co-dependente de outras forças que agem na instância como a gravidade e o atrito, e geralmente é um valor entre 0 e 1 (valores mais altos podem ser usados, mas podem dar resultados imprevisíveis). Aqui está uma ilustração de como funciona:

Sintaxe:

physics_fixture_set_restitution(fixture, restitution)

ArgumentoTipo Descrição
fixture The index of the fixture
restitution The restitution of the fixture (usually between 0 and 1)

Devoluções:

Exemplo:

physics_fixture_set_restitution(fix_Ball, 0.9);

O código acima estabelecerá a restituição da fixação indexada em "fix_ball" para 0,9.
