fisica_fixture_set_box_box_shape

Esta função define uma forma de caixa para sua fixação. Ela toma a metade da largura e da altura, já que o mundo da física usa este valor muito mais do que valores inteiros de largura/altura para determinar coisas como colisões.

Sintaxe:

physics_fixture_set_box_shape(fixture, halfWidth, halfHeight)

ArgumentoTipo Descrição
fixture the index of the fixture
halfWidth the half width of the box
halfHeight the half height of the box

Devoluções:

Exemplo:

physics_fixture_set_box_shape(fix_Border, room_width/2, 10);

O código acima aplicará um formato de caixa à fixação indexada na variável "fix_Border" com uma largura do room e uma altura de 20 pixels.
