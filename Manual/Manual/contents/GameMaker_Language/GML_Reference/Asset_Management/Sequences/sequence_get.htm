sequence_get

Com esta função você pode recuperar a Sequência Object Estruturas a partir de um índice sequence asset. Você fornece o índice asset para o sequence conforme definido no navegador asset, e a função retornará o sequence object struct que poderá ser acessado, ou -1 se o sequence não existir ou se o índice asset dado não for um sequence.

Sintaxe:

sequence_get(sequence_index);

Argumento Tipo Descrição
sequence_index Sequência Ativo The sequence asset as defined in the Asset Browser.

Devoluções:

Sequence Object Struct or -1

Exemplo:

var _seq = sequence_get(seq_Logo);
seq.loopmode = seqplay_pingpong;

O código acima recupera o sequence object struct do sequence object "seq_Loop" e depois define o modo loop para o sequence (todas as instâncias subseqüentemente criadas sequence agora usarão este modo loop ).
