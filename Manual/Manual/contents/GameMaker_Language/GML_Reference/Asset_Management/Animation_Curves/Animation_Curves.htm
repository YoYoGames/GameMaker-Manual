Curvas de Animação

As curvas de animação podem ser configuradas no Navegador de Ativos como assets usando o Editor de Curvas de Animação e podem então ser usadas no Editor de Seqüências para controlar faixas em um sequence. No entanto, elas também podem ser usadas em seu código para alterar valores ao longo do tempo em seus jogos para qualquer coisa, desde efeitos de entrelaçamento em botões, até movimentação, mudança de cor e muito mais. Você também pode criar seu próprio animation curves em runtime usando as funções listadas aqui, tornando-as uma ferramenta muito flexível para seus jogos.

As funções disponíveis para as Curvas de Animação são as seguintes:

  • animcurve_get
  • animcurve_get_channel_index
  • animcurve_get_channel
  • animcurve_channel_evaluate
  • animcurve_create
  • animcurve_existe
  • animcurve_channel_new
  • animcurve_point_new
  • animcurve_destroy
