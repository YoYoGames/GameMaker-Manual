shader_get_sampler_index

Como você não pode alterar o valor de um amostrador shader dentro do próprio shader, você tem que configurá-lo antes de ligar para o shader usando uma das funções do conjunto uniforme disponível. Entretanto, para poder fazer isso, você deve primeiro chamar esta função para obter o "cabo" do amostrador shader que você desejará configurar.

NOTA Embora um shader seja composto de dois programas discretos (vertex e fragmento) esta função não diferenciará entre os dois e devolverá o cabo da amostra shader de nenhum deles.

Sintaxe:

shader_get_sampler_index(shader, uniform);

Argumento Tipo Descrição
shader Ativo Shader The index of the shader to use.
uniform Cordão The shader sampler to get the handle of (a string).

Devoluções:

Shader Sampler Handle

Exemplo:

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

O código acima obterá o cabo para o amostrador dentro do shader indexado como "GlassShader" e então definirá que shader constante para o dado sprite texture.
