shader_set

Com esta função, você pode definir a meta de desenho para o dado shader e todos os outros desenhos serão feitos usando isso. Você pode terminar o uso do shader com a função shader_reset().

Sintaxe:

shader_set(shader);

ArgumentoTipo Descrição
shader The index of the shader to use.

Devoluções:

Exemplo:

shader_set(shader_Glass);
draw_self();
shader_reset();

O código acima estabelecerá um shader a ser usado para desenhar, depois desenhe o atual sprite usado para o exemplo usando-o.
