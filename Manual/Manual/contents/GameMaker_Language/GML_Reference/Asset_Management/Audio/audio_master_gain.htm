audio_master_gain

Com esta função você pode definir o valor absoluto para o volume global de todos os sons e músicas. É baseado em uma escala linear de 0 (silencioso) a qualquer valor maior que 0, embora normalmente você consideraria o volume máximo como 1. Qualquer coisa acima de 1 pode ser usada mas, dependendo do som usado e da plataforma sendo compilada, você pode obter distorção ou recorte quando o som for reproduzido. Esta função afetará o volume relativo de todos os sons e músicas reproduzidos a partir de seu jogo.

Sintaxe:

audio_master_gain(gain);

Argumento Tipo Descrição
gain Real Value for the global volume (0 to 1).

Devoluções:

N/A

Exemplo:

if keyboard_check(vk_up)
{
    if vol < 1 vol += 0.05;
    audio_master_gain(vol);
}
if keyboard_check(vk_down)
{
    if vol > 0 vol -= 0.05;
    audio_master_gain(vol);
}

O código acima verifica se as teclas das setas para cima e para baixo estão pressionadas, o que aumenta ou diminui a variável "vol". Esta variável é então usada para definir o ganho global de todo o som e música no jogo.
