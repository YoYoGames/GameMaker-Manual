audio_group_unload

Esta função irá descarregar todos os sons que estão marcados como pertencentes ao grupo de áudio em questão na memória. A função retornará true se a descarga for iniciada e false se a ID do grupo for inválida, ou se já tiver sido marcada para descarga. Note que qualquer áudio que esteja sendo tocado atualmente quando esta função for chamada, será interrompido.

Sintaxe:

audio_group_unload(groupID);

Argumento Tipo Descrição
groupID ID do grupo de áudio The index of the audio group to unload (as defined in the Audio Groups Window)

Devoluções:

Boolean

Exemplo:

if audio_group_is_loaded(audiogroup_level1)
{
    audio_group_unload(audiogroup_level1);
}

O código acima verifica se um grupo de áudio foi carregado e se ele o descarrega.
