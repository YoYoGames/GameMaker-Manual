audio_group_set_gain

Com esta função você pode desbotar um grupo de sons dentro ou fora durante um determinado período de tempo, ou pode ser usado para definir o ganho do grupo instantaneamente.

O tempo é medido em milissegundos, e a função requer que você insira um nível final de ganho para que o grupo tenha alcançado até o final desse tempo. Este ganho pode ser entre 0 (silencioso) e 1 (volume completo) e a escala é linear, de tal forma que um valor de 0,5 seria metade do volume. Para alterar instantaneamente o ganho, basta definir o argumento de tempo para 0. Observe que em algumas plataformas você pode ter um ganho maior que 1, embora um valor de 1 seja considerado "volume completo" e qualquer coisa maior possa introduzir clipping de áudio.

Sintaxe:

audio_group_set_gain(groupID, volume, time);

Argumento Tipo Descrição
groupID ID do grupo de áudio The index of the audio group to stop (as defined in the Audio Groups Window)
volume Real The final value for the group volume.
time Real The length of the change in gain in milliseconds.

Devoluções:

N/A

Exemplo:

if keyboard_check_pressed(vk_space)
{
    audio_group_set_gain(audiogroup1, 0, 5000);
}

O código acima verifica a tecla "espaço" e depois desbota todo o áudio do "audiogrupo1" para 0 em 5 segundos.
