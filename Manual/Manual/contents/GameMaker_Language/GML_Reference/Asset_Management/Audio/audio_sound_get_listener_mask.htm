audio_sound_get_listener_mask

Esta função retornará os dados da máscara de bits que define de quais ouvintes de áudio um som deve ser reproduzido. Consulte a seção sobre os ouvintes de áudio para mais informações.

Sintaxe:

audio_sound_get_listener_mask(soundID);

Argumento Tipo Descrição
soundID Identificação da instância sonora The unique ID of the sound to get the mask of

Devoluções:

Real

Exemplo:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
if audio_sound_get_listener_mask(snd) != global.PlayerMask
{
    audio_sound_set_listener_mask(snd, global.PlayerMask);
}

O código acima toca um som e então verifica os dados da máscara do ouvinte quanto ao som, e se não for o mesmo que o armazenado em uma variável global, ele define o(s) ouvinte(s) para tocar a partir do uso dos dados da máscara armazenados na variável global.
