audio_emitter_get_pitch

Esta função retorna o valor do tom atual definido para o emissor de áudio em questão.

Sintaxe:

audio_emitter_get_pitch(emitter);

Argumento Tipo Descrição
emitter ID do emissor de áudio The index of the emitter to use.

Devoluções:

Real

Exemplo:

if audio_emitter_get_pitch(emitter_player) != 1
{
    audio_emitter_pitch(emitter_player, 1);
}

O código acima verifica o passo atual de um determinado emissor e, se não for igual a 1, é ajustado para 1.
