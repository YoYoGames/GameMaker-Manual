audio_listener_set_velocity

Esta função pode ser usada para dar ao ouvinte dado efeitos Doppler e simular movimento de áudio com base no vetor que é resolvido a partir das posições relativas x, y e z dadas. O índice padrão do ouvinte é 0, mas você pode usar a função audio_get_listener_info() para obter os diferentes índices disponíveis para a plataforma alvo.

Se o ouvinte dado não vai se mover, ou o movimento não é um movimento constante, você normalmente não precisaria definir estes valores, mas, por exemplo, se você estiver fazendo um jogo de rolagem onde o jogador tem um movimento constante de baixo para cima e os inimigos um movimento constante de cima para baixo, você definiria as velocidades do ouvinte e do emissor (para emissores, você usaria audio_emitter_velocity()) para os vetores apropriados para simular o efeito Doppler correto à medida que eles passam da instância do jogador.

A imagem abaixo mostra como isto poderia ser configurado para o jogo de exemplo dado acima:

Sintaxe:

audio_listener_set_velocity(index, x, y, z);

Argumento Tipo Descrição
index ID do ouvintereal ou de áudio The listener to change the velocity of (default 0).
x Real The x velocity of the listener (default 0).
y Real The y velocity of the listener (default 0).
z Real The z velocity of the listener (default 0).

Devoluções:

N/A

Exemplo:

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var data = audio_listener_get_data(info[? "index"]);
    if data[? "vx"] != 0
    {
        audio_listener_set_velocity(info[? "index"], 0, 0, 0);
    }
    ds_map_destroy(info);
    ds_map_destroy(data);
}

O código acima verifica o número de ouvintes disponíveis então loops através deles e se sua velocidade x não for 0, ele define seus valores de velocidade para 0, 0, 0, 0.
