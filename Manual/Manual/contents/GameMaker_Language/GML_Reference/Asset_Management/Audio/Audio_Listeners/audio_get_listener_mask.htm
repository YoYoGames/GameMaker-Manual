audio_get_listener_mask

Esta função retornará os dados da máscara de bits que define a máscara padrão (global) atual para os ouvintes de áudio.

Sintaxe:

audio_get_listener_mask();

Devoluções:

Exemplo:

var g_mask = audio_get_listener_mask();
if g_mask != global.Audio_Mask
{
    var num = audio_get_listener_count();
    global.Audio_Mask = 0;

    for(var i = 0; i < num; ++i;)
    {
        var info = audio_get_listener_info(i);
        var m = audio_listener_get_data(info[? "mask"]);
        global.Audio_Mask = global.Audio_Mask | m;
        ds_map_destroy(info);
    }
    audio_set_listener_mask(mask);
}

O código acima obtém os dados atuais da máscara do ouvinte e os compara com os dados armazenados em uma variável global. Se não forem os mesmos, o código verifica o número de ouvintes disponíveis, então loops através deles e recebe seus bits de máscara, que são então combinados para criar uma única máscara que é aplicada ao sistema de áudio para definir os ouvintes globais.
