audio_resume_grupo_sync

Esta função retomará o grupo de sincronização dado se ele estiver tocando e tiver sido previamente pausado (usando a função audio_pause_sync_group). O índice do grupo é o valor retornado quando você criou o grupo usando a função audio_create_sync_group().

NOTA: Esta funcionalidade não está disponível para a plataforma de destino HTML5.

Sintaxe:

audio_resume_sync_group(group_index);

Argumento Tipo Descrição
group_index ID do Grupo de Sincronização de Áudio The group index to resume.

Devoluções:

N/A

Exemplo:

if (keyboard_check_pressed((ord)"P"))
{
    global.Pause = !global.Pause
    if global.Pause
    {
        audio_pause_sync_group(sg);
    }
    else
    {
        audio_resume_sync_group(sg);
    }
}

O código acima verifica se a tecla "P" está pressionada e, se for detectada, alterna a variável "global.Pause" e, em seguida, verifica-a para pausar ou retomar o grupo de sincronização indexado na variável "sg".
