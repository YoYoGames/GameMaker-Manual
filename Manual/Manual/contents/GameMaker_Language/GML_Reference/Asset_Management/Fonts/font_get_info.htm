font_get_info

Esta função é utilizada para recuperar informações para o dado font. Você fornece um índice font (que pode ser um asset adicionado através do Navegador de Ativos ou um font adicionado em runtime) e a função retorna uma estrutura com as seguintes variáveis:

Estruturas de informação da fonte
Nome da variável Tipo de dados Descrição
ascenderOffset Real O deslocamento máximo da linha de base para o topo do font (em pixels)
size Real O tamanho aproximado do site font (em pixels)
spriteIndex Ativo Sprite O índice sprite para o font se foi criado a partir de um duende, caso contrário -1
texture Pointer -1 se o font foi criado a partir de um sprite, caso contrário o texture ID da fonte
name Cordão O nome da fonte
bold Booleano true se o font for ousado, caso contrário false (ver NOTA abaixo)
italic Booleano true se o font é itálico, caso contrário false (ver NOTA abaixo)
glyphs Fonte Estrutura do Glifo Um struct contendo informações para cada glifo no font (mais informações são dadas abaixo)

NOTA As variáveis bold e italic refletem apenas as configurações do usuário para o font, portanto podem não ser precisas para fonts que aparecem em negrito ou itálico por padrão.

Se o font fornecido não existir, a função retornará undefined.

Observe também que a alteração dos valores de qualquer uma destas variáveis, ou de quaisquer variáveis contidas no glyphs struct, não mudará a forma como o font é apresentado, portanto, esta informação deve ser considerada somente leitura.

Estrutura dos Glifos

A variável glyphs no retorno struct será um struct por si só, contendo informações para cada glifo incluído no font. Cada variável nesta estrutura será um nome de caráter de glifo, que conterá um struct contendo informações para aquele glifo. Você pode imaginar estes aninhados structs no seguinte formato (a partir do principal struct):

{
    glyphs:
    {
        A: {},
        B: {},
        C: {},
        // ...other glyphs
    },
    // ...other font info
}

Você pode usar o nome de um caractere do glifo para recuperar suas informações do struct, tal como no código a seguir:

var _font_info = font_get_info(Font1);

var _info_A = _font_info.glyphs[$ "A"]; // $ is a struct accessor

show_debug_message(_info_A);

O código acima recebe o glifo struct para o caractere "A" e o imprime no log de saída. Você também pode substituir o string ("A") por uma variável que contenha um string, o que lhe permitirá obter informações para qualquer glifo através dessa variável em runtime.

O struct retornado para um glifo conterá as seguintes variáveis:

OBSERVAÇÃO Em HTML5, os glifos struct estarão vazios para qualquer arquivo carregado fonts (como *.ttf ou *.otf), pois o runtime não terá informações sobre esses glifos.

Fonte Estrutura do Glifo
Nome da variável Tipo de dados Descrição
char real Se o font foi criado a partir de um sprite, este será o índice de imagem do glifo daquele sprite, caso contrário, será seu número de caracteres Unicode

Nota: Todas as variáveis abaixo disso não estarão presentes no struct se o font foi criado a partir de um duende
x real A posição X do glifo na página texture (em texels)
y real A posição em Y do glifo na página texture (em texels)
w real A largura do glifo na página texture (em texto)
h real A altura do glifo na página texture (em texels)
shift real O número de pixels a deslocar para a direita ao avançar para o próximo personagem (pode ser negativo para deslocar para a esquerda)
offset real O número de pixels para compensar horizontalmente a renderização deste glifo (pode ser positivo ou negativo)
kerning matriz Um conjunto de números inteiros contendo informações perspicazes em pares (ou grupos de 2). O primeiro inteiro em um par é o valor Unicode para um personagem, e o segundo inteiro é a quantidade a ser adicionada ao valor de deslocamento desse personagem (pode ser positivo ou negativo) se ele for precedido pelo caractere deste glifo.

NOTA Para o arquivo fonts carregado em runtime, os valores x e y para um glifo podem ser -1 se ainda não tiver sido apresentado, pois não terá posição na página texture utilizada para o cache.

Sintaxe:

font_get_info(font);

Argumento Tipo Descrição
font Fonte Ativo The font for which information should be retrieved.

Devoluções:

Font Info Struct (or undefined)

Exemplo:

var _info = font_get_info(my_font);

if (_info != undefined && _info.bold)
{
    image_xscale *= 2;
    image_yscale *= 2;
}

Este código recupera informações para a font armazenadas na variável my_font. Se o valor retornado não forundefined e o dado font for ousado, a escala da instância é duplicada.
