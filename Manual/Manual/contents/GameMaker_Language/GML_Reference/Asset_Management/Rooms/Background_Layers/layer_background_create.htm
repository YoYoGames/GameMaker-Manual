fundo_da_camada_criar

Com esta função, você pode atribuir um recurso sprite a uma camada a ser usada como fundo em seu projeto. Você fornece o ID da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho) e um índice sprite (que seria o nome do sprite como mostrado no Asset Browser), e será adicionado à camada. A função retorna o valor de identificação único para o elemento, que pode então ser usado em outras funções de camada para fundos.

Sintaxe:

layer_background_create(layer_id, sprite)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer to target (or the layer name as a string)
sprite Ativo Sprite The sprite index to be used

Devoluções:

Background Element ID

Exemplo:

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

O código acima cria uma nova camada e depois adiciona um novo elemento de fundo a ela, definindo um sprite para ser a imagem de fundo utilizada.
