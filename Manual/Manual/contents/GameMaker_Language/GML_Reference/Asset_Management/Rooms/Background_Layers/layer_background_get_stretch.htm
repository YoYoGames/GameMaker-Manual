Esta função pode ser usada para obter o estado esticado do elemento de fundo sprite. Você dá o ID do elemento de fundo (que você obtém quando cria um elemento de fundo usando layer_background_create() ou quando você usa a função layer_background_get_id()), e a função retornará ou true se o elemento sprite estiver atualmente esticado para caber no room, ou false se não estiver.

Sintaxe:

layer_background_get_stretch(background_element_id);

ArgumentoTipo Descrição
background_element_id The unique ID value of the background element to get the information from

Devoluções:

Exemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_stretch(back_id)
{
    layer_background_stretch(back_id, false);
}

O código acima obterá a identificação da camada chamada "Background_sky" e depois a usará para obter a identificação do elemento de fundo nessa camada. Este ID é então usado para verificar e ver se o elemento sprite será esticado para caber no room ou não e se for esticado, então esta propriedade é definida para false.
