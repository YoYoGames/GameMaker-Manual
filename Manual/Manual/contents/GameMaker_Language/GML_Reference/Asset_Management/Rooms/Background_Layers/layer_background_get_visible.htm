layer_background_get_visible

Esta função pode ser usada para obter o estado visível do elemento de fundo. Você dá o ID do elemento de fundo (que você obtém quando cria um elemento de fundo usando layer_background_create() ou quando você usa a função layer_background_get_id()), e a função retornará ou true se o elemento estiver visível no momento, ou false se não estiver. Note que este valor de retorno não é afetado se a camada em que o elemento está é visível ou não.

Sintaxe:

layer_background_get_visible(background_element_id);

ArgumentoTipo Descrição
background_element_id The unique ID value of the background element to get the information from

Devoluções:

Exemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_visible(back_id)
{
    layer_background_visible(back_id, false);
}

O código acima obterá a identificação da camada chamada "Background_sky" e depois a utilizará para obter a identificação do elemento de fundo nessa camada. Este ID é então usado para verificar a visibilidade do elemento e se ele for visível, então esta propriedade é definida para false.
