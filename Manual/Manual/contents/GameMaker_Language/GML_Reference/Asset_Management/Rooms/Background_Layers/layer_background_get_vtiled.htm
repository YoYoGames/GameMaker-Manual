fundo_camada_get_vtiled

Esta função pode ser usada para descobrir se o elemento de fundo está ou não ladrilhado verticalmente. Você dá a identificação do elemento de fundo (que você obtém quando cria um elemento de fundo usando layer_background_create() ou quando você usa a função layer_background_get_id()), e a função retornará ou true se o elemento estiver atualmente azulejado, ou false se não estiver.

Sintaxe:

layer_background_get_vtiled(background_element_id);

ArgumentoTipo Descrição
background_element_id The unique ID value of the background element to get the information from

Devoluções:

Exemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if !layer_background_get_vtiled(back_id)
{
    layer_background_vtiled(back_id, true);
}

O código acima obterá a identificação da camada chamada "Background_sky" e depois a usará para obter a identificação do elemento de fundo nessa camada. Este ID é então usado para verificar se o elemento está mosaico através do eixo vertical, e se não estiver, então está configurado para fazê-lo.
