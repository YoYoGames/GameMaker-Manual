mapa_get_móvel_largura_do_móvel

Usando esta função você pode recuperar a largura (em pixels) de cada célula tile do elemento do mapa tile. Você dá o ID do elemento do mapa tile (que você obtém ao criar um elemento do mapa tile usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), e a função retornará a tile set largura da célula.

Sintaxe:

tilemap_get_tile_width(tilemap_element_id);

ArgumentoTipo Descrição
tilemap_element_id The unique ID value of the tile map element to get the tile cell width of

Devoluções:

Exemplo:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
global.snap_x = tilemap_get_tile_width(map_id);
global.snap_y = tilemap_get_tile_height(map_id);

O código acima usa a ID do mapa tile recuperada para obter a largura e altura da célula tile e usá-las para definir duas variáveis globais.
