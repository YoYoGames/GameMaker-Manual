tilemap_get_global_mask

Esta função pode ser usada para recuperar o valor da máscara de bit para todos os mapas tile, retornando o valor atual da máscara ou -1 se houver um erro ou 0 se nenhuma máscara for especificada. Para mais informações sobre as máscaras de bit do mapa tile global, veja a função tilemap_set_global_mask().

Sintaxe:

tilemap_get_global_mask();

Devoluções:

(0 for no mask, -1 for an error, mask value otherwise)

Exemplo:

var mask = tilemap_get_global_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if mask != new_mask
{
    tilemap_set_global_mask(new_mask);
}

O código acima obtém o valor global da máscara associado a todos os mapas tile. Se não for o mesmo que o valor definido na variável "new_mask", então ele é definido para esse valor.
