layer_tilemap_create

Com esta função, você pode atribuir um recurso de conjunto de azulejos a uma camada a ser usada como um mapa tile em seu projeto. Você fornece o ID da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho) e então uma posição inicial (x, y) para adicionar o mapa tile ao room, o recurso tile set a ser usado, e então a largura e altura do mapa tile em células (ou seja: uma largura de 20 e uma altura de 10 irá criar um mapa tile com 200 células que é 20 células de largura e 10 células de altura), com o tamanho das próprias células sendo definido pelo tile set escolhido. Vale notar que não se pode colocar tiles em posições negativas dentro do mapa tile, portanto todo tiles deve ser colocado dentro dos espaços das células 0 à largura - 1, 0 à altura - 1.

Sintaxe:

layer_tilemap_create(layer_id, x, y, tileset, width, height)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer to target (or the layer name as a string)
x Real The x position of the tile map in the room
y Real The y position of the tile map in the room
tileset Conjunto de azulejos Ativo The Tile Set asset to be used
width Real The width tile map (in cells)
height Real The height of the tile map (in cells)

Devoluções:

Tile Map Element ID

Exemplo:

global.back_layer = layer_create(10000);
global.back_tiles = layer_tilemap_create(global.back_layer, 0, 0, tmap_Trees, 16, 32);

O código acima cria uma nova camada e depois adiciona um novo elemento do mapa tile, definindo a posição do mapa tile para (0,0), assim como o tile set a ser usado e a largura e altura do mapa tile.
