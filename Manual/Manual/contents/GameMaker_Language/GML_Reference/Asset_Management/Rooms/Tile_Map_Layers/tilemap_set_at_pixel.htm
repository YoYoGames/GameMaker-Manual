mapa_do_cartão_de_fixos_na_pixel

Esta função pode ser usada para definir uma célula dentro do elemento do mapa tile na camada para um novo tile usando a posição real do tile para mudar dentro do room. Você dá o ID do elemento do mapa tile (que você obtém quando cria um elemento do mapa tile usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), e depois os dados tile a definir, bem como a posição dentro do room. Você pode recuperar os dados tile usando a função tilemap_get_at_pixel() e depois usar as funções tile_ para mudá-lo antes de defini-lo novamente usando esta função. A função retornará true se o tile foi configurado com sucesso e false se houve um problema e não foi configurado.

Sintaxe:

tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y)

Argumento Tipo Descrição
tilemap_element_id Identificação do Elemento do Mapa de Telha The unique ID value of the tile map element to change
tiledata Dados do Ladrilho (Real) The tile set data to set
xcell Real The x position (in the room)
ycell Real The y position (in the room)

Devoluções:

Boolean

Exemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 4, 4);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 4, 4);

O código acima obtém a identificação para o mapa tile na camada "Tiles_Sky" e depois a utiliza para obter os dados do tile na posição (4, 4). Estes dados do tile são então virados antes de serem usados para colocar novamente o tile no mapa tile.
