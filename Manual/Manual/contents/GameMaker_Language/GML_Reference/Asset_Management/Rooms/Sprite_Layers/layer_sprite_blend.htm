layer_sprite_blend

Esta função controla a mistura (ou "tinting") do sprite sprite e o valor padrão é -1 (que representa a constante c_white, que também pode ser usada). Qualquer outro valor (incluindo constantes de cor internas como c_red, ou c_aqua) irá misturar a cor especificada com o original sprite. Você dá o ID do elemento sprite (que você recebe quando cria um elemento sprite usando layer_sprite_create() ou quando você usa a função layer_sprite_get_id()), e então definir a cor da mistura a ser utilizada. Abaixo você pode ver um exemplo de um sprite que foi misturado com cores diferentes:

Favor observar que você deve tentar limitar a mistura na plataforma HTML5 (a menos que utilize WebGL), pois cada mistura sprite tem de ser colocada em cache separadamente e, portanto, ter muitas misturas sprites pode afetar negativamente o desempenho (você também pode definir o tamanho cache usando a função sprite_set_cache_size()).

Sintaxe:

layer_sprite_blend(sprite_element_id, blend);

ArgumentoTipo Descrição
sprite_element_id The unique ID value of the sprite element to change
blend The colour to blend with the sprite sprite (default is c_white)

Devoluções:

Exemplo:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_blend(spr_id, c_gray);

O código acima recebe o valor de identificação do sprite chamado "Clouds" atribuído à camada "Asset_sky" e depois o tinge com uma cor.
