layer_sequence_play

Com esta função você pode iniciar a reprodução do dado sequence. Você fornece o ID do elemento sequence conforme retornado por layer_sequence_create() ou por uma das funções de elementos de camada e a função desempenhará o sequence, que você pode então pausar, se necessário, usando a função layer_sequence_pause().

IMPORTANTE! Se seu sequence tem alguma instância, estas instâncias não devem alterar suas variáveis image_xscale / image_yscale / image_angle / x / y, pois elas serão sobrescritas quando o sequence atualizar cada passo após começar a ser reproduzido.

Sintaxe:

layer_sequence_play(sequence_element_id)

Argumento Tipo Descrição
sequence_element_id Identificação do Elemento de Sequência The unique ID value of the sequence element to target

Devoluções:

N/A

Exemplo:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    var a = layer_get_all_elements(layer);
    for (var i = 0; i < array_length(a); i++;)
    {
        if layer_get_element_type(a[i]) == layerelementtype_sequence
        {
            if global.Pause
            {
                layer_sequence_pause(a[i]);
            }
            else
            {
                layer_sequence_play(a[i]);
            }
        }
    }
}

O código acima verifica se o jogo foi pausado ou não quando uma tecla é pressionada. Se o jogo for pausado, então ele loops através de todos os elementos sequence na camada atual (a camada da instância de chamada) e pausa sua reprodução, e se o jogo não for pausado, então o loop começará sua reprodução novamente.
