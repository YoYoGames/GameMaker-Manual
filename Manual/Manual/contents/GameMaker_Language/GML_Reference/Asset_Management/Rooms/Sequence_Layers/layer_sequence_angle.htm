layer_sequence_angle

Com esta função você pode definir o ângulo de rotação para o elemento sequence dado. Você fornece o ID do elemento sequence conforme retornado por layer_sequence_create() ou por uma das funções dos elementos da camada junto com o novo ângulo a ser definido e o sequence será girado sobre sua origem para a nova posição. Os ângulos em GameMaker são calculados com 0º para a direita, e vão no sentido anti-horário - assim 90º é para cima, 180º para a esquerda e 270º para baixo - e o ângulo padrão para um sequence seria de 0º.

x Observe que você não deve alterar as variáveis image_xscale image_yscale image_angle y para qualquer instância em um sequence que utilize esta função, pois elas serão sobrescritas assim que o sequence for atualizado após a chamada à função.

Sintaxe:

layer_sequence_angle(sequence_element_id, angle)

Argumento Tipo Descrição
sequence_element_id Identificação do Elemento de Sequência The unique ID value of the sequence element to target
angle Real The new angle to rotate the sequence element to

Devoluções:

N/A

Exemplo:

if current_angle < 90
{
    current_angle += 1;
    layer_sequence_angle(my_seq, current_angle);
}

O código acima verifica o valor mantido na variável current_angle e, se for inferior a 90, adiciona o valor para definir o ângulo do elemento sequence referenciado na variável my_seq.
