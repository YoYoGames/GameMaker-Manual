layer_sequence_get_headpos

Com esta função você fornece o ID do elemento sequence - como retornado por layer_sequence_create() ou por uma das funções dos elementos da camada - e ele retornará a posição atual do playhead (a moldura atual em que o playhead está).

Sintaxe:

layer_sequence_get_headpos(sequence_element_id)

ArgumentoTipo Descrição
sequence_element_id The unique ID value of the sequence element to target

Devoluções:

Exemplo:

if layer_sequence_get_headpos(title_sequence) != 0
{
    layer_sequence_headpos(title_sequence, 0);
}

O código acima verifica a posição atual da cabeça de jogo do elemento ID sequence armazenado na variável "title_sequence", e se não estiver definido como 0, está definido para este valor.
