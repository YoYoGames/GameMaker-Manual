room_get_camera

Com esta função, você pode obter a identificação única do índice da câmera atribuída a uma visão específica em um room diferente da atual. Você dá o room para usar, a porta de visualização para usar (de 0 a 7) e a função retornará um índice de câmera.

Sintaxe:

room_get_camera(rm, vind);

Argumento Tipo Descrição
rm Ativos da sala The index of the room to get the view camera of
vind Real The index of the view port to get the camera of

Devoluções:

Camera ID

Exemplo:

var cam = room_get_camera(rm_Game, 0);
if cam != global.MainCam
{
    room_set_camera(rm_Game, 0, global.MainCam);
}

O código acima atribui uma câmera em uma recém-criada room para visualizar a porta [0].
