layer_get_target_room

Esta função devolverá o atual room sendo alvo das funções de camada. Ver layer_set_target_room() para mais informações.

Sintaxe:

layer_get_target_room()

Devoluções:

Room Asset

Exemplo:

if layer_get_target_room() != room
{
    layer_reset_target_room();
}

O código acima verifica o alvo atual room e se não for o atual room então o alvo da camada room é redefinido.
