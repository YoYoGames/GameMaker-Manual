layer_get_hspeed

Você pode usar esta função para recuperar a velocidade horizontal (em pixels por frame de jogo) da camada dentro do escopo atualmente escopado room. Você fornece a identificação da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho) e a função retorna um número real para a velocidade horizontal, onde um valor positivo é para a direita e um valor negativo para a esquerda. O valor padrão é 0 (a menos que definido no editor room ).

Sintaxe:

layer_get_hspeed(layer_id)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer to get the horizontal speed from

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

O código acima verifica as velocidades horizontal e vertical da camada dada e, se ambas não estiverem ambas ajustadas a 0, então é ajustado para 0.
