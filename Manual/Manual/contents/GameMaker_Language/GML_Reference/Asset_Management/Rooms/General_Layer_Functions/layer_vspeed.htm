layer_vspeed

Você pode usar esta função para definir a velocidade vertical (em pixels por quadro de jogo) da camada dentro do escopo atual room. Você fornece a identificação da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho) e o valor de velocidade a ser definido, onde um valor positivo é para baixo e um valor negativo para cima.

Sintaxe:

layer_vspeed(layer_id, vspd)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer to set the vertical speed of
vspd Real The vertical speed (in pixels per game frame) to set

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

O código acima verifica as velocidades horizontal e vertical da camada dada e, se ambas não estiverem ambas ajustadas a 0, então é ajustado para 0.
