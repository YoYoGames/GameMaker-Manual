path_get_closed

Esta função pode ser usada para retornar se o path está marcado como fechado (verdadeiro) ou aberto (falso), ou seja, se o path loops ou se tem um início e fim definitivo.

Sintaxe:

path_get_closed(index);

Argumento Tipo Descrição
index Ativo do caminho The index of the path to check.

Devoluções:

Boolean

Exemplo:

state = path_get_closed(pth_Patrol);

Isto definirá "estado" para true ou false dependendo do estado fechado do path indexado em "pth_Patrol".
