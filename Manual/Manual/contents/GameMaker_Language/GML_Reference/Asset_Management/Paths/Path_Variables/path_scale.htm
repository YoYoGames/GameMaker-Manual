escala_do_caminho

Este valor pode ser usado para obter ou para definir a escala do path atualmente atribuído para a instância (como definido pela função path_start()) com um valor padrão de 1. Este é um valor escalar, portanto 1 é uma escala de 1:1, enquanto que fixá-la em 2, por exemplo, será o dobro da escala e fixá-la em 0,5 seria reduzir pela metade a escala.

Sintaxe:

path_scale;

Devoluções:

Exemplo:

path_scale = 1 + random(2);

O código acima definirá a escala do caminho atual para um tamanho aleatório entre 1 (seu tamanho original) e 3 (3 vezes seu tamanho original).
