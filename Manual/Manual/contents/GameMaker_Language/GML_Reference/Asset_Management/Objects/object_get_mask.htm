object_get_mask

Esta função lhe dirá se o object que você está verificando tem ou não um índice de máscara, e se tiver, então retornará o índice dessa máscara (que é um sprite asset), ou -1 se não tiver. Note que esta não é uma função de instância! Você pode ter um object sem máscara enquanto uma instância desse mesmo object pode ter uma e vice-versa, ou eles podem até mesmo ter máscaras diferentes. Você pode definir um índice de máscaras de instâncias individuais usando a função mask_index variável de instância.

Sintaxe:

object_get_mask(obj);

ArgumentoTipo Descrição
obj The index of the object to check

Devoluções:

Sprite Asset

Exemplo:

if (mask_index != object_get_mask(object_index))
{
    mask_index = object_get_mask(object_index);
}

O exemplo acima verificará o índice da máscara da instância em relação à máscara do índice do objeto_da instância. Se eles não forem os mesmos, então atribuirá a mesma máscara que o índice object tem à instância.
