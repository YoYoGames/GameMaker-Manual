object_get_visible

Esta função lhe dirá se o object que você está verificando foi assinalado como "visível" (realiza seu evento de sorteio) ou não (não realiza seu evento de sorteio). Note que esta não é uma função de instância! Portanto, você pode ter um object visível e uma instância invisível do mesmo object e vice-versa. Você pode definir a visibilidade de uma instância individual usando a função visible variável de instância.

Sintaxe:

object_get_visible(obj);

ArgumentoTipo Descrição
obj The index of the object to check.

Devoluções:

Exemplo:

if (!visible) && (object_get_visible(object_index))
{
    visible = true;
}

O código acima verificará a instância em execução para ver se ela está visível ou não, bem como o índice object da instância para ver se ela está sinalizada como visível ou não. Se a instância ainda não estiver visível, o índice object será sinalizado como em, ele definirá "visível" como verdadeiro para aquela instância.
