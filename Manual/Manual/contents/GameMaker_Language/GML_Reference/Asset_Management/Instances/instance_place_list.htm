lista_de_lugares_da_classe

Com esta função você pode verificar uma posição para uma colisão com todas as instâncias de uma object usando a máscara de colisão da instância que executa o código para a verificação. Quando você usa esta função você está efetivamente pedindo ao GameMaker para mover a instância para a nova posição, verificar se há uma colisão e, em seguida, voltar e dizer-lhe se uma colisão foi encontrada ou não. Isto funcionará para colisões precisas, mas somente se tanto a instância e as instâncias da object que está sendo verificada tiverem máscaras de colisão precisas selecionadas, caso contrário somente serão aplicadas colisões de caixas delimitadoras.

Você também fornece uma lista DS, portanto o id valores de quaisquer instâncias que estejam colidindo com a instância de chamada serão adicionados ao final da lista dada. Você pode limpar a lista antes de chamar esta função para que ela contenha apenas os resultados desta chamada de função. Você também tem a opção de ordenar a lista com base na distância da origem da instância fazendo a verificação até a origem das instâncias que se encontram em colisão.

Note que a função também aceita a palavra-chave especial allNeste caso, todas as instâncias encontradas em colisão serão listadas. A função retorna o número de instâncias encontradas, ou 0 se nenhuma for encontrada.

Sintaxe:

instance_place_list(x, y, obj, list, ordered);

Argumento Tipo Descrição
x Real The x position to check for instances.
y Real The y position to check for instances.
obj Objeto Ativo ou ID da instância The object to check for instances of.
list DS List ID The DS list to use to store the IDs of colliding instances.
ordered Booleano Whether the list should be ordered by distance (true) ou não (false).

Devoluções:

Real (The number of instances found to be in collision)

Exemplo:

var _list = ds_list_create();
var _num = instance_place_list(x, y, obj_Enemy, _list, false);

if _num > 0
{
    for (var i = 0; i < _num; ++i;)
    {
        instance_destroy(_list[| i]);
    }
}

ds_list_destroy(_list);

O código acima verificará se há uma colisão com todas as instâncias encontradas na posição de instâncias de chamada. Estas serão adicionadas a uma lista DS, que é então ligada em loop para destruir cada uma das instâncias em colisão.
