instance_id_get

Com esta função você pode obter o valor de identificação único de qualquer instância a partir da lista de instâncias atualmente ativa. Você fornece o índice na lista de instâncias para obter o ID e a função retornará o valor para armazenamento em uma variável.

Sintaxe:

instance_id_get(index);

Argumento Tipo Descrição
index Real The index within the instance list from 0 para a contagem de instâncias - 1.

Devoluções:

Instance ID

Exemplo:

for (var i = 0; i < instance_count; ++i;)
{
    var temp_id = instance_id_get(i);
    with (temp_id)
    {
        speed += 0.1;
    }
}

O código acima será loop através de todas as instâncias dentro do room e adicionará 0,1 à sua velocidade.
