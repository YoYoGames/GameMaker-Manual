direção

Todas as instâncias no GameMaker têm certas propriedades "embutidas" que você pode usar e definir para governar o seu aspecto e comportamento. Direction é uma dessas propriedades e pode ser usada para definir a direção do movimento do instância quando a instância tem um speed Note que as direções no GameMaker são normalmente calculadas como 0° para a direita, 90° para cima, 180° para a esquerda e 270° para baixo, e que o gravity e gravity_direction podem modificar o valor direction quando são utilizados em seus jogos.

Sintaxe:

direction;

Devoluções:

(single precision floating point value)

Exemplo:

if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

O código acima mudará a direção de movimento da instância com base em qual tecla (esquerda ou direita) é pressionada.
