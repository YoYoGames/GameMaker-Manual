xstart

Esta variável armazena a posição inicial x da instância quando ela é criada pela primeira vez no room. Esta não é uma variável somente de leitura e pode ser definida tão bem quanto de leitura.

Sintaxe:

xstart;

Devoluções:

(single precision floating point value)

Exemplo:

if place_meeting(x, y, obj_spike)
{
    score -= 100;
    x = xstart;
    y = ystart;
}

O código acima verificará uma colisão com uma instância de "obj_spike" e se houver uma, ela deduz 100 da pontuação e move a instância de volta para sua posição inicial.
