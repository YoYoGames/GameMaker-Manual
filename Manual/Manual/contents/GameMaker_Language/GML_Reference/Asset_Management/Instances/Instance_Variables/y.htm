y

O valor y de uma instância é a posição vertical na atual room, medida em pixels. Este valor pode ser 0, positivo ou negativo, onde 0 é o topo do room e se movendo para baixo aumenta y, se movendo para cima diminui y (um valor negativo para y significa que a instância foi para fora do topo do room). Você pode até mesmo dar um valor real como 12.345, mas como você não pode obter .345 de um pixel, a instância pode não parecer ter se movido, embora estes valores sejam mantidos e usados internamente por GameMaker e são perfeitamente válidos.

Ao definir os valores y (e x) você pode fazer o object saltar em torno do room para a posição de sua escolha, ou você pode adicionar e subtrair a eles por quantidades menores para dar a ilusão de movimento sem realmente usar o embutido speed e direction funções. Isto deve ser levado em consideração ao fazer seus jogos, pois muitas vezes um tipo de movimento se presta a um estilo particular de jogo mais do que outro.

Sintaxe:

y;

Devoluções:

(single precision floating point value)

Exemplo:

if keyboard_check(vk_down) y += 5;
if keyboard_check(vk_up) y -= 5;

O código acima usará as teclas de seta para adicionar ou subtrair à posição y e assim criar a ilusão de movimento.
