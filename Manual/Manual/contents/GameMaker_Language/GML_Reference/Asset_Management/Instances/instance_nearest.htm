exemplo_nearest

Esta função verificará todas as instâncias do dado object para ver qual é a mais próxima ao ponto x/y dado. Todas as verificações serão da posição x/y dada até a origem (a posição x/y) das instâncias do object especificado. Caso não existam instâncias do object, a função retornará a palavra-chave noonemas, se houver casos, devolverá o id da instância encontrada. Observe que se a instância executando o código tiver o mesmo índice object que o object sendo verificado, então ele será incluído na verificação (isto inclui verificações para os pais objects se a instância chamadora for também um filho dos pais).

Sintaxe:

instance_nearest(x, y, obj);

ArgumentoTipo Descrição
x The x position to check from.
y The y position to check from.
obj The object to check for instances of.

Devoluções:

Instance ID or noone

Exemplo:

var inst, xx;
xx = x;
x -= 10000;
inst = instance_nearest(xx, y, object_index);
if inst != id
{
    draw_line(x, y, inst.x, inst.y);
}
x += 10000;

O código acima move a instância atual 10000 pixels e depois verifica sua posição anterior para encontrar a instância mais próxima do mesmo tipo object. Se essa instância for ela mesma, nada mais fará do que voltar à sua posição original, mas se a instância encontrada for diferente, ela traçará uma linha entre as duas.
