conjunto_esqueleto_de_squema_cores

Com esta função você pode definir uma cor e alfa da ranhura de fixação para que todos sprites anexados a ela sejam desenhados com estes valores de mistura. Tenha em mente que a instância sprite pode ter uma cor de mistura e um ajuste alfa (image_blend e image_alpha), assim como o anexo (ver a função skeleton_attachment_create_colour()), e assim a cor final e alfa que o anexo designado sprite para o slot terá será um composto de todos esses valores.

Sintaxe:

skeleton_slot_colour_set(slot, colour, alpha);

Argumento Tipo Descrição
slot Cordão The slot name to set, a string
colour Cor The colour to set, either an integer, a constant, or a hex value.
alpha Real The alpha to set from 0 to 1.

Devoluções:

N/A

Exemplo:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if !ds_list_empty(slot_list)
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_colour_get(slot_list[| i]) != c_white
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

O código acima cria uma lista DS e depois a preenche com os nomes dos slots encontrados na posição do mouse no room. Em seguida, loops através da lista de slots e repõe a cor dos slots encontrados para branco com um alfa de 1 se a cor ainda não for branca.
