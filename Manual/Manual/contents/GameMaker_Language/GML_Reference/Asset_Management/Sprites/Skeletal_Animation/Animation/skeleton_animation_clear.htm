esqueleto_animação_clara

Esta função limpará a pista de animação especificada de todas as animações, pronta para ser reatribuída.

Sintaxe:

skeleton_animation_clear(track);

Argumento Tipo Descrição
track Real The animation track to clear.

Devoluções:

N/A

Exemplo:

if mouse_check_button(mb_right)
{
    skeleton_animation_clear(1);
}

O código acima limpará a faixa de animação 1 se o botão direito do mouse for pressionado.
