tempo_esqueleto_de_sorteio

Esta função desenhará a animação dada usando a pele dada em um momento específico da animação. O valor do tempo deve estar entre 0 (o início) e a duração final da animação, que você pode encontrar usando a função skeleton_animation_get_duration(). Você pode definir o valor do tempo para valores superiores à duração total da animação e a animação loop voltará ao início, mas corre o risco de perder a precisão do ponto flutuante à medida que o tempo acumulado se torna maior.

Sintaxe:

draw_skeleton_time(sprite, animname, skinname, time, x, y, xscale, yscale, rot, colour)

Argumento Tipo Descrição
sprite Ativo Sprite The index of the sprite to draw.
animname Cordão The name of the animation to get the frame from (a string).
skinname Cordão The name of the skin to use for drawing (a string).
time Real The time-frame to draw the animation at (from 0 to the end duration, in seconds).
x Real The x coordinate of where to draw the sprite.
y Real The y coordinate of where to draw the sprite.
xscale Real The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Real The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Real The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour Cor The colour with which to blend the sprite.

Devoluções:

N/A

Exemplo:

var time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

O código acima desenhará a animação esquelética dada sprite usando o tempo delta para definir o frame que está sendo desenhado.
