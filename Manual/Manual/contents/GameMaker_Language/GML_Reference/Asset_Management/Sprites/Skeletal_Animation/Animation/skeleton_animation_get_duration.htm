esqueleto_animação_get_duração

Esta função retornará o tempo necessário para que o conjunto de animação em questão seja executado antes do looping de volta ao início. O valor de retorno é em segundos.

Sintaxe:

skeleton_animation_get_duration(animname);

ArgumentoTipo Descrição
animname The name (a string) of the animation set to use.

Devoluções:

Exemplo1:

time += delta_time / 1000000;
var duration = skeleton_animation_get_duration(skeleton_animation_get());
var frame = floor((image_number * (mTime / duration)) + 0.5) % image_number;
image_index = frame;
draw_self();

O código acima definirá o image_index como o valor correto para a animação esquelética atualmente atribuída sprite.

Exemplo2:

time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

O código acima desenhará a animação esquelética dada sprite usando o tempo delta para definir o frame que está sendo desenhado.
