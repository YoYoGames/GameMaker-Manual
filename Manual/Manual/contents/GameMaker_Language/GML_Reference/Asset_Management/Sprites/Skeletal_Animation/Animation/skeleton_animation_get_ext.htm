esqueleto_animação_get_ext

Uma única animação esquelética sprite pode ter vários conjuntos de animação, e a estes conjuntos podem ser atribuídas diferentes pistas para que você possa "misturar e combinar" conjuntos de animação. Essa função retornará o nome do conjunto de animação atualmente usado pelo número da faixa dada (como definido pela função skeleton_animation_set_ext).

Sintaxe:

skeleton_animation_get_ext(track);

Argumento Tipo Descrição
track Real The track number to get the animation set name of.

Devoluções:

String

Exemplo:

if skeleton_animation_get_ext(1) != "bodyfight"
{
    skeleton_animation_set_ext("bodyfight", 1);
}

O código acima mudará o conjunto de animação sendo usado pela faixa 1 para "bodyfight" se ainda não o for.
