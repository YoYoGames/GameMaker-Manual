bbox_top

Esta variável lida somente retorna a posição dentro do room (ao longo do eixo y) do topo da caixa de delimitação da instância, onde a caixa de delimitação é definida pela largura e altura máximas da máscara para a instância (conforme definido pelo sprite_index ou pelo mask_index). Mesmo quando um sprite tem uma máscara de colisão precisa, a caixa de delimitação existe e é usada para certas coisas, e assim você pode usar esta variável para encontrá-la. Observe que quando a instância não tem sprite atribuído, o valor retornado será o mesmo que a posição Y da instância.

Sintaxe:

bbox_top;

Devoluções:

(integer)

Exemplo:

if bbox_top < 0
{
    y = sprite_yoffset;
}

O código acima verifica se a caixa de delimitação da instância está fora do room e se é ela que coloca a instância em uma nova posição.
