sprite_index

Esta variável retorna o índice do atual sprite para a instância, ou -1 se a instância não tiver nenhum sprite associado a ela. Você pode alterá-la para dar à instância um sprite diferente, dando-lhe o nome de um sprite da árvore de recursos ou usando uma variável que tenha um sprite indexado externamente nela. A alteração do sprite não altera o índice do quadro atualmente visível, portanto, se você alterar o sprite no quadro número 3, o novo sprite será desenhado com esse quadro visível (assumindo que ele tenha o mesmo número de quadros).

Sintaxe:

sprite_index;

Devoluções:

Sprite Asset

Exemplo:

with (obj_Check)
{
    if !collision_line(x, y, other.x, other.y, obj_Wall, false, true)
    {
        sprite_index = spr_spotted;
    }
    else
    {
        sprite_index = spr_clear;
    }
}

O código acima irá loop através de todas as instâncias de "obj_Check" verificando se há uma linha de colisão entre elas e a instância que executa o código. O sprite dessas instâncias será alterado de acordo com o valor de retorno (verdadeiro ou falso) para a linha de colisão.
