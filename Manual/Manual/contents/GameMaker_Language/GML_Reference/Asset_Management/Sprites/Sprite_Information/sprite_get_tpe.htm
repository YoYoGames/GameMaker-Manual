sprite_get_tpe

Esta função retornará um valor especial relacionado à posição do dado sprite e subimagem na página texture utilizada por seu jogo. Desta forma, você pode passar o valor para extensions para uso fora do jogo compilado, contornando efetivamente as funções de desenho do GameMaker e permitindo que o sprite seja usado no conteúdo DOM que pode então ser desenhado em qualquer lugar dentro da janela que contém a tela do jogo. Esta função é de particular interesse para aqueles que desejam criar botões e outras mídias interativas fora do elemento de tela do GameMaker em sua página principal, utilizando a função clickable_add().

NOTA: Esta função é apenas para HTML5.

Sintaxe:

sprite_get_tpe(sprite, index);

Argumento Tipo Descrição
sprite Ativo Sprite The index of the sprite to find the texture page entry of.
index Real The image index of the sprite.

Devoluções

Texture Page Entry

Exemplo:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

O código acima cria um botão DOM na posição (32, 32) da página em que a tela do jogo está rodando. O botão usa o sprite referido da página texture como "spr_MS_Home" e quando clicado o botão abrirá uma nova janela para a URL especificada e com as propriedades definidas para aquela janela.
