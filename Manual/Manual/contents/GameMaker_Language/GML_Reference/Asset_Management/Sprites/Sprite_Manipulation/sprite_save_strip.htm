sprite_save_strip

Esta função criará uma imagem de faixa de todas as sub-imagens em um sprite, salvando-a em disco com o nome de arquivo especificado. O sprite deve ter sido adicionado em runtime (você não pode salvar sprites adicionado através do IDE) e o arquivo deve ser salvo com um *.png extension.

NOTA: Dependendo da plataforma de destino escolhida, você está limitado quanto ao local de onde pode salvar e carregar arquivos. Consulte Manuseio de arquivos para mais informações.

Sintaxe:

sprite_save_strip(ind, filename);

Argumento Tipo Descrição
ind Ativo Sprite The index of the sprite to save.
filename Cordão The filename for the saved sprite strip.

Devoluções

N/A

Exemplo:

var spr, i;
spr = sprite_create_from_surface(0, 0, 32, 32, true, true, 16, 16);

for (i = 1; i < 8; i +=1)
{
    sprite_add_from_surface(spr, i, 0, 32, 32, true, true, 16, 16);
}

sprite_save_strip(spr, "Custom_Player_Sprite.png");
sprite_delete(spr);

O código acima cria um sprite da superfície indexada na variável "surf", atribuindo seu índice à variável local "spr_Custom", e então usa um for loop para se mover através da superfície e capturar várias seções que são adicionadas no sprite como sub-imagens. este novo sprite é então salvo como uma tira de png antes de ser removido da memória.
