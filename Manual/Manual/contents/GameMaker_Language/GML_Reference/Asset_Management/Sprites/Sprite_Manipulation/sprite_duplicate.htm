sprite_duplicate

Esta função retornará o índice de um sprite recentemente criado que é uma duplicata (cópia) de uma entrada como o argumento "índice". Se a função falhar, -1 é retornado. Esta função deve ser usada para copiar qualquer sprites do original assets antes que qualquer transformação possa ser feita neles. Por exemplo, se você desejar mudar a caixa delimitadora para um sprite, ou definir seu alfa a partir de outro sprite, você deve primeiro duplicá-la, depois realizar a operação na duplicata sprite e utilizá-la. Uma sprite duplicada será colocada em sua própria página única texture quando criada, o que significa que a duplicação de múltiplas sprites criará múltiplas páginas texture e terá um impacto no desempenho, portanto, use esta função somente quando necessário.

NOTA: Ao duplicar um sprite no GameMaker você deve se lembrar de removê-lo novamente (com sprite_delete()) quando não for mais necessário, caso contrário existe o risco de um vazamento de memória que irá retardar e, eventualmente, quebrar seu jogo.

Sintaxe:

sprite_duplicate(index);

ArgumentoTipo Descrição
index The index of the sprite to duplicate.

Devoluções

Sprite Asset or -1

Exemplo:

new_spr = sprite_duplicate(sprite_index)

O código acima duplica o sprite que está sendo usado atualmente como o índice sprite da instância e armazena o índice para este novo sprite na variável "new_spr".
