linha do tempo_velocidade

Normalmente, em cada etapa, a posição na linha de tempo é aumentada em 1, porém é possível alterar este valor, definindo esta variável para um valor diferente. Você pode usar números reais (como 0,5, ou 2,4 por exemplo) e se o valor for maior que um, vários momentos podem acontecer dentro da mesma etapa de tempo (todos eles serão executados na mesma ordem definida para a linha de tempo, portanto nenhuma ação será pulada).

Sintaxe:

timeline_speed;

Devoluções:

(single precision floating point value)

Exemplo:

if !timeline_running
{
    timeline_running = true;
    timeline_speed = 0.5;
}

O código acima verificará e verá se a instância está rodando sua linha de tempo associada e, se não estiver, iniciará a linha de tempo rodando a meia velocidade.
