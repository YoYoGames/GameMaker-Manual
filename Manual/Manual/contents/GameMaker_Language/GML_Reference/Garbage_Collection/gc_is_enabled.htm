gc_is_enabled

Com esta função, você pode verificar se o coletor de lixo está ou não habilitado. A função retornará true se estiver ativada ou false caso contrário.

Sintaxe:

gc_is_enabled();

Devoluções:

Exemplo:

if (!gc_is_enabled())
{
    gc_enable(true);
}

O código acima verifica se o coletor de lixo está habilitado e se não está habilitado.
