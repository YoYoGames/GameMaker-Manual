network_resolve

Esta função retornará o endereço IP do URL dado.

Sintaxe:

network_resolve(url);

ArgumentoTipo Descrição
url The URL to get the IP of (a string).

Devoluções:

Exemplo:

game_ip = network_resolve("www.macsweeneygames.com");

O código acima retornará o endereço IP do URL dado e o armazenará na variável "game_ip".
