network_create_socket_ext

Esta função é utilizada para criar um novo soquete de cliente para que seu jogo se comunique através da rede. Você deve definir o tipo de socket (veja a lista de constantes abaixo) e dar uma porta para usar, e a função retornará uma identificação única que deve ser usada em todas as outras chamadas de função para aquele socket, ou um valor inferior a 0 se a conexão falhar.

Sintaxe:

network_create_socket_ext(protocol, port);

Argumento Tipo Descrição
protocol Tipo de soquete Constante The network protocol to use
port Real The port to use

Devoluções:

Network Socket ID

Exemplo:

client = network_create_socket_ext(network_socket_udp, 6510);

O código acima irá criar um novo soquete UDP na porta 6510.
