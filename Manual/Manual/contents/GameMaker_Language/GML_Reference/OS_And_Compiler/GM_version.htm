GM_versão

Esta constante mantém o número da versão, conforme definido nas Opções de Jogo para cada plataforma alvo. O valor é armazenado como um string.

Sintaxe:

GM_version;

Devoluções:

Exemplo:

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);

O código acima pega a data GM e as constantes da versão e as desenha para a tela.
