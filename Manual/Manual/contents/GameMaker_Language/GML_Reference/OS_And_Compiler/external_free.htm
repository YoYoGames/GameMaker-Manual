externo_livre

Esta função libera a memória associada com o dll ou dylib com o nome dado. Isto deve ser feito sempre que o arquivo em questão não for mais necessário no jogo, normalmente (por exemplo) em um evento Game End.

Sintaxe:

external_free(id);

Argumento Tipo Descrição
id Função externa The name of the dll ou dylib que você quer liberar

Devoluções:

N/A

Exemplo:

external_free("MyDLL.dll");

O código de exemplo acima irá liberar a memória associada à dll dada.
