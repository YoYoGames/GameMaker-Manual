GM_projeto_nome_do_filme

Esta constante contém o path completo e o nome do arquivo do projeto YYP.

Sintaxe:

GM_project_filename;

Devoluções:

String

Exemplo:

draw_text(32, 32, GM_project_filename);

O código acima desenha o valor da constante do nome do arquivo do projeto GM.
