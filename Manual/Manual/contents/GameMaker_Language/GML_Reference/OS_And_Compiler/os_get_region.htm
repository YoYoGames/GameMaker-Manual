os_get_region

Esta função retorna um string com as duas ou três letras Código Regional para o SO que está rodando o jogo, conforme definido pela norma ISO3166-1. Se a informação não estiver disponível, ela conterá simplesmente um string "" vazio.

NOTA: Este não é o código regional de localização que é devolvido, mas o código do idioma regional do sistema operacional.

Sintaxe:

os_get_region()

Devoluções:

Exemplo:

switch (os_get_language())
{
    case "zh":
        var region = os_get_region();
        if (region == "HK" || region == "MO" || region == "TW")
        {
            ini_open("chinese_traditional.ini");
        }
        else
        {
            ini_open("chinese_simplified.ini");
        }
    break;

    case "fr":
        ini_open("french.ini");
    break;

    case "it":
        ini_open("italian.ini");
    break;

    default:
        ini_open("english.ini");
    break;
}

O código acima verifica o idioma do SO e se for chinês, então verifica a região do SO, abrindo um arquivo *.ini diferente, dependendo dos valores retornados.
