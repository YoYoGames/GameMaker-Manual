display_set_ui_visibility

Esta função pode ser usada para mostrar ou ocultar a interface do sistema apenas no Android. A função requer que você forneça uma ou mais bandeiras do sistema como um valor inteiro. Ao usar mais de um único flag, estes precisam ser fundidos usando o bitwise "ou", como mostrado no exemplo abaixo. Você pode encontrar uma lista do sistema Android flags aqui.

Sintaxe:

display_set_ui_visibility(flags);

Argumento Tipo bandeiras
flags Real The Android SYSTEM flags to use (as an integer value).

Devoluções:

N/A

Exemplo:

var flags = 1024|4096;
display_set_ui_visibility(flags);

O código acima usará o sistema Android flags "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" e "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN" para definir a visibilidade da tela.
