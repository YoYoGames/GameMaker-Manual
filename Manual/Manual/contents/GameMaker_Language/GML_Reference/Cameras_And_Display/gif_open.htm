gif_open

Com esta função você pode criar uma imagem em formato GIF vazia, pronta para ter dados adicionados a ela. Quando você chama a função, você precisa especificar a largura e a altura do GIF (em pixels, e não há limite superior de tamanho, exceto para a memória disponível), e a função retornará o valor de identificação único usado para identificar o presente em funções subseqüentes, ou retornará -1 se o gif não puder ser inicializado (por exemplo, se a largura/altura for muito grande para a memória disponível). Você também pode especificar um argumento opcional para definir a "cor clara" para o GIF. Este é um valor RGB (sem componente alfa), e limpará o gif para esta cor antes que qualquer coisa seja adicionada a ele. Se você não fornecer um argumento de cor clara, será usada a cor padrão de preto.

Note que ao utilizar esta função você deve chamar gif_save() para finalizar a criação do GIF antes de abrir outro arquivo para gravação (assim, para cada gif_open() deve haver um arquivo de acompanhamento gif_save()).

Sintaxe:

gif_open(width, height, [clear_colour]);

Argumento Tipo Descrição
width Real The width of the gif to create
height Real The height of the gif to create
[clear_colour] Cor OPTIONAL A cor (RGB) para limpar o gif para

Devoluções:

GIF ID

Exemplo:

if save_gif == true
{
    if count == 0
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if count < 30
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

O código acima criará um arquivo de imagem GIF com 30 quadros tirados da superfície da aplicação e depois o salvará.
