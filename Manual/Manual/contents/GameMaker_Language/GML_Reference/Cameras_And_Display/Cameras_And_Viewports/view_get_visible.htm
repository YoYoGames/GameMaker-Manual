view_get_visible

Esta função pode ser usada para verificar a visibilidade do porto de vista dado. A função retornará true se estiver visível e false se não estiver.

Sintaxe:

view_get_visible(view_port)

ArgumentoTipo Descrição
view_port The view port to target (0 - 7)

Devoluções:

Boolean

Exemplo:

if !view_get_visible(0)
{
    view_set_visible(0, true);
}

O código acima verificará se a porta de visualização[0] está visível e se não está definida para visível.
