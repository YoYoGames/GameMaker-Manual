camera_get_end_script

Esta função pode ser usada para recuperar a identificação da função de script atribuída como a função final para a câmera em questão. Se nenhuma função script for atribuída, então a função retornará -1.

Sintaxe:

camera_get_end_script(camera_id)

ArgumentoTipo Descrição
camera_id The unique camera ID value returned when you created the camera

Devoluções:

Script Function or -1 if no function is assigned

Exemplo:

var scr = camera_get_end_script(camera_view[0]);
if scr != cutScene
{
    camera_set_update_script(camera_view[0], cutScene);
}

O código acima verifica a função script atribuída como início script para a câmera designada para visualizar a porta[0] e se ela não for "cutScene" está definida para essa função.
