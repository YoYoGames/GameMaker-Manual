câmera_set_view_size_size

Você pode usar esta função para atualizar o tamanho da câmera de visão dentro do room. Você dá o valor único de identificação da câmera (como retornado pelos diferentes camera_create() funções) e depois dar a largura e a altura (em pixels) para ajustar a câmera para.

Sintaxe:

camera_set_view_size(camera_id, width, height)

ArgumentoTipo Descrição
camera_id The unique camera ID value returned when you created the camera.
width The width of the camera view in pixels
height The height of the camera view in pixels

Devoluções:

Exemplo:

camera_set_view_size(view_camera[0], view_wport[0], view_hport[0]);

O código acima definirá o tamanho da câmera de visualização atribuído à porta de visualização[0].
