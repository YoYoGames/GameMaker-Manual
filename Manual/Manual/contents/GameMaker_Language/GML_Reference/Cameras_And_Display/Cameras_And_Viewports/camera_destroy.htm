camera_destroy

Com esta função, você pode destruir qualquer câmera. Ao chamar a função você fornece o valor único de identificação da câmera, que você obtém das funções camera_create_* ou do view_camera se usando o editor Room para configurar a porta de visualização e a visualização. Você nunca deve destruir uma câmera que está atualmente atribuída a uma vista visível, a menos que você esteja atribuindo uma nova câmera a essa vista no mesmo passo, e você deve sempre destruir quaisquer câmeras que você tenha criado através de código quando não for mais necessário para evitar vazamentos de memória, e você também pode destruir as câmeras padrão se você tiver alguma atribuída no Editor Room, mas você deve atribuir uma nova câmera à porta de vista (ou desabilitá-la) caso contrário você obterá resultados estranhos.

Sintaxe:

camera_destroy(camera_id)

ArgumentoTipo Descrição
camera_id The unique camera ID value returned when you created the camera.

Devoluções:

N/A

Exemplo:

camera_destroy(view_camera[0]);
view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

O código acima destrói a câmera atualmente designada para visualizar a porta [0] e então cria uma nova câmera e atribui sua identificação a essa porta.
