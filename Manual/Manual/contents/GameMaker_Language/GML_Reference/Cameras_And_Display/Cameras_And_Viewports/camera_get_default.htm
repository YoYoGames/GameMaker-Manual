camera_get_default

Esta função pode ser usada para recuperar o valor único de identificação da câmera padrão (a câmera que o GameMaker usa quando não há nenhuma visualização ou porta de câmera ativa em um jogo room).

Sintaxe:

camera_get_default()

Devoluções:

Camera ID

Exemplo:

var def = camera_get_default();
view_camera[0] = def;

O código acima obtém o ID da câmera para a câmera padrão e define a câmera de visualização para a porta[0] para usá-la.
