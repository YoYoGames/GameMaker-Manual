camera_set_view_border

Você pode usar esta função para definir a seguinte borda da câmera de visão dentro do room. Você dá o valor único de identificação da câmera (como retornado pelas diferentes funções camera_create ) e então dá o tamanho da borda x e o tamanho da borda y (horizontal e vertical). Estes valores só serão usados quando a câmera de visualização tiver sido designada um alvo de instância a seguir (seja no Editor Room ou usando a função camera_set_view_target()) e relacionar para quão longe da fronteira da visão a instância precisa estar antes que a visão atualize sua posição a seguir.

Sintaxe:

camera_set_view_border(camera_id, x_border, y_border)

ArgumentoTipo Descrição
camera_id The unique camera ID value returned when you created the camera.
x_border The border (in pixels) for the view camera along the horizontal axis
y_border The border (in pixels) for the view camera along the vertical axis

Devoluções:

Exemplo:

camera_set_view_border(view_camera[0], 64, 64);

O código acima definirá a borda da câmera de visualização para a câmera designada para visualizar a porta[0].
