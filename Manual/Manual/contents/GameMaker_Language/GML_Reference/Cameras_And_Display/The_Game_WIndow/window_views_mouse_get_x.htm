window_views_mouse_get_x

Esta função retorna a coordenada x do mouse com respeito a todas as vistas ativas e retorna o mesmo valor mouse_x.

NOTA: Para funções regulares do mouse, veja a seção sobre Entrada do Rato.

Sintaxe:

window_views_mouse_get_x();

Devoluções:

Exemplo:

myx = window_views_mouse_get_x();

Isto colocaria o myx na coordenada x do mouse na janela em relação a todas as vistas ativas.
