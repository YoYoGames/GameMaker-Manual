O Steam Overlay é o display visual que pode ser trazido para exibir a interface Steam para o usuário. Isto normalmente é feito pelo próprio usuário usando uma combinação de teclas, mas você também tem a possibilidade de fazê-lo de dentro de seu jogo, para que você possa mapear um botão ou um evento para mostrar a sobreposição, usando as seguintes funções:

  • steam_is_overlay_enabled
  • steam_is_overlay_activated
  • steam_activate_overlay
  • steam_activate_overlay_browser
  • steam_activate_overlay_store
  • steam_activate_overlay_user
