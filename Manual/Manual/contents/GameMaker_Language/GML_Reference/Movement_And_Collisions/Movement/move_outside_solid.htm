move_outide_solid

Com esta função você pode dizer a uma instância para sair de uma colisão com qualquer instância assinalada como sólido em qualquer direção e qualquer número de pixels a cada passo, com um valor de -1 ou 0 para o maxdist sendo 1000px padrão, ou seja: o GameMaker movimentará a instância continuamente até 1000 pixels até que ela esteja fora de colisão.

Sintaxe:

move_outside_solid(dir, maxdist);

ArgumentoTipo Descrição
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

Devoluções:

Exemplo:

if other.solid
{
    var pdir;
    pdir = point_direction(other.x, other.y, x, y);
    move_outside_solid(pdir, -1);
}

O código acima iria em um evento de colisão e verifica se a "outra" instância na colisão está marcada como "sólida". Se for, ela então moverá a instância para fora da colisão.
