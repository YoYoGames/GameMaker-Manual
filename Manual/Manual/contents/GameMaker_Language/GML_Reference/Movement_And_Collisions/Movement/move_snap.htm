move_snap

Esta função é usada para "estalar" a instância para uma grade de um determinado tamanho. Ela será encaixada na posição correspondente mais próxima na grade "invisível" que os valores hsnap e vsnap definem.

Sintaxe:

move_snap(hsnap, vsnap);

ArgumentoTipo Descrição
hsnap The horizontal snapping (the size in pixels between 'cells').
vsnap The vertical snapping (the size in pixels between 'cells').

Devoluções:

Exemplo:

with (obj_Pieces)
{
    if !place_snapped(32, 32)
    {
        move_snap(32, 32);
    }
}

O código acima verifica todas as instâncias de "obj_Pieces" para ver se elas estão encaixadas em uma grade de 32x32 pixels, e se elas não estão encaixadas na posição mais próxima daquela grade.
