place_meeting

Com esta função você pode verificar uma posição para uma colisão com outra instância ou todas as instâncias de uma object usando a máscara de colisão da instância que executa o código para a verificação. Quando você usa esta função, você está efetivamente pedindo ao GameMaker para mover a instância para a nova posição, verificar se há uma colisão, voltar atrás e dizer se uma colisão foi encontrada ou não. Isto funcionará para colisões precisas, mas somente se tanto a instância quanto a object que está sendo verificada tiverem máscaras de colisão precisas selecionadas, caso contrário, somente serão aplicadas colisões de caixas delimitadoras. Esta função também aceita as palavras-chave especiais todas e outras. Observe que caso você precise obter a identificação única da instância com a qual o object está sendo colidido, você deve usar instance_place().

Observe que as coordenadas x/y dadas serão niveladas com o inteiro mais próximo antes da verificação ser realizada.

Sintaxe:

place_meeting(x, y, obj);

Argumento Tipo Descrição
x Real The x position to check.
y Real The y position to check.
obj Objeto Ativo ou ID da instância The instance or object to check for.

Devoluções:

Boolean

Exemplo:

if keyboard_check(vk_left)
{
    if !place_meeting(x - 5, y, obj_wall) x -=5;
}

O código acima verifica se não há uma colisão à esquerda da instância e move a instância se não houver nenhuma.
