mp_grid_clear_all

Com esta função você pode limpar uma grade MP de todas as células "proibidas".

Sintaxe:

mp_grid_clear_all(id);

Argumento Tipo Descrição
id ID da grade MP Index of the mp_grid that is to be used

Devoluções:

Boolean

Exemplo:

if !instance_exists(obj_Player)
{
    mp_grid_clear_all(grid);
}

O código acima limpará a mp_grid indexada na variável "grid", marcando todas as células como livres, se uma instância da object "obj_Player" não existir mais no room.
