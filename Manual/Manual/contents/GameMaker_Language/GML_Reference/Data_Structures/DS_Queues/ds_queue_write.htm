ds_queue_write

Esta função retorna um string que pode então ser armazenado ou transferido para outra estrutura de dados usando o ds_queue_read() função.

OBSERVAÇÃO: O retorno string não é uma leitura humana string, mas sim um despejo do conteúdo da estrutura de dados

Sintaxe:

ds_queue_write(id);

ArgumentoTipo Descrição
id The id of the data structure to write.

Devoluções:

Exemplo:

ini_open("save.ini");
var str =ds_queue_write(queue);
ini_write_string("Queues", "0", str);
ds_queue_clear(queue);
ini_close();

O código acima abre um arquivo ini e então escreve um string contendo as informações armazenadas na fila DS indexada na variável "queue". O queue é então limpo e o arquivo ini é fechado.
