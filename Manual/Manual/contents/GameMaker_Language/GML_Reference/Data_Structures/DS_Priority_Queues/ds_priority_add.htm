ds_priority_add

Com esta função você pode adicionar um valor (um número real ou um string) a uma prioridade queue, ao mesmo tempo em que lhe atribui um valor prioritário.

Sintaxe:

ds_priority_add(id, val, priority);

Argumento Tipo Descrição
id DS Identificação prioritária The id of the priority queue to add to.
val Variável The value to add to the priority queue.
priority Real The priority of the value to add.

Devoluções:

N/A

Exemplo:

ds_priority_add(ai_priority, AI_Search, 5);

O código acima acrescenta uma função script à prioridade queue indexada na variável "ai_priority" e lhe atribui uma prioridade de 5.
