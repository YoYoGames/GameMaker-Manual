ds_priority_clear

Com esta função você pode limpar todos os dados da estrutura de dados queue dada prioridade. Isto NÃO destrói a estrutura de dados (para isso você deve usar ds_priority_destroy()) ele apenas limpa todos os dados e retorna uma prioridade vazia queue.

Sintaxe:

ds_priority_clear(id);

ArgumentoTipo Descrição
id The id of the data structure to clear.

Devoluções:

Exemplo:

if (count = 15) && (!ds_priority_empty(command_queue))
{
    ds_priority_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
}

O código acima verifica uma variável para ver se ela atingiu um valor específico e se ela tem a prioridade DS queue indexada na variável "command_queue", define um alarme e redefine a variável para 0.
