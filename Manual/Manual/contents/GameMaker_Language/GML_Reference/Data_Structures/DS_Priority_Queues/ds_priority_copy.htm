ds_priority_copy

Esta função pode ser usada para copiar o conteúdo de uma prioridade queue para outra. Note que isto NÃO remove o conteúdo da prioridade original queue, nem destrói a prioridade original queue. Ao utilizar esta função, a prioridade queue a ser copiada deve ter sido previamente criada e se ela contivesse algum item antes da cópia, então estes serão limpos primeiro (significando que esta informação será perdida).

Sintaxe:

ds_priority_copy(id, source);

ArgumentoTipo Descrição
id The id of the priority queue to copy to.
source The id of the priority queue to copy from.

Devoluções:

Exemplo:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    p_queue = ds_priority_create();
    ds_priority_copy(p_queue, other.p_queue);
}

A função acima cria uma nova instância e, nessa instância, cria uma nova prioridade DS queue e copia o conteúdo da prioridade queue na instância que executa o bloco de código, para a nova prioridade de instância criada queue.
