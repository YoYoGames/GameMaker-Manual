ds_grid_resize

Com esta função você pode redimensionar a grade DS dada para ter uma largura e/ou altura diferente. Se o tamanho da grade for maior do que a grade atual, as novas células terão um valor base de 0, e se o tamanho for menor do que os valores mantidos nas células que não estão mais dentro do novo tamanho será perdido. Todas as outras células serão deixadas intocadas.

Sintaxe:

ds_grid_resize(index, w, h);

ArgumentoTipo Descrição
index This index of the grid to resize.
w The new width of the grid.
h The new height of the grid.

Devoluções:

Exemplo:

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

O código acima redimensionará a grade DS indexada na variável global "Grid" e então a limpará para que cada célula mantenha o valor -1.
