ds_grid_add_region

Esta função pode ser usada para adicionar um determinado valor (real ou string) a todos os valores das células encontradas dentro da área definida de uma grade. O valor a ser adicionado deve ser do mesmo tipo do que é mantido dentro das células da grade, ou seja: não se pode adicionar um string a um real ou vice-versa, e para strings isto corresponde a concatenação.

Sintaxe:

ds_grid_add_region(index, x1, y1, x2, y2, val);

ArgumentoTipo Descrição
index The index of the grid.
x1 The x position of the left of the region in the grid.
y1 The y position of the top of the region in the grid.
x2 The x position of the right of the region in the grid.
y2 The y position of the bottom of the region in the grid.
val The value to add to the region cells.

Devoluções:

Exemplo:

ds_grid_add_region(grid, 2, 4, 5, 5, ".")

Isto acrescentaria "..." a todos os strings mantidos nas células dentro da região definida da grade DS referenciada pela variável "grid".
