ds_grid_set

Esta função pode ser usada para definir uma determinada célula dentro da grade DS dada para qualquer valor, que pode ser um número real ou um string. A imagem abaixo ilustra isto:

Sintaxe:

ds_grid_set(index, x, y, value);

ArgumentoTipo Descrição
index This index of the grid.
x The x position of the cell to set.
y The y position of the cell to set.
value The value with which to set the cell.

Devoluções:

Exemplo:

grid = ds_grid_create(5, 5);
var i = 0;
var j = 0;

repeat (ds_grid_width(grid))
{
    repeat (ds_grid_height(grid))
    {
        ds_grid_set(grid, i, j, irandom(9));
        j += 1;
    }

    j = 0;
    i += 1;
}

O código acima cria uma grade e armazena seu índice na variável "grade". Ele então preenche esta grade com números inteiros aleatórios de 0 a 9.
