ds_grid_set_disk

Com esta função você pode definir uma região circular de uma grade para um determinado valor. Você precisa fornecer uma célula inicial da grade (como coordenada dos eixos x e y), bem como o raio do disco a ser definido e o valor que você deseja definir o disco também, como mostrado pela ilustração abaixo:

Sintaxe:

ds_grid_set_disk(index, xm, ym, r, val);

ArgumentoTipo Descrição
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.
val The value to set with the cells within the disk.

Devoluções:

Exemplo:

ds_grid_set_disk(grid, ds_grid_width(grid) div 2, ds_grid_height(grid) div 2, 5, -4)

O código acima definirá uma região circular com um raio de 5 células na grade DS indexada na variável "grade" a um valor de -4.
