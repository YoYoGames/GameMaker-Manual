ds_grid_multiply

Esta função multiplicará o valor da célula de grade dada pela quantidade especificada.

NOTA Esta função só funcionará com números reais, não strings.

Sintaxe:

ds_grid_multiply(index, x, y, val);

Argumento Tipo Descrição
index DS Grid ID The index of the grid.
x Real The x position of the cell in the grid.
y Real The y position of the cell in the grid.
val Real The value to multiply with the cell.

Devoluções:

N/A

Exemplo:

ds_grid_multiply(mygrid, 5, 5, 2)

O código acima multiplicará o valor armazenado na célula de grade DS dada por 2.
