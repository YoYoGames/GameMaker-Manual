ds_grid_multiply_disk

Esta função tomará todos os valores em uma determinada região em forma de disco da grade DS, e multiplicará cada um deles pela quantidade determinada.

NOTA Esta função só funcionará com números reais, não strings.

Sintaxe:

ds_grid_multiply_disk(index, xm, ym, r, val);

Argumento Tipo Descrição
index DS Grid ID The index of the grid.
xm Real The x position of the disk on the grid.
ym Real The y position of the disk on the grid.
r Real The radius of the disk on the grid.
val Real The value to multiply the cells within the disk with.

Devoluções:

N/A

Exemplo:

ds_grid_multiply_disk(mygrid, 5, 5, 5, 2)

O código acima tomará todos os valores encontrados dentro da área da grade circular e multiplicará cada um por 2.
