ds_grid_sort

Esta função pode ser usada para classificar uma grade DS com base nos valores de uma determinada coluna (assim como você classificaria os arquivos por data, tamanho, etc... no explorador de arquivos do sistema operacional). A imagem a seguir mostra um exemplo:

Sintaxe:

ds_grid_sort(index, column, ascending);

ArgumentoTipo Descrição
index The index of the grid to sort.
column The column to use for sorting the rows
ascending Whether to sort lowest to highest (true), ou do mais alto para o mais baixo (false).

Devoluções:

Exemplo:

ds_grid_sort(grid, 3, false)

Isto pegaria todos os valores da grade DS indexados na variável "grid" e os classificaria de acordo com os valores encontrados na 3ª coluna da grade (como mostrado na imagem acima).
