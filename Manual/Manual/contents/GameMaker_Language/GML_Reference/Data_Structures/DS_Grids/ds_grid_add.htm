ds_grid_add

Esta função pode ser usada para adicionar um determinado valor (real ou string) ao valor da célula dada dentro da grade. O valor a ser adicionado deve ser do mesmo tipo que o mantido dentro da célula da grade, ou seja: não se pode adicionar um string a um real ou vice-versa, e para strings isto corresponde à concatenação.

Sintaxe:

ds_grid_add(index, x, y, val);

ArgumentoTipo Descrição
index The index of the grid.
x The x position of the cell in the grid.
y The y position of the cell in the grid.
val The value to add to the cell.

Devoluções:

Exemplo:

ds_grid_add(grid, 5, 5, 6)

Isto acrescentaria 6 à célula dada dentro da grade DS referenciada pela variável "grid".
