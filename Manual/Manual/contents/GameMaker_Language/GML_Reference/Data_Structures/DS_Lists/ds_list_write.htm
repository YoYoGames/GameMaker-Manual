ds_list_write

Esta função retorna um string que pode então ser armazenado ou transferido para outra estrutura de dados usando o ds_list_read() função.

OBSERVAÇÃO: O retorno string não é uma leitura humana string, mas sim um despejo do conteúdo da estrutura de dados.

Sintaxe:

ds_list_write(id);

ArgumentoTipo Descrição
id The id of the data structure to write.

Devoluções:

Exemplo:

ini_open("save.ini");
var str = ds_list_write(list);
ini_write_string("Lists", "0", str);
ds_list_clear(list);
ini_close();

O código acima abre um arquivo ini e então escreve um string contendo as informações armazenadas na lista DS indexada na variável "lista". A lista é então limpa e o arquivo ini é fechado.
