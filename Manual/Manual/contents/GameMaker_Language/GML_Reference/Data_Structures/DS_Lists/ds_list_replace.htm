ds_list_replace

Esta função substituirá o valor na posição dada por outro.

Sintaxe:

ds_list_replace(id, pos, val);

Argumento Tipo Descrição
id DS List ID The id of the list to change.
pos Real The position to replace the value, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.
val Variável The new value to replace the given value with.

Devoluções:

N/A

Exemplo:

ds_list_replace(n_list, 3, name);

O código acima substituirá o valor armazenado na posição 3 da lista por aquele armazenado na variável "name".
