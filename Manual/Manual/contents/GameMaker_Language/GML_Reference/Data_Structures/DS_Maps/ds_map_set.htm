ds_map_set

Com esta função, você pode definir o valor de uma chave dentro de um determinado mapa DS. Você fornece o valor do DS map ID (como retornado pela função ds_map_create()), então dê a chave que você quer definir e o valor a ser definido. As chaves podem ser inteiros ou strings, e se a chave dada não existir, então ela será criada para você e ajustada para o valor. Esta função é a mesma que a de usar o acessor de mapa DS para definir/criar um par chave de mapa/valor. A função não retorna nada, portanto, se você precisar verificar se o valor da chave foi substituído ou se uma nova chave foi criada, então você deve usar a função ds_map_replace().

Sintaxe:

ds_map_set(id, key, value)

ArgumentoTipo Descrição
id The id of the map to use.
key The key to set.
value The value to set the key to.

Devoluções:

Exemplo:

if is_undefined(ds_map_find_value(map, "score"))
{
    ds_map_set(map, "score", 0);
}

O código acima verificará se a chave dada existe e se não existe, então ela é criada e definida.
