ds_map_is_map

Com esta função, você pode verificar se um mapa DS está armazenado na chave do mapa dado. Se a chave indicada contém um DS map ID, então a função retornará true caso contrário, retornará false.

Note que isto só detectará mapas que foram adicionados usando a função ds_map_add_map() .

Sintaxe:

ds_map_is_map(id, key)

ArgumentoTipo Descrição
id The id of the map to use.
key The key to replace.

Devoluções:

Exemplo:

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++)
{
    if ds_map_is_map(inventory, key)
    {
        ds_map_destroy(inventory[? key]);
    }
    key = ds_map_find_next(inventory);
}
ds_map_destroy(inventory);

O código acima loops através de um mapa DS e verifica se alguma das chaves dentro dele é para outros mapas DS. Se forem, então o mapa DS armazenado é destruído, e no final do loop o mapa DS principal também é destruído.
