ds_map_replace_map

Com esta função você pode substituir um DS Map que foi armazenado na "chave" dada por outro mapa que foi criado anteriormente. Esta função foi projetada para criar mapas compatíveis com JSON que você então codificaria usando json_encode() e só deve ser usado em conjunto com essa funcionalidade.

Sintaxe:

ds_map_replace_map(id, key, value)

Argumento Tipo Descrição
id DS Map ID The id of the ds_map to use.
key Cordão The key to replace.
value DS Map ID The id of the ds_map to use to replace the one previously stored in the given key.

Devoluções:

N/A

Exemplo:

var temp_map = ds_map_create();
ds_map_add_list(temp_map, "list", j_list);
ds_map_add(temp_map, "array", j_array);
ds_map_replace_map(j_map, "maps", temp_map);
var j = json_encode(j_map);
ds_map_destroy(temp_map);

O código acima criará um DS Map e o preencherá com um array e um DS List antes de substituir um mapa previamente armazenado no DS Map "j_map".
