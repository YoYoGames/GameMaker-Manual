ds_map_delete

Com esta função você pode remover qualquer chave (e seu valor correspondente) do mapa DS dado, previamente criado.

Sintaxe:

ds_map_delete(id, key);

Argumento Tipo Descrição
id DS Map ID The id of the map to change.
key Cordão The key to delete (along with its associated value).

Devoluções:

N/A

Exemplo:

ds_map_delete(inventory, "shield");

O código acima apagará a chave "escudo" (e o valor com o qual está emparelhado) do mapa DS (inventário).
