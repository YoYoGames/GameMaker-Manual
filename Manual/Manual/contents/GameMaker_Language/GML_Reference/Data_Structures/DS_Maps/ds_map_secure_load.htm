ds_map_secure_load

Esta função carregará um mapa DS previamente gravado do arquivo em questão. Este arquivo deve ter sido previamente criado usando o ds_map_secure_save() e, quando carregada, a função retornará o índice do mapa DS que foi criado a partir dos dados carregados. Este índice de mapa DS deve ser armazenado em uma variável e usado para todas as chamadas de funções adicionais a este mapa. Note que se o DS map que está sendo carregado foi salvo com um array como um dos valores chave, este array terá sido convertido em uma lista DS em carga.

IMPORTANTE! Uma das características de um arquivo seguro salvo é que ele está travado no dispositivo em que foi criado, de modo que você não pode carregar um arquivo salvo em um dispositivo em um projeto em execução em outro dispositivo.

Sintaxe:

ds_map_secure_load(filename);

ArgumentoTipo Descrição
filename The name of the file to load the map data from.

Devoluções:

DS Map ID

Exemplo:

p_map = ds_map_secure_load("p_data.dat");

O código acima irá carregar um mapa DS salvo com segurança e armazenar seu valor de índice em uma variável para uso futuro.
