draw_clear_alpha

Esta função pode ser usada para limpar a tela inteira com uma determinada cor e o componente alfa do destino é definido para o valor que você definiu - esta função não faz nenhuma mistura como funciona, mas qualquer operação subseqüente de mistura pode ser configurado para usar o alfa de destino que você definiu. Isto é apenas para uso no evento de sorteio de uma instância (não será mostrado se usado em qualquer outro evento), e também pode ser muito útil para limpar superfícies quando elas são criadas recentemente.

Vale notar que ao usar o módulo HTML5, você pode usar esta função para tornar o fundo da tela transparente, o que permitirá que qualquer imagem sendo usada no HTML hospedeiro seja vista com o jogo correndo sobre elas. Para conseguir este efeito você deve primeiro entrar no editor room e remover quaisquer camadas de fundo do editor da sala, depois desmarcar a opção Clear View Background encontrada nas propriedades View Properties. Finalmente, você precisará fazer um novo object, colocá-lo no primeiro room do jogo, e atribuir-lhe uma profundidade muito alta (por exemplo, 10000). Depois, no evento Pre Draw use esta função para ajustar o alfa da tela à cor e transparência que que você deseja que tenha.

Sintaxe:

draw_clear_alpha(col, alpha);

ArgumentoTipo Descrição
col The colour with which the screen will be cleared
alpha The transparency of the colour with which the screen will be cleared

Devoluções:

Exemplo:

draw_clear_alpha(c_red, 0.5);

Isto irá limpar a tela com 50% (0,5) de transparência com a cor vermelha.
