cor_get_hue

Esta função devolverá a tonalidade da cor dada. Este é o tom de cor "pura" que faz parte do método de tonalidade, saturação e valor (luminosidade) para definir uma cor. A imagem a seguir ilustra como este valor corresponde ao Escala de cor HSV:

Sintaxe:

colour_get_hue(col);

ArgumentoTipo Descrição
col The colour to check

Devoluções:

Exemplo:

col = merge_colour(colour_get_hue(c_teal), c_white, 0.5);

O código acima obtém a tonalidade usada para fazer a cor constante "c_teal" e depois a funde com o branco na proporção de 50:50, armazenando a cor resultante na variável "col".
