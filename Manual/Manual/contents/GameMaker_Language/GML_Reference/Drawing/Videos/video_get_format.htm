video_get_format

Esta função retorna o formato de cor do vídeo carregado atualmente. Esta pode ser qualquer uma das seguintes constantes:

Formato do vídeo Constante
Constante Descrição
video_format_rgba A superfície de vídeo utiliza o modelo em cores RGBA
video_format_yuv A superfície de vídeo utiliza o modelo YUV colorido

Windows, macOS, Opera GX, Android, iOS, e HTML5 usam o modelo RGBA para vídeos. Todas as outras plataformas utilizam o modelo YUV.

Como os métodos de desenho para vídeos RGBA e YUV são diferentes, o valor de retorno desta função pode ser usado para executar códigos diferentes com base no formato do vídeo que está sendo reproduzido. Veja o vídeo de desenho para um exemplo.

Sintaxe:

video_get_format();
