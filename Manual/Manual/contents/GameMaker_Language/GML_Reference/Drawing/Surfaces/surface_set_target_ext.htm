superfície_set_alvo_texto

Esta função é para uso com as Funções Shader e define o MRT (0 - 3) para nível nativo shaders (DX9, DX11, OpenGL).

NOTA: Quando se trabalha com superfícies há a possibilidade de que elas possam deixar de existir a qualquer momento devido ao fato de serem armazenadas na memória texture. Você deve SEMPRE verificar se uma superfície existe usando surface_exists() antes de referenciá-los diretamente.

Sintaxe:

surface_set_target_ext(index, surface_id);

ArgumentoTipo Descrição
index The render target index to use (from 0 to 3).
surface_id The ID of the surface to use.

Devoluções:

Exemplo:

surface_set_target_ext(0, surf);

O código acima definirá a meta de renderização 0 para a identificação da superfície indexada na variável "surf".
