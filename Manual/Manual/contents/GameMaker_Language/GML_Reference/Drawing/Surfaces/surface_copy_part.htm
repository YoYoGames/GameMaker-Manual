parte_da_cópia_de_superfície

Esta função simplesmente tira a imagem de uma superfície e a copia para outra na posição local especificada dentro daquela superfície (onde a posição (0,0) é o canto superior esquerdo da superfície de destino). Você pode especificar uma posição local x e y para copiar, assim como a largura e a altura da seção. Observe que estas são coordenadas baseadas no tamanho da superfície e não na posição em que a superfície está sendo desenhada no room. Se a superfície de destino já tiver informações, estas serão sobrescritas pela cópia, e a função não altera de forma alguma a superfície de origem.

NOTA: Quando se trabalha com superfícies há a possibilidade de que elas possam deixar de existir a qualquer momento devido ao fato de serem armazenadas na memória texture. Você deve SEMPRE verificar se uma superfície existe usando surface_exists() antes de referenciá-los diretamente.

Sintaxe:

surface_copy_part(destination, x, y, source, xs, ys, ws, hs);

Argumento Tipo Descrição
destination Identificação da superfície The ID of the surface to copy the other surface to.
x Real The x position to copy to.
y Real The y position to copy to.
source Identificação da superfície The ID of the surface to be copied.
xs Real The x position in the source surface to copy from.
ys Real The y position in the source surface to copy from.
ws Real The width of the area in the source surface to copy from.
hs Real The height of the area in the source surface to copy from.

Devoluções:

N/A

Exemplo:

if view_current == 0
{
    surface_copy_part(surf, 0, 0, temp_surf, 0, 0, view_xview[1] - mouse_x, view_yview[1] - mouse_y);
}
else
{
    draw_surface(surf, 0, 0);
}

O código acima verificará a vista atual sendo desenhada e se é vista[0] copia a superfície indexada na variável "temp_surf" para a superfície indexada na variável "surf". A área copiada corresponde a um retângulo formado pela posição relativa do mouse dentro da superfície como seria desenhada na vista[1]. Se a vista atual for algo diferente da vista[0], a superfície "surf" é desenhada para a tela.
