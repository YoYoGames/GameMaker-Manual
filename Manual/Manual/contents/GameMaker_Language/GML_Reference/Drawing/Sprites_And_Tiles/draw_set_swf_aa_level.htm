nível_de_sorteio_swf_aa

Esta função pode ser usada para definir o nível de anti-aliasing (AA) para o formato vetor SWF sprites. Este pode ser um valor real de 0 a 1 e irá "suavizar" as bordas destes sprites. Note que para ver este efeito, você deve primeiro ter habilitado AA usando a função draw_enable_swf_aa().

Sintaxe:

draw_set_swf_aa_level(AA);

ArgumentoTipo Descrição
AA The anti-aliasing value to use from 0 to 1.

Devoluções:

Exemplo:

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

O código acima verificará o valor AA para o formato SWF sprites, e se for 0, habilita AA e define o valor para 0,5.
