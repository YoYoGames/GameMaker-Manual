textura_get_width

Retorna a largura do texture com o id dado, que é sempre um valor dentro da faixa 0 - 1. Isto pode então ser usado no mapeamento textures para modelos ou primitivos.

Sintaxe:

texture_get_width(tex);

Argumento Tipo Descrição
tex Ponteiro de Textura The texture page asset pointer to use

Devoluções:

Real

Exemplo:

tex_w = texture_get_width(sprite_get_texture(spr_Model_tex, 0));

O código acima terá a largura do texture retirado de um sprite asset.
