part_system_get_layer

Esta função pode ser usada para recuperar o valor de identificação da camada única para o sistema de partículas em questão. Você fornece o valor de identificação do sistema de partícula (como retornado pela função part_system_create_layer()) e a função devolverá o valor de identificação da camada.

Sintaxe:

part_system_get_layer(ind);

Argumento Tipo Descrição
ind Identificação do Sistema de Partículas The particle system ID value to get the layer ID of

Devoluções:

Layer ID or 0 (if the layer is an internally managed one)

Exemplo:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
    part_system_layer(global.p_sys, "effects_layer";
}

O código acima verificará um sistema de partículas para ver em que camada está e, se não estiver na camada dada, será alterado.
