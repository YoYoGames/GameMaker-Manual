part_type_colour_mix

Com esta função, você pode definir o tipo de partícula dado para ser uma mistura aleatória de duas cores.

Sintaxe:

part_type_colour_mix(ind, colour1, colour2);

Argumento Tipo Descrição
ind Identificação do tipo de partícula The index of the particle type to change.
colour1 Cor The first colour the blend will take from.
colour2 Cor The second colour the blend will take from.

Devoluções:

N/A

Exemplo:

part_type_colour_mix(global.P_Damage, c_red, c_yellow);

O código acima definirá a cor para cada partícula emitida do tipo de partícula indexada na variável global "P_Damage" para ser uma mistura aleatória entre as cores vermelha e amarela.
