part_type_alpha1

Esta função é usada para definir um tipo de partícula a ter um único valor alfa (transparência) para a duração total da vida útil de cada partícula individual, e este pode ser de 0 (transparente) a 1 (opaco).

Sintaxe:

part_type_alpha1(ind, alpha1);

Argumento Tipo Descrição
ind Identificação do tipo de partícula The index of the particle type to change.
alpha1 Real The alpha of the particle.

Devoluções:

N/A

Exemplo:

part_type_alpha1(global.Snow_Part, 0.5);

O código acima definirá todas as partículas criadas do tipo de partícula indexada na variável global "Snow_Part" para ter um valor alfa de 0,5 (semi-transparente).
