part_type_colour_rgb

Com esta função você pode definir a mistura de cores vermelha, verde e azul para todas as partículas criadas do tipo dado. Você fornece um valor mínimo e um valor máximo para cada um dos três componentes e as partículas criadas terão uma cor aleatória com base na faixa de parâmetros dada. Todos os valores devem estar entre 0 e 255.

Sintaxe:

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

Argumento Tipo Descrição
ind Identificação do tipo de partícula The index of the particle type to change.
rmin Real The minimum the final colour's red component can be.
rmax Real The maximum the final colour's red component can be.
gmin Real The minimum the final colour's green component can be.
gmax Real The maximum the final colour's green component can be.
bmin Real The minimum the final colour's blue component can be.
bmax Real The maximum the final colour's blue component can be.

Devoluções:

N/A

Exemplo:

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0);

O código acima define cada partícula emitida do tipo de partícula indexada na variável global "Blood_Part" para ser apenas diferentes tonalidades de vermelho.
