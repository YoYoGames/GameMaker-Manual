part_type_gravity

Esta função definirá a gravidade que deve afetar cada partícula do tipo dado que é criada. O valor da força da gravidade é adicionado à velocidade da partícula a cada passo e normalmente é um valor pequeno como 0,5, enquanto a direção é a direção da gravidade "puxar" e segue as direções padrão do GameMaker de 0° para a direita, 90° para cima, 180° para a esquerda e 270° para baixo.

Sintaxe:

part_type_gravity(ind, grav_amount, grav_direction);

Argumento Tipo Descrição
ind Identificação do tipo de partícula The index of the particle type to change.
grav_amount Real Strength of the gravity.
grav_direction Real The direction of the gravity.

Devoluções:

N/A

Exemplo:

part_type_shape(global.p1, pt_shape_pixel);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

O código acima estabelecerá vários valores de partícula, incluindo a gravidade que adicionará 0,2 à velocidade de cada passo com uma direção de 270º, assim, puxará a partícula "para baixo" em direção à parte inferior da tela.
