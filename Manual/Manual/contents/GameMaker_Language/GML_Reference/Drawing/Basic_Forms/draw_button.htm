draw_button

Esta função desenhará um "botão" muito simples e retangular usando a cor e o alfa atualmente selecionados, onde o argumento para cima define o aspecto do efeito de borda biselada, como mostrado na imagem abaixo:

NOTA: Se você estiver querendo desenhar uma forma usando um shader, você deve estar ciente de que a maioria shaders espera as seguintes entradas: vertex, texture, Cor. Entretanto, ao usar esta função, apenas vertex e dados de cor estão sendo passados, e assim o shader não pode desenhar nada (ou desenhar algo, mas não corretamente). Se você precisar desenhar formas desta forma, então o shader deve ser personalizado com isto em mente.

Sintaxe:

draw_button(x1, y1, x2, y2, up);

Argumento Tipo Descrição
x1 Real The x coordinate of the left of the button
y1 Real The y coordinate of the top of the button
x2 Real The x coordinate of the right of the button
y2 Real The y coordinate of the bottom of the button
up Booleano Whether the button is up (true) ou para baixo (false)

Devoluções:

N/A

Exemplo:

draw_button(100, 100, 200, 150, !mouse_check_button(mb_left));

Isto atrairá um botão que aparecerá pressionado se o botão esquerdo do mouse for mantido pressionado.
