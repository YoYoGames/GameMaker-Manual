gpu_set_tex_mip_enable

Com esta função você pode mudar se o mipmapping está desligado, ligado para tudo, ou ligado somente para texture grupos selecionados no Gerente do Grupo de Textura. A função requer uma das constantes listadas abaixo.

Constante Descrição
mip_off O Mipmapping está desativado.
mip_on O Mipmapping para todos textures está habilitado.
mip_markedonly O Mipmapping está habilitado para textures que o tem habilitado nas opções do grupo Texture (default).

Sintaxe:

gpu_set_tex_mip_enable(setting);

Argumento Tipo Descrição
setting Constante de Mipmapping The mipmap setting (a constant, default: mip_markedonly)

Devoluções:

N/A

Exemplo:

if gpu_get_tex_mip_enable != mip_on
{
    gpu_set_tex_mip_enable(mip_on);
}

O código acima verificará se o mipmapping está habilitado e se não estiver, ele o habilitará.
