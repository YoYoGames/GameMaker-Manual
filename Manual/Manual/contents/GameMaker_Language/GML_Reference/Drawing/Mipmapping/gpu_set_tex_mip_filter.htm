gpu_set_tex_mip_filter

Com esta função você pode ajustar o modo atual de filtro mip para um dos três tipos suportados. Você dá a constante que se refere ao modo de filtragem mip que você precisa, onde você pode escolher entre filtragem pontual (configuração padrão, significando sem filtragem), filtragem linear (também conhecida como filtragem trilinear) ou filtragem aniostrópica (veja a tabela de constantes abaixo).

Constante Descrição
tf_point Isto desativará a filtragem entre os níveis do mipmap, o que pode causar transições visíveis texture, mas dá o melhor desempenho.
tf_linear Isto permitirá a filtragem linear entre níveis mipmap (também conhecida como filtragem trilinear), o que suaviza as transições texture, mas dará um pequeno golpe no desempenho.
tf_anisotropic Isto permitirá uma filtragem anisotrópica entre níveis mipmap, o que melhora muito a qualidade de transição texture e pode reduzir o embaçamento visível com outros modos de filtragem, mas tem o maior impacto no desempenho.

Sintaxe:

gpu_set_tex_mip_filter(filter);

Argumento Tipo Descrição
filter Filtro de Mipmapping Constante The mip filter mode to use (a constant, default: tf_point).

Devoluções:

N/A

Exemplo:

if keyboard_check(vk_enter)
{
    switch(gpu_get_tex_mip_filter())
    {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
    }
}

O código acima verifica o teclado e, se a tecla especificada for pressionada, ele obterá o filtro mip atual e alternará o valor para o próximo, pedalando pelos diferentes modos.
