gpu_set_tex_mip_bias_ext

Com esta função você pode definir o valor do viés do mipmap para um determinado amostrador shader. Você fornece o valor do índice para o amostrador shader (como retornado pela função shader_get_sampler_index()e depois o valor do viés, onde 0 é para nenhum viés, 1 é igual ao primeiro mipmap, 2 é igual ao segundo mipmap etc... Isto controla a taxa na qual o mapa mip é trocado e geralmente fará com que o shader textures fique mais borrado quanto maior o valor e maior a "distância" sendo visualizada. Note que esta função também pode assumir valores negativos, caso em que shader textures será mais nítido em uma distância maior quanto menor for o valor.

Sintaxe:

gpu_set_tex_mip_bias_ext(sampler_index, bias);

ArgumentoTipo Descrição
sampler_index The index of the shader sampler to get
bias The mipmap bias value to use (default: 0)

Devoluções:

Exemplo:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_mip_bias_ext(_sampleIndex) != 0
{
    gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

O código acima define o viés do filtro mip como 0 para o sampler shader texture dado, caso ainda não tenha sido definido como 0.
