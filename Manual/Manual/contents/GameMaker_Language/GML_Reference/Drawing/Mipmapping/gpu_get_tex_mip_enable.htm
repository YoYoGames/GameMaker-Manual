gpu_get_tex_mip_enable

Com esta função você pode obter se o mipmapping está desligado, ligado para tudo ou ligado somente para grupos texture selecionados no Gerente de Grupos de Textura. A função retornará uma das constantes listadas abaixo, com a configuração padrão sendo mip_markedonly.

Sintaxe:

gpu_get_tex_mip_enable();

Devoluções:

Constant

Constante de Mipmapping
Constante Descrição
mip_off O Mipmapping está desativado.
mip_on O Mipmapping para todos textures está habilitado.
mip_markedonly O Mipmapping está habilitado para textures que o tem habilitado nas opções do grupo Texture (default).

Exemplo:

if gpu_get_tex_mip_enable != mip_on
{
    gpu_set_tex_mip_enable(mip_on);
}

O código acima verificará se o mipmapping está habilitado e se não estiver, ele o habilitará.
