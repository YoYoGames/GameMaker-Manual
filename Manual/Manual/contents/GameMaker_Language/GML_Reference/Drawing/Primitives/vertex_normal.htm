vértice_normal

Esta função adicionará dados normais de superfície ao vertex que está sendo definido atualmente para os primitivos personalizados. Você fornece o buffer para escrever os dados, assim como as partes componentes x, y e z do normal.

Sintaxe:

vertex_normal(buffer, nx, ny, nz);

Argumento Tipo Descrição
buffer Vertex Buffer ID The buffer to write the information to.
nx Real The x component of the surface normal.
ny Real The y component of the surface normal.
nz Real The z component of the surface normal.

Devoluções:

N/A

Exemplo:

vertex_normal(buff, 0, 1, 1);

O código acima definirá a superfície normal do vertex que está sendo definido.
