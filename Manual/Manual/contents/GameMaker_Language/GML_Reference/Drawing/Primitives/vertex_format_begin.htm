vértice_format_begin

Antes de poder definir seu novo formato vertex você deve dizer ao GameMaker que está fazendo isso usando esta função. Você deve chamar isto primeiro, depois definir os valores do formato usando as funções apropriadas, e finalmente chamar vertex_format_end() para terminar a definição e devolver o novo formato "handle".

Sintaxe:

vertex_format_begin();

Devoluções:

Exemplo:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

O código acima criará um novo formato vertex com apenas valores de cor e posição e então armazenará o id de formato na variável "my_format".
