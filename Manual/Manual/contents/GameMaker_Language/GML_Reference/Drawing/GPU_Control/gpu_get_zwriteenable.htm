gpu_get_zwriteenable

Esta função pode ser usada para recuperar se a escrita em z está habilitada (a função retorna true) ou não (a função retorna false). O valor padrão é que a escrita em z está ativada, portanto a função retorna true.

Sintaxe:

gpu_get_zwriteenable();

Devoluções:

Exemplo:

if gpu_get_zwriteenable() == false
{
    gpu_set_zwriteenable(true);
}

O código acima verifica se a escrita em z está habilitada ou não e se está desabilitada, então ela é habilitada novamente.
