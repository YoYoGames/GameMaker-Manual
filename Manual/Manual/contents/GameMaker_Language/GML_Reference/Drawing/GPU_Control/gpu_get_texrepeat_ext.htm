gpu_get_texrepeat_ext

Com esta função você pode verificar se a repetição texture está habilitada (retorna true) ou não (retorna false) para um determinado coletor de amostras shader texture .

Sintaxe:

gpu_get_texrepeat_ext(sampler_id);

ArgumentoTipo Descrição
sampler_id The sampler id from the shader.

Devoluções:

Exemplo:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if !gpu_get_texrepeat_ext(s_tex)
{
    gpu_set_texrepeat_ext(s_tex, true);
}

O código acima verifica se texture está filtrando para uma identificação específica do amostrador (armazenado em uma variável local) e o liga se não estiver.
