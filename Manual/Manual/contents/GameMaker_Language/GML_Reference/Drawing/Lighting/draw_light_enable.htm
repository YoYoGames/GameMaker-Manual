draw_enable_light_enable

Esta função é utilizada para permitir uma luz definida. Quando você define uma luz posicional ou direcional, você deve atribuir-lhe um número índice que é então utilizado por esta função para switch a luz acesa ou apagada. O padrão está desativado (false).

NOTA: Há apenas 8 luzes de hardware disponíveis, portanto apenas 8 luzes definidas podem ser habilitadas a qualquer momento (embora mais possam ser definidas).

Sintaxe:

draw_light_enable(ind, enable);

Argumento Tipo Descrição
ind Real The index number of the light (from 0 to 7)
enable Booleano Enable or disable all lighting (true ou false)

Devoluções:

N/A

Exemplo:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

O código acima permitirá a iluminação de toda a cena, então definirá uma luz direcional branca no espaço room, e então finalmente acenderá essa luz.
