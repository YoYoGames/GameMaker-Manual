draw_text_transformed

Esta função irá desenhar texto de forma semelhante a draw_text() somente agora você pode escolher escalar o texto ao longo do eixo horizontal ou vertical (efetivamente esticando-o ou encolhendo-o) e também ter O GameMaker desenha-o em um ângulo (onde 0 é normal e cada grau sobre 0 gira o texto no sentido anti-horário).

Sintaxe:

draw_text_transformed(x, y, string, xscale, yscale, angle);

ArgumentoTipo Descrição
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
xscale The horizontal scale (default 1).
yscale The vertical scale(default 1).
angle The angle of the text.

Devoluções:

Exemplo:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed(room_width / 2, room_height / 2, "GAME OVER!", 2, 2, image_angle);

O código acima desenhará o texto dado no meio do room, girando redondo e escalonado para o dobro de seu tamanho original.
