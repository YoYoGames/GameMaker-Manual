draw_set_halign

Esta função é utilizada para alinhar o texto ao longo do eixo horizontal e alterar o alinhamento horizontal mudará a posição e a direção na qual todo o texto adicional é desenhado, sendo o valor padrão fa_left. As seguintes constantes são aceitas:

Constante Alinhamento
fa_left
fa_center
fa_right

Sintaxe:

draw_set_halign(halign);

Argumento Tipo Descrição
halign Constante de Alinhamento Horizontal Horizontal alignment constant (from the table above).

Devoluções:

N/A

Exemplo:

draw_set_halign(fa_left);
draw_text(100, 32, "Score: " + string(score));
draw_set_halign(fa_right);
draw_text(room_width - 100, 32, "Health: " + string(health));

O código acima irá empatar dois strings na mesma linha, com a pontuação alinhada à esquerda e a saúde alinhada à direita.
