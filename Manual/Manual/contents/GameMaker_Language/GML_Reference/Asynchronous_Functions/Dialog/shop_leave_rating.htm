shop_leave_rating

Esta função abre um diálogo dependente do SO onde você pode pedir ao usuário para publicar uma classificação ou comentário em uma determinada página. Você pode definir o texto que deve aparecer no diálogo, assim como o texto que deseja aparecer nos dois botões e a URL onde o comentário deve ser postado.

NOTA: Esta função é apenas para alvos Android e iOS.

IMPORTANTE! No iOS, ao utilizar o TestFlight, em desenvolvimento, o prompt será exibido, mas você não poderá fazer nada. Uma vez que seu aplicativo esteja na App Store, este botão se torna ativo e o usuário tem a opção de escrever uma revisão ou deixar uma avaliação. Note que se você distribuir o jogo através do TestFlight, esta janela pode nem mesmo ser mostrada.

Sintaxe:

shop_leave_rating(text, yes_string, no_string, url)

Argumento Tipo Descrição
text Cordão Text that appears on the dialog.
yes_string Cordão Text that appears on the "yes" button.
no_string Cordão Text that appears on the "no" button.
url Cordão The full URL where the comment has to be sent.

Devoluções:

N/A

Exemplo:

if timer <= 0
{
    shop_leave_rating("Thanks for playing the game! Why not leave a comment?", "Okay!", "Not Today!", "http://MacSweeney/comments");
    timer = 10000000;
}
else
{
    timer -= 1;
}

O código acima pedirá ao usuário que deixe um comentário se a variável "timer" tiver contado até 0.
