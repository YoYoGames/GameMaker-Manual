achievement_post_score

Você pode usar esta função para enviar sua pontuação para o quadro de líderes escolhido e serviço de realizações. Em Android, você envia o ID da tabela de pontuação como um string (este é o ID exclusivo do quadro de líderes que foi designado quando você montou o quadro de líderes), enquanto em todas as outras plataformas que você fornece o nome do quadro de líderes definido como um string, e então você dá o valor real da pontuação. A função desencadeará um Evento Assíncrono Social onde o retorno async_load O mapa DS terá os seguintes pares chave/valor:

  • "tipo" - Este é o tipo de evento que foi disparado, que será o string "achievement_post_score_resultado" para esta função.
  • "id" - Este é o ID do evento, e, para esta função, ele retornará uma constante GML GooglePlayServices_EVENT_ID_POST_SCORE.
  • "leaderboard_id" - Esta é a identificação única do leaderboard que foi atualizada.
  • "status" - Este será 1 se a pontuação for atualizada com sucesso, ou 0 se a solicitação falhar.
  • "daily_new_best" - Este será 1 se a pontuação apresentada foi a melhor do usuário no último dia, ou 0 caso contrário.
  • "daily_best" - Isto manterá a pontuação máxima do usuário desde o último dia de jogo.
  • "semanal_new_best" - Este será 1 se a pontuação apresentada foi a melhor do usuário na última semana, ou 0 caso contrário.
  • "semanal_best" - Isto detém a pontuação máxima do usuário da última semana.
  • "all_time_new_best" - Este será 1 se a pontuação apresentada foi a melhor de todos os tempos, ou 0 caso contrário.
  • "all_time_best" - Isto detém a pontuação máxima do usuário de todos os tempos.

Sintaxe:

achievement_post_score(leaderboard, score)

Argumento Descrição
leaderboard O nome/ID do quadro de líderes.
score A pontuação a ser enviada.

Devoluções:

N/A

Exemplo ampliado:

if achievement_available()
    {
    achievement_post_score(global.LeaderboardID, 1000);
    }

O código acima verifica se o quadro de classificação e o serviço de realizações escolhido está disponível e, em caso afirmativo, publica uma pontuação na tabela de pontuação chamada "Quadro de Classificação". Isto desencadeará um Evento Social Async que pode ser analisado desta forma:

if async_load[? "id"] == GooglePlayServices_EVENT_ID_POST_SCORE
    {
    if async_load[? "type"] == "achievement_post_score_result"
        {
        if async_load[? "status"] == 1
            {
            global.Best_Day = async_load[? "daily_best"];
            global.Best_Week = async_load[? "weekly_best"];
            global.Best = async_load[? "all_time_best"];
            }
        }
    }

O código acima verifica o mapa DS async_load para as chaves "id" e "type" que correspondem a esta função, e então verifica o status do evento. Se o status for 1, então alguns valores do mapa DS são armazenados em variáveis.
