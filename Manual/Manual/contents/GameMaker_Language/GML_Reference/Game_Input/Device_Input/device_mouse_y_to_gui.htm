dispositivo_mouse_y_to_gui

Esta função retorna a posição y do toque no dispositivo em relação à camada GUI. Como a camada GUI pode ser definida para ser um tamanho diferente independentemente da resolução da tela e do tamanho room, às vezes pode ser complicado obter o ponto exato na camada GUI em que um usuário tocou a tela. Com esta função é possível obter essa posição com base no tamanho real da GUI, tornando os botões GUI e elementos HUD mais fáceis de criar e interagir. Se você estiver executando isto em um módulo HTML5 ou PC e Mac, então este valor é atualizado constantemente, desde que o dispositivo (geralmente um mouse) esteja conectado, porém para dispositivos móveis, isto só será atualizado enquanto a tela estiver sendo tocada, e note que o número máximo de toques que podem ser detectados dependerá muito do dispositivo que estiver sendo usado e do sistema operacional que estiver rodando.

Sintaxe:

device_mouse_y_to_gui(device);

ArgumentoTipo Descrição
device The device (from 0 - n) that is being checked.

Devoluções:

Exemplo:

if device_mouse_check_button(0, mb_left)
{
    if device_mouse_y_to_gui(0) > 0 && device_mouse_y_to_gui(0) < 32
    {
        pressed = true;
    }
    else
    {
        pressed = false;
    }
}

O código acima verifica se o mouse do dispositivo está sendo pressionado e, em caso afirmativo, ele então sonda a posição x do dispositivo na camada GUI para ver se ele está dentro dos parâmetros. Se for, ele ajusta a variável "pressionada" para true, outros ajustes são feitos para false.
