gamepad_is_supported

Com esta função, você pode descobrir se a plataforma alvo suporta Game Pads (retorna true) ou não (retorna false).

Sintaxe:

gamepad_is_supported();

Devoluções:

Exemplo:

global.GP = gamepad_is_supported();

O código acima verifica se um controle de jogo é suportado e armazena o valor de retorno em uma variável global para verificações futuras.
