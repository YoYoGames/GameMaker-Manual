gamepad_get_option

Esta função pode ser usada para recuperar o valor da opção dada. Você fornece o "slot" do controle de jogo para verificar, assim como a opção string e a função retornará o valor associado. A opção disponível string dependerá da plataforma na qual o projeto está sendo executado, conforme listado abaixo:

Opção Plataforma Descrição Devoluções
"allow_rotation" (permitir_rotação) tvOS Retorna se o Siri Remote será bloqueado por rotação (ou seja: só pode ser usado verticalmente) ou não. Quando destravado, a rotação do controle mudará a entrada, de tal forma que uma rotação de 90º para a esquerda fará com que o lado "direito" do touch pad retorne "para cima", etc... de tal forma que toda a entrada será relativa à orientação do dispositivo Booleano
"dpad_absolute" tvOS Retorna se o touchpad Siri Remote aceita entrada absoluta, ou entrada relativa. No modo absoluto, o centro do touchpad é considerado a posição (0,0) - ou seja: onde o bastão analógico está em repouso ou onde estaria o centro do dpad - e o movimento em torno desta posição gerará entrada. Quando definida como relativa, então a posição (0,0) será considerada como estando onde quer que o toque inicial no pad seja feito e toda a entrada será gerada a partir dessa posição, não o centro do pad. Booleano

Sintaxe:

gamepad_get_option(device, option_name);

ArgumentoTipo Descrição
device Which gamepad device "slot" to check.
option_name The name of the option to get the value from (a string, see the table above)

Devoluções:

Boolean, Real, String, or N/A (depends on platform and option)

Exemplo:

for(var i = 0; i < 12; ++i;)
{
    if gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote"
    {
        if gamepad_get_option(i, "allow_rotation") == false
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

O código acima loops através de todos os slots e verificações para o "Siri Remote" na plataforma tvOS. Se for detectado, ele então define o controle remoto para permitir a rotação de entrada.
