gamepad_hat_count

Esta função devolverá o número total de chapéus disponíveis para o gamepad conectado ao "slot" do dispositivo em questão. Os chapéus geralmente se referem aos botões acima/abaixo/esquerda/direita, e note que no alvo Windows, os chapéus só estão disponíveis no DirectInput controladores (portanto, do slot 4 para cima).

Sintaxe:

gamepad_hat_count(device);

ArgumentoTipo Descrição
device Which gamepad device "slot" to check.

Devoluções:

Exemplo:

h_num = gamepad_hat_count(4);

O código acima retornará o número de chapéus disponíveis no controle de jogo conectado ao dispositivo "slot" 4 e armazenará o valor na variável "h_num".
