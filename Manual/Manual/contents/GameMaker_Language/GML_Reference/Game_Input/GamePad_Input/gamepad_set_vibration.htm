gamepad_set_vibration

Com esta função você pode ajustar a vibração dos motores gamepad, com qualquer dos motores usando um valor de 0 (sem vibração) a 1 (vibração total). Note que não há limite de tempo nesta função, portanto você precisará usar uma variável ou um alarme para switch fora da vibração (ajustar motores a 0) depois de um determinado tempo, caso contrário o controle de jogo continuará a vibrar indefinidamente.

NOTA: Esta função está atualmente disponível apenas para os módulos-alvo padrão Windows, PS4 e Xbox One.

Sintaxe:

gamepad_set_vibration(device, left_motor, right_motor);

ArgumentoTipo Descrição
device Which gamepad device "slot" to check.
left_motor The amount of vibration from the left motor from 0 to 1.
right_motor The amount of vibration from the right motor from 0 to 1.

Devoluções:

Exemplo:

if lives = 0
{
    gamepad_set_vibration(0, 1, 1);
    alarm[0] = room_speed / 2;
}

O código acima seria usado (por exemplo) em um evento de colisão para fazer com que o controle de jogo fosse ligado no "slot" 0 vibrar por meio segundo, com o alarme que está ajustado sendo usado para switch uma vez que o tempo tenha passado.
