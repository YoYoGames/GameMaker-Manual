altura_virtual_do_ teclado

Esta função retornará a altura atual em pixels do teclado virtual, com base no tamanho da tela. Se o teclado não for visível, 0 será retornado.

Sintaxe

keyboard_virtual_height();

Devoluções:

Exemplo:

if keyboard_virtual_status() == true
{
    key_h = keyboard_virtual_height();
}

O código acima verificará o status do teclado virtual do SO, e se estiver visível, definirá uma variável para a altura do teclado.
