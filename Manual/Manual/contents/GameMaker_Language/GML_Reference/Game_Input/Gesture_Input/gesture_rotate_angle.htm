gesto_rotate_angle

Esta função é usada para definir o ângulo que um par de toques deve exceder para acionar um gesto de partida rotativo. O ângulo é medido em graus e tem um valor padrão de 5°.

Sintaxe:

gesture_rotate_angle();

Devoluções:

(inches)

Exemplo:

if gesture_get_rotate_angle() != 5
{
    gesture_rotate_angle(5);
}

O código acima verifica se o ângulo de rotação dos gestos está ajustado a 5° e se não está ajustado a esse valor.
