teclado_get_map

Às vezes você pode desejar obter o código ascii para uma chave mapeada (para ver se ela já está mapeada, por exemplo), que é o que esta função retorna. A função terá um valor de código de chave como retornado pela função ord() (somente letras maiúsculas de A-Z ou números de 0-9), ou qualquer uma das constantes vk_* listadas na página principal de Entrada de Teclado.

Sintaxe:

keyboard_get_map(key);

ArgumentoTipo Descrição
key This is the key that you wish to get the mapped value from

Devoluções:

Exemplo:

if keyboard_get_map(ord("A")) = ord("A")
{
    keyboard_set_map(ord("A"), keyboard_lastkey);
}

O código do exemplo acima vai primeiro verificar e ver se "A" foi mapeado para outra tecla, e se não foi, vai mapeá-lo para a última tecla que o usuário pressionou.
