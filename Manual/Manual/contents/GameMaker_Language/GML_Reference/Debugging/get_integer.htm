get_integer

Isto cria uma janela pop-up mostrando uma mensagem personalizada, com um botão rotulado "Ok", e solicita que o usuário insira um valor inteiro. A função retornará o número inteiro digitado, ou o valor padrão, se nada tiver sido inserido.

NOTA Esta função é para uso de depuração apenas no alvo Windows, mas é depreciada em todos os outros alvos.

Sintaxe:

get_integer(str, def);

Argumento Tipo Descrição
str Cordão The string to show in the pop-up message.
def Real The default value in the text box.

Devoluções:

Real

Exemplo:

global.level = get_integer("Level to test?", 1);

O código acima exibirá uma mensagem solicitando ao usuário que selecione um nível para testes. O valor de retorno será armazenado na variável global "global.level".
