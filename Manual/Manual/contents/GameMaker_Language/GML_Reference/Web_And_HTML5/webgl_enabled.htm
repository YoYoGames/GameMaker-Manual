webgl_enabled

Esta variável somente leitura retornará se a WebGL estiver habilitada (true) ou não (false) para seu jogo. Ela só funcionará para aqueles jogos rodando através de um navegador (ou seja: HTML5), e para todas as outras plataformas ela retornará verdadeiro.

Sintaxe:

webgl_enabled

Devoluções:

Exemplo:

if webgl_enabled
{
    global.quality = 1;
}
else global.quality = 0;

O código acima verifica o WebGL flag e então define a variável global "qualidade" de acordo.
