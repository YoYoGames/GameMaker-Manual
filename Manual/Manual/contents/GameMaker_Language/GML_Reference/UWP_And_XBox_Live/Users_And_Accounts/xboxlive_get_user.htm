xboxlive_get_user

Com esta função você pode recuperar o ponteiro de identificação do usuário para o usuário indexado. Se o usuário não existir, a função retornará a constante pointer_null em seu lugar. Você pode encontrar o número de usuários atualmente logados com a função xboxlive_get_user_count().

IMPORTANTE: Esta função só deve ser utilizada com xboxlive_get_user_count() - não a utilize com controles de jogo. Use xboxlive_user_for_pad() em seu lugar.

Sintaxe:

xboxlive_get_user(index);

Argumento Tipo Descrição
index Real The index (an integer) to get the User ID from.

Devoluções:

Xbox User ID or pointer_null

Exemplo:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

O acima loops através de todos os usuários assinados e armazena seu ponteiro de identificação único em uma matriz.
