xboxlive_matchmaking_session_leave

Esta função faz com que o usuário que criou ou encontrou a sessão especificada saia dela. Você pode obter o ponteiro de identificação da sessão a partir do async_load Mapa DS que é retornado no Evento Social Assíncrono quando você criou ou encontrou uma sessão (veja xboxlive_matchmaking_create() para mais detalhes).

Sintaxe:

xboxlive_matchmaking_session_leave(session_id);

ArgumentoTipo Descrição
session_id The Session ID pointer of the session to leave.

Devoluções:

Exemplo:

if gamepad_button_check_pressed(0, gp_start)
{
    xboxlive_matchmaking_session_leave(global.SessionID);
}

O código acima verifica se há um botão de controle de jogo pressionado e, se for detectado, o usuário sairá da sessão de matchmaking.
