xboxlive_achievement_load_leaderboard

Esta função enviará um pedido ao server para obter informações sobre o quadro de líderes em questão. O quadro de líderes deve ter sido criado anteriormente no painel de desenvolvedores para seu jogo e o ID dado é o que você deve fornecer como primeiro argumento da função. Ele acionará um Evento Assíncrono Social de retorno que contém o async_load mapa povoado com os pares chave/valor relevantes. A chave de id deste DS Map é usada para identificar o retorno de chamada correto (pode haver mais de uma função de acionamento para qualquer evento assíncrono), e será emparelhada com a constante xboxlive_achievement_leaderboard_info, bem como um número de outros pares de chaves/valores para cada jogador. O conteúdo exato do mapa é mostrado abaixo:

  • "id" - Para esta função deve ser xboxlive_achievement_leaderboard_info
  • "leaderboardid" - A identificação única para o leaderboard, conforme definido no painel do fornecedor.
  • "numentries" - O número de entradas no quadro de líderes que você recebeu.
  • "PlayerN" - O nome do jogador, onde "N" é um valor inteiro correspondente à sua posição dentro da lista de entradas do quadro de líderes.
  • "PlayeridN" - A identificação única do usuário do jogador, "N".
  • "RankN" - A classificação do jogador "N" dentro da tabela de liderança.
  • Pontuação "N"- A pontuação do jogador "N".

Você pode especificar a porção exata do quadro de líderes a mostrar, dando um valor de índice mínimo e máximo, bem como solicitar que os resultados sejam filtrados para recuperar somente as pontuações de seus amigos ou aquelas postadas por todos os jogadores usando as seguintes constantes de filtro:

Constante do filtro Xbox Live Achievement
Constante Descrição
xboxlive_achievement_filter_friends_only  Obtenha apenas as pontuações dos amigos jogadores.
xboxlive_achievement_filter_all_players  Obtenha todas as pontuações para todos os jogadores.

Sintaxe:

xboxlive_achievement_load_leaderboard(ident, minindex, maxindex, filter)

Argumento Tipo Descrição
ident Cordão The unique ID of the leaderboard as shown on the developer dashboard.
minindex Real The starting index value to get the leaderboard data from.
maxindex Real The maximum index value to get the leaderboard data from.
filter Constante do filtro Xbox Live Achievement Whether to filter the results to those that are on your friends list or to all players (see the Description above)

Exemplo ampliado:

O seguinte código provavelmente seria chamado depois que o jogador entrasse em sua conta de jogo para obter uma lista de todas as posições do quadro de líderes de amigos para o jogo:

xboxlive_achievement_load_leaderboard("CgkIs9_51u0PEAIQBw", 1, 100, xboxlive_achievement_filter_friends_only);

Isto enviará um pedido de informações sobre a tabela de liderança e gerará uma chamada assíncrona de retorno com o mapa especial async_load DS contendo os seguintes dados:

var ident = ds_map_find_value(async_load, "id");
if ident == xboxlive_achievement_leaderboard_info
{
    var lbid = ds_map_find_value(async_load,"leaderboardid");
    if lbid == "CgkIs9_51u0PEAIQBw" || lbid == "leaderboard1id"
    {
        global.numentries = ds_map_find_value(async_load,"numentries");
        for(var i = 0; i < numentries; i++;)
        {
            global.playername[i] = ds_map_find_value(async_load, "Player" + string(i));
            global.playerid[i] = ds_map_find_value(async_load, "Playerid" + string(i));
            global.playerrank[i] = ds_map_find_value(async_load, "Rank" + string(i));
            global.playerscore[i] = ds_map_find_value(async_load, "Score" + string(i));
        }
    }
}

O código acima verifica o retorno DS Map no Evento Assíncrono Social e se seu "id" corresponde à constante a ser verificada, então verifica se o quadro de líderes a ser enviado corresponde à chamada de função original antes de percorrer o mapa e armazenar todos os diferentes valores em uma série de arrays.
