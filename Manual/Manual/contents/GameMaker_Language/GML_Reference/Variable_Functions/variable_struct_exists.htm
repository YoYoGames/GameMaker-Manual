variável_estruto_existe

Com esta função você pode verificar se existe ou não uma variável dentro do struct dado. Você fornece a referência struct a ser usada, bem como o nome da variável a ser verificada como uma string (veja o código de exemplo abaixo). A função retornará true se uma variável com o nome dado existir para os endereços struct e false caso contrário.

Sintaxe:

variable_struct_exists(struct, name);

Argumento Tipo Descrição
struct Estrutura The struct reference to check
name Cordão The name of the struct variable to check for (as a string)

Devoluções:

Boolean

Exemplo:

if !variable_struct_exists(mystruct, "shields")
{
    mystruct.shields = 0;
}

O código acima verificará se a variável chamada "shields" existe no dado struct e se não existe, então ela é criada e rubricada a 0.
