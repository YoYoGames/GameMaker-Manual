array_delete

Com esta função você pode apagar um valor (ou valores) de uma matriz em qualquer posição. A função requer que você forneça uma variável que contenha o array, o índice (posição) do array a ser excluído, bem como o número de valores a serem excluídos.

Sintaxe:

array_delete(variable, index, number);

ArgumentoTipo Descrição
variable The variable that holds the array.
index The index (position) in the array to delete the value(s) from.
number The number of values to delete.

Devoluções:

Exemplo:

array_delete(score_array, 0, 10);

O código acima apagará os primeiros 10 valores do array dado começando no índice 0 (assim os índices 0 a 9 serão removidos do array).
