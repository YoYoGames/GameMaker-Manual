variável_struct_get

Com esta função você pode obter o valor de uma determinada variável nomeada dentro de um struct. Você fornece a referência struct assim como o nome da variável para obter o valor de uma string (veja o código de exemplo abaixo). A função retornará o valor mantido pela variável ou undefined se a variável nomeada não existir.

IMPORTANTE! Se a variável que você está recebendo não existir, então a função retornará a palavra-chave undefined e você poderá receber erros que impedirão o funcionamento do jogo, portanto, em caso de dúvida, use a função variable_struct_exists() primeiro.

Sintaxe:

variable_struct_get(struct, name);

ArgumentoTipo Descrição
struct The struct reference to use
name The name of the variable to get (as a string)

Devoluções:

Variable (any data type) or undefined (if the named variable does not exist)

Exemplo:

if variable_struct_exists(mystruct, "shields")
{
    var ss = variable_struct_get(mystruct, "shields");
}
else
{
    var ss = -1;
}

O código acima verificará se uma variável existe no dado struct e se existe, então o valor que ela detém é recuperado e armazenado em uma variável local. Se não existir, a variável local é definida como -1.
