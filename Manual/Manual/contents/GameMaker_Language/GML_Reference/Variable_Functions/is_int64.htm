is_int64

Esta função retorna se uma determinada variável é um inteiro de 64 bits ou não. Em alguns casos, você quer verificar e ver que tipo de dado uma variável contém no GameMaker e é quando você usaria esta função. Ela retorna true ou false, dependendo se o valor é um int64 ou não.

Sintaxe:

is_int64(n);

Argumento Tipo Descrição
n Variável The argument to check.

Devoluções:

Boolean

Exemplo:

if !is_int64(val)
{
    show_debug_message("Not a 64bit integer!");
}

O código acima verifica a variável "val" para ver se ela contém uma int64 e se não contém, então ela mostra uma mensagem no console de depuração.
