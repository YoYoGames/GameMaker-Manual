time_source_stop

Esta função interrompe a Fonte de Tempo dada e restabelece seu timer. Você não pode parar uma Fonte de Tempo que ou não começou, ou está em um estado acabado.

Sintaxe:

time_source_stop(id);

Argumento Tipo Descrição
id Identificação da fonte de tempo The Time Source para parar

Devoluções:

N/A

Exemplo:

if (time_source_get_state(time_source) != time_source_state_stopped)
{
    time_source_stop(time_source);
}

O código acima irá parar uma Fonte de Tempo quando ela ainda não tiver parado.
