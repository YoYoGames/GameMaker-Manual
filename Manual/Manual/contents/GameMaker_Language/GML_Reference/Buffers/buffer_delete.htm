buffer_delete

Com esta função você pode excluir um buffer criado anteriormente usando a função buffer_create() da memória, liberando os recursos utilizados para criá-la e removendo quaisquer dados que ela possa conter atualmente.

OBSERVAÇÃO: É importante sempre remover da memória quaisquer recursos criados dinamicamente quando você não precisa mais deles para evitar vazamentos de memória.

IMPORTANTE! Quando você cria um buffer, o valor do índice usado para identificá-lo é um valor inteiro começando em 0. Estes índices são reutilizados por GameMaker, de modo que um valor de índice destruído buffer pode ser usado por um índice recém-criado posteriormente, e recomendamos que você defina qualquer variável que contenha um índice tampão para -1 após ter destruído o buffer.

Sintaxe:

buffer_delete(buffer)

ArgumentoTipo Descrição
buffer The index of the buffer to delete.

Devoluções:

Exemplo:

buffer_delete(player_buffer);
player_buffer = -1;

O código acima apagará o código anteriormente criado buffer com o id armazenado na variável "player_buffer" e então define a variável como -1.
