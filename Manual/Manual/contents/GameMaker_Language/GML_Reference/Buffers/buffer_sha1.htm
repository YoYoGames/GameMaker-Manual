buffer_sha1

Na criptografia, SHA-1 é uma função de hashing criptográfica projetada pela Agência de Segurança Nacional dos Estados Unidos e é empregada em várias aplicações e protocolos amplamente utilizados, como o popular Git onde é usado para verificar mudanças de arquivo. Esta função irá pegar os dados de entrada de um buffer e retorna um resumo de 160 bits de mensagem em formato ASCII. Desta forma, você pode gerar uma chave segura que pode ser armazenada e usada para verificar a integridade das informações enviadas para (ou recebidas de) um server externo (por exemplo). Ao aplicar isto a buffers usando esta função, você deve especificar o id buffer do buffer para usar, depois um valor de offset (em bytes) para onde começar, e então um tamanho (novamente em bytes) para a região a ser apressada.

NOTA: O SHA-1 não é completamente seguro e pode ser quebrado. Veja esta página para mais informações.

Sintaxe:

buffer_sha1(buffer, offset, size);

ArgumentoTipo Descrição
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Devoluções:

Exemplo:

check_string = buffer_sha1(buff, 0, buffer_get_size(buff));

O código acima criará um hash sha1 para os dados completos armazenados no buffer indexado pela variável "buff", e armazenará o hash retornado na variável "check_string".
