buffer_base64_decode

Com esta função você pode decodificar uma base64 codificada string (criada usando o buffer_base64_encode() buffer Esta função criará o buffer (como um buffer de "crescimento" alinhado a 1 byte) e retornará o índice único para o buffer que deve ser usado em todas as outras chamadas de função.

NOTA: É importante que você remova da memória qualquer recurso criado dinamicamente como este quando não precisar mais deles para evitar vazamentos de memória, então quando você terminar de usar o buffer que você criou você deve liberá-lo novamente usando buffer_delete().

Sintaxe:

buffer_base64_decode(string);

ArgumentoTipo Descrição
string The base64 encoded string to decode

Devoluções:

Buffer ID

Exemplo:

ini_open("Save.ini");
buff = buffer_base64_decode(ini_read_string("Save", "Slot1", ""));
ini_close();

O código acima abrirá um arquivo ini e então lerá um string dele para a função de decodificação. Esta função irá retornar um índice buffer, que é armazenado na variável "buff", contendo os dados previamente codificados e salvos. O arquivo ini é então fechado.
