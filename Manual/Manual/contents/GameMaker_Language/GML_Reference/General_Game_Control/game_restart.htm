jogo_restart

Com esta função, você pode reiniciar o jogo. Isto é essencialmente o mesmo que executar o jogo pela primeira vez e assim o Evento de Início do Jogo será acionado, assim como o Evento de Fim do Jogo.

Deve-se notar que certas coisas não serão reiniciadas quando esta função for chamada:

  • As variáveis globais não serão reinicializadas a menos que sejam explicitamente codificadas como tal - por exemplo, a variável global incorporada score não começará a zero após um reinício do jogo se já tiver sido modificada no jogo.
  • O estado da GPU não será alterado (portanto, se você tiver definido a cor do sorteio ou alfa, por exemplo, ela permanecerá no valor alterado).
  • A velocidade do jogo permanecerá no que quer que você tenha definido em seu código de jogo (se você o alterar, esta alteração será perpetuada).
  • Qualquer asset do navegador Asset que tenha sido alterado em tempo de execução dentro do jogo - por exemplo, se você mudar a origem para um recurso sprite ou mudar a posição de um recurso path - não será redefinido.
  • Recursos dinâmicos como buffers, superfícies, estruturas de dados ou importados sprites também não serão limpos ou removidos (embora você possa perder referências a eles, portanto tenha cuidado ao usar esta função para usar referências globais para o recurso dinâmico, ou para limpá-los antes que a função seja chamada).

Sintaxe:

game_restart();

Devoluções:

N/A

Exemplo:

if keyboard_check_pressed(ord("R")) game_restart();

Isto reiniciaria o jogo quando o jogador pressionasse a tecla "R".
