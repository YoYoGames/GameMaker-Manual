Características lingüísticas

O GameMaker Language (GML) tem uma série de características que fazem parte da estrutura essencial de todos os programas escritos com a língua. Cada um deles será usado em qualquer código que você escrever várias vezes, portanto você deve saber o que são e como funcionam antes de começar a programar um projeto.

  • repeat
  • while
  • do / until
  • for
  • switch
  • break
  • continue
  • exit
  • with
  • return
  • begin / end
  • try / catch / finally
  • throw
  • new
  • delete
