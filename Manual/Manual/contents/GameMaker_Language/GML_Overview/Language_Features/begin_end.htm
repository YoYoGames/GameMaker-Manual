início / fim

O GameMaker Language permite que você utilize as palavras-chave begin e end em vez dos parênteses mais usuais {} ao criar blocos de código. O exemplo de código abaixo mostra como isto funciona:

if (!visible)
    begin
    exit;
    end
while (place_meeting(x, y))
    begin
    x -= lengthdir_x(1, direction - 180);
    y -= lengthdir_y(1, direction - 180);
    end

Note que o uso dessas palavras-chave não é típico e é fornecido como parte do idioma mais para suporte legado do que para qualquer outra coisa, e a qualquer momento no futuro elas podem ser depreciadas.
