Ir para Sala Anterior

Esta ação fará com que seu jogo vá para o room anterior, conforme listado no Gerente de Sala no momento em que o jogo foi compilado. Se este room não existir, um erro será lançado e o jogo será forçado para fechar. Note que o room não mudará até o final do evento ou script onde a ação foi chamada, portanto, qualquer código após esta ação ainda será executado.
