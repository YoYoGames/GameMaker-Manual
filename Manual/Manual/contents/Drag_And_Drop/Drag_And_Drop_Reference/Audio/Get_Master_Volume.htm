Obter volume mestre

Esta ação devolve o volume mestre para o sistema de som. Você dá uma variável alvo para retornar o valor do volume para (você pode flag isto como sendo uma variável local temporária). O valor retornado será entre 0 e 1, onde 0 é silencioso e 1 é completo. volume.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Target A variável alvo para armazenar o volume retornado

Exemplo:

O código de bloco de ação acima verifica uma variável global para ver se ela é true ou falso. Se for true então o volume mestre para o sistema de som é recuperado e armazenado em uma variável local temporária. O valor é então verificado para ver se não é igual a 0,5, e se não for, então o volume mestre do sistema de som está ajustado para 0,5. Se a variável global for avaliada para false, então o volume mestre é ajustado para 1.
