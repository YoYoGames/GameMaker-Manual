Remover entrada do mapa

Com esta ação você pode remover uma chave (e seu valor correspondente) de uma estrutura de dados cartográficos. Você fornece a variável que contém o índice do mapa (como retornado pela ação Criar Mapa) e então dá uma "chave" (que é o identificador dentro do mapa para um valor), e a ação removerá essa chave do mapa. Note que esta ação não retorna nada, e se você precisar do valor para a chave dada, você deve usar a ação Obter Valor do Mapa antes de removê-la.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Map O índice (armazenado em uma variável) do mapa para remover o valor de
Key A chave a usar (real ou string)
