Obtenha Valor do Mapa

Com esta ação você pode recuperar o valor associado a uma determinada chave em uma estrutura de dados cartográficos. Você fornece a variável que contém o índice do mapa (como retornado pela ação Criar Mapa) e então dá uma "chave" (que é o identificador dentro do mapa para um valor), e também fornecer uma variável alvo para manter o valor retornado (que pode ser sinalizada como uma variável local temporária).

NOTA: Se você der uma chave que não existe no mapa, a ação retornará undefined. Você pode verificar isso usando a ação If Undefined.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Map O índice (armazenado em uma variável) do mapa para obter o valor de
Key A chave para obter (real ou string)
Target A variável alvo para manter o valor do mapa retornado

Exemplo:

O código de bloco de ação acima verifica se há uma colisão na posição da instância e se um é encontrado, então um mapa é procurado para obter um valor da chave "dano", e este valor é então usado para mudar a variável de instância de chamada "hp".
