Estrutura de dados clara

Esta ação limpará todos os valores a partir da estrutura de dados fornecida. Você deve primeiro selecionar o tipo de estrutura de dados para limpar e depois dar o índice da estrutura de dados (o índice é o valor que é retornado e armazenado em uma variável quando você criou a estrutura de dados originalmente). Uma estrutura de dados limpa é considerada "vazia" (ou seja: não tem entradas), exceto as grades. Com as grades esta ação irá simplesmente limpar todas as células da grade para 0.

NOTA: Isto não liberta a estrutura de dados da memória e apenas a limpa de qualquer informação. Para liberar a memória atribuída à estrutura de dados, deve-se usar a ação Free Data Structure (Estrutura de dados livre).
