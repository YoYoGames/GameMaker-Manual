Jogo de Carga

Esta ação irá carregar um jogo que foi salvo anteriormente usando a ação Salvar Jogo. Você dá o nome do arquivo para carregar (como um string, e incluindo o extension) e o jogo será carregado e continuará rodando. a partir do ponto salvo. Note que isto não foi projetado como um sistema universal de salvar/carregar, e tentar carregar um jogo salvo de uma execução anterior do projeto pode dar erros (especialmente se você tiver usado coisas como Estruturas de Dados ou Partículas), e como tal, você só deve tentar carregar jogos que foram salvos no progresso de um único jogo (para coisas como pontos de verificação). Observe também que o arquivo não será carregado até o final do evento atual ou script, portanto, quaisquer ações após a ação de carregamento ainda serão executadas. Para uma abordagem mais abrangente para carregar dados de jogo salvos, veja o Arquivo de Ações.
