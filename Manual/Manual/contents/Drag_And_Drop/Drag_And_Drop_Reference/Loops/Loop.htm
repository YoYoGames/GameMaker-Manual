Loop

A ação de Loop irá loop infinitamente até que uma ação de Break seja chamada. Deve-se tomar cuidado com esta ação para garantir que, independentemente das ações colocadas dentro do loop, uma Pausa será sempre chamada, caso contrário o jogo causará um erro de "não resposta" no sistema operacional alvo e deixará de funcionar. Você adiciona esta ação no código de ação workspace e depois adiciona as ações a serem executadas dentro do loop, assim como adicionaria uma ação a um "se", ou seja: deixá-la cair para o lado da ação em vez de sob ela:
