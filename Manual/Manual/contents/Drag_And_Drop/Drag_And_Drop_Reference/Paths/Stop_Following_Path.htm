Pare de seguir o caminho

Esta ação encerrará o movimento ao longo do atual path que a instância está seguindo.

Sintaxe de ação:

Exemplo:

O código de bloco de ação acima verifica o valor de um alarme e se ele for inferior a 0 a Switch é verificado. Dependendo do string armazenado na variável da instância que é passada para o switch, a instância parará o atual path em que está ligada, ou iniciará um novo path antes de ajustar o alarme novamente.
