Obtenha Path Siga a velocidade

Esta ação terá a velocidade, em pixels por quadro de jogo, de uma instância seguir um path (note que você deve ter chamado a ação Start Following Path para atribuir um path à instância para que esta devolver um valor diferente de 0). O valor retornado será armazenado na variável de destino que você der, ou você pode flag a variável como uma variável local temporária para criá-la e atribuir o valor.
