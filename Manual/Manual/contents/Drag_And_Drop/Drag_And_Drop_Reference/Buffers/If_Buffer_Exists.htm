Se Buffer existir

Esta ação pode ser usada para verificar se um determinado ID buffer é válido e o buffer realmente existe na memória. Se o ID buffer for válido e existir, a ação retornará true caso contrário, retornará false. Você pode flag "não". para fazer a verificação da ação, caso não exista um buffer.
