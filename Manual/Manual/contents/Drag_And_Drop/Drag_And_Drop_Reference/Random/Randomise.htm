Randomise

Esta ação define a "semente" para o gerador de números aleatórios para um valor aleatório. Ao utilizar as funções de número aleatório no GameMaker, a semente inicial é sempre a mesma, pois isso torna os erros de rastreamento e debugging muito mais fáceis (compilados jogos não têm este comportamento). Caso você deseje testar com verdadeiro aleatório, você deve chamar esta ação uma vez no início do jogo.
