Definir o Estado da Janela

Com esta ação você pode definir o estado da janela do jogo para tela cheia ou com janela. Há três opções aqui:

  • Toggle - Quando esta opção é selecionada, cada vez que você chamar a ação ela irá alternar a janela entre os estados de tela cheia e janela
  • Tela cheia - Isto fará com que a janela fique em tela cheia a partir de um estado de janela
  • Janela - Isto fará a janela voltar a um estado de janela se em tela cheia

Observe que a aparência do jogo enquanto estiver em tela cheia dependerá das Opções do jogo - Configurações gráficas para as plataformas alvo, pois se o jogo não tiver algum mecanismo de ajuste a diferentes proporções de aspecto e resoluções você pode obter alongamentos (se a opção de jogo estiver definida para Escala Total), ou você pode obter a adição de barras pretas (se a opção de jogo estiver definida para Manter a Proporção de Aspecto). Em geral, se você não tiver nada no lugar para lidar com diferentes resoluções, você deve estar usando Keep Aspect Ratio para as plataformas alvo para as quais você deseja compilar.
