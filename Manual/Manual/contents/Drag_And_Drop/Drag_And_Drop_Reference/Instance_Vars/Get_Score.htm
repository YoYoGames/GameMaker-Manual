Obter pontuação

Você pode usar esta ação para recuperar o valor da variável de instância criada para a "pontuação". Você fornece a variável de destino para armazenar o valor da "pontuação" retornada (que pode ser marcada como uma variável local temporária), mas observe que se a variável não tiver sido definida anteriormente (usando Set Score) então a variável será criada para você e a ação retornará 0 (ao contrário das variáveis de instância regular, que darão um erro se você tentar acessá-las sem tê-las definido primeiro).
