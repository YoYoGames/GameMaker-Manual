Arquivo de cópia

Com esta ação, você pode copiar qualquer arquivo que tenha sido salvo por seu jogo ou que tenha sido empacotado como parte dos Arquivos Incluídos. Você dá o nome do arquivo a copiar (como um string) e depois o nome do arquivo a criar como uma cópia do arquivo original (também como um string e incluindo o arquivo extension). O dado arquivo será duplicado e receberá o novo nome.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Filename O nome (como um string) do arquivo a ser copiado
Copy To O nome (como um string) do arquivo a ser criado como uma cópia
