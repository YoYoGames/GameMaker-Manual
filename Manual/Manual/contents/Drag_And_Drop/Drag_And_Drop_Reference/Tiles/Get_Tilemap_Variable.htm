Obter a Variável do Mapa de Telha

Com esta ação você pode recuperar qualquer uma de um determinado número de variáveis para o elemento do mapa de azulejos em uma camada. Quando você cria uma camada tile no editor room, esta camada contém um elemento do mapa de azulejo que é preenchido com tiles a partir de um conjunto de azulejos. Este elemento do mapa tile tem certos valores associados a ele, como uma posição de offset, o conjunto de azulejos sendo usado, a largura, altura, etc... A lista completa de valores que você pode obter é:

  • X Coordenada: A posição ao longo do eixo x na camada onde o elemento do mapa tile foi colocado.
  • Coordenada Y: A posição ao longo do eixo y na camada onde o elemento do mapa tile foi colocado.
  • Colunas: A largura (definida como o número de colunas de tile células) do elemento do mapa tile.
  • Linhas: A altura (definida como o número de filas de tile células) do elemento do mapa tile.
  • Largura da telha: A largura de uma única célula tile.
  • Altura do azulejo: A altura de uma única célula tile.
  • Recurso do conjunto de azulejos: O recurso de conjunto de azulejos que foi atribuído ao elemento do mapa tile para desenho.
  • Moldura atual: A moldura atual sendo desenhada para animado tiles.
  • Máscara: Os dados da máscara para o elemento do mapa tile.
  • Máscara Global: Os dados da máscara global para todos os elementos do mapa tile.

Ao utilizar esta ação, você fornece o nome da camada (um string, conforme definido no editor Room ) para obter os dados do elemento do mapa tile, depois o tipo de dados que você deseja recuperar (conforme mostrado na lista acima). O valor retornado será então armazenado em a variável alvo que pode ter sido criada anteriormente ou pode ser uma nova variável temporária (se você marcar a caixa de seleção "Temp"). Observe que você pode recuperar valores adicionais clicando no ícone "plus ao lado de a ação, e selecionando outra variável e dando outra variável para armazenar o valor retornado.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Layer A camada para obter dados de
Variable A variável para recuperar o valor de (como listado acima)
