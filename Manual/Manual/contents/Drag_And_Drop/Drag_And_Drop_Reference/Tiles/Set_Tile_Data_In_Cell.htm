Conjunto Tile Dados na Célula

Com esta ação você pode configurar os dados tile para uma célula tile em uma camada de mapa de azulejos. Os dados tile são simplesmente um valor que reflete o índice do tile junto com os valores de inverter, espelhar e girar, assim como quaisquer máscaras personalizadas tile que você tenha usado. Com esta ação você primeiro fornece a camada a ser alvo usando o nome da camada (a string) como definido no editor room, e depois a coluna e linha na camada do mapa tile para definir os dados da telha para. A "célula" é a área na grade do mapa tile que contém o tile para o qual você deseja definir os dados, portanto, se seu tiles for 16x16, por exemplo, e o room for 1024x768, o mapa tile terá 64 colunas e 48 linhas.

NOTA: A camada selecionada deve ter sido definida como um Tile Map Layer no editor room, caso contrário, você poderá receber erros.

O argumento final para a ação são os próprios dados do azulejo. Você normalmente recuperaria os dados tile para a célula do mapa tile usando a ação Obter Tile Data In Cell e depois manipulá-los usando o conjunto de ação Tile Data Transform antes de configurá-los novamente com esta ação. Para mais informações sobre os dados tile, consulte a seção GML sobre Funções do mapa de azulejos.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Layer A camada com o mapa tile ao alvo
Columns A posição da coluna da célula ao longo do eixo horizontal para definir os dados da telha para
Row A posição da linha de células ao longo do eixo vertical para definir os dados da telha
Data Os dados tile a serem utilizados para a configuração da célula
