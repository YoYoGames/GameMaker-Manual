Preferências do Editor de Curvas de Animação

As Preferências das Curvas de Animação são usadas para definir certas propriedades para quando você estiver trabalhando com o Editor de Curvas de Animação. As opções disponíveis são:

  • Resposta automática à eliminação de curvas: Esta opção pode ser usada para controlar a mensagem pop up que é mostrada quando se apaga uma curva do editor de Curvas de Animação. A ação padrão aqui é mostrar uma mensagem de aviso, mas você pode definir isto para "Ok" e o diálogo não será mostrado.
