O Gerente Room

Ao utilizar o Navegador de Ativos, você pode criar seu rooms e movê-los entre os diferentes grupos de pastas para organizar o projeto da maneira que melhor lhe convier. Entretanto, os jogos no GameMaker geralmente têm rooms que seguem uma certa ordem, e no mínimo devem ter uma sala de partida específica (depois o jogo começa neste room você pode usar código ou ações para ir a qualquer room no jogo, se desejar), que é onde o Gerente de Sala entra em jogo.

Você pode abrir o gerenciador room clicando no botão "Extras" na parte superior do navegador Asset e selecionando-o no menu que se abre, ou você pode basta clicar em no ícone à esquerda de um room no navegador asset:

Uma vez aberto, o Gerente Room lhe apresenta duas abas diferentes, uma para Pedido de Quarto e outra para Herança de Quarto. A ordem room define a ordem em que rooms será acessada se você estiver atravessando o jogo literalmente, usando a função room_goto_next() em o final de cada room, e o próprio topo room listado será sempre sinalizado pelo ícone "home" , o que mostra que este room é o designado espaço de partida para o jogo. Este ícone também será mostrado no navegador Asset para que você possa ver claramente qual room é, independentemente de como você tenha filtrado ou configurado seu assets.

Nota: Os IDs de seu room assets não estarão na mesma ordem definida nesta lista e, portanto, você não deve fornecer um valor numérico diretamente para alterar o room em runtime. Use sempre a constante da sala para o asset que você deseja referir (que é simplesmente seu nome) ou recuperá-la através de uma função.

Você pode alterar a ordem de rooms simplesmente clicando e arrastando o room que deseja mover para cima ou para baixo da lista, e você pode usar e/ou / para selecionar vários rooms e depois clicar e arrastá-los para movê-los por completo:

A aba " Room Inheritance" é para a criação da "criança" rooms. Uma criança room é uma criança que foi copiada de um "pai" room e definir para inherit todas as suas propriedades. Se você não fizer nada a este room, então este room será de fato uma cópia do original room, mas você pode então entrar no Room Editor e alternar herança para coisas específicas como camadas ou instâncias, assim como adicionar novas camadas, instâncias e propriedades ao room.

Como na aba Room, você pode clicar em e arrastar para reordenar o rooms, mas note que selecionar um filho room e mudá-lo para um pai diferente quebrará a herança room com o pai original e, em vez disso, inherit as propriedades do novo room. Note também que você pode converter uma criança em uma room "cheia" sem herança, arrastando-a para a esquerda da lista (a borda da lista será destacada), e nesse momento ela simplesmente se tornará uma room em seu próprio direito.

Finalmente, o gerente room tem algumas ferramentas extras para ajudar você a navegar e organizar seu rooms. No topo você tem uma Barra de Busca que lhe permite adicionar um termo de busca e depois ir rapidamente ao rooms que tem o termo de busca em seu Rooms que têm o termo de busca será destacado, e então você poderá navegar até eles instantaneamente usando os botões para cima/para baixo na lateral:

Há também alguns botões úteis na parte inferior do Room Manager que permitem mover rooms para cima ou para baixo na lista de pedidos room:

Ao lado destes botões você também tem um botão Criar Grupo que pode ser usado para criar uma pasta de grupo. Estas pastas podem ser usadas para organizar melhor o rooms em seu jogo (room ordem ainda será respeitada), e você pode clicar e arrastar rooms para a pasta para adicioná-los. Você também pode selecionar vários rooms clicando enquanto segura e/ou / , depois clique no botão Criar Grupo e todos os rooms selecionados serão adicionados ao grupo. Note que isto mudará a ordem do rooms em seu jogo, e também note que se você optar por excluir a pasta do grupo posteriormente, qualquer rooms dentro da pasta também será excluído (por padrão GameMaker irá adverti-lo sobre isso).

Você também deve observar que pode clicar com o botão direito do mouse em qualquer room no Gerente Room para abrir o menu a seguir:

Este menu lhe dá as seguintes opções:

  • Criar quarto para crianças: Isto criará um novo room como uma criança do room que foi clicado, com o novo room herdando as propriedades dos pais room.
  • Edição: Isto abrirá o Editor de Sala para o room.
  • Editar Etiquetas: Isto abrirá o editor de tags para que você possa adicionar tags ao room.
  • Renomear: Isto permite renomear o room asset (você também pode fazer um duplo clique lento no room para renomeá-lo).
  • Duplicar: Isto irá duplicar o selecionado room, colocando o novo room sob o selecionado room.
  • Favorito: Isto adicionará o selecionado room à seção de Favoritos do Navegador de Ativos.
  • Excluir: Clicando aqui, serão excluídos os room (ou rooms) que foram selecionados. Se usado em uma pasta de grupo, todo rooms dentro da pasta será excluído junto com a pasta. Se usado em uma pasta pai room, somente o pai será excluído, e todas as crianças rooms permanecerão, embora agora sejam independentes e não mais crianças rooms.
  • Expandir as crianças: Se uma pasta de grupo for selecionada, esta opção expandirá o conteúdo da pasta.
  • Expandir tudo: Esta opção expandirá todos os grupos de pastas.
  • Colapsar tudo: Esta opção irá colapsar todos os grupos de pastas.
  • Organizar: O submenu de arranjos pode ser usado para mover rooms para cima ou para baixo no gerenciador room, muito parecido com os botões na parte inferior da janela do gerenciador.
  • Abrir no Explorer: Isto abrirá um explorador de arquivos de OS no asset no projeto do jogo.
  • Asset Colour: Aqui você pode selecionar uma cor para o room (ou rooms) para ajudá-los a identificá-los no Gerente Room. Note que esta cor também será usada no navegador Asset e substituirá qualquer cor que tenha sido definida anteriormente.
