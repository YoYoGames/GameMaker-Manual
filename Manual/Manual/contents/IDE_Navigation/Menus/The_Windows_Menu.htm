O menu Windows

Este menu é para abrir as janelas da doca geral IDE. Como as docas podem ser reorganizadas e o conteúdo pode ser removido como janelas autônomas ou mesmo totalmente fechadas, esta janela permite que você as abra novamente se achar que precisa delas para alguma coisa. As janelas que podem ser abertas a partir daqui são a janela do Navegador de Ativos e todas as diferentes janelas de Saída (Output, Search, Source Control, Breakpoints e Erros de Compilação/Syntax).

Em seguida, você pode abrir a janela do Inspetor. Esta janela é usada para inspecionar as propriedades do elemento atualmente selecionado IDE, e estas propriedades podem então ser editadas conforme necessário.

Após a opção Inspetor você tem a lista de Janelas Recentes, e há também a opção de abrir a janela de Variáveis de Ambiente onde você pode ver o diferente paths que os vários ambientes variáveis utilizadas nas Preferências apontam para. Estas variáveis não podem ser editadas (nem seu nome nem seu caminho/configuração), mas você pode selecionar e copiar/colar qualquer uma delas para uso em outro lugar.

Finalmente, você tem a opção de abrir as janelas do Tutorial. Isto reabrirá quaisquer janelas que estejam relacionadas a um projeto tutorial você abriu, mas somente se o projeto foi criado como parte de um tutorial (se não, esta opção não estará disponível). Para informações sobre como criar seus próprios projetos de tutorial, consulte a seção sobre o Marketplace (como todos os tutoriais são criados, hospedados e baixados de lá).
