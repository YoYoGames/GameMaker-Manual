Informações adicionais

Esta seção do manual contém uma coleção de artigos diversos relacionados à programação e à forma de funcionamento do GameMaker Language. Os artigos a seguir foram concebidos como artigos complementares para expandir ainda mais sua compreensão de como o GameMaker funciona e como obter o máximo das diferentes características de linguagem disponíveis:

  • Melhores práticas ao programar
  • O sistema de arquivo
  • Operadores Bitwise
  • Tabelas de tipo
  • Vetores
  • Guia para a utilização de amortecedores
  • Guia para o uso de partículas
  • Guia para o uso de Shaders
  • Guia para Primitivos e Vertex Edifício
  • Guia para a utilização de emendas
  • Relatório de erros
  • Funções de Compatibilidade
  • Roteiros de Compatibilidade
  • Funções obsoletas
