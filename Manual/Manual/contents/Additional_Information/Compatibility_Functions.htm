Funções de Compatibilidade

Quando você estiver importando um projeto de uma versão anterior de GameMaker você poderá ver que o projeto tem scripts de compatibilidade adicionados. Em geral, estes scripts utilizam as funções atualizadas GameMaker Language (GML) para converter funções obsoletas na nova maneira de fazer as coisas, e são uma ótima maneira de aprender como as coisas devem ser feitas no GameMaker. Entretanto, existem algumas funções que são utilizadas apenas para compatibilidade e que não devem ser utilizadas em qualquer outro lugar, e certamente não em seus próprios projetos. Para ajudá-lo a identificar estas funções de compatibilidade, elas estão listadas aqui, mas devemos salientar que elas nunca devem ser usadas fora dos scripts de compatibilidade:

  • layer_tile_exists
  • layer_tile_create
  • layer_tile_destroy
  • layer_tile_change
  • layer_tile_xscale
  • layer_tile_yscale
  • layer_tile_blend
  • layer_tile_alpha
  • layer_tile_x
  • layer_tile_y
  • layer_tile_region
  • layer_tile_visible
  • layer_tile_get_sprite
  • layer_tile_get_xscale
  • layer_tile_get_yscale
  • layer_tile_get_blend
  • layer_tile_get_alpha
  • layer_tile_get_x
  • layer_tile_get_y
  • layer_tile_get_region
  • layer_tile_get_visible
